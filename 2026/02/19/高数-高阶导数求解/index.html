еёёи§Ғзҡ„жі°еӢ’еұ•ејҖе…¬ејҸ

\[\begin{aligned} e^{x}&=\sum_{n=0}^{\infty} \frac{1}{n !} x^{n}=1+x+\frac{1}{2 !} x^{2}+\cdots \in(-\infty,+\infty) \\ \sin x&=\sum_{n=0}^{\infty} \frac{(-1)^{n}}{(2 n+1) !} x^{2 n+1}=x-\frac{1}{3 !} x^{3}+\frac{1}{5 !} x^{5}+\cdots, x \in(-\infty,+\infty) \\ \cos x&=\sum_{n=0}^{\infty} \frac{(-1)^{n}}{(2 n) !} x^{2 n}=1-\frac{1}{2 !} x^{2}+\frac{1}{4 !} x^{4}+\cdots, x \in(-\infty,+\infty) \\ \ln (1+x)&=\sum_{n=0}^{\infty} \frac{(-1)^{n}}{n+1} x^{n+1}=x-\frac{1}{2} x^{2}+\frac{1}{3} x^{3}+\cdots, x \in(-1,1] \\ \frac{1}{1-x}&=\sum_{n=0}^{\infty} x^{n}=1+x+x^{2}+x^{3}+\cdots, x \in(-1,1) \\ \frac{1}{1+x}&=\sum_{n=0}^{\infty}(-1)^{n} x^{n}=1-x+x^{2}-x^{3}+\cdots, x \in(-1,1)\\ (1+x)^{\alpha}&=1+\sum_{n=1}^{\infty} \frac{\alpha(\alpha-1) \cdots(\alpha-n+1)}{n !} x^{n}=1+\alpha x+\frac{\alpha(\alpha-1)}{2 !} x^{2}+\cdots, x \in(-1,1) \\ \arctan x&=\sum_{n=0}^{\infty} \frac{(-1)^{n}}{2 n+1} x^{2 n+1}=x-\frac{1}{3} x^{3}+\frac{1}{5} x^{5}+\cdots+ x \in[-1,1] \\ \arcsin x&=\sum_{n=0}^{\infty} \frac{(2 n) !}{4^{n}(n !)^{2}(2 n+1)} x^{2n+1}=x+\frac{1}{6} x^{3}+\frac{3}{40} x^{5}+\frac{5}{112} x^{7}+\frac{35}{1152} x^{9}+\cdots+, x \in(-1,1)\\ \tan x&=\sum_{n=1}^{\infty} \frac{B_{2 n}(-4)^{n}\left(1-4^{n}\right)}{(2 n) !} x^{2 n-1}=x+\frac{1}{3} x^{3}+\frac{2}{15} x^{5}+\frac{17}{315} x^{7}+\frac{62}{2835} x^{9}+\cdots,x\in (-\frac{\pi}{2},\frac{\pi}{2}) \end{aligned}\]

й«ҳйҳ¶еҜјж•°жұӮи§Јж–№жі•

жұӮи§Јй«ҳйҳ¶еҜјж•° \(y^{(n)}\) дё»иҰҒжңүд»ҘдёӢеӣӣз§Қеёёз”Ёж–№жі•.

йҖӮз”ЁеңәжҷҜ йҖӮз”ЁдәҺеҪўејҸз®ҖеҚ•гҖҒи§„еҫӢжҳҺжҳҫзҡ„еҮҪж•°.

жӯҘйӘӨ

  1. жұӮеҜј з®—еҮә \(y', y'', y'''\) зӯүеүҚеҮ йҳ¶еҜјж•°.
  2. жүҫи§„еҫӢ и§ӮеҜҹзі»ж•°гҖҒз¬ҰеҸ·гҖҒжҢҮж•°зӯүеҸҳеҢ–и§„еҫӢ.
  3. жҖ»з»“ еҶҷеҮә \(y^{(n)}\) зҡ„йҖҡйЎ№е…¬ејҸ.
    • ( дёҘи°Ёиө·и§ҒпјҢеҸҜдҪҝз”Ёж•°еӯҰеҪ’зәіжі•иҜҒжҳҺпјҢдҪҶиҖғиҜ•йҖҡеёёеҸӘиҰҒжұӮеҶҷеҮәз»“жһң ).

еёёз”ЁеҹәзЎҖе…¬ејҸ

  • \((x^\mu)^{(n)} = \mu(\mu-1)\cdots(\mu-n+1)x^{\mu-n}\)
  • \((\sin ax)^{(n)} = a^n \sin(ax + \frac{n\pi}{2})\)
  • \((\cos ax)^{(n)} = a^n \cos(ax + \frac{n\pi}{2})\)
  • \((e^{ax})^{(n)} = a^n e^{ax}\)
  • \((\ln x)^{(n)} = (-1)^{n-1} \frac{(n-1)!}{x^n}\)

е…ёеһӢзӨәдҫӢ и®ҫ \(y = \arctan x\)пјҢжұӮ \(y^{(n)}(0)\).

  1. еҸҳйҮҸд»ЈжҚў д»Ө \(y = \arctan x\)пјҢеҲҷ \(x = \tan y\). \[ y' = \frac{1}{1+x^2} = \frac{1}{1+\tan^2 y} = \cos^2 y \]
  2. жұӮй«ҳйҳ¶еҜјж•° ( жіЁж„Ҹ \(y\) жҳҜ \(x\) зҡ„еҮҪж•°пјҢжұӮеҜјйңҖд№ҳ \(y'\) )
    • \(y' = \cos^2 y = 0! \cdot \cos^1 y \cdot \sin(y + \frac{\pi}{2})\) ( еҲ©з”Ё \(\cos y = \sin(y+\frac{\pi}{2})\) )
    • \(y'' = 2\cos y (-\sin y) \cdot y' = -2\sin y \cos^3 y = 1! \cdot \cos^2 y \cdot \sin[2(y+\frac{\pi}{2})]\)
    • \(y''' = -2(\cos 2y \cdot y' \cdot \cos^2 y + \sin 2y \cdot 2\cos y (-\sin y) \cdot y')\)
    • и§„еҫӢ \[ y^{(n)} = (n-1)! \cos^n y \sin[n(y+\frac{\pi}{2})] \]
  3. д»Је…Ҙ \(x=0\) еҪ“ \(x=0\) ж—¶пјҢ\(y=0\). \[ y^{(n)}(0) = (n-1)! \cos^n 0 \cdot \sin(\frac{n\pi}{2}) = (n-1)! \sin(\frac{n\pi}{2}) \]
  4. еҲҶзұ»и®Ёи®ә
    • \(n\) дёәеҒ¶ж•° (\(n=2k\)) \(\sin(k\pi) = 0 \implies y^{(2k)}(0) = 0\).
    • \(n\) дёәеҘҮж•° (\(n=2k+1\)) \(\sin(\frac{(2k+1)\pi}{2}) = (-1)^k \implies y^{(2k+1)}(0) = (-1)^k (2k)!\).
    • жҚўеӣһ \(n\) \(y^{(n)}(0) = (-1)^{\frac{n-1}{2}} (n-1)!\).

йҖӮз”ЁеңәжҷҜ йҖӮз”ЁдәҺдёӨдёӘеҮҪж•°д№ҳз§Ҝзҡ„й«ҳйҳ¶еҜјж•° \((uv)^{(n)}\). йҖҡеёёе…¶дёӯдёҖдёӘеҮҪж•° \(v(x)\) жҳҜ еӨҡйЎ№ејҸ ( жұӮеҜјеҮ ж¬ЎеҗҺдёә 0 ).

е…¬ејҸ \[ (uv)^{(n)} = \sum_{k=0}^n C_n^k u^{(n-k)} v^{(k)} \] еұ•ејҖеҚі \[(uv)^{(n)} = u^{(n)}v + C_n^1 u^{(n-1)}v' + C_n^2 u^{(n-2)}v'' + \cdots + uv^{(n)}\]

жҠҖе·§ и®ҫе®ҡ (v(x)) дёәйӮЈдёӘ вҖңжұӮеҜјеҮ ж¬Ўе°ұдјҡеҸҳжҲҗ 0вҖқ зҡ„еӨҡйЎ№ејҸ ( еҰӮ (x, x^2) )пјҢиҝҷж ·е…¬ејҸеҗҺйқўзҡ„йЎ№йғҪдјҡж¶ҲеӨұпјҢеӨ§еӨ§з®ҖеҢ–и®Ўз®—.

е…ёеһӢзӨәдҫӢ и®ҫ \(y = (x^2+1) \sin 2x\)пјҢжұӮ \(y^{(n)}\).

  1. йҖүе®ҡ \(u, v\) д»Ө \(u = \sin 2x\) ( жұӮеҜји§„еҫӢжҳҺжҳҫ ), \(v = x^2+1\) ( жұӮеҜјдёүж¬Ўдёә 0 ).

  2. и®Ўз®—еҗ„йҳ¶еҜјж•°

    • \(u^{(k)} = 2^k \sin(2x + \frac{k\pi}{2})\) ( жіЁж„ҸеӨҚеҗҲеҮҪж•°жұӮеҜјзҡ„зі»ж•° \(2^k\) )
    • \(v = x^2+1, v' = 2x, v'' = 2, v''' = 0\)

  3. д»Је…ҘиҺұеёғе°јиҢЁе…¬ејҸ \[ y^{(n)} = C_n^0 u^{(n)} v + C_n^1 u^{(n-1)} v' + C_n^2 u^{(n-2)} v'' \] \[ = 2^n \sin(2x + \frac{n\pi}{2}) \cdot (x^2+1) + n \cdot 2^{n-1} \sin(2x + \frac{(n-1)\pi}{2}) \cdot 2x + \frac{n(n-1)}{2} \cdot 2^{n-2} \sin(2x + \frac{(n-2)\pi}{2}) \cdot 2 \]

  4. еҢ–з®ҖдёҺеҗҲе№¶ жҸҗеҸ–е…¬еӣ ејҸ \(2^{n-2}\)пјҢе№¶еҲ©з”ЁиҜұеҜје…¬ејҸ \(\sin(\alpha - \frac{\pi}{2}) = -\cos\alpha, \sin(\alpha - \pi) = -\sin\alpha\)

    \[ y^{(n)} = 2^{n-2} \left[ 4(x^2+1)\sin(2x+\frac{n\pi}{2}) - 4nx\cos(2x+\frac{n\pi}{2}) - n(n-1)\sin(2x+\frac{n\pi}{2}) \right] \]

    ж•ҙзҗҶеҗҢзұ»йЎ№ ( еҗҲе№¶ \(\sin\) йЎ№ ) \[ y^{(n)} = 2^{n-2} \left[ (4x^2 + 4 - n^2 + n) \sin(2x + \frac{n\pi}{2}) - 4nx \cos(2x + \frac{n\pi}{2}) \right] \]

йҖӮз”ЁеңәжҷҜ дё“й—Ёз”ЁдәҺжұӮ \(x=0\) еӨ„ зҡ„й«ҳйҳ¶еҜјж•° \(f^{(n)}(0)\).

еҺҹзҗҶ е°Ҷ \(f(x)\) еңЁ \(x=0\) еӨ„еұ•ејҖдёәйәҰе…ӢеҠіжһ—зә§ж•° \[ f(x) = \sum_{n=0}^\infty a_n x^n = \sum_{n=0}^\infty \frac{f^{(n)}(0)}{n!} x^n \] йҖҡиҝҮеҜ№жҜ” \(x^n\) зҡ„зі»ж•°пјҢеҸҜеҫ— \[ a_n = \frac{f^{(n)}(0)}{n!} \implies f^{(n)}(0) = n! \cdot a_n \]

жӯҘйӘӨ 1. еҲ©з”Ёй—ҙжҺҘеұ•ејҖжі• ( д»ЈжҚўжі• ) е°ҶеҮҪж•°еұ•ејҖдёәе№Ӯзә§ж•°. 2. жүҫеҲ° \(x^n\) йЎ№зҡ„зі»ж•° \(a_n\). 3. и®Ўз®— \(f^{(n)}(0) = n! \cdot a_n\).

и®ҫ \(y = \arctan x\)пјҢжұӮ \(y^{(n)}(0)\).

  1. еҲ©з”Ёе·ІзҹҘеұ•ејҖејҸ е·ІзҹҘ \(\frac{1}{1+x} = \sum_{n=0}^{\infty} (-1)^n x^n\).
  2. жұӮеҜјж•°еұ•ејҖ \[ f'(x) = \frac{1}{1+x^2} = \sum_{n=0}^{\infty} (-1)^n (x^2)^n = \sum_{n=0}^{\infty} (-1)^n x^{2n} \]
  3. з§ҜеҲҶиҝҳеҺҹеҮҪж•° йҖҗйЎ№з§ҜеҲҶ ( жіЁж„Ҹ \(f(0)=\arctan 0 = 0 \implies C=0\) ) \[ f(x) = \int_0^x \frac{1}{1+t^2} dt = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1} \] еҚі \[ f(x) = \sum_{k=0}^{\infty} (-1)^k \frac{x^{2k+1}}{2k+1} \]
  4. жҸҗеҸ–зі»ж•° \(a_n\)
    • еҒ¶ж¬ЎйЎ№ (\(n=2k\)) зә§ж•°дёӯж—  \(x^{2k}\) йЎ№пјҢзі»ж•° \(a_{2k} = 0\).
    • еҘҮж¬ЎйЎ№ (\(n=2k+1\)) \(x^{2k+1}\) зҡ„зі»ж•°дёә \(a_{2k+1} = \frac{(-1)^k}{2k+1}\).
  5. и®Ўз®—й«ҳйҳ¶еҜјж•° \(f^{(n)}(0) = n! \cdot a_n\).
    • еҪ“ \(n\) дёәеҒ¶ж•° \(f^{(n)}(0) = 0\).
    • еҪ“ \(n\) дёәеҘҮж•° (\(n=2k+1\)) \[ f^{(n)}(0) = (2k+1)! \cdot \frac{(-1)^k}{2k+1} = (-1)^k (2k)! \] д»Јеӣһ \(k = \frac{n-1}{2}\) \[ f^{(n)}(0) = (-1)^{\frac{n-1}{2}} (n-1)! \]

йҖӮз”ЁеңәжҷҜ йҖӮз”ЁдәҺ \(y^{(n)}\) дёҺ \(y^{(n-1)}, y^{(n-2)}\) зӯүжңүжҳҺжҳҫдҫқиө–е…ізі»зҡ„еҮҪж•°. еёёз”ЁдәҺ \(y = \arctan x, y = (\arcsin x)^2\) жҲ–ж»Ўи¶ізү№е®ҡеҫ®еҲҶж–№зЁӢзҡ„еҮҪж•°.

жӯҘйӘӨ

  1. жұӮеҜје»әз«Ӣ \(y', y'', y\) д№Ӣй—ҙзҡ„ж–№зЁӢ ( еҰӮ \((1+x^2)y' = 1\) ).
  2. еҜ№иҜҘж–№зЁӢдёӨиҫ№еҗҢж—¶жұӮ \(n\) йҳ¶еҜј ( еҲ©з”ЁиҺұеёғе°јиҢЁе…¬ејҸ ).
  3. еҫ—еҲ°е…ідәҺ \(y^{(n+k)}\) зҡ„йҖ’жҺЁе…¬ејҸ.
  4. д»Је…Ҙзү№е®ҡеҖјжұӮи§Ј.

е…ёеһӢзӨәдҫӢ и®ҫ \(y = \arctan x\)пјҢжұӮ \(y^{(n)}(0)\).

  1. е»әз«Ӣж–№зЁӢ
    \[ y' = \frac{1}{1+x^2} \implies (1+x^2)y' = 1 \]
  2. жұӮ \(n-1\) йҳ¶еҜј
    еҜ№зӯүејҸдёӨиҫ№еҗҢж—¶жұӮ \(n-1\) йҳ¶еҜј ( еҲ©з”ЁиҺұеёғе°јиҢЁе…¬ејҸ ) \[ [(1+x^2)y']^{(n-1)} = (1)^{(n-1)} = 0 \] еұ•ејҖе·Ұиҫ№ ( жіЁж„Ҹ \(1+x^2\) жұӮеҜјдёӨж¬ЎеҗҺдёә 0 ) \[ C_{n-1}^0 y^{(n)} (1+x^2) + C_{n-1}^1 y^{(n-1)} (2x) + C_{n-1}^2 y^{(n-2)} (2) = 0 \] \[ y^{(n)}(1+x^2) + 2(n-1)x y^{(n-1)} + (n-1)(n-2)y^{(n-2)} = 0 \]
  3. д»Је…Ҙ \(x=0\) \[ y^{(n)}(0) \cdot 1 + 0 + (n-1)(n-2)y^{(n-2)}(0) = 0 \] \[ y^{(n)}(0) = -(n-1)(n-2)y^{(n-2)}(0) \]
  4. еҲҶзұ»и®Ёи®әжұӮи§Ј
    • еҪ“ \(n\) дёәеҒ¶ж•° (\(n=2k\)) з”ұдәҺ \(y=\arctan x\) жҳҜеҘҮеҮҪж•°пјҢеҒ¶ж•°йҳ¶еҜјж•°еңЁ 0 еӨ„дёә 0. \[ y^{(2k)}(0) = 0 \]
    • еҪ“ \(n\) дёәеҘҮж•° (\(n=2k+1\)) \[ y^{(n)}(0) = -(n-1)(n-2)y^{(n-2)}(0) \] йҖ’жҺЁеҸҜеҫ— ( \(n=2k+1\) ж—¶пјҢ\(k=\frac{n-1}{2}\) ) \[ y^{(n)}(0) = (-1)^{\frac{n-1}{2}} (n-1)! \]
