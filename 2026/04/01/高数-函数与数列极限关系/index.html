函数与数列极限关系

极限问题里，数列与函数并不是两套互相独立的语言 : 很多“函数极限”的结论可以用数列来判定，很多“数列极限”的结论也可以转写为函数极限. 本文以充分必要条件为主线，给出最实用的序列化判别 ( 海涅定理 )、无穷远处版本、常见反例与题型.

总结与速查 ( Cheat Sheet )

结论 形式 关键词
数列极限定义 \(a_n\to A\) \(\varepsilon\)-\(N\)
函数极限定义 \(f(x)\to A\ (x\to x_0)\) \(\varepsilon\)-\(\delta\)
海涅定理 \(f(x)\to A\ (x\to x_0)\iff \forall x_n\to x_0,\ x_n\ne x_0,\ f(x_n)\to A\) 充分必要
无穷远版本 \(f(x)\to A\ (x\to\infty)\iff \forall x_n\to\infty,\ f(x_n)\to A\) 充分必要
否定极限 \(\exists x_n,y_n\to x_0\) 使 \(f(x_n),f(y_n)\) 极限不同 一票否决
连续的序列刻画 \(f\) 在 \(x_0\) 连续 \(\iff \forall x_n\to x_0,\ f(x_n)\to f(x_0)\) 连续 \(\Leftrightarrow\) 序列
柯西准则 极限存在 \(\iff\) “足够靠近时函数值足够接近” 不用猜极限值

一、两类极限的“语言”对照

数列极限 ( \(\varepsilon\)-\(N\) )

\[a_n\to A\iff \forall \varepsilon>0,\ \exists N,\ \forall n\ge N,\ |a_n-A|<\varepsilon\]

函数极限 ( \(\varepsilon\)-\(\delta\) )

\[\lim_{x\to x_0}f(x)=A\iff \forall \varepsilon>0,\ \exists \delta>0,\ 0<|x-x_0|<\delta\Rightarrow |f(x)-A|<\varepsilon\]

这里的 \(0<|x-x_0|\) 表示 \(x\) 趋近但不取到 \(x_0\)，因此 \(f\) 不必在 \(x_0\) 有定义.

最容易混淆的一点

数列极限里，“趋近”的对象是整数下标 \(n\to\infty\);
函数极限里，“趋近”的对象是自变量 \(x\to x_0\) 或 \(x\to\infty\).

它们的共同点是 : 都用“任意小 \(\varepsilon\)”来描述“最终足够接近”.

二、海涅定理 ( 函数极限的序列化判别 )

定理 ( 充分必要条件 )

设 \(f\) 在 \(x_0\) 的去心邻域内有定义. 则 \[\lim_{x\to x_0}f(x)=A\iff \forall (x_n),\ x_n\to x_0,\ x_n\ne x_0,\ \lim_{n\to\infty}f(x_n)=A\]

怎么理解

“\(x\to x_0\)” 的所有方式，都可以用“选一条趋近 \(x_0\) 的数列 \(x_n\)”来表达.
因此，函数极限存在且等于 \(A\) 等价于 : 沿任何趋近路径 ( 用数列表示 )，函数值都趋于同一个 \(A\).

直接推论 ( 常用到爆 )

若 \(\displaystyle\lim_{x\to x_0}f(x)=A\)，则对任意 \(x_n\to x_0\) 且 \(x_n\ne x_0\)，都有 \[\lim_{n\to\infty}f(x_n)=A\] 这句话通常用来把“函数极限题”转成“数列极限题”，或者把复杂的 \(x\to x_0\) 替换成易算的 \(x_n\to x_0\) 来做反例.

例 : 由函数极限推出数列极限

已知 \(\displaystyle\lim_{x\to 0}\frac{\sin x}{x}=1\)，取 \(x_n=\frac{1}{n}\to 0\)，则 \[\lim_{n\to\infty}\frac{\sin(1/n)}{1/n}=1\quad\Rightarrow\quad \lim_{n\to\infty}n\sin\frac{1}{n}=1\]

用来否定极限的标准套路

想证明 \(\displaystyle\lim_{x\to x_0}f(x)\) 不存在，只要构造两列 \(x_n\to x_0,\ y_n\to x_0\)，使 \[\lim f(x_n)\ne \lim f(y_n)\] 即可断言极限不存在.

经典反例

\[f(x)=\sin\frac{1}{x},\quad x\ne 0\]

取 \[x_n=\frac{1}{2\pi n+\frac{\pi}{2}}\to 0,\qquad y_n=\frac{1}{2\pi n+\frac{3\pi}{2}}\to 0\] 则 \[f(x_n)=1,\qquad f(y_n)=-1\] 两列函数值极限不同，所以 \(\lim_{x\to 0}\sin\frac{1}{x}\) 不存在.

三、无穷远处的版本 ( \(x\to\infty\) 与 \(n\to\infty\) 的桥 )

定理 ( 充分必要条件 )

设 \(f\) 在某个 \([M,+\infty)\) 上有定义. 则 \[\lim_{x\to\infty}f(x)=A\iff \forall (x_n),\ x_n\to\infty,\ \lim_{n\to\infty}f(x_n)=A\]

立即得到的推论

若 \(\displaystyle\lim_{x\to\infty}f(x)=A\)，则对任意趋于无穷的数列 \(x_n\) 都有 \(f(x_n)\to A\).
特别地，取 \(x_n=n\) 得 \[\lim_{x\to\infty}f(x)=A\Rightarrow \lim_{n\to\infty}f(n)=A\]

例 : 把数列极限看成函数无穷远极限

求 \(\displaystyle\lim_{n\to\infty}\left(1+\frac{1}{n}\right)^n\).
考虑函数 \(f(x)=\left(1+\frac{1}{x}\right)^x\)，则 \[\lim_{n\to\infty}\left(1+\frac{1}{n}\right)^n=\lim_{n\to\infty}f(n)\] 而若已知 \(\lim_{x\to\infty}\left(1+\frac{1}{x}\right)^x=e\)，则立刻得到数列极限为 \(e\).

反过来不成立

只知道 \(f(n)\to A\) 并不能推出 \(f(x)\to A\ (x\to\infty)\).

反例 \[f(x)=\sin(\pi x)\] 则 \(f(n)=0\to 0\)，但 \(\sin(\pi x)\) 在无穷远处不收敛.

四、从“序列判别”得到的充分必要条件

极限存在的等价表述

设 \(f\) 在 \(x_0\) 去心邻域内有定义. 则 \[\lim_{x\to x_0}f(x)\ \text{存在}\iff \forall x_n\to x_0,\ x_n\ne x_0,\ \{f(x_n)\}\ \text{收敛且极限相同}\]

极限不存在的两种“典型原因”

  1. 沿不同路径趋近，极限不同 ( 用两列否定 )
  2. 函数值本身不有界或振荡加剧，使得任何序列化都无法统一到同一极限

柯西准则 ( 不用猜极限值 )

设 \(f\) 在 \(x_0\) 去心邻域内有定义，则 \[\lim_{x\to x_0}f(x)\ \text{存在}\iff \forall\varepsilon>0,\ \exists\delta>0,\ 0<|x-x_0|,0<|y-x_0|<\delta\Rightarrow |f(x)-f(y)|<\varepsilon\]

它对应数列的柯西准则 : \(\{a_n\}\) 收敛 \(\iff\) \(\forall\varepsilon>0,\exists N,\forall m,n\ge N,\ |a_n-a_m|<\varepsilon\).

五、连续性的序列刻画 ( 常用推论 )

结论 ( 充分必要条件 )

若 \(f\) 在 \(x_0\) 有定义，则 \[f\ \text{在 }x_0\text{ 连续}\iff \forall x_n\to x_0,\ f(x_n)\to f(x_0)\]

应用一句话

一旦知道 \(x_n\to x_0\) 且 \(f\) 在 \(x_0\) 连续，就可以直接写 \[\lim_{n\to\infty}f(x_n)=f(x_0)\] 不用再做 \(\varepsilon\)-\(N\) 推导.

六、典型题型与方法选择

题型 A : 否定函数极限

目标 : 证明 \(\displaystyle\lim_{x\to x_0}f(x)\) 不存在
方法 : 构造两列 \(x_n,y_n\to x_0\) 使 \(f(x_n),f(y_n)\) 极限不同.

题型 B : 由函数极限推出数列极限

目标 : 求 \(\displaystyle\lim_{n\to\infty}a_n\)，其中 \(a_n=f(n)\)
方法 : 若能先求出 \(\displaystyle\lim_{x\to\infty}f(x)\)，则直接得到 \(a_n\to A\).

题型 C : 由数列反推函数极限 ( 常见于“反证/否定” )

若怀疑 \(\displaystyle\lim_{x\to x_0}f(x)\) 不存在，可以“先找数列证据” 尝试让 \(f(x_n)\) 变成常数、交替、或逼近不同值，这通常比直接处理 \(x\to x_0\) 更容易.

七、函数与数列嵌套的情形

情形 1 : 把 \(x\to x_0\) “嵌”进数列 \(x_n\to x_0\)

若 \(f\) 在 \(x_0\) 去心邻域内有定义，且 \[\lim_{x\to x_0}f(x)=A\] 则对任意 \(x_n\to x_0\) 且 \(x_n\ne x_0\)，都有 \[\lim_{n\to\infty}f(x_n)=A\]

这是最常见的 “函数极限 \(\Rightarrow\) 数列极限” 的嵌套方式.

情形 1 的逆命题一般不成立

仅仅知道对某一条数列 \(x_n\to x_0\) 有 \(f(x_n)\to A\)，并不能推出 \(\displaystyle\lim_{x\to x_0}f(x)=A\).

要想反推出函数极限，必须是 “对所有 \(x_n\to x_0\) 都成立”，这正是海涅定理的必要条件.

例 : 常见嵌套一行做完

若 \(x_n\to 0\) 且 \(x_n\ne 0\)，由 \(\displaystyle\lim_{x\to 0}\frac{\sin x}{x}=1\) 得 \[\lim_{n\to\infty}\frac{\sin x_n}{x_n}=1\] 例如取 \(x_n=\frac{1}{n}\)，立刻得到 \(\displaystyle\lim_{n\to\infty}n\sin\frac{1}{n}=1\).

情形 2 : 连续性把“嵌套极限”变成“直接代入”

若 \(a_n\to a\) 且 \(f\) 在 \(a\) 连续，则 \[\lim_{n\to\infty}f(a_n)=f(a)\]

更强的是 : \(f\) 在 \(a\) 连续 \(\iff\) 对任意 \(a_n\to a\) 都有 \(f(a_n)\to f(a)\).

例 : 极限题常见套路

若 \(a_n\to 0\)，则 \[\lim_{n\to\infty}\cos a_n=\cos 0=1,\qquad \lim_{n\to\infty}\sin a_n=\sin 0=0\] 因为 \(\sin,\cos\) 在 \(\mathbb{R}\) 上处处连续.

补充 : 数列不收敛，但复合 \(f(a_n)\) 可能收敛

有时 \(a_n\) 本身没有极限，但 \(f(a_n)\) 仍然有极限，原因是 \(f\) 把 \(a_n\) 的“多聚点”压成了同一个值.

一个实用判别是 : 设 \(a_n\) 的所有收敛子列极限构成集合 \(\Omega\) ( 聚点集 ).
若 \(f\) 在 \(\Omega\) 上连续，且在 \(\Omega\) 上取同一个值 \(L\) ( 即 \(f(x)\equiv L,\ x\in\Omega\) )，则 \[\lim_{n\to\infty}f(a_n)=L\]

更常见的简化版 : 如果 \(a_n\) 只有有限个聚点 \(\alpha_1,\dots,\alpha_k\)，并且 \[f(\alpha_1)=\cdots=f(\alpha_k)=L\] 且 \(f\) 在这些点连续，那么同样得到 \(f(a_n)\to L\).

例 1 : 振荡被平方“抹平”

取 \(a_n=(-1)^n\)，则 \(a_n\) 不收敛，但取 \(f(x)=x^2\) 有 \[f(a_n)=(-1)^{2n}=1\ \Rightarrow\ \lim_{n\to\infty}f(a_n)=1\]

例 2 : 数列趋于无穷 ( 在实数意义下“无极限” )，但复合有极限

若 \(a_n\to\infty\)，且 \(\displaystyle\lim_{x\to\infty}f(x)=L\)，则取 \(x_n=a_n\to\infty\) 得 \[\lim_{n\to\infty}f(a_n)=L\] 例如 \(a_n=n\)，\(f(x)=\frac{1}{x}\)，则 \(a_n\) 没有有限极限，但 \(f(a_n)=\frac{1}{n}\to 0\).

情形 3 : 把 \(n\to\infty\) “嵌”进函数的无穷远极限

若 \[\lim_{x\to\infty}f(x)=A\] 则取 \(x_n=n\) 得 \[\lim_{n\to\infty}f(n)=A\]

更一般地，只要 \(x_n\to\infty\)，就有 \(f(x_n)\to A\).

情形 3 的逆命题也不成立

\(f(n)\to A\) 只说明在离散点 \(n\) 上收敛，不足以推出 \(x\to\infty\) 时收敛.

反例 : \(f(x)=\sin(\pi x)\)，则 \(f(n)=0\)，但 \(x\to\infty\) 时不收敛.

情形 4 : 两层嵌套 \(a_n=g(n)\) 再代入 \(f\)

常见结构 \[\lim_{n\to\infty}f(g(n))\] 先看 \(g(n)\) 的极限

  • 若 \(g(n)\to x_0\)，再用 “\(x_n=g(n)\) 嵌入 \(x\to x_0\)” 的思想
  • 若 \(g(n)\to\infty\)，再用 “\(x_n=g(n)\) 嵌入 \(x\to\infty\)” 的思想

一句话 : 先把里面那层当成 \(x_n\)，外面那层当成 \(f(x_n)\).

例 : 双层嵌套 ( 先看里面 )

\[\lim_{n\to\infty}\sin\left(\frac{1}{n}+\frac{(-1)^n}{n^2}\right)\] 因 \(\frac{1}{n}+\frac{(-1)^n}{n^2}\to 0\)，而 \(\sin\) 在 0 连续，所以极限为 \(\sin 0=0\).

加餐 : 更复杂的嵌套 \(f_n(x_n)\)

形如 \(\displaystyle\lim_{n\to\infty}f_n(x_n)\) 的题，通常需要额外条件，否则无法仅凭 \(x_n\to x_0\) 判断.
常见可用条件是 : \(f_n\to f\) 在 \(x_0\) 附近一致收敛，且 \(f\) 在 \(x_0\) 连续，则 \(f_n(x_n)\to f(x_0)\).

八、易错点清单

别踩坑

  1. 用海涅定理时，必须要求 \(x_n\ne x_0\) ( 对应去心邻域 )
  2. “沿所有数列都趋于同一值”是充分必要条件; 只验证几条不够
  3. \(f(n)\to A\) 只能说明一条路径 \(x=n\) 上的行为，不能代表 \(x\to\infty\) 的全部
  4. 讨论 \(x\to x_0\) 的极限时，要先确认 \(f\) 在 \(x_0\) 附近 ( 去心 ) 是有定义的

九、做题技巧

技巧 1 : 否定函数极限 = 两条数列一票否决

看到“证明 \(\lim_{x\to x_0}f(x)\) 不存在”，优先想把 \(f(x)\) 变成常数/交替

  • 若有 \(\sin\frac{1}{x},\cos\frac{1}{x}\) : 取 \(x_n=\frac{1}{2\pi n+c}\) 让三角函数固定到 \(1,0,-1\)
  • 若有 \(|x|/x\) : 取 \(x_n=\frac{1}{n},\ y_n=-\frac{1}{n}\)
  • 若有 \(x\sin\frac{1}{x}\) : 取 \(x_n\to 0\) 看能否把振荡压到 0

技巧 2 : 看到 \(f(a_n)\) 先做“内层极限”

结构 \(\lim_{n\to\infty}f(a_n)\)

  1. 先算 \(a_n\to a\) 还是 \(a_n\to\infty\)
  2. 若 \(a_n\to a\) 且 \(f\) 在 \(a\) 连续，直接写 \(f(a_n)\to f(a)\)
  3. 若 \(a_n\to a\) 但 \(f\) 在 \(a\) 不连续，改用“海涅思维” : 把 \(a_n\) 当作一条逼近方式，必要时找子列/反例

例 : 先算里面

\[\lim_{n\to\infty}\frac{\sin(2/n)}{2/n}\] 因为 \(2/n\to 0\)，由 \(\lim_{x\to 0}\frac{\sin x}{x}=1\) 得极限为 1.

技巧 3 : 用“聚点集”判断 \(a_n\) 不收敛但 \(f(a_n)\) 收敛

先找 \(a_n\) 可能的聚点

  • \((-1)^n\) 的聚点是 \(\{-1,1\}\)
  • \(\sin n\) 的聚点是 \([-1,1]\) ( 稠密 )
  • 周期/三角类经常只有有限个聚点

若这些聚点在 \(f\) 下都被映到同一个值，就容易推出 \(f(a_n)\) 收敛.

技巧 4 : \(f(n)\to A\) 不能代替 \(\lim_{x\to\infty}f(x)\)

求 \(\displaystyle\lim_{x\to\infty}f(x)\) 时，代 \(x=n\) 只能得到必要条件，不是充分条件.
反过来，若已知 \(\displaystyle\lim_{x\to\infty}f(x)=A\)，那代 \(x=n\) 才是可靠推论.

技巧 5 : 需要“存在性”就用柯西准则，不用猜极限值

题目问“证明极限存在”时，不一定要先猜 \(A\) 再证明.
直接用 \[0<|x-x_0|,0<|y-x_0|<\delta\Rightarrow |f(x)-f(y)|<\varepsilon\] 在估计题里通常更顺手.
