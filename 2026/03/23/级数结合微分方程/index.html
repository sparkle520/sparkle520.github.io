级数结合微分方程问题

本专题重点掌握 递推式与微分方程的双向转化，深入理解 构造微分方程求和函数 与 设幂级数解求递推式 这两种核心出题模式及其套路.

总结与速查 ( Cheat Sheet )

出题模式 已知 目标 关键操作
模式一 借助微分方程求级数 递推式 ( + 可能有微分方程 ) 和函数 \(S(x)\) → 通项 \(a_n\) 构造微分方程，求解 \(S(x)\) 后展开
模式二 借助级数解微分方程 微分方程 系数 \(a_n\) → 和函数 \(y(x)\) 设 \(y = \sum a_n x^n\) 代入方程，令系数为零

核心关联框架

\[\text{递推式} \underset{}{\overset{}{\rightleftharpoons}} \text{微分方程}\]

\[\downarrow \qquad\qquad\qquad \downarrow\]

\[\text{通项 } a_n \underset{}{\overset{}{\rightleftharpoons}} \text{和函数 } S(x)\]

一. 出题模式一 : 借助微分方程求解级数

1. 条件与目标

情形 A 已知递推式 + 微分方程

\[\text{递推式} + \text{微分方程} \xrightarrow{\text{验证}} S(x) \text{ 满足微分方程} \xrightarrow{\text{求解}} S(x) \xrightarrow{\text{展开}} a_n\]

情形 B 仅已知递推式

\[\text{递推式} \xrightarrow{\text{构造}} S(x) \text{ 满足的微分方程} \xrightarrow{\text{求解}} S(x) \xrightarrow{\text{展开}} a_n\]

2. 构造微分方程的套路 ( 情形 B )

步骤 ① 将递推式套上 \(\sum\) 和 \(x^n\) ( 数列 → 级数 )

将含 \(a_n\) 的递推式两端同乘以 \(x^n\)，再对 \(n\) 求和，即令 \(S(x) = \displaystyle\sum_{n=0}^{\infty} a_n x^n\)，将递推关系转化为关于 \(S(x)\) 的等式.

  • 注意 \(\sum\) 的起始值需与递推式保持一致.
  • \(x^n\) 可改写为 \(x^{n+1}\) 或 \(x^{n-1}\)，为便于凑出和函数均可.

步骤 ② 对 \(S(x)\) 求导或积分，凑出各级数，得到微分方程

利用幂级数逐项求导 / 积分性质

\[S'(x) = \sum_{n=1}^{\infty} n a_n x^{n-1}, \qquad \int_0^x S(t)\,dt = \sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}\]

通过适当的求导或积分操作，将递推关系两端化为同一形式，从而得到关于 \(S(x)\) 的微分方程.

⚠️ 易错点

  • \(\sum\) 的起始值改变时，需补充或扣除对应项，不能随意调换!
  • 换元 ( 如 \(n \to n+k\) ) 后，\(\sum\) 的起始值和 \(x^n\) 的次数要同步调整.

二. 出题模式二 : 借助级数解微分方程

1. 条件与目标

已知微分方程，设幂级数解

\[\text{已知微分方程} \xrightarrow{\text{设}} y(x) = \sum_{n=0}^{\infty} a_n x^n \xrightarrow{\text{代入}} \text{求满足递推式的} a_n \xrightarrow{\text{求和}} y(x)\]

2. 求递推式的套路

步骤 ① 将 \(y(x) = \displaystyle\sum_{n=0}^{\infty} a_n x^n\) 代入微分方程

对 \(y(x)\) 逐项求导得 \(y', y'', \ldots\)，全部代入微分方程，展开各幂级数.

步骤 ② 将各幂级数统一到同一个 \(\sum\) 中

  • 统一 \(\sum\) 的起始值 : 所有级数统一到相同下标 ( 通常需换元 \(n \to n+1\) 或 \(n \to n-1\) ).
  • 统一 \(x^n\) 次数 : 通过换元使各项 \(x^n\) 的幂次相同，才能合并同类项.
  • 单独处理 : 起始值以外多出的有限项 ( 如常数项、一次项 ) 单独提出合并.

步骤 ③ 令各项 \(x^n\) 的系数等于零，得递推式

将合并后的级数写成 \(\displaystyle\sum_{n=N}^{\infty} (\cdots) x^n = 0\)，由幂级数系数唯一性，逐项令括号内表达式 \(= 0\)，即得 \(a_n\) 的递推关系.

  • 若微分方程含初始条件，代入 \(x=0\) 确定 \(a_0, a_1\) 等初始系数.

⚠️ 易错点

  • 统一 \(x^n\) 次数时，须同步调整 \(\sum\) 的起始值，两者必须一起换元.
  • 起始值以外多余的项 ( 如 \(n=0\) 或 \(n=1\) 项 ) 必须单独写出，不能遗漏.
  • 微分方程含初始条件时，求和函数须代入 \(x=0\) 处的值来确定任意常数.

三. 两种模式对比

平行对比速查

模式一 模式二
已知 递推式 ( 含 \(a_n\) 的关系式 ) 微分方程 ( 含 \(y', y''\) 等 )
求解目标 和函数 \(S(x)\)，进而得 \(a_n\) 系数 \(a_n\)，进而得 \(y(x)\)
关键操作 套 \(\sum x^n\)，构造微分方程 设 \(y = \sum a_n x^n\)，代入方程
核心工具 幂级数求导/积分 + 微分方程求解 幂级数逐项求导 + 系数比较
终止步骤 解微分方程，代初值，展开 \(S(x)\) 由递推式求 \(a_n\)，写出 \(\sum a_n x^n\)

四. 常用工具公式

幂级数逐项求导 / 积分

\[S(x) = \sum_{n=0}^{\infty} a_n x^n, \quad x \in (-R, R)\]

\[S'(x) = \sum_{n=1}^{\infty} n a_n x^{n-1} = \sum_{n=0}^{\infty} (n+1) a_{n+1} x^n\]

\[S''(x) = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n\]

\[\int_0^x S(t)\,dt = \sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}\]

换元技巧

\[\sum_{n=1}^{\infty} n a_n x^{n-1} \xrightarrow{m=n-1} \sum_{m=0}^{\infty} (m+1) a_{m+1} x^m\]

\[\sum_{n=0}^{\infty} a_n x^{n+1} \xrightarrow{m=n+1} \sum_{m=1}^{\infty} a_{m-1} x^m\]

五. 例题精讲

例题 1　模式一 — 由递推式构造微分方程求 \(S(x)\)

已知 \(a_0 = 1\)，\(a_{n+1} = \dfrac{a_n}{n+1}\) ( \(n \ge 0\) )，求幂级数 \(S(x) = \displaystyle\sum_{n=0}^{\infty} a_n x^n\) 的和函数.

分析 仅给出递推式，属于模式一情形 B. 将递推式两端乘以 \(x^{n+1}\) 并对 \(n\) 求和，构造关于 \(S(x)\) 的微分方程.

解

1. 套 \(\sum\) 构造关系式

递推式两端乘以 \(x^{n+1}\)

\[a_{n+1} x^{n+1} = \frac{a_n}{n+1} x^{n+1}\]

对 \(n \ge 0\) 求和

\[\sum_{n=0}^{\infty} a_{n+1} x^{n+1} = \sum_{n=0}^{\infty} \frac{a_n}{n+1} x^{n+1}\]

2. 识别左右两端

左端 \(= S(x) - a_0 = S(x) - 1\) ( 去掉 \(n=0\) 项 ).

右端 \(= \displaystyle\int_0^x \displaystyle\sum_{n=0}^{\infty} a_n t^n\,dt = \int_0^x S(t)\,dt\).

故得微分方程

\[S(x) - 1 = \int_0^x S(t)\,dt\]

3. 转化为常微分方程

两端对 \(x\) 求导

\[S'(x) = S(x), \quad S(0) = a_0 = 1\]

4. 求解

这是一阶线性 ODE，解为 \(S(x) = e^x\)，\(x \in (-\infty, +\infty)\).

例题 2　模式二 — 由微分方程求幂级数解

已知 微分方程 \(y'' - xy' - y = 0\)，初始条件 \(y(0) = 1\)，\(y'(0) = 0\)，求幂级数解 \(y(x) = \displaystyle\sum_{n=0}^{\infty} a_n x^n\).

分析 已知微分方程，属于模式二. 设幂级数解代入，统一 \(\sum\) 后令系数为零得递推式.

解

1. 由初始条件定 \(a_0, a_1\)

\(y(0) = a_0 = 1\)，\(y'(0) = a_1 = 0\).

2. 代入方程

\[y = \sum_{n=0}^{\infty} a_n x^n\]

\[y' = \sum_{n=1}^{\infty} n a_n x^{n-1}\]

\[y'' = \sum_{n=2}^{\infty} n(n-1) a_n x^{n-2}\]

代入 \(y'' - xy' - y = 0\)

\[\sum_{n=2}^{\infty} n(n-1) a_n x^{n-2} - x \sum_{n=1}^{\infty} n a_n x^{n-1} - \sum_{n=0}^{\infty} a_n x^n = 0\]

3. 统一 \(x^n\) 次数与 \(\sum\) 起始值

第一项令 \(m = n-2\)，即 \(n = m+2\)

\[\sum_{m=0}^{\infty} (m+2)(m+1) a_{m+2} x^m\]

第二、三项 \(x^n\) 次数已一致

\[-\sum_{n=1}^{\infty} n a_n x^n - \sum_{n=0}^{\infty} a_n x^n\]

合并 ( 第一项 \(m\) 改回 \(n\) )

\[\sum_{n=0}^{\infty} \left[(n+2)(n+1) a_{n+2} - n a_n - a_n\right] x^n = 0\]

4. 令系数为零得递推式

\[(n+2)(n+1) a_{n+2} - (n+1) a_n = 0\]

\[a_{n+2} = \frac{a_n}{n+2}, \quad n \ge 0\]

5. 由递推式求 \(a_n\)

由 \(a_0 = 1\)，\(a_1 = 0\)

奇数项 : \(a_1 = a_3 = a_5 = \cdots = 0\).

偶数项 : \(a_2 = \dfrac{1}{2}\)，\(a_4 = \dfrac{1}{4 \cdot 2}\)，\(a_{2k} = \dfrac{1}{2^k \cdot k!}\).

6. 写出幂级数解

\[y(x) = \sum_{k=0}^{\infty} \frac{x^{2k}}{2^k \cdot k!} = \sum_{k=0}^{\infty} \frac{1}{k!} \left(\frac{x^2}{2}\right)^k = e^{x^2/2}\]

例题 3　模式一 — 验证级数满足微分方程再求和函数

已知 \(a_0 = 1\)，\(a_1 = 0\)，\((n+2)a_{n+2} + a_n = 0\) ( \(n \ge 0\) )，求 \(S(x) = \displaystyle\sum_{n=0}^{\infty} a_n x^n\) 的和函数.

分析 递推式含 \((n+2)a_{n+2}\) 和 \(a_n\)，若配上 \(x^{n+1}\)，则前项对应积分或求导. 我们尝试直接两端同乘 \(x^{n+1}\)，然后对 \(n\) 从 \(0\) 开始求和.

解

1. 套 \(\sum\) 构造关系式

递推式两端同乘 \(x^{n+1}\)，并对 \(n \ge 0\) 求和 \[\sum_{n=0}^{\infty} (n+2) a_{n+2} x^{n+1} + \sum_{n=0}^{\infty} a_n x^{n+1} = 0\]

2. 识别各项并建立方程

第一项 \[\sum_{n=0}^{\infty} (n+2) a_{n+2} x^{n+1} = 2a_2 x + 3a_3 x^2 + 4a_4 x^3 + \cdots\] 由于 \(S(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \cdots\) 其导数为 \(S'(x) = a_1 + 2a_2 x + 3a_3 x^2 + 4a_4 x^3 + \cdots\) 所以第一项正是 \(S'(x) - a_1\). 已知 \(a_1 = 0\)，故第一项即为 \(S'(x)\).

第二项 \[\sum_{n=0}^{\infty} a_n x^{n+1} = x \sum_{n=0}^{\infty} a_n x^n = x S(x)\]

代入原式，得到关于 \(S(x)\) 的微分方程 \[S'(x) + x S(x) = 0\]

3. 求解初值问题

由定义知 \(S(0) = a_0 = 1\). 这是一个可分离变量的微分方程 \[\frac{dS}{S} = -x dx\] 两边积分得 \[\ln |S| = -\frac{1}{2}x^2 + C\] \[S(x) = C_1 e^{-x^2/2}\] 代入初值 \(S(0) = 1\)，得 \(C_1 = 1\).

4. 结论

所求和函数为 \[S(x) = e^{-x^2/2}, \quad x \in (-\infty, +\infty)\]

递推式与微分方程，级数与函数，看似两个世界，其实通过 \(\sum a_n x^n\) 这座桥梁紧密相连. 掌握”套 \(\sum x^n\) 构造方程”与”设 \(\sum a_n x^n\) 代入方程”这两把钥匙，此类综合题便迎刃而解.
