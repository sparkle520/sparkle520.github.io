一、核心思想：微元法

定积分应用的核心原理是 微元法 (Element Method).

微元法思想

将一个复杂的几何量 \(Q\) 看作是由许多微小的部分 \(dQ\) 组成的.

  1. 选取微元 : 在区间 \([x, x+dx]\) 上，近似地用简单的几何量 ( 如矩形面积、圆柱体积 ) 代替复杂的几何量，得到微分 \(dQ = f(x)dx\).
  2. 求和取极限 : \(Q = \displaystyle\int_a^b f(x)dx\).

二、重要技巧 : 利用对称性

在定积分应用中，充分利用 对称性 可以极大地简化计算，尤其是在求面积、弧长等几何量时.

对称性

1. 奇偶性 ( Even / Odd ) 若积分区间为 \([-a, a]\) ( 关于原点对称 ) :

  • 偶函数 \(f(-x) = f(x)\) : \(\displaystyle\int_{-a}^a f(x) dx = 2 \int_0^a f(x) dx\). ( 如面积、弧长通常涉及偶函数特性 )
  • 奇函数 \(f(-x) = -f(x)\) : \(\displaystyle\int_{-a}^a f(x) dx = 0\).

2. 几何对称性 ( Geometric Symmetry )

  • 关于坐标轴对称 : 若图形关于 \(x\) 轴或 \(y\) 轴对称，只需计算一半 ( 或象限部分 ) 再乘以相应倍数.
    • 例 : 摆线、星形线、椭圆.
  • 关于原点对称 : 若图形关于原点对称 ( 如双纽线 )，可计算第一象限部分再乘以 4 ( 或 2 ).

3. 极坐标系下的对称性 ( Symmetry in Polar Coordinates )

  • 关于极轴 ( x 轴 ) 对称 : \(r(-\theta) = r(\theta)\). ( 只需算 \(0 \sim \pi\) 或 \(0 \sim \text{max}\) 的一半 )
  • 关于 \(\theta = \frac{\pi}{2}\) ( y 轴 ) 对称 : \(r(\pi-\theta) = r(\theta)\).
  • 关于极点 ( 原点 ) 对称 : \(r(\pi+\theta) = r(\theta)\).
    • 例 : 双纽线 \(r^2 = a^2 \cos 2\theta\) ( 三者皆对称 ).
    • 例 : 心脏线 \(r = a(1+\cos\theta)\) ( 关于极轴对称 ).

4. 周期性 ( Periodicity ) 若 \(f(x)\) 是周期为 \(T\) 的周期函数 :

  • \(\displaystyle\int_a^{a+T} f(x) dx = \int_0^T f(x) dx\). ( 积分值与起点无关 )
  • 常用于三角函数相关的面积、弧长计算.

5. 区间再现公式 ( King’s Property )

  • \(\displaystyle\int_a^b f(x) dx = \displaystyle\int_a^b f(a+b-x) dx\).
  • 特别地 : \(\displaystyle\int_0^{\pi} x f(\sin x) dx = \frac{\pi}{2} \displaystyle\int_0^{\pi} f(\sin x) dx\). ( 极重要结论，常用于消除被积函数中的 \(x\) )

6. 华里士公式 ( Wallis Formula )

  • \(\displaystyle I_n = \int_0^{\frac{\pi}{2}} \sin^n x dx = \int_0^{\frac{\pi}{2}} \cos^n x dx\).
  • \(\displaystyle I_n = \frac{n-1}{n} I_{n-2}\).
    • \(n\) 为偶数 : \(\frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{1}{2} \cdot \frac{\pi}{2}\).
    • \(n\) 为奇数 : \(\frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{2}{3} \cdot 1\).

三、平面图形面积

3.1 直角坐标系情形

微元推导 ( Element Construction ) 取 \(x\) 为积分变量，在 \([x, x+dx]\) 上取微元. 微元近似为 矩形，底为 \(dx\)，高为 \(f(x)-g(x)\) ( 上曲线减下曲线 ). \[ dA = [f(x) - g(x)] dx \] \[ S = \int_a^b dA = \int_a^b [f(x) - g(x)] dx \]

公式 ( X 型区域 ) \[S=\int_{a}^{b}\left[f(x)-g(x)\right]dx\]

公式 ( Y 型区域 ) \[S=\int_{a}^{b}\left[\Phi(y)-\phi(y)\right]dy\]

3.2 参数方程情形

微元推导 设 \(x=x(t), y=y(t)\). \[ dA = y dx = y(t) \cdot x'(t) dt \] 注意 : 积分限需从左端点对应参数积分到右端点对应参数.

公式 \[S=\int_{\alpha}^by\mathrm{~dx~}=\int_\alpha^\beta |y(t)\cdot x'(t)|\mathrm{~dt}\]

注意

  1. 积分限 : 需对应 \(x\) 从 \(a\) 到 \(b\) 时参数 \(t\) 的变化范围.
  2. 符号 : 面积恒为正，注意 \(x'(t)\) 的符号，通常取绝对值.

示例 1 ( 摆线面积 ) 求摆线的一拱 \(\begin{cases}x=a(t-\sin t)\\[.5em]y=a(1-\cos t)\end{cases} (0 \le t \le 2\pi)\) 与 \(x\) 轴围成的面积.

解 : \[ dx = a(1-\cos t) dt \] \[ \begin{aligned} S &= \int_{0}^{2\pi} y(t) x'(t) dt \\ &= \int_{0}^{2\pi} a(1-\cos t) \cdot a(1-\cos t) dt \\ &= a^2 \int_{0}^{2\pi} (1 - 2\cos t + \cos^2 t) dt \\ &= a^2 \int_{0}^{2\pi} (1 - 2\cos t + \frac{1+\cos 2t}{2}) dt \\ &= a^2 \left[ \frac{3}{2}t - 2\sin t + \frac{1}{4}\sin 2t \right]_0^{2\pi} \\ &= a^2 \cdot 3\pi = 3\pi a^2. \end{aligned} \]

3.3 极坐标系情形

微元推导 在极坐标系下，微元是 扇形. 半径为 \(r(\theta)\)，圆心角为 \(d\theta\). 扇形面积公式 : \(dA = \frac{1}{2} r^2 d\theta\).

公式 ( 扇形微元 ) \(S=\displaystyle\int_{\alpha}^{\beta} \frac{1} {2} r^{2} ( \theta) d \theta\)

示例 2 ( 双纽线面积 ) 求双纽线 \(r^2 = a^2 \cos 2\theta\) 所围图形的面积.

解 : 双纽线关于极轴和极点对称. 考虑第一象限部分 : \(r^2 \ge 0 \implies \cos 2\theta \ge 0 \implies 0 \le 2\theta \le \frac{\pi}{2} \implies 0 \le \theta \le \frac{\pi}{4}\). 全区域面积 \(S = 4 \times S_1\).

\[ \begin{aligned} S &= 4 \int_{0}^{\frac{\pi}{4}} \frac{1}{2} r^2 d\theta \\ &= 2 \int_{0}^{\frac{\pi}{4}} a^2 \cos 2\theta d\theta \\ &= 2a^2 \cdot \frac{1}{2} \sin 2\theta \Big|_0^{\frac{\pi}{4}} \\ &= a^2 \cdot \sin \frac{\pi}{2} = a^2. \end{aligned} \]

四、体积计算

4.1 已知截面面积的立体体积

微元推导 ( Element Construction ) 建立坐标系，取 \(x\) 为积分变量. 在 \([x, x+dx]\) 上取微元，该微元近似为 柱体. 底面积为 \(A(x)\)，高为 \(dx\). \[ dV = A(x) dx \] \[ V = \int_a^b dV = \int_a^b A(x) dx \]

公式 \[ V=\int_{a}^{b} A ( x ) d x \]

示例 3 ( 公共底面体积 ) 求以半径为 \(R\) 的圆为底，平行于底面一直径的截面都是等边三角形的立体体积.

解 : 取圆所在平面为 \(xOy\) 面，圆心为原点，方程 \(x^2+y^2=R^2\). 取 \(x\) 轴垂直于截面底边. 在 \(x\) 处截面为等边三角形，边长 \(a(x) = 2y = 2\sqrt{R^2-x^2}\). 截面面积 \(A(x) = \frac{\sqrt{3}}{4} [a(x)]^2 = \frac{\sqrt{3}}{4} (4(R^2-x^2)) = \sqrt{3}(R^2-x^2)\). \[ V = \int_{-R}^{R} \sqrt{3}(R^2-x^2) dx = 2\sqrt{3} \int_0^R (R^2-x^2) dx = 2\sqrt{3} [R^2x - \frac{x^3}{3}]_0^R = \frac{4\sqrt{3}}{3}R^3. \]

4.2 旋转体体积

4.2.1 绕 x 轴旋转 ( 圆盘法 )

微元推导 ( Disk Method ) 由 \(y=f(x)\)， \(x=a, x=b\) 及 \(x\) 轴围成的曲边梯形绕 \(x\) 轴旋转. 取 \(x\) 为积分变量，在 \([x, x+dx]\) 上取微元. 微元旋转形成 圆柱 ( 圆盘 ). 底面半径 \(r = |f(x)|\)，高 \(h = dx\). \[ dV = \pi r^2 h = \pi [f(x)]^2 dx \]

公式 \[ V_x =\int_{a}^{b} \pi [f ( x )]^2 d x \]

4.2.2 绕 y 轴旋转 ( 柱壳法 )

微元推导 ( Shell Method ) 由 \(y=f(x)\)， \(x=a, x=b\) 及 \(x\) 轴围成的曲边梯形绕 \(y\) 轴旋转. 取 \(x\) 为积分变量，在 \([x, x+dx]\) 上取微元. 微元旋转形成 圆柱壳. 展开后近似为长方体 : 长 \(2\pi x\) (周长)，高 \(|f(x)|\)，厚 \(dx\). \[ dV = 2\pi x |f(x)| dx \]

公式 \[ V_y = \int_a^b 2\pi x |f(x)| dx \]

4.2.3 二重积分观点 ( 通用性更强 )

利用二重积分计算旋转体体积，思路更统一. 设平面区域 \(D\) 绕轴 \(L\) 旋转. 微元面积 \(d\sigma\)，旋转半径 \(r(x,y)\).

公式 ( 古尔丁定理微元形式 ) \[ dV = 2\pi r(x,y) d\sigma \implies V = \iint_D 2\pi r(x,y) d\sigma \]

  • 绕 x 轴 : \(r = |y|\), \(dV = 2\pi |y| d\sigma\)
  • 绕 y 轴 : \(r = |x|\), \(dV = 2\pi |x| d\sigma\)
  • 绕直线 x=k : \(r = |x-k|\), \(dV = 2\pi |x-k| d\sigma\)

示例 4 设平面图形由 \(x^2+y^2\leq 2x\) 与 \(y \ge x\) 围成. 求该平面图形绕 \(x=2\) 旋转形成的旋转体体积.

解 : 区域 \(D\) 为圆 \((x-1)^2 + y^2 \le 1\) 被直线 \(y=x\) 截得的上半弓形. 旋转轴为 \(x=2\)，旋转半径 \(r = 2-x\). \[ dV = 2\pi (2-x) d\sigma \]

\[ \begin{aligned} V &= \iint_{D} 2\pi (2-x) d\sigma \\ &= 2\pi \int_0^1 dx \int_{x}^{\sqrt{2x-x^2}} (2-x) dy \\ &= 2\pi \int_0^1 (2-x)(\sqrt{2x-x^2} - x) dx \\ &= \frac{1}{2}\pi^2 - \frac{2}{3}\pi. \end{aligned} \]

4.2.4 绕任意直线旋转 ( Rotation about Arbitrary Axis )

方法 1 : 平移法 ( Translation Method ) 深度解析

核心原理 将旋转轴 \(x=k\) ( 或 \(y=k\) ) 看作新的坐标原点，或者直接从 几何距离 的角度理解旋转半径.

关键步骤

  1. 确定半径 \(r\) 图形上任一点 \((x,y)\) 到旋转轴的垂直距离.
    • 绕水平轴 \(y=k\) : 半径 \(R(x) = |y - k| = |f(x) - k|\).
    • 绕铅直轴 \(x=k\) : 半径 \(r(x) = |x - k|\).
  2. 辨别类型
    • 实心体 旋转轴是图形的边界之一.
    • 空心体 ( 圆环/管 ) 旋转轴与图形有一定距离，需用 大圆减小圆 ( Washer Method ).
  3. 套用公式
    • 绕 \(y=k\) ( 圆盘/圆环法 ) \(V = \int_a^b \pi (R_{out}^2 - R_{in}^2) dx\).
    • 绕 \(x=k\) ( 柱壳法 ) \(V = \int_a^b 2\pi r(x) \cdot h(x) dx\).

注意 旋转轴通常不能穿过平面图形的内部，否则旋转体会重叠，需分段积分.

进阶示例 1 ( 绕水平轴平移 - 圆环法 ) 求由 \(y = \sin x \; (0 \le x \le \pi)\) 与 \(x\) 轴围成的图形绕直线 \(y = -1\) 旋转一周的体积.

解

  1. 分析几何 图形位于 \(y=0\) 到 \(y=\sin x\) 之间. 旋转轴 \(y=-1\) 在图形下方.
  2. 确定半径
    • 外半径 ( Outer Radius ) : 上边界 \(y=\sin x\) 到轴距离 \(R_{out} = | \sin x - (-1) | = \sin x + 1\).
    • 内半径 ( Inner Radius ) : 下边界 \(y=0\) 到轴距离 \(R_{in} = | 0 - (-1) | = 1\).
  3. 微元面积 \[ dA = \pi (R_{out}^2 - R_{in}^2) = \pi [(\sin x + 1)^2 - 1^2] = \pi (\sin^2 x + 2\sin x) \]
  4. 积分计算 \[ \begin{aligned} V &= \int_0^\pi \pi (\sin^2 x + 2\sin x) dx \\ &= \pi \left[ \int_0^\pi \frac{1-\cos 2x}{2} dx + 2 \int_0^\pi \sin x dx \right] \\ &= \pi \left[ \frac{\pi}{2} + 2(-\cos x)\Big|_0^\pi \right] \\ &= \pi ( \frac{\pi}{2} + 4 ) = \frac{\pi^2}{2} + 4\pi. \end{aligned} \]

进阶示例 2 ( 绕铅直轴平移 - 柱壳法 ) 求由 \(y = \ln x\), \(x\) 轴与 \(x=e\) 围成的图形绕直线 \(x=e\) 旋转一周的体积.

解

  1. 分析几何 区域 \(D = \{ (x,y) | 1 \le x \le e, 0 \le y \le \ln x \}\). 旋转轴 \(x=e\) 是图形右边界.
  2. 选用柱壳法 取 \(x\) 为积分变量 ( \(1 \to e\) ).
    • 旋转半径 微元 \(x\) 到轴 \(x=e\) 的距离 \(r(x) = e - x\).
    • 微元高度 \(h(x) = y = \ln x\).
  3. 微元体积 \(dV = 2\pi r(x) h(x) dx = 2\pi (e-x) \ln x dx\).
  4. 积分计算 \[ V = 2\pi \int_1^e (e \ln x - x \ln x) dx \]
    • 第一部分 \(\displaystyle\int_1^e \ln x dx = [x\ln x - x]_1^e = 1\).
    • 第二部分 \(\displaystyle\int_1^e x \ln x dx = [\frac{x^2}{2}\ln x]_1^e - \displaystyle\int_1^e \frac{x}{2} dx = \frac{e^2}{2} - (\frac{e^2}{4} - \frac{1}{4}) = \frac{e^2+1}{4}\). \[ V = 2\pi [ e(1) - \frac{e^2+1}{4} ] = \frac{\pi}{2} (4e - e^2 - 1). \]

补充 : 形心 ( Centroid ) 概念与公式

1. 物理意义与几何意义

  • 物理意义 若平面图形是由均匀材质 ( 密度 \(\rho\) 为常数 ) 制成的薄片，则形心即为 质心 ( Center of Mass ). 它是图形的“平衡点”.
  • 几何意义 几何图形的几何中心.

2. 计算公式 ( 直角坐标系 ) 设平面区域 \(D\) 由 \(y_1(x) \le y \le y_2(x)\) 及 \(x=a, x=b\) 围成，面积为 \(A\). 利用 力矩 ( Moment ) 原理

  • 对 y 轴的静力矩 \(M_y = \displaystyle\int_a^b x \cdot [y_2(x) - y_1(x)] dx\).
  • 对 x 轴的静力矩 \(M_x = \displaystyle\int_a^b \frac{y_1(x) + y_2(x)}{2} \cdot [y_2(x) - y_1(x)] dx = \frac{1}{2} \int_a^b [y_2^2(x) - y_1^2(x)] dx\).

形心坐标 \((\bar{x}, \bar{y})\) \[ \bar{x} = \frac{M_y}{A} = \frac{\displaystyle\int_a^b x [y_2(x)-y_1(x)] dx}{\int_a^b [y_2(x)-y_1(x)] dx} \] \[ \bar{y} = \frac{M_x}{A} = \frac{\frac{1}{2} \displaystyle\int_a^b [y_2^2(x)-y_1^2(x)] dx}{\displaystyle\int_a^b [y_2(x)-y_1(x)] dx} \]

3. 重要性质

  • 对称性 若图形关于某直线对称，则形心必在该对称轴上. ( 如圆心、椭圆中心、矩形中心 ).
  • 可加性 若图形由几部分组成，总形心坐标可通过各部分形心的加权平均求得 ( 权为面积 ).

方法 2 古尔丁第二定理 ( Pappus-Guldinus Theorem II ) 设平面图形面积为 \(A\)，形心坐标为 \((\bar{x}, \bar{y})\). 若该图形绕其平面内一条不穿过内部的直线 \(L: ax+by+c=0\) 旋转一周. 则旋转体体积 \(V\) 等于面积 \(A\) 乘图形形心走过的路程 \(2\pi d\).

\[ V = 2\pi \cdot d \cdot A \]

其中 \(d\) 为形心到直线 \(L\) 的距离 : \(d = \frac{|a\bar{x} + b\bar{y} + c|}{\sqrt{a^2+b^2}}\).

适用场景 旋转轴倾斜 ( 如 \(y=x\) ) 或求环体体积.

示例 ( 任意轴旋转 ) 求由 \(y=x^2\) 与 \(y=1\) 围成的图形绕直线 \(y=x-2\) 旋转一周所得旋转体的体积.

解

  1. 求面积 \(A\) 利用对称性 ( 或直接积分 ) \(A = \displaystyle\int_{-1}^1 (1-x^2) dx = [x - \frac{x^3}{3}]_{-1}^1 = \frac{4}{3}\).

  2. 求形心 \((\bar{x}, \bar{y})\) 分析对称性 边界曲线 \(y=x^2\) ( 偶函数 ) 和直线 \(y=1\) 均关于 \(y\) 轴 ( 即 \(x=0\) ) 对称，故围成的图形关于 \(y\) 轴对称. 根据形心性质，形心必在对称轴上，故直接得 \(\bar{x} = 0\). \(\bar{y} = \frac{1}{A} \displaystyle\int_{-1}^1 \frac{1}{2} [(1)^2 - (x^2)^2] dx = \frac{3}{4} \cdot \frac{1}{2} \displaystyle\int_{-1}^1 (1-x^4) dx = \frac{3}{8} \cdot 2 [x - \frac{x^5}{5}]_0^1 = \frac{3}{4} (1-\frac{1}{5}) = \frac{3}{5}\). 故形心为 \((0, \frac{3}{5})\).

  3. 求距离 \(d\) 直线 \(x-y-2=0\). 形心 \((0, \frac{3}{5})\) 到直线的距离 \[ d = \frac{|0 - \frac{3}{5} - 2|}{\sqrt{1^2+(-1)^2}} = \frac{|-2.6|}{\sqrt{2}} = \frac{13}{5\sqrt{2}}. \]

  4. 求体积 \(V\) \[ V = 2\pi \cdot \frac{13}{5\sqrt{2}} \cdot \frac{4}{3} = \frac{104\pi}{15\sqrt{2}} = \frac{52\sqrt{2}\pi}{15}. \]

五、平面曲线弧长

微元推导 ( Arc Differential ) 取 \(x\) 为积分变量，在 \([x, x+dx]\) 上取微元. 微元近似为 线段 ( 切线段 ). 由勾股定理 \((ds)^2 = (dx)^2 + (dy)^2\). \[ ds = \sqrt{(dx)^2 + (dy)^2} = \sqrt{1 + (\frac{dy}{dx})^2} dx \] \[ s = \int ds \]

公式汇总

  1. 直角坐标 \(y=f(x)\) \[ s = \int_a^b \sqrt{1+[f'(x)]^2} dx \]
  2. 参数方程 \(x=x(t), y=y(t)\) \[ s = \int_\alpha^\beta \sqrt{[x'(t)]^2 + [y'(t)]^2} dt \]
  3. 极坐标 \(r=r(\theta)\) \[ s = \int_\alpha^\beta \sqrt{r^2(\theta) + [r'(\theta)]^2} d\theta \]

示例 5 ( 星形线全长 ) 求星形线 \(\begin{cases} x= \cos^3t \\ y= \sin^3t \end{cases}\) 全长.

解 利用对称性，计算第一象限 (\(0 \le t \le \frac{\pi}{2}\)) 的 4 倍. \(x' = -3\cos^2 t \sin t, \quad y' = 3\sin^2 t \cos t\). \[ \begin{aligned} [x']^2 + [y']^2 &= 9\cos^4 t \sin^2 t + 9\sin^4 t \cos^2 t \\ &= 9\sin^2 t \cos^2 t (\cos^2 t + \sin^2 t) = 9\sin^2 t \cos^2 t. \end{aligned} \]

\[ \begin{aligned} l &= 4 \int_0^{\frac{\pi}{2}} \sqrt{9\sin^2 t \cos^2 t} dt \\ &= 12 \int_0^{\frac{\pi}{2}} \sin t \cos t dt = 12 \cdot \frac{1}{2} \sin^2 t \Big|_0^{\frac{\pi}{2}} \\ &= 6. \end{aligned} \]

示例 6 ( 心脏线全长 ) 求心脏线 \(r=1+\cos\theta\) 全长.

解 利用对称性，计算上半部分 (\(0 \le \theta \le \pi\)) 的 2 倍. \(r' = -\sin\theta\). \[ \begin{aligned} r^2 + (r')^2 &= (1+\cos\theta)^2 + (-\sin\theta)^2 \\ &= 1 + 2\cos\theta + \cos^2\theta + \sin^2\theta \\ &= 2 + 2\cos\theta = 4\cos^2\frac{\theta}{2}. \end{aligned} \]

\[ \begin{aligned} l &= 2 \int_{0}^{\pi} \sqrt{4\cos^{2}\frac{\theta}{2}} d\theta \\ &= 4 \int_{0}^{\pi} \cos\frac{\theta}{2} d\theta \quad (\because 0 \le \frac{\theta}{2} \le \frac{\pi}{2}, \cos > 0) \\ &= 4 \cdot 2 \sin\frac{\theta}{2} \Big|_0^\pi = 8. \end{aligned} \]

六、旋转体侧面积

微元推导 ( Surface Area Element ) 取弧长微元 \(ds\). 该微元绕 \(x\) 轴旋转一周形成 圆台侧面 ( 或圆柱侧面 ). 展开后近似为矩形 : 长为旋转圆周长 \(2\pi y\)，宽为弧长 \(ds\). \[ dA = 2\pi |y| ds \] 将 \(ds\) 用 \(x\) 表示 : \(dA = 2\pi |f(x)| \sqrt{1+[f'(x)]^2} dx\).

公式汇总 ( 绕 x 轴旋转 )

  1. 直角坐标 \(y=f(x)\) \[ S = \int_a^b 2\pi |y| \sqrt{1+(y')^2} dx \]
  2. 参数方程 \(x=x(t), y=y(t)\) \[ S = \int_\alpha^\beta 2\pi |y(t)| \sqrt{[x'(t)]^2 + [y'(t)]^2} dt \]
  3. 极坐标 \(r=r(\theta)\) \[ S = \int_\alpha^\beta 2\pi |r\sin\theta| \sqrt{r^2(\theta) + [r'(\theta)]^2} d\theta \] ( 注 : 弧长微元 \(ds = \sqrt{r^2+(r')^2} d\theta\), 旋转半径 \(y = r\sin\theta\) )

示例 7 计算由 \(x^2+y^2=R^2\) 与 \(x=x_1, x=x_2 \; (-\mathbb{R}\leq x_1\leq x_2\leq\mathbb{R})\) 所围成的图形绕 \(x\) 轴所形成的球台的侧面积.

解 : 上半圆方程 \(y = \sqrt{R^2-x^2}\). \[ y' = \frac{-x}{\sqrt{R^2-x^2}} \implies \sqrt{1+(y')^2} = \sqrt{1 + \frac{x^2}{R^2-x^2}} = \frac{R}{\sqrt{R^2-x^2}} \]

\[ \begin{aligned} A &= \int_{x_1}^{x_2} 2\pi y \sqrt{1+(y')^2} dx \\ &= \int_{x_1}^{x_2} 2\pi \sqrt{R^2-x^2} \cdot \frac{R}{\sqrt{R^2-x^2}} dx \\ &= \int_{x_1}^{x_2} 2\pi R dx \\ &= 2\pi R(x_2-x_1). \end{aligned} \]

示例 8 ( 极坐标侧面积 - 心脏线 ) 求心脏线 \(r = a(1+\cos\theta) \; (a>0)\) 绕极轴旋转一周所得旋转体的侧面积.

解

  1. 分析对称性 心脏线关于极轴对称，\(\theta\) 范围 \(0 \to \pi\) ( 上半部分 ) 旋转一周即可.
  2. 准备微元
    • \(r' = -a\sin\theta\).
    • 弧长微元 \(ds = \sqrt{r^2+(r')^2} d\theta = \sqrt{a^2(1+\cos\theta)^2 + a^2\sin^2\theta} d\theta\) \[ = a \sqrt{1+2\cos\theta+\cos^2\theta+\sin^2\theta} d\theta = a\sqrt{2+2\cos\theta} d\theta = 2a\cos\frac{\theta}{2} d\theta \] ( 注意 : \(0 \le \theta \le \pi \implies \cos\frac{\theta}{2} \ge 0\) ).
    • 旋转半径 \(y = r\sin\theta = a(1+\cos\theta)\sin\theta\).
  3. 积分计算 \[ S = \int_0^\pi 2\pi y ds = \int_0^\pi 2\pi [a(1+\cos\theta)\sin\theta] \cdot [2a\cos\frac{\theta}{2}] d\theta \] \[ = 4\pi a^2 \int_0^\pi (1+\cos\theta)\sin\theta \cos\frac{\theta}{2} d\theta \] 利用半角公式 \(1+\cos\theta = 2\cos^2\frac{\theta}{2}, \sin\theta = 2\sin\frac{\theta}{2}\cos\frac{\theta}{2}\). \[ S = 4\pi a^2 \int_0^\pi (2\cos^2\frac{\theta}{2}) (2\sin\frac{\theta}{2}\cos\frac{\theta}{2}) \cos\frac{\theta}{2} d\theta \] \[ = 16\pi a^2 \int_0^\pi \cos^4\frac{\theta}{2} \sin\frac{\theta}{2} d\theta \] 令 \(u = \cos\frac{\theta}{2}, du = -\frac{1}{2}\sin\frac{\theta}{2} d\theta\). 当 \(\theta=0, u=1\); 当 \(\theta=\pi, u=0\). \[ S = 16\pi a^2 \int_1^0 u^4 (-2du) = 32\pi a^2 \int_0^1 u^4 du = 32\pi a^2 [\frac{u^5}{5}]_0^1 = \frac{32}{5}\pi a^2. \]

七、定积分物理应用

定积分在物理学中有着广泛的应用，核心依然是 微元法 \(dQ = f(x) dx \implies Q = \displaystyle\int f(x) dx\).

7.1 变力做功 ( Work )

物理原理 如果物体在变力 \(F(x)\) 的作用下，沿 \(x\) 轴从 \(x=a\) 移动到 \(x=b\).

功的微元 \(dW = F(x) dx\).

总功 \[ W = \int_a^b F(x) dx \]

常见模型

  1. 弹簧做功 胡克定律 \(F = kx\). \[ W = \int_a^b kx dx = \frac{1}{2}k(b^2-a^2) \]
  2. 抽水做功
    • 将薄层水 ( 重 \(dG\) ) 提升高度 \(h(y)\).
    • \(dW = h(y) dG = \rho g A(y) h(y) dy\).
    • \(W = \displaystyle\int \rho g A(y) h(y) dy\).
  3. 缆绳/链条提升
    • 考虑自身重力，微元重 \(dG = \rho dy\).

示例 9 ( 抽水做功 )

一圆柱形蓄水池，直径 \(20 \text{m}\)，深 \(15 \text{m}\)，盛满水. 现将水全部抽出池外，求克服重力所做的功 ( 水的密度 \(\rho = 1000 \text{kg/m}^3\), \(g\) 取 \(9.8 \text{m/s}^2\) ).

解 建立坐标系，取圆柱底面中心为原点，向上为 \(y\) 轴正方向. 则水面方程为 \(y=15\). 池底 \(y=0\).

  1. 选取微元 在 \([y, y+dy]\) 处取一层薄水层.
    • 体积微元 \(dV = \pi R^2 dy = \pi (10)^2 dy = 100\pi dy\).
    • 重力微元 \(dG = \rho g dV = 100\pi \rho g dy\).
  2. 确定位移 该层水被提升的高度为 \(15 - y\) ( 或抽出池外即到达 \(y=15\) 高度 ).
    • 功的微元 \(dW = (15-y) dG = 100\pi \rho g (15-y) dy\).
  3. 积分计算 \[ W = \int_0^{15} 100\pi \rho g (15-y) dy \] \[ = 100\pi \rho g [15y - \frac{y^2}{2}]_0^{15} \] \[ = 100\pi \rho g (225 - \frac{225}{2}) = 100\pi \rho g \cdot \frac{225}{2} = 11250\pi \rho g. \] 代入数值 \(W \approx 3.46 \times 10^8 \text{J}\).

7.2 静水压力 ( Hydrostatic Force )

物理原理 垂直浸没在液体中的平面板，其受到的侧压力. 压强 \(P = \rho g h\) ( \(h\) 为水深 ). 取水平窄条微元 ( 面积 \(dA = w(y) dy\) )，深度为 \(h(y)\). 压力微元 \(dF = P \cdot dA = \rho g h(y) w(y) dy\).

总压力 \[ F = \int \rho g h(y) w(y) dy \]

示例 10 ( 三角形闸门压力 ) 一个等腰三角形闸门，底宽 \(b=4\text{m}\)，高 \(H=3\text{m}\)，垂直放置在水中. 顶点向上且在水面下 \(2\text{m}\) 处，底边平行于水面. 求闸门一侧所受的静水压力.

解 建立坐标系，取三角形顶点为原点，向下为 \(y\) 轴正方向. 则三角形顶点 \(y=0\) ( 水深 \(2\text{m}\) )，底边 \(y=3\) ( 水深 \(5\text{m}\) ). 水深 \(h(y) = y + 2\).

  1. 选取微元 在 \([y, y+dy]\) 处取水平窄条.
  2. 确定宽度 利用相似三角形 : \(\frac{w(y)}{b} = \frac{y}{H} \implies w(y) = \frac{4}{3}y\).
    • 面积微元 \(dA = w(y) dy = \frac{4}{3}y dy\).
  3. 确定压强 微元处深度为 \(y+2\).
    • 压强 \(P = \rho g (y+2)\).
  4. 压力微元 \[ dF = P dA = \rho g (y+2) \cdot \frac{4}{3}y dy = \frac{4}{3}\rho g (y^2 + 2y) dy \]
  5. 积分计算 \[ F = \int_0^3 \frac{4}{3}\rho g (y^2 + 2y) dy \] \[ = \frac{4}{3}\rho g [\frac{y^3}{3} + y^2]_0^3 \] \[ = \frac{4}{3}\rho g (9 + 9) = 24\rho g. \] 代入数值 \(F \approx 2.35 \times 10^5 \text{N}\).

7.3 引力 ( Gravitation )

物理原理 细杆 ( 线密度 \(\rho(x)\) ) 对质点 \(m\) 的引力. 万有引力定律 \(F = G \frac{Mm}{r^2}\).

微元引力 \(dF = G \frac{m \cdot (\rho dx)}{r^2(x)}\).

总引力 \[ F = \int G \frac{m \rho(x)}{r^2(x)} dx \]

示例 11 ( 细杆引力 ) 长为 \(L\)，质量为 \(M\) 的均匀细杆，位于 \(x\) 轴上区间 \([0, L]\). 在 \(x\) 轴上点 \(A(L+a, 0)\) 处 ( \(a>0\) ) 有一质量为 \(m\) 的质点. 求细杆对质点的引力.

解 细杆线密度 \(\mu = \frac{M}{L}\).

  1. 选取微元 在细杆上 \([x, x+dx]\) 处取微元 ( \(0 \le x \le L\) ).
    • 微元质量 \(dM = \mu dx\).
  2. 确定距离 微元 \(x\) 到质点 \(L+a\) 的距离 \(r = (L+a) - x\).
  3. 引力微元 \[ dF = G \frac{m \cdot dM}{r^2} = G \frac{m \mu dx}{(L+a-x)^2} \] 方向沿 \(x\) 轴负方向 ( 指向细杆 ).
  4. 积分计算 \[ F = \int_0^L G m \mu \frac{1}{(L+a-x)^2} dx \] \[ = G m \mu [\frac{1}{L+a-x}]_0^L \] \[ = G m \mu (\frac{1}{a} - \frac{1}{L+a}) = G m \mu \frac{L}{a(L+a)}. \] 代入 \(\mu = \frac{M}{L}\) \[ F = G \frac{Mm}{a(L+a)}. \] ( 注 : 当 \(a \gg L\) 时， \(F \approx G \frac{Mm}{a^2}\), 退化为质点间引力公式 ).

八、综合难题选讲 ( Selected Hard Problems )

难题 1 ( 摆线下的面积与旋转体 ) 设摆线 \(\begin{cases}x=a(t-\sin t)\\[0.5em]y=a(1-\cos t)\end{cases} (0 \le t \le 2\pi)\).

  1. 求一拱摆线与 \(x\) 轴围成的平面图形的面积 \(S\).
  2. 求该图形绕 \(x\) 轴旋转一周所得旋转体的体积 \(V_x\).
  3. 求该图形绕 \(y\) 轴旋转一周所得旋转体的体积 \(V_y\).

解 (1) 面积 \(S = 3\pi a^2\) ( 见前文示例 1 ).

  1. 绕 \(x\) 轴旋转 ( 圆盘法 ) \[ V_x = \int_0^{2\pi} \pi y^2(t) x'(t) dt = \pi \int_0^{2\pi} a^2(1-\cos t)^2 \cdot a(1-\cos t) dt \] \[ = \pi a^3 \int_0^{2\pi} (1-\cos t)^3 dt = \pi a^3 \int_0^{2\pi} (1 - 3\cos t + 3\cos^2 t - \cos^3 t) dt \] 利用周期性与积分性质 ( \(\displaystyle\int_0^{2\pi} \cos t dt = 0, \displaystyle\int_0^{2\pi} \cos^3 t dt = 0\) ) : \[ = \pi a^3 \int_0^{2\pi} (1 + 3\cos^2 t) dt = \pi a^3 [ 2\pi + 3 \cdot \pi ] = 5\pi^2 a^3. \]

  2. 绕 \(y\) 轴旋转 ( 柱壳法 ) \[ V_y = \int_0^{2\pi} 2\pi x(t) y(t) x'(t) dt \quad \text{( 注意 : 这里需细致分析，或用二重积分补集法 )} \] 更简便的方法是利用 古尔丁定理 ( Pappus-Guldinus Theorem ) \(V_y = 2\pi x_c \cdot S\).

但考研通常要求积分计算. 补集法 : 矩形体积 - 外部体积.

或者直接积分 ( 注意 \(x\) 单调增 ) \[ V_y = 2\pi \int_0^{2\pi} a(t-\sin t) \cdot a(1-\cos t) \cdot a(1-\cos t) dt \] \[ = 2\pi a^3 \int_0^{2\pi} (t - \sin t)(1 - 2\cos t + \cos^2 t) dt \] 计算较繁琐，最终结果 \(V_y = 6\pi^3 a^3\).

难题 2 ( 变限积分定义的弧长 ) 求曲线 \(y = \displaystyle\int_0^x \sqrt{\cos 2t} dt\) 上相应于 \(x\) 从 \(0\) 到 \(\frac{\pi}{4}\) 的一段弧长.

解 由变限积分求导 \(y' = \sqrt{\cos 2x}\). 弧长微元 \[ ds = \sqrt{1 + (y')^2} dx = \sqrt{1 + \cos 2x} dx = \sqrt{2\cos^2 x} dx \] 在 \(0 \le x \le \frac{\pi}{4}\) 上，\(\cos x > 0\)，故 \(ds = \sqrt{2} \cos x dx\).

\[ s = \int_0^{\frac{\pi}{4}} \sqrt{2} \cos x dx = \sqrt{2} [\sin x]_0^{\frac{\pi}{4}} = \sqrt{2} \cdot \frac{\sqrt{2}}{2} = 1. \]

难题 3 ( 极坐标面积的公共部分 ) 求圆 \(r = \sqrt{2} \sin \theta\) 与双纽线 \(r^2 = \sin 2\theta\) 公共部分的面积.

解 首先求交点. \(\sin 2\theta = (\sqrt{2}\sin\theta)^2 = 2\sin^2\theta\). \(2\sin\theta\cos\theta = 2\sin^2\theta \implies \sin\theta(\cos\theta - \sin\theta) = 0\). \(\theta = 0\) ( 极点 ) 或 \(\tan\theta = 1 \implies \theta = \frac{\pi}{4}\).

作图分析 在 \(0 \le \theta \le \frac{\pi}{4}\) 区间，圆 \(r = \sqrt{2}\sin\theta\) 在双纽线内部. 在 \(\frac{\pi}{4} \le \theta \le \frac{\pi}{2}\) 区间，双纽线在圆内部.

利用对称性 ( 或直接分段 ) \[ S = \int_0^{\frac{\pi}{4}} \frac{1}{2} (\sqrt{2}\sin\theta)^2 d\theta + \int_{\frac{\pi}{4}}^{\frac{\pi}{2}} \frac{1}{2} (\sin 2\theta) d\theta \] \[ S_1 = \int_0^{\frac{\pi}{4}} \sin^2\theta d\theta = \int_0^{\frac{\pi}{4}} \frac{1-\cos 2\theta}{2} d\theta = [\frac{\theta}{2} - \frac{\sin 2\theta}{4}]_0^{\frac{\pi}{4}} = \frac{\pi}{8} - \frac{1}{4}. \] \[ S_2 = \frac{1}{2} \int_{\frac{\pi}{4}}^{\frac{\pi}{2}} \sin 2\theta d\theta = \frac{1}{2} [-\frac{1}{2}\cos 2\theta]_{\frac{\pi}{4}}^{\frac{\pi}{2}} = -\frac{1}{4} (-1 - 0) = \frac{1}{4}. \] \[ S = S_1 + S_2 = \frac{\pi}{8} - \frac{1}{4} + \frac{1}{4} = \frac{\pi}{8}. \]
