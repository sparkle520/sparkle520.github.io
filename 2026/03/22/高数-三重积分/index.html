第十章 三重积分

本章重点掌握 三重积分的三种坐标系计算方法 ( 直角坐标、柱面坐标、球面坐标 ). 深入理解 “先一后二” 与 “先二后一” 的投影化简策略，以及利用 奇偶性 与 轮换对称性 快速化简的技巧.

总结与速查 ( Cheat Sheet )

坐标系 体积微元 \(dV\) 适用场景 变量范围
直角坐标 \(dx\,dy\,dz\) 长方体、一般区域 \(x,y,z\) 各有上下界
柱面坐标 \(r\,dr\,d\theta\,dz\) 含 \(x^2+y^2\) 的柱体、锥体 \(r\ge0,\ \theta\in[0,2\pi),\ z\in\mathbb{R}\)
球面坐标 \(\rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta\) 球体、含 \(x^2+y^2+z^2\) \(\rho\ge0,\ \varphi\in[0,\pi],\ \theta\in[0,2\pi)\)
技巧 条件 结论
奇偶性 \(\Omega\) 关于 \(xOy\) 面对称，\(f\) 关于 \(z\) 为奇函数 \(\displaystyle\iiint_\Omega f\,dV = 0\)
奇偶性 \(\Omega\) 关于 \(xOy\) 面对称，\(f\) 关于 \(z\) 为偶函数 \(\displaystyle\iiint_\Omega f\,dV = 2\iiint_{\Omega_{z\ge0}} f\,dV\)
轮换对称 \(\Omega\) 关于 \(x,y,z\) 完全对称 ( 如球体 ) \(\displaystyle\iiint x^2 dV = \iiint y^2 dV = \iiint z^2 dV\)

一. 基本概念与定义

1. 三重积分的定义

极限定义

设 \(\Omega\) 为空间中有界闭区域，\(f(x,y,z)\) 在 \(\Omega\) 上有界. 将 \(\Omega\) 任意分成 \(n\) 个小区域 \(\Delta V_i\)，在每个小区域上任取一点 \((\xi_i, \eta_i, \zeta_i)\)，作乘积 \(f(\xi_i,\eta_i,\zeta_i)\Delta V_i\) 并求和. 当各小区域直径的最大值 \(\lambda\to0\) 时，若极限存在，则称此极限为 \(f\) 在 \(\Omega\) 上的 三重积分

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \lim_{\lambda\to0}\sum_{i=1}^n f(\xi_i,\eta_i,\zeta_i)\,\Delta V_i\]

物理意义

  • 若 \(f(x,y,z) = 1\)，则 \(\displaystyle\iiint_\Omega 1\,dV\) 即为区域 \(\Omega\) 的 体积 \(V\).
  • 若 \(f(x,y,z) = \mu(x,y,z)\) 为空间体的 体密度，则 \(\displaystyle\iiint_\Omega \mu\,dV\) 为该空间体的 质量.

基本性质

  1. 线性性 \(\displaystyle\iiint_\Omega [\alpha f + \beta g]\,dV = \alpha\iiint_\Omega f\,dV + \beta\iiint_\Omega g\,dV\)
  2. 可加性 若 \(\Omega = \Omega_1 \cup \Omega_2\) ( 不重叠 )，则 \(\displaystyle\iiint_\Omega f\,dV = \iiint_{\Omega_1} f\,dV + \iiint_{\Omega_2} f\,dV\)
  3. 比较定理 若 \(f\le g\) 在 \(\Omega\) 上成立，则 \(\displaystyle\iiint_\Omega f\,dV \le \iiint_\Omega g\,dV\)
  4. 估值定理 设 \(m \le f(x,y,z) \le M\)，\(V\) 为 \(\Omega\) 的体积，则 \(mV \le \iiint_\Omega f\,dV \le MV\)
  5. 中值定理 若 \(f\) 在 \(\Omega\) 上连续，则存在 \((\xi,\eta,\zeta)\in\Omega\)，使得 \(\displaystyle\iiint_\Omega f\,dV = f(\xi,\eta,\zeta)\cdot V\)

二. 直角坐标系下的计算

1. 化为累次积分 ( “先一后二”法 )

核心思想 投影法

将三重积分化为累次积分，关键在于确定积分次序与各变量的积分限.

标准方法 ( 先对 \(z\) 积分，再对 \(x,y\) 积分 )

设区域 \(\Omega\) 在 \(xOy\) 面上的投影为 \(D_{xy}\)，对于 \((x,y)\in D_{xy}\)，\(z\) 从下方曲面 \(z_1(x,y)\) 变化到上方曲面 \(z_2(x,y)\)

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \iint_{D_{xy}}\left[\int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz\right]dxdy\]

计算口诀 “一投，二定，三积”

  • 一投 将 \(\Omega\) 向某坐标面投影，得二重积分区域 \(D\).
  • 二定 对固定的 \((x,y)\in D\)，确定第三变量 ( 如 \(z\) ) 的上下界.
  • 三积 先完成关于第三变量的定积分，再完成二重积分.

示例 \(\Omega\) 为长方体 \(\{0\le x\le1,\,0\le y\le2,\,0\le z\le3\}\)

\[\displaystyle\iiint_\Omega xyz\,dV = \int_0^1 x\,dx\int_0^2 y\,dy\int_0^3 z\,dz = \frac{1}{2}\cdot2\cdot\frac{9}{2} = \frac{9}{2}\]

2. “先二后一”法 ( 截面法 )

核心思想 用截面面积函数积分

将 \(\Omega\) 沿某一坐标轴方向 ( 如 \(z\) 轴 ) 切成无数薄片. 设 \(z\in[c,d]\)，在高度 \(z\) 处，截面为 \(D(z)\)，则

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \int_c^d\left[\iint_{D(z)} f(x,y,z)\,dxdy\right]dz\]

当截面 \(D(z)\) 形状规则 ( 圆形、椭圆、三角形 ) 时，“先二后一”法往往更简洁，尤其适合旋转体 ( 截面为圆盘 ).

💡 选择方法的技巧

情形 推荐方法
区域边界均为坐标面或简单平面 先一后二 ( 直接展开 )
截面形状规则 ( 圆、椭圆 ) 先二后一 ( 截面法 )
含 \(x^2+y^2\) 且截面为圆 先换柱面坐标，再先一后二
含 \(x^2+y^2+z^2\)，区域为球 直接换球面坐标

三. 柱面坐标系

1. 坐标变换

定义

\[\begin{cases} x = r\cos\theta \\[.5em] y = r\sin\theta \\[.5em] z = z \end{cases} \quad r\ge0,\quad \theta\in[0,2\pi),\quad z\in\mathbb{R}\]

体积微元

\[dV = r\,dr\,d\theta\,dz\]

柱面坐标就是在极坐标基础上加了一个 \(z\) 轴，因此 \(dxdy\) 变为 \(r\,dr\,d\theta\)，再乘以 \(dz\) 即得三维体积微元. 切记不要漏掉 Jacobi 行列式的因子 \(r\).

积分公式

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \iiint f(r\cos\theta,\,r\sin\theta,\,z)\cdot r\,dr\,d\theta\,dz\]

2. 适用场景与换元要点

适用场景

  • 积分区域 \(\Omega\) 含有 \(x^2+y^2\) ( 如柱面、锥面、抛物面围成的区域 ).
  • 被积函数中出现 \(x^2+y^2\)、\(\sqrt{x^2+y^2}\) 等形式.

换元步骤

  1. 令 \(x=r\cos\theta,\,y=r\sin\theta\)，将曲面方程转化为 \(r,\theta,z\) 的关系.
  2. 确定 \(\theta\) 的范围 ( 通常 \([0,2\pi]\) 或部分角度范围 ).
  3. 固定 \(\theta\)，确定 \(r\) 的范围 ( 由投影区域边界决定 ).
  4. 确定 \(z\) 的范围 ( 上下曲面给出，注意用 \(r\) 表达 ).

常用曲面的柱面坐标形式

曲面 直角方程 柱面坐标方程
圆柱面 \(x^2+y^2=a^2\) \(r=a\)
锥面 \(z=\sqrt{x^2+y^2}\) \(z=r\)
抛物面 \(z=x^2+y^2\) \(z=r^2\)
球面 \(x^2+y^2+z^2=R^2\) \(r^2+z^2=R^2\)

四. 球面坐标系

1. 坐标变换

定义

\[\begin{cases} x = \rho\sin\varphi\cos\theta \\[.5em] y = \rho\sin\varphi\sin\theta \\[.5em] z = \rho\cos\varphi \end{cases} \quad \rho\ge0,\quad \varphi\in[0,\pi],\quad \theta\in[0,2\pi)\]

其中 \(\rho = \sqrt{x^2+y^2+z^2}\) 为径向距离，\(\varphi\) 为与 \(z\) 轴正方向的夹角 ( 极角 )，\(\theta\) 为在 \(xOy\) 面上与 \(x\) 轴正方向的夹角 ( 方位角 ).

体积微元

\[dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta\]

利用 Jacobi 行列式 \(\dfrac{\partial(x,y,z)}{\partial(\rho,\varphi,\theta)}=\rho^2\sin\varphi\) 推导. 务必记住 \(\sin\varphi\) 因子，且 \(\varphi\) 是与 \(z\) 轴的夹角，而非与 \(xOy\) 面的夹角.

积分公式

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \iiint f(\rho\sin\varphi\cos\theta,\,\rho\sin\varphi\sin\theta,\,\rho\cos\varphi)\cdot\rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta\]

2. 适用场景与常用结论

适用场景

  • 积分区域为 球体 ( 全球、半球、球壳等 ) 或锥体.
  • 被积函数含 \(x^2+y^2+z^2\) 或 \(\sqrt{x^2+y^2+z^2}\).
  • 积分区域边界含球面方程.

常用球面坐标形式

曲面/区域 直角方程 球面坐标方程
球面 \(x^2+y^2+z^2=R^2\) \(\rho=R\)
上半球 \(z\ge0,\,x^2+y^2+z^2\le R^2\) \(0\le\rho\le R,\,0\le\varphi\le\frac{\pi}{2}\)
锥面 \(z=\sqrt{x^2+y^2}\) \(\varphi=\frac{\pi}{4}\)
\(z\) 轴上方锥体 \(z\ge\sqrt{x^2+y^2}\) \(0\le\varphi\le\frac{\pi}{4}\)
球心在原点的球壳 \(a^2\le x^2+y^2+z^2\le b^2\) \(a\le\rho\le b\)

常用积分结论 ( 整球 \(\Omega: x^2+y^2+z^2\le R^2\) )

\[\displaystyle\iiint_\Omega 1\,dV = \frac{4}{3}\pi R^3\]

\[\displaystyle\iiint_\Omega (x^2+y^2+z^2)\,dV = \frac{4}{5}\pi R^5\]

\[\displaystyle\iiint_\Omega x^2\,dV = \iiint_\Omega y^2\,dV = \iiint_\Omega z^2\,dV = \frac{1}{3}\cdot\frac{4}{5}\pi R^5 = \frac{4\pi R^5}{15}\]

五. 对称性与奇偶性 ( 快速化简利器 )

1. 奇偶性

关于 \(xOy\) 面 ( 即 \(z=0\) ) 对称

若空间区域 \(\Omega\) 关于 \(xOy\) 面对称 ( \((x,y,z)\in\Omega \Leftrightarrow (x,y,-z)\in\Omega\) )，则

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \begin{cases} 0 & f \text{ 关于 } z \text{ 为奇函数，即 } f(x,y,-z)=-f(x,y,z) \\[.5em] 2\displaystyle\iiint_{\Omega_{z\ge0}} f\,dV & f \text{ 关于 } z \text{ 为偶函数，即 } f(x,y,-z)=f(x,y,z) \end{cases}\]

同理，关于 \(yOz\) 面 ( 关于 \(x\) ) 和 \(zOx\) 面 ( 关于 \(y\) ) 有完全类似的结论.

判断奇偶性时，只看对称轴 ( 或面 ) 所对应的变量，其余变量视作参数.

2. 轮换对称性

若积分区域 \(\Omega\) 关于 \(x,y,z\) 具有完全的 轮换对称性 ( 交换任意两个变量，区域方程不变 )，如球体 \(x^2+y^2+z^2\le R^2\)，则

\[\displaystyle\iiint_\Omega f(x,y,z)\,dV = \iiint_\Omega f(y,z,x)\,dV = \iiint_\Omega f(z,x,y)\,dV\]

典型应用

\[\displaystyle\iiint_\Omega x^2\,dV = \iiint_\Omega y^2\,dV = \iiint_\Omega z^2\,dV = \frac{1}{3}\iiint_\Omega (x^2+y^2+z^2)\,dV\]

\[\displaystyle\iiint_\Omega x\,dV = \iiint_\Omega y\,dV = \iiint_\Omega z\,dV\]

技巧 遇到被积函数中有 \(x^2+y^2\)、\(x^2+z^2\) 等，先用轮换对称将其转化为含 \(x^2+y^2+z^2\) 的形式，往往极大简化计算.

六. 物理应用

质量

\[M = \iiint_\Omega \mu(x,y,z)\,dV \quad (\mu \text{ 为体密度})\]

质心坐标 ( 若密度均匀 \(\mu=\text{const}\)，则质心即为 形心 )

\[\bar{x} = \frac{1}{M}\iiint_\Omega x\mu\,dV,\quad \bar{y} = \frac{1}{M}\iiint_\Omega y\mu\,dV,\quad \bar{z} = \frac{1}{M}\iiint_\Omega z\mu\,dV\]

转动惯量

\[I_z = \iiint_\Omega (x^2+y^2)\mu\,dV \quad ( \text{ 绕 }z\text{ 轴旋转 } )\]

\[I_O = \iiint_\Omega (x^2+y^2+z^2)\mu\,dV \quad ( \text{ 绕原点旋转 } )\]

七. 例题精讲

专题 A 先一后二 ( 投影法 )

例 A1 标准三角域加高度 ( 奇偶与对称简化 )

设 \(\Omega : 0 \le z \le 1 - x - y,\ x \ge 0,\ y \ge 0,\ x + y \le 1\)，求 \(I = \displaystyle\iiint_\Omega ( x + y )\, dV\).

投影到 \(xOy\) 面为三角域 \(D\)，先对 \(z\) 积分 \[ I = \iint_D ( x + y ) \big( 1 - x - y \big)\, dxdy. \] 用对称性与已知积分 \(\displaystyle\int\!\!\int_D x\, dxdy = \int\!\!\int_D y\, dxdy = \frac{1}{6}\)，\(\displaystyle\int\!\!\int_D x^2\, dxdy = \int\!\!\int_D y^2\, dxdy = \frac{1}{12}\)，\(\displaystyle\int\!\!\int_D xy\, dxdy = \frac{1}{24}\) 得 \[ I = \frac{1}{3} - \frac{1}{4} = \frac{1}{12}. \]

例 A2 圆心不在原点的圆柱域 ( 平移奇零 )

设 \(\Omega : ( x - 1 )^2 + y^2 \le 1,\ 0 \le z \le h\)，求 \[ I_1 = \iiint_\Omega ( x - 1 )\, dV,\quad I_2 = \iiint_\Omega x\, dV. \] 先对 \(z\) 积分，高度为 \(h\) \[ I_1 = h \iint_{(x-1)^2+y^2 \le 1} ( x - 1 )\, dxdy = 0 \quad ( \text{ 关于 } x = 1 \text{ 奇函数 } ). \] \[ I_2 = h \iint_{(x-1)^2+y^2 \le 1} x\, dxdy = h \iint ( ( x - 1 ) + 1 )\, dxdy = h \cdot 0 + h \cdot \pi \cdot 1^2 = \pi h. \]

例 A3 平移后的抛物帽体积 ( 平移 + 极坐标 )

设 \(\Omega : 0 \le z \le 1 - ( x - 1 )^2 - y^2\)，求 \(V = \displaystyle\iiint_\Omega 1\, dV\).

先对 \(z\) 积分，得

\[ V = \iint_{(x-1)^2 + y^2 \le 1} \big( 1 - ( x - 1 )^2 - y^2 \big)\, dxdy. \] 以 \(u = x - 1,\ v = y\) 转为极坐标 \(u = r\cos\theta,\ v = r\sin\theta\) \[ V = \int_0^{2\pi}\! d\theta \int_0^1 ( 1 - r^2 ) r\, dr = 2\pi \left[ \frac{1}{2} - \frac{1}{4} \right] = \frac{\pi}{2}. \]

例 A4 球体上 \(x^2+y^2\) 的积分 ( 投影 + 轮换对称 )

设球体 \(\Omega : x^2 + y^2 + z^2 \le R^2\)，求 \(I = \displaystyle\iiint_\Omega ( x^2 + y^2 )\, dV\). 投影法 \[ I = \iint_{x^2+y^2 \le R^2} ( x^2 + y^2 ) \cdot 2\sqrt{ R^2 - x^2 - y^2 }\, dxdy. \] 用轮换对称 \(\displaystyle\iiint_\Omega x^2\, dV = \iiint_\Omega y^2\, dV = \iiint_\Omega z^2\, dV = \frac{1}{3} \iiint_\Omega ( x^2 + y^2 + z^2 )\, dV\)，

得 \(I = \dfrac{2}{3} \iiint_\Omega ( x^2 + y^2 + z^2 )\, dV = \dfrac{2}{3} \cdot \dfrac{4\pi R^5}{5} = \dfrac{8\pi R^5}{15}\).

专题 B 先二后一 ( 截面法 )

例 B1 双抛物面夹层体积 ( 分段截面 )

设 \(\Omega\) 由 \(z = x^2 + y^2\) 与 \(z = 2 - x^2 - y^2\) 围成，求 \(V\).

当 \(z \in [ 0, 1 ]\)，截面为半径 \(\sqrt{z}\) 的圆，面积 \(A(z) = \pi z\); 当 \(z \in [ 1, 2 ]\)，截面为半径 \(\sqrt{ 2 - z }\) 的圆，面积 \(A(z) = \pi ( 2 - z )\) \[ V = \int_0^1 \pi z\, dz + \int_1^2 \pi ( 2 - z )\, dz = \pi \left[ \frac{1}{2} + \frac{1}{2} \right] = \pi. \]

例 B2 圆心不在原点的球体—线性与奇偶 ( 平移 + 截面 )

设 \(\Omega : ( x - a )^2 + y^2 + z^2 \le R^2\)，求 \[ I_1 = \iiint_\Omega ( x - a )\, dV,\quad I_2 = \iiint_\Omega x\, dV. \] 固定 \(z\)，截面为圆盘 \(( x - a )^2 + y^2 \le R^2 - z^2\)，关于 \(x = a\) 对称 \[ I_1 = \int_{-R}^{R} \left[ \iint_{(x-a)^2+y^2 \le R^2 - z^2} ( x - a )\, dxdy \right] dz = 0. \] 而 \[ I_2 = \iiint_\Omega \big( ( x - a ) + a \big)\, dV = 0 + a \cdot \text{Vol} ( \Omega ) = a \cdot \frac{4\pi R^3}{3}. \]

例 B3 线性截面高度×奇偶判零

设 \(\Omega : x \in [ 0, 1 ],\ y \in [ 0, 1 ],\ z \in [ x,\ 2 - x ]\)，求 \(I = \displaystyle\iiint_\Omega ( x - \tfrac{1}{2} )\, dV\).

截面法 : 对每个 \(x\)，截面面积 \(A(x) = 1\)，高度 \(H(x) = 2 - 2x\)， \[ I = \int_0^1 ( x - \tfrac{1}{2} ) \cdot H(x)\, dx = 2 \int_0^1 ( x - \tfrac{1}{2} ) ( 1 - x )\, dx = 0 \quad ( \text{关于 } x = \tfrac{1}{2} \text{ 奇零} ). \]

例 B4 圆柱体内的截面积分

设圆柱区域 \(\Omega : x \in [ 0, h ],\ y^2 + z^2 \le R^2\)，求 \(I = \displaystyle\iiint_\Omega ( y^2 + z^2 )\, dV\).

按 \(x\) 做截面 : 每个 \(x\) 截面是半径 \(R\) 的圆盘. 用极坐标 \(y = r \cos \theta,\ z = r \sin \theta\)， \[ \iint_{y^2+z^2 \le R^2} ( y^2 + z^2 )\, dydz = \int_0^{2\pi} d\theta \int_0^R r^2 \cdot r\, dr = 2\pi \cdot \frac{R^4}{4} = \frac{\pi R^4}{2}. \] 再对 \(x\) 积分得 \[ I = \int_0^h \frac{\pi R^4}{2}\, dx = \frac{\pi R^4}{2}\, h. \]

专题 C 形心法巧用

例 C1 直圆锥的一阶矩

题设 \(\Omega : 0 \le z \le h,\ x^2 + y^2 \le \left( \dfrac{R}{h} \right)^2 z^2\)，求 \(I = \displaystyle\iiint_\Omega z\, dV\).

解法一 ( 推荐 形心法 )

  • 体积 \(V = \dfrac{1}{3}\pi R^2 h\).
  • 直圆锥形心在轴线上，距锥尖 \(z=0\) 的距离 \(\bar{z} = \dfrac{3h}{4}\).
  • 故 \(I = V \cdot \bar{z} = \dfrac{1}{3}\pi R^2 h \cdot \dfrac{3h}{4} = \dfrac{\pi R^2 h^2}{4}\).

解法二 ( 校验 截面法 )

  • 高度 \(z\) 处截面面积 \(A(z) = \pi \left( \dfrac{R}{h} \right)^2 z^2\).
  • \(I = \displaystyle\int_0^h z \, A(z)\, dz = \pi \dfrac{R^2}{h^2} \int_0^h z^3 dz = \dfrac{\pi R^2 h^2}{4}\).

附注 \(\bar{z} = \dfrac{3h}{4}\) 的来由

  • 由截面法求一阶矩与体积并相除即可 : \(\bar{z} = \dfrac{\int_0^h z A(z) dz}{\int_0^h A(z) dz} = \dfrac{\int_0^h z^3 dz}{\int_0^h z^2 dz} = \dfrac{h^4/4}{h^3/3} = \dfrac{3h}{4}\).

例 C2 半球的一阶矩 ( 形心反推 )

均匀半球 \(\Omega : x^2 + y^2 + z^2 \le R^2,\ z \ge 0\). 已知形心距底面为 \(\dfrac{3R}{8}\)，体积 \(V = \dfrac{2}{3} \pi R^3\). 则 \[ \iiint_\Omega z\, dV = V \cdot \bar{z} = \frac{2}{3} \pi R^3 \cdot \frac{3R}{8} = \frac{ \pi R^4 }{ 4 }. \] 不展开积分，直接用 “体积 × 形心坐标” 即得.

注意

  1. 忘记 Jacobi 因子 柱面坐标 \(dV=r\,dr\,d\theta\,dz\) 中的 \(r\) 和球面坐标 \(dV=\rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta\) 中的 \(\rho^2\sin\varphi\) 不能漏.

  2. 球面坐标的 \(\varphi\) 范围 \(\varphi\) 是与 \(z\) 轴正方向的夹角，取值 \([0,\pi]\) ( 不是 \([-\pi/2,\pi/2]\)，后者是纬度角 ).

  3. 积分限的上下关系 直角坐标化累次积分时，要保证内层积分的下限 \(\le\) 上限，必要时检验区域内是否满足 \(z_1(x,y)\le z_2(x,y)\).

  4. 奇偶性判断 奇偶性只针对特定变量，判断时其他变量视作常数. 如 \(f=xz\) 关于 \(x\) 为奇函数，也关于 \(z\) 为奇函数，但不是关于 \(y\) 的奇函数.

  5. 轮换对称的适用性 只有当区域关于对应变量完全对称时才能使用. 半球体只对 \(x,y\) 轮换对称，不对 \(z\) 轮换.

九. 简化计算技巧

总体速查

  • 对称判零 : 区域关于某坐标面对称且被积函数对该变量奇，三重积分为 0.
  • 成倍利用 : 区域关于某面对称且被积函数对该变量偶，可取一半区域后乘 2. 完全对称域可用轮换平均.
  • 优选坐标 : 含 \(x^2+y^2\) 选柱面，含 \(x^2+y^2+z^2\) 选球面; 截面规则优先 “先二后一”.
  • 常数散度 / 常值被积函数 : 积分可转为体积或体积分的一次常数倍.
  • 先化简后积分 : 先将曲面方程与被积函数做代换，尽量化为单变量或乘积型再积分.

换序与投影小技巧

  • 画影像 : 先把 \(\Omega\) 投影到选定坐标面，画出 \(D\) 的边界，再回代第三变量的上下界.
  • 六种次序 : \(( x \to y \to z )\) 等六种次序都可用，选择内层积分限最简单的一种.
  • “夹心” 结构 : 若 \(z\) 夹在两曲面之间，优先先对 \(z\) 积分; 若截面面积规则，优先 “先二后一”.
  • 代边界点验证 : 在边界关键点代入积分限检查上下界方向是否一致.

十. 坐标选择决策

选择决策

  • 见到 \(x^2 + y^2\) 或旋转对称轴为 \(z\) 轴 → 柱面坐标.
  • 见到 \(x^2 + y^2 + z^2\) 或球 / 球壳 / 锥体 → 球面坐标.
  • 边界为坐标平面 / 平面片段 → 直角坐标.
  • 截面规则且易求面积 → 先二后一 ( 截面法 ).
