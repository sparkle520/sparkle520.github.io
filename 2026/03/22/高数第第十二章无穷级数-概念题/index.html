一、 级数敛散性的核心推论 ( Common Corollaries )

在做概念题时，熟记以下推论可以秒杀很多选项.

1. 级数收敛的必要条件

  • 结论 若级数 \(\displaystyle \sum_{n=1}^{\infty} u_n\) 收敛，则必有 \(\displaystyle \lim_{n \to \infty} u_n = 0\).
  • 逆否命题 ( 判散神器 ) 若 \(\displaystyle \lim_{n \to \infty} u_n \neq 0\) ( 或极限不存在 )，则级数 \(\displaystyle \sum_{n=1}^{\infty} u_n\) 必发散.
  • 陷阱 \(\lim u_n = 0\) 推不出 级数收敛 ( 反例 调和级数 \(\sum \frac{1}{n}\) 发散 ).

2. 绝对收敛与条件收敛的关系

  • 强弱关系 绝对收敛 \(\Rightarrow\) 收敛. ( 即 \(\sum |u_n|\) 收 \(\Rightarrow \sum u_n\) 收 )
  • 判散推论 若 \(\sum u_n\) 发散，则 \(\sum |u_n|\) 必发散.
  • 性质
    • 绝对收敛 交换加数顺序，和不变 ( 可交换 ).
    • 条件收敛 交换加数顺序，和可能改变 ( 黎曼重排定理 ).

3. 平方级数 \(\sum u_n^2\) 与原级数 \(\sum u_n\) 的关系

  • 情况 A 正项级数 (\(u_n \ge 0\))
    • \(\sum u_n\) 收敛 \(\Rightarrow \sum u_n^2\) 必收敛. ( 理由 收敛 \(\implies u_n \to 0\)，当 \(n\) 够大时 \(u_n < 1\)，故 \(u_n^2 < u_n\) )
    • \(\sum u_n^2\) 收敛 \(\nRightarrow \sum u_n\) 收敛. ( 反例 \(\sum \frac{1}{n^2}\) 收，\(\sum \frac{1}{n}\) 散 )
  • 情况 B 一般项级数 ( \(u_n\) 可正可负 )
    • \(\sum u_n\) 收敛 \(\nRightarrow \sum u_n^2\) 收敛. ( 反例 交错调和级数 \(\sum \frac{(-1)^n}{\sqrt{n}}\) 收敛，但平方后 \(\sum \frac{1}{n}\) 发散 )
    • \(\sum u_n\) 绝对收敛 \(\Rightarrow \sum u_n^2\) 必收敛.

4. 奇偶项级数的关系

设级数 \(S = \displaystyle\sum_{n=1}^{\infty} u_n\).

  • 拆分 若 \(\sum u_{2k-1}\) ( 奇数项 ) 和 \(\sum u_{2k}\) ( 偶数项 ) 都收敛，则原级数 \(\sum u_n\) 必收敛.
  • 合并 若原级数 \(\sum u_n\) 收敛，则加括号后的级数 \((u_1+u_2) + (u_3+u_4) + \dots\) 必收敛，且和不变.
    • 逆命题不成立 加括号收敛 \(\nRightarrow\) 原级数收敛 ( 例 \((1-1)+(1-1)+\dots\) 收敛于0，但 \(1-1+1-1\dots\) 发散 ).

二、 常用反例速查表 ( Counter-Examples )

遇到 “若…则…” 的命题，尝试用以下反例进行验证.

级数名称 通项 \(u_n\) 敛散性 备注
调和级数 \(\frac{1}{n}\) 发散 \(\lim u_n = 0\)，但级数发散
P-级数 (\(p \le 1\)) \(\frac{1}{n^p}\) 发散 常用于比较判别法
P-级数 (\(p > 1\)) \(\frac{1}{n^p}\) 收敛 如 \(\frac{1}{n^2}\)
交错调和级数 \(\frac{(-1)^{n-1}}{n}\) 收敛 条件收敛
根式交错级数 \(\frac{(-1)^{n-1}}{\sqrt{n}}\) 收敛 条件收敛，平方后发散
等比级数 (\(q=-1\)) \((-1)^n\) 发散 部分和 \(S_n\) 在 \(0, -1\) 跳动
倒数阶乘 \(\frac{1}{n!}\) 收敛 收敛速度极快

三、 经典概念辨析例题

例题 1 ( 敛散性判定 - 进阶概念辨析 )

判断下列命题真假

  1. 若正项级数 \(\sum u_n\) 收敛，则 \(\displaystyle\lim_{n\to\infty} n u_n = 0\).
    • 错. ( 此命题极为经典，极易误判为真 )
    • 反例 构造序列，当 \(n = k^2\) 时 \(u_n = \frac{1}{k^2}\)，否则 \(u_n = \frac{1}{n^2}\). 此时 \(\sum u_n\) 显然收敛，但当 \(n = k^2\) 时，\(n u_n = k^2 \cdot \frac{1}{k^2} = 1 \not\to 0\).
    • 注 若增加条件 “\(u_n\) 单调递减”，则该命题为真 ( Pringsheim 定理 ).
  2. 若正项级数 \(\sum u_n\) 发散，则 \(\sum \frac{u_n}{1+u_n}\) 必发散.
    • 真.
    • 证明 若 \(u_n \not\to 0\)，则 \(\frac{u_n}{1+u_n} \not\to 0\)，级数发散; 若 \(u_n \to 0\)，则当 \(n\) 充分大时 \(\frac{u_n}{1+u_n} \sim u_n\)，由极限形式的比较判别法，两者同敛散，故 \(\sum \frac{u_n}{1+u_n}\) 发散.
  3. 若 \(\sum u_n\) 收敛且 \(u_n > 0\)，则 \(\sum \sin(u_n)\) 收敛.
    • 真. 因为 \(\sum u_n\) 收敛 \(\implies u_n \to 0\). 此时 \(\sin(u_n) \sim u_n\). 由比较判别法的极限形式，同敛散.

例题 2 ( 条件收敛的深度剖析 )

若级数 \(\sum u_n\) 条件收敛，判断以下级数的敛散性 (1) \(\sum (u_n + |u_n|)\) (2) \(\sum \frac{u_n + |u_n|}{2}\) ( 即正项部分 )

分析与结论 条件收敛意味着 \(\sum u_n\) 收敛，但 \(\sum |u_n|\) 发散.

  • 对于 (1) 假设 \(\sum (u_n + |u_n|)\) 收敛，由于 \(\sum u_n\) 收敛，由级数线性性质，其差 \(\sum |u_n|\) 必收敛，这与已知矛盾! 故 (1) 必发散.
  • 对于 (2) 实际上就是取出原级数中所有正项构成的新级数。由于 (1) 发散，且 (2) 是 (1) 的一半，故 (2) 必发散.
  • 推论 一个条件收敛的级数，其正项部分和负项部分各自构成的级数 均发散至无穷. 这也是黎曼重排定理能够成立的根本原因!

四、 含函数型通项的敛散性判断

这类题的核心不是“硬算”，而是先把函数展开成我们熟悉的等价无穷小或标准模型，再转化成 \(p\) 级数、几何级数或交错级数去判断.

1. 常见等价替换表 ( \(x \to 0\) )

  • \(\sin x \sim x\)
  • \(\tan x \sim x\)
  • \(\arctan x \sim x\)
  • \(1 - \cos x \sim \dfrac{x^2}{2}\)
  • \(e^x - 1 \sim x\)
  • \(\ln( 1 + x ) \sim x\)
  • \(( 1 + x )^\alpha - 1 \sim \alpha x\)
  • 若 \(f( x ) \sim g( x ) > 0\)，则 \(\sum f( a_n )\) 与 \(\sum g( a_n )\) 常可用极限比较判别法判断同敛散

考试口诀

  • 看见 \(\sin,\tan,\arctan,\ln(1+x),e^x-1\)，先想“能不能把里面那一坨当作小量”.
  • 看见 \(1-\cos(\cdot)\)，优先把它替成“平方量”.
  • 看见函数复合，不要急着展开外层，先判断里面是否趋于 0.

2. 判断步骤

  1. 先看通项是否趋于 0，这是所有级数收敛的必要条件.
  2. 若含函数 \(f( a_n )\)，先判断 \(a_n \to 0\) 是否成立.
  3. 若 \(a_n \to 0\)，用等价替换把 \(f( a_n )\) 化成熟悉模型.
  4. 最终归结到 \(\sum \dfrac{1}{n^p}\)、\(\sum q^n\)、\(\sum \dfrac{(-1)^n}{n^p}\) 等标准级数.
  5. 若有正负号交替，最后再判断是绝对收敛还是条件收敛.

例题 3 ( 含三角函数 )

判断 \(\displaystyle\sum_{n=1}^{\infty} \sin \frac{1}{n}\) 的敛散性.

因为 \(\dfrac{1}{n} \to 0\)，故 \[ \sin \frac{1}{n} \sim \frac{1}{n}. \]

于是 \[ \sum \sin \frac{1}{n} \quad \text{与} \quad \sum \frac{1}{n} \] 同敛散. 由于调和级数发散，所以 \[ \sum_{n=1}^{\infty} \sin \frac{1}{n} \ \text{发散}. \]

例题 4 ( 含对数函数 )

判断 \(\displaystyle\sum_{n=1}^{\infty} \ln \left( 1 + \frac{1}{n^2} \right)\) 的敛散性.

因为 \(\dfrac{1}{n^2} \to 0\)，故 \[ \ln \left( 1 + \frac{1}{n^2} \right) \sim \frac{1}{n^2}. \]

而 \(\sum \dfrac{1}{n^2}\) 收敛，所以 \[ \sum_{n=1}^{\infty} \ln \left( 1 + \frac{1}{n^2} \right) \ \text{收敛}. \]

例题 5 ( 含指数函数 )

判断 \(\displaystyle\sum_{n=1}^{\infty} \left( e^{1/n} - 1 \right)\) 的敛散性.

因为 \(\dfrac{1}{n} \to 0\)，故 \[ e^{1/n} - 1 \sim \frac{1}{n}. \]

故原级数与 \(\sum \dfrac{1}{n}\) 同敛散，因此原级数 发散.

例题 6 ( 含 \(1-\cos\) 的平方量模型 )

判断 \(\displaystyle\sum_{n=1}^{\infty} \left( 1 - \cos \frac{1}{n} \right)\) 的敛散性.

因为 \(\dfrac{1}{n} \to 0\)，且 \[ 1 - \cos x \sim \frac{x^2}{2}, \] 所以 \[ 1 - \cos \frac{1}{n} \sim \frac{1}{2n^2}. \]

故原级数与 \(\sum \dfrac{1}{n^2}\) 同敛散，所以 收敛.

例题 7 ( 含交错与函数复合 )

判断 \(\displaystyle\sum_{n=1}^{\infty} (-1)^{n-1} \ln \left( 1 + \frac{1}{n} \right)\) 的敛散性.

因为 \(\ln \left( 1 + \dfrac{1}{n} \right) \sim \dfrac{1}{n}\)，故该级数与交错调和级数同型.

  • 其绝对值级数 \(\displaystyle\sum \left| (-1)^{n-1} \ln \left( 1 + \frac{1}{n} \right) \right| = \sum \ln \left( 1 + \frac{1}{n} \right)\) 与 \(\sum \dfrac{1}{n}\) 同敛散，故 发散.
  • 原级数满足交错级数判别法条件 ( 正项递减且趋于 0 )，故 收敛.

因此该级数 条件收敛.

3. 抽象 \(a_n\) 题型的判断模板

设 \(a_n \to 0\). 在考研题里，若函数里面不给具体的 \(\dfrac{1}{n}\)，而只给一个抽象数列 \(a_n\)，通常按下列思路判断

  • 若 \(a_n > 0\) 且 \(\sum a_n\) 收敛，则 \[ \sum \sin a_n,\quad \sum \tan a_n,\quad \sum \ln( 1 + a_n ),\quad \sum ( e^{a_n} - 1 ) \] 都收敛，因为这些通项都与 \(a_n\) 等价.
  • 若 \(a_n > 0,\ a_n \to 0\) 且 \(\sum a_n\) 发散，则上面这些级数通常都发散，因为它们仍与 \(\sum a_n\) 同敛散.
  • 若只知道 \(\sum a_n\) 条件收敛，则不能随便把 \(f(a_n)\) 与 \(a_n\) 视为“完全等价”，必须继续看展开后的下一项会不会带来 \(\sum a_n^2\) 的影响.
  • 遇到 \(1 - \cos a_n\)，应立刻联想到 \[ 1 - \cos a_n \sim \frac{a_n^2}{2}, \] 所以此时本质上看的是 \(\sum a_n^2\) 而不是 \(\sum a_n\).

4. 用不等式判断收敛 ( 更适合抽象 \(a_n\) )

很多题目里并不给你足够条件去做等价展开，这时最稳的方法就是 夹估 / 比较. 常用不等式如下

  • 当 \(x \ge 0\) 时，\(0 \le \sin x \le x\)
  • 当 \(x > -1\) 时，\(\ln( 1 + x ) \le x\)
  • 当 \(x \ge 0\) 时，\(e^x - 1 \ge x\)
  • 当 \(x \ge 0\) 时，\(1 - \cos x \le \dfrac{x^2}{2}\)
  • 当 \(|x|\) 充分小时，\(|\sin x| \le |x|,\ |\arctan x| \le |x|,\ |\ln(1+x)| \le C|x|\)

使用原则

  • 想证收敛：优先找“上界”，把它压到已知收敛的级数上.
  • 想证发散：优先找“下界”，把它托到已知发散的级数上.
  • 若 \(a_n\) 可正可负，先看绝对值，再决定能否用比较判别法.
  • 若题目出现乘积型通项 \(\sum a_n b_n\)，优先考虑 柯西不等式 ( Cauchy-Schwarz ).

5. 柯西不等式在级数敛散性中的应用

柯西不等式在级数里最常见的形式是 \[ \left( \sum_{k=1}^{n} a_k b_k \right)^2 \le \left( \sum_{k=1}^{n} a_k^2 \right) \left( \sum_{k=1}^{n} b_k^2 \right). \]

它的考研用途主要有两类

  • 证明乘积级数收敛
    若 \(\sum a_n^2\)、\(\sum b_n^2\) 都收敛，则 \[ \sum |a_n b_n| \] 收敛，从而 \(\sum a_n b_n\) 绝对收敛.

  • 把陌生通项拆成两个平方可控的因子
    比如 \(\dfrac{a_n}{n}\)、\(\dfrac{a_n}{n^p}\)、\(a_n b_n\) 这类乘积型，只要能拆成 “一个平方和已知收敛” \(\times\) “另一个平方和也收敛”，就能直接判收敛.

常用结论

  1. 若 \(\displaystyle\sum a_n^2\) 收敛，则 \(\displaystyle\sum \frac{a_n}{n}\) 必收敛，因为 \(\sum \frac{1}{n^2}\) 收敛.
  2. 更一般地，若 \(\displaystyle\sum a_n^2\) 收敛，且 \(\displaystyle\sum b_n^2\) 收敛，则 \(\displaystyle\sum a_n b_n\) 必绝对收敛.
  3. 柯西不等式通常用于 判收敛，不适合直接判发散. 要判发散，仍优先考虑必要条件、比较下界或构造反例.

例题 8 ( 已知 \(\sum a_n\) 收敛，判断函数级数 )

设 \(a_n > 0\)，且 \(\displaystyle\sum a_n\) 收敛，判断下列级数的敛散性

  1. \(\displaystyle\sum \ln( 1 + a_n )\)
  2. \(\displaystyle\sum ( e^{a_n} - 1 )\)
  3. \(\displaystyle\sum \tan a_n\)

解析

由于 \(\sum a_n\) 收敛，必有 \(a_n \to 0\). 因而 \[ \ln( 1 + a_n ) \sim a_n,\qquad e^{a_n} - 1 \sim a_n,\qquad \tan a_n \sim a_n. \]

三者都与 \(\sum a_n\) 同敛散，因此这三个级数 都收敛.

例题 9 ( 已知 \(\sum a_n\) 条件收敛，不能机械等价 )

设 \(\displaystyle\sum a_n\) 条件收敛，判断 \(\displaystyle\sum \ln( 1 + a_n )\) 的敛散性.

结论 不能仅由 “\(\sum a_n\) 条件收敛” 直接判断，甚至它可以发散.

反例 取 \[ a_n = \frac{ ( -1 )^{n-1} }{ \sqrt{n} }. \] 则 \(\sum a_n\) 为交错级数，收敛，但不绝对收敛.

而 \[ \ln( 1 + a_n ) = a_n - \frac{a_n^2}{2} + o( a_n^2 ). \]

由于 \[ \sum a_n \ \text{收敛},\qquad \sum a_n^2 = \sum \frac{1}{n} \ \text{发散}, \] 故 \[ \sum \ln( 1 + a_n ) \] 受 \(-\dfrac{1}{2}\sum a_n^2\) 影响，最终 发散.

启示 \(\ln( 1 + a_n )\) 不仅看一次项 \(a_n\)，还要警惕二次项 \(- \dfrac{a_n^2}{2}\).

例题 10 ( 已知 \(\sum a_n^2\) 收敛，判断平方量模型 )

设 \(a_n \to 0\)，且 \(\displaystyle\sum a_n^2\) 收敛，判断 \[ \sum \big( 1 - \cos a_n \big) \] 的敛散性.

因为 \[ 1 - \cos a_n \sim \frac{a_n^2}{2}, \] 所以 \[ \sum \big( 1 - \cos a_n \big) \] 与 \(\sum a_n^2\) 同敛散，因此 收敛.

例题 11 ( 已知绝对收敛，函数级数也绝对收敛 )

设 \(\displaystyle\sum a_n\) 绝对收敛，判断 \(\displaystyle\sum \sin a_n\) 的敛散性.

由于绝对收敛 \(\Rightarrow a_n \to 0\)，当 \(n\) 足够大时有 \[ | \sin a_n | \le 2 | a_n |. \] 而 \(\sum |a_n|\) 收敛，故由比较判别法 \[ \sum | \sin a_n | \] 也收敛，因此 \[ \sum \sin a_n \] 绝对收敛.

同理，\(\sum \ln( 1 + a_n )\)、\(\sum ( e^{a_n} - 1 )\) 在 \(a_n\) 足够小时也可由比较判别法推出绝对收敛.

例题 12 ( 只用不等式，不用等价 )

设 \(a_n \ge 0\)，且 \(\displaystyle\sum a_n\) 收敛，判断 \(\displaystyle\sum \sin a_n\) 的敛散性.

因为 \(a_n \ge 0\)，且对任意 \(x \ge 0\) 有 \[ 0 \le \sin x \le x, \] 所以 \[ 0 \le \sin a_n \le a_n. \]

由比较判别法，\(\sum a_n\) 收敛 \(\Rightarrow \sum \sin a_n\) 收敛.

这个结论不需要写 \(\sin a_n \sim a_n\)，只靠不等式就够了，考试中更稳.

例题 13 ( 用下界判发散 )

设 \(a_n \ge 0\)，且 \(\displaystyle\sum a_n\) 发散，判断 \(\displaystyle\sum ( e^{a_n} - 1 )\) 的敛散性.

由不等式 \[ e^x - 1 \ge x \quad ( x \ge 0 ), \] 得 \[ e^{a_n} - 1 \ge a_n \ge 0. \]

由于 \(\sum a_n\) 发散，由比较判别法知 \[ \sum ( e^{a_n} - 1 ) \] 也 发散.

例题 14 ( 平方量模型的不等式判收敛 )

设 \(a_n \to 0\)，且 \(\displaystyle\sum a_n^2\) 收敛，判断 \(\displaystyle\sum ( 1 - \cos a_n )\) 的敛散性.

由不等式 \[ 0 \le 1 - \cos x \le \frac{x^2}{2}, \] 可得 \[ 0 \le 1 - \cos a_n \le \frac{a_n^2}{2}. \]

而 \(\sum a_n^2\) 收敛，故由比较判别法 \[ \sum ( 1 - \cos a_n ) \] 收敛.

这类题本质上还是看平方级数，但用不等式写法比等价展开更规范.

例题 15 ( 抽象条件收敛题的保守判断 )

设 \(\displaystyle\sum a_n\) 条件收敛，判断 \(\displaystyle\sum \sin a_n\) 是否一定收敛.

结论 不一定能仅凭条件收敛直接下结论.

原因是：若只知道 \(\sum a_n\) 条件收敛，则并不能保证 \(\sum |a_n|\) 收敛，也不能单靠 \[ |\sin a_n| \le |a_n| \] 推出绝对收敛.

进一步地，由展开式 \[ \sin a_n = a_n - \frac{a_n^3}{6} + o( a_n^3 ), \] 原级数的敛散性还会受到更高次项影响. 因此这类题若不给 \(a_n\) 的更多信息，一般应答 不能确定.

例题 16 ( 柯西不等式的直接应用 )

设 \(\displaystyle\sum a_n^2\) 收敛，判断 \(\displaystyle\sum \frac{a_n}{n}\) 的敛散性.

由柯西不等式，对部分和有 \[ \sum_{k=1}^{n} \left| \frac{a_k}{k} \right| \le \left( \sum_{k=1}^{n} a_k^2 \right)^{1/2} \left( \sum_{k=1}^{n} \frac{1}{k^2} \right)^{1/2}. \]

由于 \(\sum a_n^2\) 收敛，\(\sum \dfrac{1}{n^2}\) 也收敛，右端有界，所以 \[ \sum \left| \frac{a_n}{n} \right| \] 收敛，从而 \[ \sum \frac{a_n}{n} \] 绝对收敛.

例题 17 ( 抽象乘积型通项 )

设 \(\displaystyle\sum a_n^2\) 与 \(\displaystyle\sum b_n^2\) 都收敛，判断 \(\displaystyle\sum a_n b_n\) 的敛散性.

由柯西不等式 \[ \sum_{k=1}^{n} |a_k b_k| \le \left( \sum_{k=1}^{n} a_k^2 \right)^{1/2} \left( \sum_{k=1}^{n} b_k^2 \right)^{1/2}. \]

右边随 \(n\) 有统一上界，因此 \[ \sum |a_n b_n| \] 收敛，故 \[ \sum a_n b_n \] 绝对收敛.

这是抽象级数题里非常经典的一种结论，考试中看到 “平方收敛 + 乘积通项” 就应立刻想到柯西不等式.

6. 抽象级数组合题的判断技巧

这类题的关键词是 只给已知级数的敛散性，不给通项具体表达式. 此时不要硬凑比较，而要先判断 “哪些结论一定成立，哪些不能确定”.

常见组合规律

  1. 线性组合
    • 若 \(\sum a_n,\ \sum b_n\) 都收敛，则 \(\sum ( \alpha a_n + \beta b_n )\) 必收敛.
    • 若 \(\sum a_n\) 发散而 \(\sum b_n\) 收敛，则 \(\sum ( a_n + b_n )\) 必发散.
    • 若 \(\sum a_n,\ \sum b_n\) 都发散，则 \(\sum ( a_n + b_n )\) 不能确定. ( 可能发散，也可能通过相消而收敛，如 \(a_n = \frac{1}{n},\ b_n = -\frac{1}{n}\) )
  2. 绝对值与平方
    • \(\sum |a_n|\) 收敛 \(\Rightarrow \sum a_n\) 收敛.
    • \(\sum a_n\) 收敛 \(\nRightarrow \sum |a_n|\) 收敛.
    • \(\sum |a_n|\) 收敛 \(\Rightarrow \sum a_n^2\) 收敛.
    • \(\sum a_n^2\) 收敛 \(\nRightarrow \sum a_n\) 收敛.
  3. 奇项 / 偶项 / 分组
    • 若 \(\sum a_n\) 收敛，则奇项级数 \(\sum a_{2n-1}\) 与偶项级数 \(\sum a_{2n}\) 未必都能单独直接判断，但若它们都存在，则和为原级数.
    • 若奇项级数和偶项级数都收敛，则原级数必收敛.
    • 若原级数收敛，则有限项加括号后的分组级数仍收敛且和不变.
  4. 正项部分 / 负项部分
    • 若 \(\sum a_n\) 条件收敛，则正项部分与负项部分各自对应的级数都发散.
    • 看到 \(a_n + |a_n|,\ \dfrac{a_n + |a_n|}{2},\ \dfrac{|a_n| - a_n}{2}\)，应立刻想到“正项部分 / 负项部分拆分”.
  5. 乘积型组合
    • 若只知 \(\sum a_n\)、\(\sum b_n\) 收敛，并 不能 推出 \(\sum a_n b_n\) 一定收敛.
    • 若进一步知道 \(\sum a_n^2,\ \sum b_n^2\) 收敛，则可用柯西不等式推出 \(\sum a_n b_n\) 绝对收敛.

例题 18 ( 发散 + 收敛 的线性组合 )

设 \(\sum a_n\) 发散，\(\sum b_n\) 收敛，判断 \(\sum ( a_n - 2b_n )\) 的敛散性.

若假设 \(\sum ( a_n - 2b_n )\) 收敛，则由 \[ \sum a_n = \sum ( a_n - 2b_n ) + 2 \sum b_n \] 可知右边是“收敛 + 收敛”，从而 \(\sum a_n\) 应收敛，矛盾.

故 \[ \sum ( a_n - 2b_n ) \] 必发散.

例题 19 ( 发散 + 发散 不能确定 )

若 \(\sum a_n\)、\(\sum b_n\) 都发散，判断 \(\sum ( a_n + b_n )\) 的敛散性.

结论 不能确定.

反例 1 取 \(a_n = \dfrac{1}{n},\ b_n = \dfrac{1}{n}\)，则 \(\sum ( a_n + b_n ) = 2 \sum \dfrac{1}{n}\)，发散.

反例 2 取 \(a_n = \dfrac{1}{n},\ b_n = -\dfrac{1}{n}\)，则 \(\sum ( a_n + b_n ) = 0\)，收敛.

所以这类题不能只根据“两个都发散”就下结论.

例题 20 ( 条件收敛题中的经典组合 )

设 \(\sum a_n\) 条件收敛，判断 \[ \sum ( a_n + |a_n| ),\qquad \sum \frac{ a_n + |a_n| }{ 2 } \] 的敛散性.

因为 \[ \frac{ a_n + |a_n| }{ 2 } \] 正好是 \(a_n\) 的正项部分，而条件收敛级数的正项部分必发散.

所以这两个级数都 发散.

这类题最重要的是看出 \[ a_n + |a_n| = 2 \max \{ a_n,\ 0 \}. \]

例题 21 ( 乘积型组合不能乱推 )

若 \(\sum a_n\) 收敛，\(\sum b_n\) 收敛，判断 \(\sum a_n b_n\) 是否一定收敛.

结论 不一定.

取 \[ a_n = b_n = \frac{ ( -1 )^{n-1} }{ \sqrt{n} }, \] 则 \(\sum a_n,\ \sum b_n\) 都是交错级数，收敛. 但 \[ a_n b_n = \frac{1}{n}, \] 于是 \[ \sum a_n b_n = \sum \frac{1}{n} \] 发散.

所以“两个收敛级数的乘积级数一定收敛”是错误的，必须有更强条件 ( 如平方收敛 ) 才能用柯西不等式.

易错点总结

  • 不能只看到“函数项趋于 0”就断定级数收敛，必须继续和标准级数比较.
  • \(\ln(1+u_n)\)、\(e^{u_n}-1\)、\(\sin u_n\) 的敛散性，本质上往往取决于 \(\sum u_n\) 的敛散性.
  • \(1-\cos u_n\) 不是“一次小量”，而是“二次小量”，通常对应 \(\sum u_n^2\).
  • 含交错号时，要区分“收敛”与“绝对收敛”.
  • 若函数里面的量不趋于 0，应先用必要条件直接判散，而不是盲目做等价替换.
  • 柯西不等式通常用来证“绝对收敛”，但不能反过来只凭它去断发散.
