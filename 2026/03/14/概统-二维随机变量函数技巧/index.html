总结与速查 ( Cheat Sheet )

类型 函数 \(Z=g(X,Y)\) 核心公式 / 结论 备注
和分布 \(Z=X+Y\) \(f_Z(z) = \displaystyle\int_{-\infty}^{+\infty} f(x, z-x) dx\) 卷积公式 ( 独立时常用 )
最大值 \(M=\max(X,Y)\) \(F_M(z) = F_X(z) \cdot F_Y(z)\) 需 \(X,Y\) 独立
最小值 \(N=\min(X,Y)\) \(F_N(z) = 1 - [1-F_X(z)][1-F_Y(z)]\) 需 \(X,Y\) 独立
一般法 \(Z=g(X,Y)\) \(F_Z(z) = \displaystyle\iint_{g(x,y) \le z} f(x,y) dx dy\) 分布函数法 ( 万能 )

一、离散型随机变量函数的分布

1.1 解题步骤 ( 清晰明确 )

  1. 列出联合分布律 若 \(X,Y\) 独立，则 \(p_{ij} = p_i \cdot p_j\).
  2. 计算函数值 对每一对 \((x_i, y_j)\)，计算 \(z_{ij} = g(x_i, y_j)\).
  3. 合并同类项 找出所有相同的 \(z\) 值，将其对应的概率相加.
  4. 列表整理 按 \(z\) 从小到大排列，写出 \(Z\) 的分布律.

典型例题 1

设 \(X, Y\) 独立，且 \(X \sim B(1, 0.5), Y \sim B(1, 0.5)\). 求 \(Z = X+Y\) 的分布律.

解 \(X, Y\) 取值均为 \(0, 1\)，概率均为 \(0.5\). 联合概率 \(P(X=i, Y=j) = 0.5 \times 0.5 = 0.25\).

\((X,Y)\) \((0,0)\) \((0,1)\) \((1,0)\) \((1,1)\)
概率 \(0.25\) \(0.25\) \(0.25\) \(0.25\)
\(Z=X+Y\) \(0\) \(1\) \(1\) \(2\)

合并 \(Z=1\) 的项 \(P(Z=1) = 0.25 + 0.25 = 0.5\).

\(Z\) 的分布律为

\(Z\) \(0\) \(1\) \(2\)
\(P\) \(0.25\) \(0.5\) \(0.25\)

二、连续型随机变量函数的分布

2.1 分布函数法 ( Definition Method ) —— 万能方法

此方法适用于 所有连续型 问题，尤其是非独立或非标准形式的问题.

标准步骤

  1. 写定义 \(F_Z(z) = P(Z \le z) = P(g(X,Y) \le z)\).
  2. 化积分 \(F_Z(z) = \displaystyle\iint_{D_z} f(x,y) dx dy\)，其中 \(D_z = \{(x,y) \mid g(x,y) \le z\}\).
  3. 定限计算 根据 \(f(x,y)\) 的非零区域 (通常是矩形、三角形或圆) 与 \(g(x,y) \le z\) 的交集，确定积分限. ( 关键难点 )
  4. 求导 \(f_Z(z) = F'_Z(z)\)，注意确定 \(z\) 的取值范围.

注意

若 \(X,Y\) 独立，则 \(f(x,y) = f_X(x)f_Y(y)\).

典型例题 2 ( 考研真题难度 )

设 \((X,Y)\) 在区域 \(D: 0 < x < 1, 0 < y < 1\) 上服从均匀分布. 求 \(Z = X+Y\) 的概率密度 \(f_Z(z)\).

解

\(f(x,y) = \begin{cases} 1, & 0<x<1, 0<y<1 \\[.5em] 0, & \text{其他} \end{cases}\)

\(F_Z(z) = P(X+Y \le z) = \displaystyle\iint_{x+y \le z, 0<x<1, 0<y<1} 1 dx dy = \text{面积}(S_z)\).

分段讨论 ( 截面积法 )

  1. 当 \(z < 0\) 时，\(F_Z(z) = 0\).
  2. 当 \(0 \le z < 1\) 时，区域为左下角三角形，面积 \(\frac{1}{2}z^2\).
  3. 当 \(1 \le z < 2\) 时，区域为正方形切去右上角，面积 \(1 - \frac{1}{2}(2-z)^2\).
  4. 当 \(z \ge 2\) 时，\(F_Z(z) = 1\).

求导得

\(f_Z(z) = \begin{cases} z, & 0 \le z < 1 \\[.5em] 2-z, & 1 \le z < 2 \\[.5em] 0, & \text{其他} \end{cases}\)

2.2 卷积公式 ( Convolution Formula )

适用于 \(Z = X+Y\) 且积分容易计算的情况.

公式

\[ f_Z(z) = \int_{-\infty}^{+\infty} f(x, z-x) dx = \int_{-\infty}^{+\infty} f(z-y, y) dy \]

若 \(X,Y\) 独立 \[ f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) dx \]

解题技巧

  • 非零区间分析 积分区间由 \(f_X(x) > 0\) 和 \(f_Y(z-x) > 0\) 共同决定. 即 \(x \in D_X\) 且 \(z-x \in D_Y\).
  • 此法常用于正态分布的和 ( \(X \sim N(\mu_1, \sigma_1^2), Y \sim N(\mu_2, \sigma_2^2) \Rightarrow X+Y \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)\) ).

2.3 最值分布 ( Max & Min )

设 \(X, Y\) 独立，分布函数分别为 \(F_X(x), F_Y(y)\).

  1. 最大值 \(M = \max(X,Y)\) \[ F_M(z) = P(\max(X,Y) \le z) = P(X \le z, Y \le z) = P(X \le z)P(Y \le z) = F_X(z)F_Y(z) \] \[ f_M(z) = F'_M(z) = f_X(z)F_Y(z) + F_X(z)f_Y(z) \]

  2. 最小值 \(N = \min(X,Y)\) \[ F_N(z) = P(\min(X,Y) \le z) = 1 - P(\min(X,Y) > z) = 1 - P(X > z, Y > z) \] \[ = 1 - [1-F_X(z)][1-F_Y(z)] \]

推广到 n 个变量

  • \(F_{\max}(z) = [F(z)]^n\) ( 独立同分布 )
  • \(F_{\min}(z) = 1 - [1-F(z)]^n\) ( 独立同分布 )

三、常见非线性变换 ( 考研重点 )

3.1 乘积分布 \(Z=XY\) 与 商分布 \(Z=Y/X\)

建议直接使用 分布函数法 推导，死记公式容易出错.

商分布 \(Z=Y/X\) 推导思路

\(F_Z(z) = P(Y/X \le z)\). 若 \(X>0\)，则 \(Y \le zX\); 若 \(X<0\)，则 \(Y \ge zX\). 通常在 \(f(x,y)\) 定义域内画图，用射线 \(y=zx\) 扫描区域.

典型例题 3 ( 商分布 )

设 \(X,Y\) 独立且均服从 \(E(1)\) ( 指数分布 ). 求 \(Z = Y/X\) 的密度.

解

\(f(x,y) = e^{-x}e^{-y} = e^{-(x+y)} \quad (x>0, y>0)\). \(F_Z(z) = P(Y/X \le z)\). 当 \(z \le 0\) 时，\(F_Z(z) = 0\) (因为 \(X,Y>0 \Rightarrow Z>0\)). 当 \(z > 0\) 时 \[ F_Z(z) = \iint_{0 < y \le zx} e^{-(x+y)} dx dy = \int_0^{+\infty} e^{-x} dx \int_0^{zx} e^{-y} dy \] \[ = \int_0^{+\infty} e^{-x} (1 - e^{-zx}) dx = \int_0^{+\infty} (e^{-x} - e^{-(1+z)x}) dx \] \[ = \left[ -e^{-x} + \frac{e^{-(1+z)x}}{1+z} \right]_0^{+\infty} = 1 - \frac{1}{1+z} = \frac{z}{1+z} \]

求导得 \(f_Z(z) = \frac{1}{(1+z)^2} \quad (z>0)\).
