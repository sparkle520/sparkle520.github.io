一、基本概念与性质

1.1 二重积分的定义

定义

设 \(f(x,y)\) 在有界闭区域 \(D\) 上有界. 将 \(D\) 任意分割为 \(n\) 个小区域 \(\Delta\sigma_i\)，在每个 \(\Delta\sigma_i\) 上任取一点 \((\xi_i, \eta_i)\)，作乘积 \(f(\xi_i, \eta_i)\Delta\sigma_i\) 并求和. 若当各小区域直径的最大值 \(\lambda \to 0\) 时，该和式的极限存在且与分割方式及取点方式无关，则称此极限为 \(f(x,y)\) 在 \(D\) 上的二重积分，记作

\[\iint_D f(x,y)\,d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i)\Delta\sigma_i\]

其中 \(d\sigma\) 为面积微元，在直角坐标下 \(d\sigma = dx\,dy\).

几何意义

当 \(f(x,y) \ge 0\) 时，二重积分 \(\displaystyle\iint_D f(x,y)\,d\sigma\) 表示以 \(D\) 为底、以曲面 \(z = f(x,y)\) 为顶的曲顶柱体的体积.

当 \(f(x,y)\) 变号时，\(\displaystyle\iint_D f(x,y)\,d\sigma\) 为代数体积 ( 曲面在 \(xOy\) 平面上方的体积减去下方的体积 ) .

1.2 基本性质

性质汇总

  1. 线性性 : \(\displaystyle\iint_D [\alpha f + \beta g]\,d\sigma = \alpha \displaystyle\iint_D f\,d\sigma + \beta \displaystyle\iint_D g\,d\sigma\).

  2. 区域可加性 : 若 \(D = D_1 \cup D_2\) ( \(D_1, D_2\) 无公共内点 ) ，则 \[\iint_D f\,d\sigma = \iint_{D_1} f\,d\sigma + \iint_{D_2} f\,d\sigma.\]

  3. 保号性 : 若在 \(D\) 上 \(f(x,y) \le g(x,y)\)，则 \(\displaystyle\iint_D f\,d\sigma \le \displaystyle\iint_D g\,d\sigma\).

  4. 绝对值不等式 : \(\left|\displaystyle\iint_D f\,d\sigma\right| \le \displaystyle\iint_D |f|\,d\sigma\).

  5. 估值定理 : 设 \(m \le f(x,y) \le M\)，\(S\) 为 \(D\) 的面积，则 \[mS \le \iint_D f(x,y)\,d\sigma \le MS.\]

  6. 二重积分中值定理 : 若 \(f(x,y)\) 在有界闭区域 \(D\) 上连续，则存在 \((\xi, \eta) \in D\)，使得 \[\iint_D f(x,y)\,d\sigma = f(\xi, \eta) \cdot S.\] 其中 \(S\) 为 \(D\) 的面积.

易错点：估值定理的应用

示例 : 估计 \(I = \displaystyle\iint_D e^{x^2+y^2}\,d\sigma\)，其中 \(D : x^2 + y^2 \le 1\).

在 \(D\) 上，\(0 \le x^2+y^2 \le 1\)，故 \(1 \le e^{x^2+y^2} \le e\). 面积 \(S = \pi\)，因此 \(\pi \le I \le \pi e\).

二、直角坐标系下的计算

2.1 化为累次积分

核心思想 ( 截面法 )

对 X 型区域 \(D = \{(x,y) \mid a \le x \le b,\ \varphi_1(x) \le y \le \varphi_2(x)\}\)

\[\iint_D f(x,y)\,d\sigma = \int_a^b \left[ \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy \right] dx\]

对 Y 型区域 \(D = \{(x,y) \mid c \le y \le d,\ \psi_1(y) \le x \le \psi_2(y)\}\)

\[\iint_D f(x,y)\,d\sigma = \int_c^d \left[ \int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx \right] dy\]

示例 1 ( 直角坐标基本计算 )

计算 \(\displaystyle\iint_D xy\,d\sigma\)，其中 \(D\) 由 \(y=x\)，\(y=x^2\) 围成.

解 : 交点 : \(x=x^2 \Rightarrow x=0\) 或 \(x=1\). 在 \([0,1]\) 上，\(x^2 \le y \le x\) ( X 型 ) .

\[ \begin{aligned} I &= \int_0^1 dx \int_{x^2}^{x} xy\,dy = \int_0^1 x \cdot \left[\frac{y^2}{2}\right]_{x^2}^{x} dx \\[.5em] &= \int_0^1 x \cdot \frac{x^2 - x^4}{2}\,dx = \frac{1}{2}\int_0^1 (x^3 - x^5)\,dx \\[.5em] &= \frac{1}{2}\left[\frac{x^4}{4} - \frac{x^6}{6}\right]_0^1 = \frac{1}{2}\left(\frac{1}{4} - \frac{1}{6}\right) = \frac{1}{24}. \end{aligned} \]

2.2 交换积分次序 ( 重点难点 )

交换积分次序的方法与步骤

  1. 画图法 ( 最常用 )
    • 还原 : 根据给定的累次积分限，写出不等式组，在平面上画出积分区域 \(D\).
    • 投影 : 观察 \(D\) 向另一坐标轴的投影，确定新的外层积分限.
    • 穿线 : 作垂直于该轴的射线 ( 穿线法 ) ，确定内层积分的上下限.
  2. 不等式法 ( 辅助 )
    • 纯代数推导，适合图形简单的区域. 将 \(a \le x \le b, \varphi_1(x) \le y \le \varphi_2(x)\) 转化为 \(y\) 的范围 \(c \le y \le d\) 和 \(x\) 的范围.

适用场景

  • 内层积分原函数无法用初等函数表示 ( 如 \(\int e^{x^2}dx\), \(\int \frac{\sin x}{x}dx\) ) .
  • 积分区域分块后，换序可能使两个积分合并为一个，简化计算.

示例 2 ( 交换次序——含不可积原函数 )

计算 \(\displaystyle\int_0^1 dy \int_y^1 \frac{\sin x}{x}\,dx\).

解 积分区域 \(D = \{(x,y) \mid 0 \le y \le 1,\ y \le x \le 1\}\)，即 \(\{0 \le x \le 1,\ 0 \le y \le x\}\).

交换次序后 \[ \begin{aligned} I &= \int_0^1 dx \int_0^x \frac{\sin x}{x}\,dy = \int_0^1 \frac{\sin x}{x} \cdot x\,dx = \int_0^1 \sin x\,dx = 1 - \cos 1. \end{aligned} \]

示例 3 ( 交换次序——区域需合并 )

交换积分次序 \(\displaystyle\int_0^1 dx \int_0^x f(x,y)\,dy + \int_1^2 dx\int_0^{2-x} f(x,y)\,dy\).

解 \(D_1 : 0 \le x \le 1,\ 0 \le y \le x\); \(D_2 : 1 \le x \le 2,\ 0 \le y \le 2-x\).

合并后 \(D\) 为三角形，顶点 \((0,0),(1,1),(2,0)\). 化为 Y 型 ( \(0 \le y \le 1\) ) ：左边界 \(x = y\)，右边界 \(x = 2-y\).

\[\quad \int_0^1 dy \int_y^{2-y} f(x,y)\,dx.\]

三、极坐标系下的计算

3.1 极坐标变换

变换公式

令 \(x = r\cos\theta\)，\(y = r\sin\theta\)，Jacobi 行列式 \(J = r\)，面积微元

\[d\sigma = dx\,dy = r\,dr\,d\theta\]

二重积分化为 \[\iint_D f(x,y)\,d\sigma = \int_\alpha^\beta d\theta \int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta, r\sin\theta)\cdot r\,dr\]

适用场景 区域为圆形、扇形、环形; 被积函数含 \(x^2+y^2\)、\(\dfrac{y}{x}\) 等.

极点位置与积分限的判断

极点位置 \(r\) 的下限 典型示例
极点在 \(D\) 内部 \(0\) \(x^2+y^2 \le R^2\)
极点在 \(D\) 边界上 \(0\) \(r \le 2\cos\theta\) ( 圆过原点 )
极点在 \(D\) 外部 \(r_1(\theta) > 0\) 环形区域 \(1 \le r \le 2\)

示例 4 ( 极坐标——圆过原点 )

计算 \(\displaystyle\iint_D \sqrt{x^2+y^2}\,d\sigma\)，其中 \(D : x^2+y^2 \le 2x\).

解 \((x-1)^2+y^2 \le 1\)，极坐标下 \(r \le 2\cos\theta\)，\(\theta \in \left[-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right]\).

\[ \begin{aligned} I &= \int_{-\pi/2}^{\pi/2} d\theta \int_0^{2\cos\theta} r \cdot r\,dr = \int_{-\pi/2}^{\pi/2} \frac{8\cos^3\theta}{3}\,d\theta \\[.5em] &= \frac{16}{3}\int_0^{\pi/2} \cos^3\theta\,d\theta = \frac{16}{3} \cdot \frac{2}{3} = \frac{32}{9}. \end{aligned} \]

示例 5 ( 极坐标——环形域 + 特殊函数 )

计算 \(\displaystyle\iint_D \arctan\frac{y}{x}\,d\sigma\)，其中 \(D : 1 \le x^2+y^2 \le 4,\ x \ge 0,\ y \ge 0\).

解 第一象限环形区域，\(1 \le r \le 2\)，\(0 \le \theta \le \dfrac{\pi}{2}\). 注意 \(\arctan\dfrac{y}{x} = \theta\).

\[ I = \int_0^{\pi/2} \theta\,d\theta \int_1^2 r\,dr = \frac{\pi^2}{8} \cdot \frac{3}{2} = \frac{3\pi^2}{16}. \]

3.2 广义极坐标 ( 椭圆区域 )

广义极坐标变换

对椭圆区域 \(\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2} \le 1\)，令 \(x = ar\cos\theta\)，\(y = br\sin\theta\)，

则 \(|J| = abr\)，\(d\sigma = abr\,dr\,d\theta\)，区域变为 \(0 \le r \le 1\)，\(0 \le \theta \le 2\pi\).

示例 6 ( 广义极坐标 )

计算 \(\displaystyle\iint_D e^{\frac{x^2}{a^2}+\frac{y^2}{b^2}}\,d\sigma\)，其中 \(D : \dfrac{x^2}{a^2}+\dfrac{y^2}{b^2} \le 1\).

解 令 \(x=ar\cos\theta\)，\(y=br\sin\theta\)，被积函数 \(= e^{r^2}\)，\(d\sigma = abr\,dr\,d\theta\).

\[ I = 2\pi ab \int_0^1 r e^{r^2}\,dr = 2\pi ab \cdot \frac{e-1}{2} = \pi ab(e-1). \]

四、对称性技巧

4.1 奇偶对称性

奇偶对称性定理

设积分区域 \(D\) 关于 \(y\) 轴对称，则：

\[\iint_D f(x,y)\,d\sigma = \begin{cases} 2\displaystyle\iint_{D_+} f\,d\sigma, & f(-x,y) = f(x,y) \text{ ( 关于 } x \text{ 为偶函数 ) } \\[8pt] 0, & f(-x,y) = -f(x,y) \text{ ( 关于 } x \text{ 为奇函数 ) } \end{cases}\]

其中 \(D_+\) 为 \(D\) 在 \(x \ge 0\) 的部分. 类似地，若 \(D\) 关于 \(x\) 轴对称，则分析 \(f(x,-y)\) 的奇偶性.

示例 7 ( 奇偶对称性——圆域上的混合型被积函数 )

计算 \(\displaystyle I = \displaystyle\iint_D (x^2 + xy + y^2)\,d\sigma\)，其中 \(D : x^2+y^2 \le R^2\).

解

  • \(xy\) 关于 \(y\) 轴为奇函数 \(\Rightarrow \displaystyle\iint_D xy\,d\sigma = 0\).
  • 由轮换对称性 \(\displaystyle\iint_D x^2\,d\sigma = \displaystyle\iint_D y^2\,d\sigma\).

\[\iint_D (x^2+y^2)\,d\sigma = \int_0^{2\pi}d\theta\int_0^R r^3\,dr = 2\pi \cdot \frac{R^4}{4} = \frac{\pi R^4}{2}\]

\[I = \frac{1}{2}\cdot\frac{\pi R^4}{2} + 0 + \frac{1}{2}\cdot\frac{\pi R^4}{2} = \frac{\pi R^4}{2}.\]

4.2 轮换对称性 ( 考研重要技巧 )

轮换对称性定理

若积分区域 \(D\) 关于直线 \(y=x\) 对称，则

\[\iint_D f(x,y)\,d\sigma = \iint_D f(y,x)\,d\sigma\]

由此 \(\displaystyle\iint_D f(x,y)\,d\sigma = \frac{1}{2}\displaystyle\iint_D [f(x,y)+f(y,x)]\,d\sigma\).

常见区域 圆形、正方形 ( \(|x| \le a,|y| \le a\) ) 、顶点关于 \(y=x\) 对称的三角形.

示例 8 ( 轮换对称性——化简分式被积函数 )

计算 \(\displaystyle I = \displaystyle\iint_D \frac{x^2}{x^2+y^2}\,d\sigma\)，其中 \(D : x^2+y^2 \le 1\).

解 \(D\) 关于 \(y=x\) 对称，由轮换对称性 \(\displaystyle\iint_D \dfrac{x^2}{x^2+y^2}\,d\sigma = \displaystyle\iint_D \dfrac{y^2}{x^2+y^2}\,d\sigma\).

\[2I = \iint_D \frac{x^2+y^2}{x^2+y^2}\,d\sigma = \iint_D 1\,d\sigma = \pi \implies I = \frac{\pi}{2}.\]

示例 9 ( 奇偶 + 轮换联合——快速判零 )

计算 \(\displaystyle I = \displaystyle\iint_D (x + y + x^2 y + xy^2)\,d\sigma\)，其中 \(D : x^2+y^2 \le 1\).

解 \(D\) 关于两坐标轴及 \(y=x\) 均对称.

  • \(\displaystyle\iint_D x\,d\sigma = 0\) ( \(D\) 关于 \(y\) 轴对称，\(x\) 奇 )
  • \(\displaystyle\iint_D y\,d\sigma = 0\) ( \(D\) 关于 \(x\) 轴对称，\(y\) 奇 )
  • \(\displaystyle\iint_D x^2 y\,d\sigma = 0\) ( 关于 \(x\) 轴对称，\(x^2 y\) 关于 \(y\) 奇 )
  • \(\displaystyle\iint_D xy^2\,d\sigma = 0\) ( 关于 \(y\) 轴对称，\(xy^2\) 关于 \(x\) 奇 )

\[ \quad I = 0.\]

五、一般换元法

二重积分换元公式

设 \(x = x(u,v)\)，\(y = y(u,v)\) 将 \(D'\) 一一映射到 \(D\)，Jacobi 行列式

\[J(u,v) = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} x_u & x_v \\[.5em] y_u & y_v \end{vmatrix} \ne 0\]

则

\[\iint_D f(x,y)\,dx\,dy = \iint_{D'} f(x(u,v),y(u,v))\,|J(u,v)|\,du\,dv\]

换元动机 将斜边、椭圆、平行四边形等复杂区域变换为矩形或圆形.

示例 10 ( 线性换元——平行四边形区域 )

计算 \(\displaystyle\iint_D (x+y)e^{x-y}\,d\sigma\)，其中 \(D\) 由 \(x+y=0,\ x+y=1,\ x-y=0,\ x-y=1\) 围成.

解 令 \(u = x+y\)，\(v = x-y\)，\(D' : 0 \le u \le 1,\ 0 \le v \le 1\).

\(x = \dfrac{u+v}{2}\)，\(y = \dfrac{u-v}{2}\)，\(|J| = \dfrac{1}{2}\).

\[ I = \int_0^1 du \int_0^1 u e^v \cdot \frac{1}{2}\,dv = \frac{1}{2} \cdot \frac{1}{2} \cdot (e-1) = \frac{e-1}{4}. \]

示例 11 ( 换元法——双曲型边界区域 )

计算 \(\displaystyle I = \displaystyle\iint_D xy\,d\sigma\)，其中 \(D\) 由 \(xy=1,\ xy=2,\ y=x,\ y=4x\) 围成 ( \(x>0,y>0\) ) .

解 令 \(u = xy\)，\(v = \dfrac{y}{x}\)，则 \(D' : 1 \le u \le 2,\ 1 \le v \le 4\).

由 \(x = \sqrt{\dfrac{u}{v}}\)，\(y = \sqrt{uv}\)：

\[J = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \frac{1}{2\sqrt{uv}} & -\frac{\sqrt{u}}{2v^{3/2}} \\[4pt] \frac{\sqrt{v}}{2\sqrt{u}} & \frac{\sqrt{u}}{2\sqrt{v}} \end{vmatrix} = \frac{1}{4v} + \frac{1}{4v} = \frac{1}{2v}\]

被积函数 \(xy = u\)

\[ I = \int_1^2 du\int_1^4 u \cdot \frac{1}{2v}\,dv = \frac{1}{2}\cdot\frac{3}{2}\cdot\ln 4 = \frac{3\ln 2}{2}. \]

示例 12 ( 利用二重积分计算含参积分——Frullani 积分 )

计算 \(\displaystyle I = \int_0^1 \frac{x^a - x^b}{\ln x}\,dx\)，其中 \(a > b > -1\).

解 注意 \(\dfrac{x^a - x^b}{\ln x} = \displaystyle\int_b^a x^t\,dt\).

\[ I = \int_0^1 \int_b^a x^t\,dt\,dx = \int_b^a \int_0^1 x^t\,dx\,dt = \int_b^a \frac{1}{t+1}\,dt = \ln\frac{a+1}{b+1}. \]

六、几何与物理应用

6.1 曲面面积

曲面面积公式

设曲面 \(z = f(x,y)\) 在区域 \(D_{xy}\) 上有连续偏导数，则曲面面积

\[S = \iint_{D_{xy}} \sqrt{1 + f_x^2 + f_y^2}\,d\sigma\]

示例 13 ( 球面被柱面截取的面积 )

求球面 \(x^2+y^2+z^2=R^2\) 被柱面 \(x^2+y^2=Rx\) ( \(R>0\) ) 截取的面积.

解 取上半球面 \(z = \sqrt{R^2-x^2-y^2}\)，总面积 \(= 2\times\) 上半部分.

\[\sqrt{1+z_x^2+z_y^2} = \frac{R}{\sqrt{R^2-x^2-y^2}}\]

投影区域极坐标 \(r \le R\cos\theta\)，\(\theta \in \left[-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right]\).

\[ \begin{aligned} S &= 2\int_{-\pi/2}^{\pi/2} d\theta \int_0^{R\cos\theta} \frac{Rr}{\sqrt{R^2-r^2}}\,dr \\[.5em] &= 2R\int_{-\pi/2}^{\pi/2} \left(R - R|\sin\theta|\right)d\theta = 2R^2(\pi - 2). \end{aligned} \]

6.2 质心与转动惯量

公式汇总

设平面薄板占区域 \(D\)，密度为 \(\rho(x,y)\)，则：

  • 质量 : \(M = \displaystyle\iint_D \rho\,d\sigma\)
  • 质心 : \(\bar{x} = \dfrac{1}{M}\displaystyle\iint_D x\rho\,d\sigma\)，\(\bar{y} = \dfrac{1}{M}\displaystyle\iint_D y\rho\,d\sigma\)
  • 对 \(x\) 轴转动惯量 : \(I_x = \displaystyle\iint_D y^2\rho\,d\sigma\)
  • 对 \(y\) 轴转动惯量 : \(I_y = \displaystyle\iint_D x^2\rho\,d\sigma\)
  • 对原点 : \(I_O = I_x + I_y = \displaystyle\iint_D (x^2+y^2)\rho\,d\sigma\)

七、含参变量二重积分

示例 14 ( 变限二重积分求导 )

设 \(f(x)\) 连续，\(F(t) = \displaystyle\iint_{x^2+y^2 \le t^2} f\!\left(\sqrt{x^2+y^2}\right)d\sigma\) ( \(t>0\) ) ，求 \(F'(t)\).

解 极坐标下 \[F(t) = 2\pi\int_0^t r f(r)\,dr \implies F'(t) = 2\pi t f(t).\]

八、抽象函数的二重积分处理

抽象函数的二重积分是指被积函数 \(f(x,y)\) 形式未具体给出，仅已知其连续性或对称性. 解决此类问题需紧扣 积分次序交换 与 坐标变换.

8.1 交换积分次序

对于含抽象函数 \(f(x,y)\) 的累次积分，关键在于准确画出积分区域 \(D\).

  1. 分段函数的合并
    • 若积分形式为 \(\int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy + \int_b^c dx \int_{\psi_1(x)}^{\psi_2(x)} f(x,y)\,dy\)，需画出两个区域 \(D_1, D_2\).
    • 观察 \(D_1 \cup D_2\) 是否构成一个简单的 Y 型区域，若能，则合并为一个积分.
  2. 复杂区域的分割
    • 若区域 \(D\) 既非标准 X 型也非 Y 型，需作辅助线将其分割.

8.2 利用极坐标转化

若积分区域 \(D\) 为圆形、扇形或圆环，且被积函数含 \(f(x^2+y^2)\) 或 \(f(\sqrt{x^2+y^2})\)，应优先转化为极坐标.

\[ \iint_D f(x^2+y^2)\,d\sigma = \int_\alpha^\beta d\theta \int_{r_1}^{r_2} f(r^2) r\,dr \]

注意 : 转化后内层积分往往变为 \(\int f(r^2) r\,dr = \frac{1}{2} \int f(u)\,du\)，从而使抽象函数可积.

8.3 利用对称性简化

若 \(D\) 关于 \(y=x\) 对称，则有轮换对称性： \[ \iint_D f(x)g(y)\,d\sigma = \iint_D f(y)g(x)\,d\sigma \] 进而 \[ \iint_D f(x)g(y)\,d\sigma = \frac{1}{2}\iint_D [f(x)g(y) + f(y)g(x)]\,d\sigma \] 此性质常用于证明积分等式或计算.

九、二重积分通用解题技巧 ( Tips )

9.1 “看” —— 观察区域与函数

  1. 看对称性
    • 区域关于 \(x\) 轴对称 \(\to\) 查被积函数关于 \(y\) 的奇偶性 ( 奇函数积分为 0 ) .
    • 区域关于 \(y\) 轴对称 \(\to\) 查被积函数关于 \(x\) 的奇偶性.
    • 区域关于 \(y=x\) 对称 \(\to\) 查轮换对称性 ( \(x,y\) 互换积分值不变 ) .
  2. 看形状
    • 圆域、扇形、环形 \(\to\) 极坐标.
    • 椭圆域 \(\to\) 广义极坐标.
    • 平行四边形域 \(\to\) 线性换元 ( 令 \(u,v\) 为两组平行边的方程 ) .
  3. 看函数
    • 含 \(x^2+y^2\), \(y/x\) \(\to\) 极坐标.
    • 含 \(f(ax+by)\) \(\to\) 换元.
    • 含 \(e^{x^2}, \frac{\sin x}{x}\) \(\to\) 交换积分次序.

9.2 “切” —— 分割与填补

  • 分块积分 : 若区域 \(D\) 复杂 ( 如 “L” 型、挖洞型 ) ，用辅助线将其切割为简单的 X 型或 Y 型区域.
  • 大域减小域 : 环形区域积分 = 大圆积分 - 小圆积分 ( 适用于被积函数在原点有定义的情况 ) .

9.3 “换” —— 变量代换

  • 旋转变换 : 若区域倾斜，可作旋转变换 \(x = u\cos\theta - v\sin\theta, y = u\sin\theta + v\cos\theta\).
  • 平移变换 : 若圆心不在原点 ( 如 \((x-1)^2+(y-2)^2 \le 1\) ) ，令 \(u=x-1, v=y-2\).

十、综合难题选讲 ( Selected Hard Problems )

难题 1 ( 极坐标 + 奇偶对称 )

计算 \(\displaystyle I = \displaystyle\iint_D \left(\frac{x^3 \sin y}{x^2+y^2} + \sqrt{x^2+y^2}\right)d\sigma\)，其中 \(D : x^2+y^2 \le 4\).

解 \(D\) 关于 \(y\) 轴对称，\(\dfrac{x^3 \sin y}{x^2+y^2}\) 关于 \(x\) 为奇函数，故其积分为 \(0\).

极坐标下 \[\iint_D \sqrt{x^2+y^2}\,d\sigma = \int_0^{2\pi}d\theta\int_0^2 r^2\,dr = 2\pi \cdot \frac{8}{3} = \frac{16\pi}{3}.\]

\[ \quad I = \frac{16\pi}{3}.\]

难题 2 ( 含绝对值——分区域 )

计算 \(\displaystyle I = \displaystyle\iint_D |x^2+y^2-1|\,d\sigma\)，其中 \(D : x^2+y^2 \le 4\).

解 分为 \(D_1 : r \le 1\) ( 内圆 ) 和 \(D_2 : 1 \le r \le 2\) ( 环形 ) .

\[I_1 = 2\pi\int_0^1 (1-r^2)r\,dr = 2\pi\left[\frac{r^2}{2}-\frac{r^4}{4}\right]_0^1 = \frac{\pi}{2}\]

\[I_2 = 2\pi\int_1^2 (r^2-1)r\,dr = 2\pi\left[\frac{r^4}{4}-\frac{r^2}{2}\right]_1^2 = 2\pi\cdot\frac{9}{4} = \frac{9\pi}{2}\]

\[ \quad I = 5\pi.\]

难题 3 ( 二重积分与 Gauss 积分 )

计算 \(I = \displaystyle\int_0^{+\infty} e^{-x^2}\,dx\).

解 \[I^2 = \iint_{x\ge 0,y\ge 0} e^{-(x^2+y^2)}\,d\sigma = \int_0^{\pi/2}d\theta\int_0^{+\infty} e^{-r^2} r\,dr = \frac{\pi}{2}\cdot\frac{1}{2} = \frac{\pi}{4}\]

\[ \quad I = \frac{\sqrt{\pi}}{2}.\]

难题 4 ( 综合：广义极坐标 + 几何意义 )

计算 \(\displaystyle I = \displaystyle\iint_D \sqrt{1-\frac{x^2}{4}-\frac{y^2}{9}}\,d\sigma\)，其中 \(D : \dfrac{x^2}{4}+\dfrac{y^2}{9} \le 1\).

解 令 \(x = 2r\cos\theta\)，\(y = 3r\sin\theta\)，\(|J| = 6r\)，被积函数 \(= \sqrt{1-r^2}\)，\(D' : 0 \le r \le 1\)，\(0 \le \theta \le 2\pi\).

\[I = 12\pi\int_0^1 r\sqrt{1-r^2}\,dr = 12\pi \cdot \left[-\frac{1}{3}(1-r^2)^{3/2}\right]_0^1 = 4\pi.\]

几何意义 : 此积分表示以椭圆为底、以半椭球面 \(\dfrac{x^2}{4}+\dfrac{y^2}{9}+z^2=1\) ( \(z \ge 0\) ) 为顶的曲顶柱体体积.

难题 5 ( 考研真题类型：二重积分极限 )

求 \(\displaystyle\lim_{t \to 0^+} \frac{1}{t^4}\iint_{x^2+y^2 \le t^2} \left(x^2+y^2\right) e^{-(x^2+y^2)}\,d\sigma\).

解 极坐标下 \[\iint_{x^2+y^2 \le t^2} r^2 e^{-r^2}\,d\sigma = 2\pi\int_0^t r^3 e^{-r^2}\,dr = F(t)\]

令分子为 \(F(t)\)，分母为 \(t^4\)，两者均趋于 \(0\)，用 L’Hopital 法则 :

\[\lim_{t \to 0^+}\frac{F(t)}{t^4} = \lim_{t \to 0^+}\frac{F'(t)}{4t^3} = \lim_{t \to 0^+}\frac{2\pi t^3 e^{-t^2}}{4t^3} = \frac{2\pi}{4} = \frac{\pi}{2}.\]

难题 6 ( 二重积分不等式证明 )

设 \(f(x,y)\) 在 \(D:[0,1]\times[0,1]\) 上有连续偏导数，且 \(f(x,0)=0\). 证明

\[\left|\iint_D f(x,y)\,d\sigma\right| \le \frac{1}{2}\iint_D \left|\frac{\partial f}{\partial y}\right|d\sigma.\]

证明 由 \(f(x,0)=0\) 及牛顿-莱布尼茨公式有 \(f(x,y) = \displaystyle\int_0^y \frac{\partial f}{\partial t}(x,t)\,dt\).

故 \(|f(x,y)| \le \displaystyle\int_0^y \left|\frac{\partial f}{\partial t}\right|dt\).

\[\left|\iint_D f\,d\sigma\right| \le \int_0^1 dx \int_0^1 \int_0^y \left|\frac{\partial f}{\partial t}(x,t)\right|dt\,dy\]

交换 \(t,y\) 积分次序 ( \(0 \le t \le y \le 1\) 换为 \(0 \le t \le 1,\ t \le y \le 1\) )

\[= \int_0^1 dx \int_0^1 (1-t)\left|\frac{\partial f}{\partial y}\right|dt \le \frac{1}{2}\iint_D\left|\frac{\partial f}{\partial y}\right|d\sigma. \qquad\]

( 最后一步因 \(\max_{t\in[0,1]}(1-t) = 1\)，但更精确地 \(\displaystyle\int_0^1(1-t)h(t)dt \le \frac{1}{2}\int_0^1 h(t)dt\) 对非负 \(h\) 不直接成立，需利用 \(1-t \le 1\) 再乘系数 \(\frac{1}{2}\) 的均值论证，详见分析教材. )

难题 7 ( 递推关系 )

设 \(I_n = \displaystyle\iint_{x^2+y^2 \le 1} (x^2+y^2)^n\,d\sigma\)，求 \(I_n\).

解 极坐标下 \[I_n = \int_0^{2\pi}d\theta\int_0^1 r^{2n}\cdot r\,dr = 2\pi \cdot \frac{1}{2n+2} = \frac{\pi}{n+1}.\]

验证 \(I_0 = \pi\) ( 单位圆面积 ) \(\checkmark\); \(I_1 = \dfrac{\pi}{2}\) ( \(R=1\) 时 \(\iint (x^2+y^2)\,d\sigma = \dfrac{\pi R^4}{2} = \dfrac{\pi}{2}\) ) \(\checkmark\).

难题 8 ( Chebyshev 积分不等式 )

设 \(f(x),g(x)\) 在 \([0,1]\) 上连续且均单调递增，证明 :

\[\int_0^1 f(x)\,dx \cdot \int_0^1 g(x)\,dx \le \int_0^1 f(x)g(x)\,dx.\]

证明 由 \(f,g\) 均单调递增，对任意 \(x,y \in [0,1]\)，\((f(x)-f(y))(g(x)-g(y)) \ge 0\).

在 \([0,1]^2\) 上对 \(x,y\) 二重积分 \[\iint_{[0,1]^2}(f(x)-f(y))(g(x)-g(y))\,dx\,dy \ge 0\]

展开 \[\iint f(x)g(x)\,d\sigma - \iint f(x)g(y)\,d\sigma - \iint f(y)g(x)\,d\sigma + \iint f(y)g(y)\,d\sigma \ge 0\]

注意 \(\iint_{[0,1]^2} f(x)g(y)\,dx\,dy = \int_0^1 f\,dx \cdot \int_0^1 g\,dy\)，各项整理后得

\[2\int_0^1 fg\,dx \ge 2\int_0^1 f\,dx \cdot \int_0^1 g\,dx. \qquad\]
