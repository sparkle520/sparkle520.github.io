二、一阶微分方程

2.1 可分离变量的微分方程

形式 : \(f(x)dx = g(y)dy\) 或 \(\frac{dy}{dx} = f(x)g(y)\)

解法 :

  1. 分离变量 : \(\frac{1}{g(y)}dy = f(x)dx\)
  2. 两边积分 : \(\displaystyle\int \frac{1}{g(y)}dy = \displaystyle\int f(x)dx + C\)

2.2 齐次微分方程

形式 : \(\frac{dy}{dx} = \varphi(\frac{y}{x})\)

解法 :

  1. 令 \(u = \frac{y}{x} \implies y = ux \implies \frac{dy}{dx} = u + x\frac{du}{dx}\)
  2. 代入原方程 : \(u + x\frac{du}{dx} = \varphi(u)\)
  3. 分离变量求解 : \(\frac{du}{\varphi(u)-u} = \frac{dx}{x}\)

示例 求 \((x^2+y^2)dx - xy dy = 0\) 的通解.

解 : 变形 \(\frac{dy}{dx} = \frac{x^2+y^2}{xy} = \frac{x}{y} + \frac{y}{x}\).

令 \(u=y/x \implies y=ux \implies y'=u+xu'\).

代入 : \(u+xu' = \frac{1}{u} + u \implies xu' = \frac{1}{u} \implies u du = \frac{dx}{x}\).

积分 : \(\frac{1}{2}u^2 = \ln|x| + C \implies \frac{y^2}{2x^2} = \ln|x| + C\).

整理 : \(y^2 = 2x^2(\ln|x| + C)\).

2.3 一阶线性微分方程

形式 : \(\frac{dy}{dx} + P(x)y = Q(x)\)

通解公式 :

\[ y = e^{-\int P(x)dx} \left( \int Q(x) e^{\int P(x)dx} dx + C \right) \]

( 口诀 先积 \(P\) 后乘 \(Q\)，积分号内再乘 \(e\) )

2.4 伯努利方程 ( Bernoulli )

形式 : \(\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (n \neq 0, 1)\)

解法 :

  1. 两边同除 \(y^n\) : \(y^{-n}y' + P(x)y^{1-n} = Q(x)\)
  2. 换元 : 令 \(z = y^{1-n} \implies z' = (1-n)y^{-n}y'\)
  3. 化为一阶线性 : \(\frac{1}{1-n}z' + P(x)z = Q(x)\)

示例 求 \(y' + \frac{y}{x} = y^2 \frac{\ln x}{x}\) 的通解.

解 : 这是 \(n=2\) 的伯努利方程. 两边除 \(y^2\) : \(y^{-2}y' + \frac{1}{x}y^{-1} = \frac{\ln x}{x}\).

令 \(z = y^{1-2} = y^{-1} \implies z' = -y^{-2}y'\).

代入 : \(-z' + \frac{1}{x}z = \frac{\ln x}{x} \implies z' - \frac{1}{x}z = -\frac{\ln x}{x}\).

这是一阶线性方程, \(P(x)=-\frac{1}{x}, Q(x)=-\frac{\ln x}{x}\).

通解 \(z = e^{\int \frac{1}{x}dx} (\displaystyle\int -\frac{\ln x}{x} e^{-\int \frac{1}{x}dx} dx + C) = x (\displaystyle\int -\frac{\ln x}{x} \frac{1}{x} dx + C)\).

\(\displaystyle\int -\frac{\ln x}{x^2} dx = \displaystyle\int \ln x d(\frac{1}{x}) = \frac{\ln x}{x} - \int \frac{1}{x^2} dx = \frac{\ln x}{x} + \frac{1}{x}\).

故 \(z = x (\frac{\ln x}{x} + \frac{1}{x} + C) = \ln x + 1 + Cx\).

即 \(\frac{1}{y} = \ln x + 1 + Cx\).

2.5 全微分方程

形式 : \(P(x,y)dx + Q(x,y)dy = 0\)，满足 \(\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}\)

解法 :

  1. 凑全微分法 分组观察，凑成 \(d(u(x,y)) = 0\).
  2. 偏积分法 \(u(x,y) = \int_{x_0}^x P(x,y)dx + \int_{y_0}^y Q(x_0,y)dy = C\).

示例 求 \((e^y + 2xy)dx + (xe^y + x^2)dy = 0\) 的通解.

解 : \(P = e^y + 2xy, Q = xe^y + x^2\).

\(\frac{\partial P}{\partial y} = e^y + 2x, \frac{\partial Q}{\partial x} = e^y + 2x\). 相等, 为全微分方程.

凑全微分 : \((e^y dx + xe^y dy) + (2xy dx + x^2 dy) = 0\). \(d(xe^y) + d(x^2y) = 0 \implies d(xe^y + x^2y) = 0\).

通解 : \(xe^y + x^2y = C\).

三、可降阶的高阶微分方程

3.1 \(y^{(n)} = f(x)\) 型

解法 : 连续积分 \(n\) 次.

3.2 \(y'' = f(x, y')\) 型 ( 缺 \(y\) )

解法 :

  1. 令 \(y' = p(x)\)，则 \(y'' = p'\)
  2. 原方程化为一阶 : \(p' = f(x, p)\)
  3. 解出 \(p\) 后，再积分一次 : \(y = \displaystyle\int p(x)dx + C_2\)

3.3 \(y'' = f(y, y')\) 型 ( 缺 \(x\) )

解法 :

  1. 令 \(y' = p(y)\) ( 注意 \(p\) 是 \(y\) 的函数 )
  2. 则 \(y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}\)
  3. 原方程化为 : \(p \frac{dp}{dy} = f(y, p)\) ( 可分离或一阶 )
  4. 解出 \(p(y)\) 后，解 \(y' = p(y) \implies \frac{dy}{p(y)} = dx\)

四、高阶线性微分方程解的结构

4.1 齐次方程解的结构

对于 \(y'' + p(x)y' + q(x)y = 0\) : 若 \(y_1(x), y_2(x)\) 是两个 线性无关 的特解，则通解为 : \[ Y(x) = C_1 y_1(x) + C_2 y_2(x) \]

4.2 非齐次方程解的结构

对于 \(y'' + p(x)y' + q(x)y = f(x)\) : 若 \(y^*(x)\) 是一个特解，\(Y(x)\) 是对应齐次方程的通解，则非齐次通解为 : \[ y(x) = Y(x) + y^*(x) = C_1 y_1(x) + C_2 y_2(x) + y^*(x) \]

叠加原理 :

若 \(y_1^*\) 是对应 \(f_1(x)\) 的特解，\(y_2^*\) 是对应 \(f_2(x)\) 的特解， 则 \(y_1^* + y_2^*\) 是对应 \(f_1(x) + f_2(x)\) 的特解.

五、常系数齐次线性微分方程

形式 : \(y'' + py' + qy = 0\) 特征方程 : \(r^2 + pr + q = 0\)

特征根情况 \(r_1, r_2\) 通解 \(y(x)\)
两个不相等实根 \(r_1 \neq r_2\) \(C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
两个相等实根 \(r_1 = r_2 = r\) \((C_1 + C_2 x) e^{r x}\)
一对共轭复根 \(\alpha \pm i\beta\) \(e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)\)

对于 \(n\) 阶方程，若 \(r\) 是 \(k\) 重实根，对应项为 \((C_1 + \cdots + C_k x^{k-1})e^{rx}\).

六、常系数非齐次线性微分方程

形式 : \(y'' + py' + qy = f(x)\) 通解 : \(y = Y + y^*\) ( 齐次通解 + 非齐次特解 )

6.1 类型 I : \(f(x) = e^{\lambda x} P_m(x)\)

特解设法 : \(y^* = x^k Q_m(x) e^{\lambda x}\)

  • \(Q_m(x)\) : 与 \(P_m(x)\) 同次的一般多项式.
  • \(k\) : 修正系数 ( 看 \(\lambda\) 是特征方程的根的重数 ).
    • \(\lambda\) 不是根 \(\implies k=0\)
    • \(\lambda\) 是单根 \(\implies k=1\)
    • \(\lambda\) 是重根 \(\implies k=2\)

6.2 类型 II : \(f(x) = e^{\alpha x} [P_l(x)\cos\beta x + P_n(x)\sin\beta x]\)

特解设法 : \(y^* = x^k e^{\alpha x} [R_m^{(1)}(x)\cos\beta x + R_m^{(2)}(x)\sin\beta x]\)

  • \(m = \max\{l, n\}\).
  • \(R_m^{(1)}, R_m^{(2)}\) : 两个不同的 \(m\) 次一般多项式.
  • \(k\) : 修正系数 ( 看 \(\alpha \pm i\beta\) 是特征根的重数 ).
    • \(\alpha \pm i\beta\) 不是根 \(\implies k=0\)
    • \(\alpha \pm i\beta\) 是根 \(\implies k=1\)

示例 1 ( 类型 I 共振 ) 求 \(y'' - 2y' + y = x e^x\) 的通解.

解 : 特征方程 \(r^2 - 2r + 1 = 0 \implies (r-1)^2 = 0 \implies r_{1,2} = 1\) ( 二重根 ).

齐次通解 \(Y = (C_1 + C_2 x)e^x\). \(f(x) = x e^x \implies \lambda = 1, P_m(x)=x\) ( 1次 ).

因 \(1\) 是二重特征根 ( \(k=2\) ), 设特解 \(y^* = x^2 (Ax+B) e^x = (Ax^3 + Bx^2)e^x\).

代入原方程比较系数 : \(6Ax + 2B = x\).

\(\implies 6A=1, 2B=0 \implies A=1/6, B=0\).

特解 \(y^* = \frac{1}{6}x^3 e^x\).

通解 \(y = (C_1 + C_2 x)e^x + \frac{1}{6}x^3 e^x\).

示例 2 ( 类型 II 混合 ) 求 \(y'' + y = x \cos x + 2 \sin x\) 的特解形式 ( 无需解出系数 ).

解 : 特征方程 \(r^2 + 1 = 0 \implies r = \pm i\).

\(f(x) = x \cos x + 2 \sin x \implies \alpha=0, \beta=1 \implies \alpha \pm i\beta = \pm i\) ( 是单根, \(k=1\) ).

\(P_l(x)=x\) ( 1次 ), \(P_n(x)=2\) ( 0次 ) \(\implies m = \max\{1, 0\} = 1\). 设特解 \(y^* = x^1 e^{0x} [(Ax+B)\cos x + (Cx+D)\sin x] = x[(Ax+B)\cos x + (Cx+D)\sin x]\).

七、欧拉方程

形式 : \(x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + \cdots + a_n y = f(x)\)

解法 : 令 \(x = e^t\) ( \(t = \ln x, x>0\) )，将变系数化为常系数. 转换关系 :

  • \(xy' = \frac{dy}{dt} = Dy\)
  • \(x^2y'' = \frac{d^2y}{dt^2} - \frac{dy}{dt} = D(D-1)y\)
  • \(x^3y''' = D(D-1)(D-2)y\)

示例 求 \(x^2 y'' + xy' - 4y = 0\) 的通解.

解 : 这是一个欧拉方程. 令 \(x = e^t\), 则 \(xy' = Dy, x^2 y'' = D(D-1)y\).

代入 : \(D(D-1)y + Dy - 4y = 0 \implies (D^2 - D + D - 4)y = 0 \implies (D^2 - 4)y = 0\).

特征方程 \(r^2 - 4 = 0 \implies r = \pm 2\). \(y(t) = C_1 e^{2t} + C_2 e^{-2t}\).

回代 \(t = \ln x\) : \(y(x) = C_1 x^2 + C_2 x^{-2}\).

八、总结与速查

常见方程类型判别与解法速查

方程类型 形式特征 核心解法
可分离变量 \(f(x)dx = g(y)dy\) 直接积分
齐次方程 \(y' = \varphi(y/x)\) 令 \(u=y/x\)
一阶线性 \(y' + P(x)y = Q(x)\) 公式法 ( 先积 \(P\) 后乘 \(Q\) )
伯努利方程 \(y' + P(x)y = Q(x)y^n\) 令 \(z=y^{1-n}\)
全微分方程 \(Pdx + Qdy = 0, P_y=Q_x\) 凑全微分 / 偏积分
缺 \(y\) 二阶 \(y'' = f(x, y')\) 令 \(y'=p(x)\)
缺 \(x\) 二阶 \(y'' = f(y, y')\) 令 \(y'=p(y), y''=p \frac{dp}{dy}\)
欧拉方程 \(x^n y^{(n)} + \cdots\) 令 \(x=e^t\)

常系数非齐次特解设法口诀

  • 指数函数 \(e^{\lambda x}\) 看 \(\lambda\) 是不是根，是根就乘 \(x\)，几重乘几次.
  • 三角函数 \(\cos \beta x\) 看 \(\alpha \pm i\beta\) 是不是根 ( \(\alpha\) 为 \(e^{\alpha x}\) 指数 )，是根就乘 \(x\).
  • 多项式 \(P_m(x)\) 设一般形式 \(Q_m(x)\)，次数要最高.
