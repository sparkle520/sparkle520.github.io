дәҢгҖҒдёҖйҳ¶еҫ®еҲҶж–№зЁӢ

2.1 еҸҜеҲҶзҰ»еҸҳйҮҸзҡ„еҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(f(x)dx = g(y)dy\) жҲ– \(\frac{dy}{dx} = f(x)g(y)\)

и§Јжі• :

  1. еҲҶзҰ»еҸҳйҮҸ : \(\frac{1}{g(y)}dy = f(x)dx\)
  2. дёӨиҫ№з§ҜеҲҶ : \(\displaystyle\int \frac{1}{g(y)}dy = \displaystyle\int f(x)dx + C\)

2.2 йҪҗж¬Ўеҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(\frac{dy}{dx} = \varphi(\frac{y}{x})\)

и§Јжі• :

  1. д»Ө \(u = \frac{y}{x} \implies y = ux \implies \frac{dy}{dx} = u + x\frac{du}{dx}\)
  2. д»Је…ҘеҺҹж–№зЁӢ : \(u + x\frac{du}{dx} = \varphi(u)\)
  3. еҲҶзҰ»еҸҳйҮҸжұӮи§Ј : \(\frac{du}{\varphi(u)-u} = \frac{dx}{x}\)

зӨәдҫӢ жұӮ \((x^2+y^2)dx - xy dy = 0\) зҡ„йҖҡи§Ј.

и§Ј : еҸҳеҪў \(\frac{dy}{dx} = \frac{x^2+y^2}{xy} = \frac{x}{y} + \frac{y}{x}\).

д»Ө \(u=y/x \implies y=ux \implies y'=u+xu'\).

д»Је…Ҙ : \(u+xu' = \frac{1}{u} + u \implies xu' = \frac{1}{u} \implies u du = \frac{dx}{x}\).

з§ҜеҲҶ : \(\frac{1}{2}u^2 = \ln|x| + C \implies \frac{y^2}{2x^2} = \ln|x| + C\).

ж•ҙзҗҶ : \(y^2 = 2x^2(\ln|x| + C)\).

2.3 дёҖйҳ¶зәҝжҖ§еҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(\frac{dy}{dx} + P(x)y = Q(x)\)

йҖҡи§Је…¬ејҸ :

\[ y = e^{-\int P(x)dx} \left( \int Q(x) e^{\int P(x)dx} dx + C \right) \]

( еҸЈиҜҖ : е…Ҳз§Ҝ \(P\) еҗҺд№ҳ \(Q\)пјҢз§ҜеҲҶеҸ·еҶ…еҶҚд№ҳ \(e\) )

2.4 дјҜеҠӘеҲ©ж–№зЁӢ ( Bernoulli )

еҪўејҸ : \(\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (n \neq 0, 1)\)

и§Јжі• :

  1. дёӨиҫ№еҗҢйҷӨ \(y^n\) : \(y^{-n}y' + P(x)y^{1-n} = Q(x)\)
  2. жҚўе…ғ : д»Ө \(z = y^{1-n} \implies z' = (1-n)y^{-n}y'\)
  3. еҢ–дёәдёҖйҳ¶зәҝжҖ§ : \(\frac{1}{1-n}z' + P(x)z = Q(x)\)

зӨәдҫӢ жұӮ \(y' + \frac{y}{x} = y^2 \frac{\ln x}{x}\) зҡ„йҖҡи§Ј.

и§Ј : иҝҷжҳҜ \(n=2\) зҡ„дјҜеҠӘеҲ©ж–№зЁӢ. дёӨиҫ№йҷӨ \(y^2\) : \(y^{-2}y' + \frac{1}{x}y^{-1} = \frac{\ln x}{x}\).

д»Ө \(z = y^{1-2} = y^{-1} \implies z' = -y^{-2}y'\).

д»Је…Ҙ : \(-z' + \frac{1}{x}z = \frac{\ln x}{x} \implies z' - \frac{1}{x}z = -\frac{\ln x}{x}\).

иҝҷжҳҜдёҖйҳ¶зәҝжҖ§ж–№зЁӢ, \(P(x)=-\frac{1}{x}, Q(x)=-\frac{\ln x}{x}\).

йҖҡи§Ј \(z = e^{\int \frac{1}{x}dx} (\displaystyle\int -\frac{\ln x}{x} e^{-\int \frac{1}{x}dx} dx + C) = x (\displaystyle\int -\frac{\ln x}{x} \frac{1}{x} dx + C)\).

\(\displaystyle\int -\frac{\ln x}{x^2} dx = \displaystyle\int \ln x d(\frac{1}{x}) = \frac{\ln x}{x} - \int \frac{1}{x^2} dx = \frac{\ln x}{x} + \frac{1}{x}\).

ж•… \(z = x (\frac{\ln x}{x} + \frac{1}{x} + C) = \ln x + 1 + Cx\).

еҚі \(\frac{1}{y} = \ln x + 1 + Cx\).

2.5 е…Ёеҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(P(x,y)dx + Q(x,y)dy = 0\)пјҢж»Ўи¶і \(\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}\)

и§Јжі• :

  1. еҮ‘е…Ёеҫ®еҲҶжі• : еҲҶз»„и§ӮеҜҹпјҢеҮ‘жҲҗ \(d(u(x,y)) = 0\).
  2. еҒҸз§ҜеҲҶжі• : \(u(x,y) = \int_{x_0}^x P(x,y)dx + \int_{y_0}^y Q(x_0,y)dy = C\).

зӨәдҫӢ жұӮ \((e^y + 2xy)dx + (xe^y + x^2)dy = 0\) зҡ„йҖҡи§Ј.

и§Ј : \(P = e^y + 2xy, Q = xe^y + x^2\).

\(\frac{\partial P}{\partial y} = e^y + 2x, \frac{\partial Q}{\partial x} = e^y + 2x\). зӣёзӯү, дёәе…Ёеҫ®еҲҶж–№зЁӢ.

еҮ‘е…Ёеҫ®еҲҶ : \((e^y dx + xe^y dy) + (2xy dx + x^2 dy) = 0\). \(d(xe^y) + d(x^2y) = 0 \implies d(xe^y + x^2y) = 0\).

йҖҡи§Ј : \(xe^y + x^2y = C\).

дёүгҖҒеҸҜйҷҚйҳ¶зҡ„й«ҳйҳ¶еҫ®еҲҶж–№зЁӢ

3.1 \(y^{(n)} = f(x)\) еһӢ

и§Јжі• : иҝһз»ӯз§ҜеҲҶ \(n\) ж¬Ў.

3.2 \(y'' = f(x, y')\) еһӢ ( зјә \(y\) )

и§Јжі• :

  1. д»Ө \(y' = p(x)\)пјҢеҲҷ \(y'' = p'\)
  2. еҺҹж–№зЁӢеҢ–дёәдёҖйҳ¶ : \(p' = f(x, p)\)
  3. и§ЈеҮә \(p\) еҗҺпјҢеҶҚз§ҜеҲҶдёҖж¬Ў : \(y = \displaystyle\int p(x)dx + C_2\)

3.3 \(y'' = f(y, y')\) еһӢ ( зјә \(x\) )

и§Јжі• :

  1. д»Ө \(y' = p(y)\) ( жіЁж„Ҹ \(p\) жҳҜ \(y\) зҡ„еҮҪж•° )
  2. еҲҷ \(y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}\)
  3. еҺҹж–№зЁӢеҢ–дёә : \(p \frac{dp}{dy} = f(y, p)\) ( еҸҜеҲҶзҰ»жҲ–дёҖйҳ¶ )
  4. и§ЈеҮә \(p(y)\) еҗҺпјҢи§Ј \(y' = p(y) \implies \frac{dy}{p(y)} = dx\)

еӣӣгҖҒй«ҳйҳ¶зәҝжҖ§еҫ®еҲҶж–№зЁӢи§Јзҡ„з»“жһ„

4.1 йҪҗж¬Ўж–№зЁӢи§Јзҡ„з»“жһ„

еҜ№дәҺ \(y'' + p(x)y' + q(x)y = 0\) : иӢҘ \(y_1(x), y_2(x)\) жҳҜдёӨдёӘ зәҝжҖ§ж— е…і зҡ„зү№и§ЈпјҢеҲҷйҖҡи§Јдёә : \[ Y(x) = C_1 y_1(x) + C_2 y_2(x) \]

4.2 йқһйҪҗж¬Ўж–№зЁӢи§Јзҡ„з»“жһ„

еҜ№дәҺ \(y'' + p(x)y' + q(x)y = f(x)\) : иӢҘ \(y^*(x)\) жҳҜдёҖдёӘзү№и§ЈпјҢ\(Y(x)\) жҳҜеҜ№еә”йҪҗж¬Ўж–№зЁӢзҡ„йҖҡи§ЈпјҢеҲҷйқһйҪҗж¬ЎйҖҡи§Јдёә : \[ y(x) = Y(x) + y^*(x) = C_1 y_1(x) + C_2 y_2(x) + y^*(x) \]

еҸ еҠ еҺҹзҗҶ :

иӢҘ \(y_1^*\) жҳҜеҜ№еә” \(f_1(x)\) зҡ„зү№и§ЈпјҢ\(y_2^*\) жҳҜеҜ№еә” \(f_2(x)\) зҡ„зү№и§ЈпјҢ еҲҷ \(y_1^* + y_2^*\) жҳҜеҜ№еә” \(f_1(x) + f_2(x)\) зҡ„зү№и§Ј.

дә”гҖҒеёёзі»ж•°йҪҗж¬ЎзәҝжҖ§еҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(y'' + py' + qy = 0\) зү№еҫҒж–№зЁӢ : \(r^2 + pr + q = 0\)

зү№еҫҒж №жғ…еҶө \(r_1, r_2\) йҖҡи§Ј \(y(x)\)
дёӨдёӘдёҚзӣёзӯүе®һж № \(r_1 \neq r_2\) \(C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
дёӨдёӘзӣёзӯүе®һж № \(r_1 = r_2 = r\) \((C_1 + C_2 x) e^{r x}\)
дёҖеҜ№е…ұиҪӯеӨҚж № \(\alpha \pm i\beta\) \(e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)\)

еҜ№дәҺ \(n\) йҳ¶ж–№зЁӢпјҢиӢҘ \(r\) жҳҜ \(k\) йҮҚе®һж №пјҢеҜ№еә”йЎ№дёә \((C_1 + \cdots + C_k x^{k-1})e^{rx}\).

е…ӯгҖҒеёёзі»ж•°йқһйҪҗж¬ЎзәҝжҖ§еҫ®еҲҶж–№зЁӢ

еҪўејҸ : \(y'' + py' + qy = f(x)\) йҖҡи§Ј : \(y = Y + y^*\) ( йҪҗж¬ЎйҖҡи§Ј + йқһйҪҗж¬Ўзү№и§Ј )

6.1 зұ»еһӢ I : \(f(x) = e^{\lambda x} P_m(x)\)

зү№и§Ји®ҫжі• : \(y^* = x^k Q_m(x) e^{\lambda x}\)

  • \(Q_m(x)\) : дёҺ \(P_m(x)\) еҗҢж¬Ўзҡ„дёҖиҲ¬еӨҡйЎ№ејҸ.
  • \(k\) : дҝ®жӯЈзі»ж•° ( зңӢ \(\lambda\) жҳҜзү№еҫҒж–№зЁӢзҡ„ж №зҡ„йҮҚж•° ).
    • \(\lambda\) дёҚжҳҜж № \(\implies k=0\)
    • \(\lambda\) жҳҜеҚ•ж № \(\implies k=1\)
    • \(\lambda\) жҳҜйҮҚж № \(\implies k=2\)

6.2 зұ»еһӢ II : \(f(x) = e^{\alpha x} [P_l(x)\cos\beta x + P_n(x)\sin\beta x]\)

зү№и§Ји®ҫжі• : \(y^* = x^k e^{\alpha x} [R_m^{(1)}(x)\cos\beta x + R_m^{(2)}(x)\sin\beta x]\)

  • \(m = \max\{l, n\}\).
  • \(R_m^{(1)}, R_m^{(2)}\) : дёӨдёӘдёҚеҗҢзҡ„ \(m\) ж¬ЎдёҖиҲ¬еӨҡйЎ№ејҸ.
  • \(k\) : дҝ®жӯЈзі»ж•° ( зңӢ \(\alpha \pm i\beta\) жҳҜзү№еҫҒж №зҡ„йҮҚж•° ).
    • \(\alpha \pm i\beta\) дёҚжҳҜж № \(\implies k=0\)
    • \(\alpha \pm i\beta\) жҳҜж № \(\implies k=1\)

зӨәдҫӢ 1 ( зұ»еһӢ I е…ұжҢҜ ) жұӮ \(y'' - 2y' + y = x e^x\) зҡ„йҖҡи§Ј.

и§Ј : зү№еҫҒж–№зЁӢ \(r^2 - 2r + 1 = 0 \implies (r-1)^2 = 0 \implies r_{1,2} = 1\) ( дәҢйҮҚж № ).

йҪҗж¬ЎйҖҡи§Ј \(Y = (C_1 + C_2 x)e^x\). \(f(x) = x e^x \implies \lambda = 1, P_m(x)=x\) ( 1ж¬Ў ).

еӣ  \(1\) жҳҜдәҢйҮҚзү№еҫҒж № ( \(k=2\) ), и®ҫзү№и§Ј \(y^* = x^2 (Ax+B) e^x = (Ax^3 + Bx^2)e^x\).

д»Је…ҘеҺҹж–№зЁӢжҜ”иҫғзі»ж•° : \(6Ax + 2B = x\).

\(\implies 6A=1, 2B=0 \implies A=1/6, B=0\).

зү№и§Ј \(y^* = \frac{1}{6}x^3 e^x\).

йҖҡи§Ј \(y = (C_1 + C_2 x)e^x + \frac{1}{6}x^3 e^x\).

зӨәдҫӢ 2 ( зұ»еһӢ II ж··еҗҲ ) жұӮ \(y'' + y = x \cos x + 2 \sin x\) зҡ„зү№и§ЈеҪўејҸ ( ж— йңҖи§ЈеҮәзі»ж•° ).

и§Ј : зү№еҫҒж–№зЁӢ \(r^2 + 1 = 0 \implies r = \pm i\).

\(f(x) = x \cos x + 2 \sin x \implies \alpha=0, \beta=1 \implies \alpha \pm i\beta = \pm i\) ( жҳҜеҚ•ж №, \(k=1\) ).

\(P_l(x)=x\) ( 1ж¬Ў ), \(P_n(x)=2\) ( 0ж¬Ў ) \(\implies m = \max\{1, 0\} = 1\). и®ҫзү№и§Ј \(y^* = x^1 e^{0x} [(Ax+B)\cos x + (Cx+D)\sin x] = x[(Ax+B)\cos x + (Cx+D)\sin x]\).

дёғгҖҒж¬§жӢүж–№зЁӢ

еҪўејҸ : \(x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + \cdots + a_n y = f(x)\)

и§Јжі• : д»Ө \(x = e^t\) ( \(t = \ln x, x>0\) )пјҢе°ҶеҸҳзі»ж•°еҢ–дёәеёёзі»ж•°. иҪ¬жҚўе…ізі» :

  • \(xy' = \frac{dy}{dt} = Dy\)
  • \(x^2y'' = \frac{d^2y}{dt^2} - \frac{dy}{dt} = D(D-1)y\)
  • \(x^3y''' = D(D-1)(D-2)y\)

зӨәдҫӢ жұӮ \(x^2 y'' + xy' - 4y = 0\) зҡ„йҖҡи§Ј.

и§Ј : иҝҷжҳҜдёҖдёӘж¬§жӢүж–№зЁӢ. д»Ө \(x = e^t\), еҲҷ \(xy' = Dy, x^2 y'' = D(D-1)y\).

д»Је…Ҙ : \(D(D-1)y + Dy - 4y = 0 \implies (D^2 - D + D - 4)y = 0 \implies (D^2 - 4)y = 0\).

зү№еҫҒж–№зЁӢ \(r^2 - 4 = 0 \implies r = \pm 2\). \(y(t) = C_1 e^{2t} + C_2 e^{-2t}\).

еӣһд»Ј \(t = \ln x\) : \(y(x) = C_1 x^2 + C_2 x^{-2}\).

  • дёүи§’еҮҪж•° \(\cos \beta x\) : зңӢ \(\alpha \pm i\beta\) жҳҜдёҚжҳҜж № ( \(\alpha\) дёә \(e^{\alpha x}\) жҢҮж•° )пјҢжҳҜж №е°ұд№ҳ \(x\).
  • еӨҡйЎ№ејҸ \(P_m(x)\) : и®ҫдёҖиҲ¬еҪўејҸ \(Q_m(x)\)пјҢж¬Ўж•°иҰҒжңҖй«ҳ.
