第一章 随机事件及其概率

本章是概率论的基础，重点掌握 事件的关系与运算、全概率公式与贝叶斯公式 以及 事件的独立性.

一、随机事件与样本空间

1.1 基本概念

  1. 样本点 ( Sample Point, \(\omega\) ) ：随机试验中每一个可能结果.
  2. 样本空间 ( Sample Space, \(\Omega\) ) ：随机试验所有可能结果组成的集合.
  3. 随机事件 ：样本空间的子集.
  4. 基本事件 ：由单个样本点构成的集合.
  5. 不可能事件 ：\(\varnothing\)；必然事件 ：\(\Omega\).

1.2 事件的关系与运算

关系/运算 记号 含义 备注
包含 \(A \subset B\) A 发生必导致 B 发生 \(A=B \iff A \subset B\) 且 \(B \subset A\)
和事件 \(A \cup B\) A, B 至少有一个发生 若互不相容，记为 \(A+B\)
积事件 \(A \cap B\) 或 \(AB\) A, B 同时发生
差事件 \(A - B\) A 发生且 B 不发生 \(A - B = A\overline{B}\)
互不相容 \(AB = \varnothing\) A, B 不能同时发生 也称 “互斥”
对立事件 \(\overline{A}\) A 不发生 \(A \cup \overline{A} = \Omega, A \cap \overline{A} = \varnothing\)

运算律 ( Laws of Operation )

  1. 交换律、结合律、分配律 ( 类比集合运算 ).
  2. 对偶律 ( De Morgan’s Laws ) :
    • \(\overline{A \cup B} = \overline{A} \cap \overline{B}\) ( “长棍截断变短棍，开口方向要改变” )
    • \(\overline{A \cap B} = \overline{A} \cup \overline{B}\)
    • 推广 : \(\overline{\bigcup A_i} = \bigcap \overline{A_i}\) ; \(\overline{\bigcap A_i} = \bigcup \overline{A_i}\)

二、概率的定义与性质

2.1 公理化定义

  1. 非负性 ：\(P(A) \ge 0\).
  2. 规范性 ：\(P(\Omega) = 1\).
  3. 可列可加性 ：若 \(A_1, A_2, \dots\) 两两互不相容，则 \(P(\displaystyle\bigcup_{i=1}^{\infty} A_i) = \displaystyle\sum_{i=1}^{\infty} P(A_i)\).

2.2 重要性质与公式

  1. 有限可加性 ：若 \(A_1, \dots, A_n\) 两两互不相容，则 \(P(\sum A_i) = \sum P(A_i)\).
  2. 减法公式 ：\(P(A - B) = P(A) - P(AB)\).
    • 若 \(B \subset A\)，则 \(P(A - B) = P(A) - P(B)\).
  3. 加法公式 ( 容斥原理 ) ：
    • \(P(A \cup B) = P(A) + P(B) - P(AB)\).
    • \(P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)\).
    • 口诀：奇加偶减.
  4. 互补性 ：\(P(A) = 1 - P(\overline{A})\).
    • 技巧 : 当正面计算困难 ( 如 “至少有一个” ) 时，考虑对立事件.
  5. 重要不等式 :
    • 单调性 : 若 \(A \subset B\)，则 \(P(A) \le P(B)\).
    • Boole 不等式 ( 次可加性 ) : \(P(A \cup B) \le P(A) + P(B)\). ( 可推广至可列个 : \(P(\bigcup A_i) \le \sum P(A_i)\) )
    • Bonferroni 不等式 : \(P(AB) \ge P(A) + P(B) - 1\).
      • 推广 : \(P(\displaystyle\bigcap_{i=1}^n A_i) \ge \displaystyle\sum_{i=1}^n P(A_i) - (n-1)\).
  6. 隐含的包含关系 :
    • \(AB \subset A \subset A \cup B\); \(AB \subset B \subset A \cup B\).
    • 推论 : \(P(AB) \le P(A) \le P(A \cup B)\). ( “夹逼” 思想 )
    • 推论 : \(P(AB) \le \min\{P(A), P(B)\}, P(A \cup B) \ge \max\{P(A), P(B)\}\).

三、条件概率与独立性

3.1 条件概率

定义 ：\(P(A|B) = \dfrac{P(AB)}{P(B)}\) ( 要求 \(P(B) > 0\) ).

3.2 三大公式 ( 核心 )

  1. 乘法公式 :
    • 推导 : 由 \(P(B|A) = \dfrac{P(AB)}{P(A)} \implies P(AB) = P(A)P(B|A)\).
    • 推广 : \(P(ABC) = P(A)P(B|A)P(C|AB)\). ( 归纳法 )
  2. 全概率公式 ( “由因推果” ) :
    • 条件 : \(B_1, \dots, B_n\) 为完备事件组 ( Partition )，即 \(\bigcup B_i = \Omega, B_i B_j = \varnothing (i \neq j), P(B_i) > 0\).
    • 推导 :
      1. 分解 : \(A = A\Omega = A(\bigcup B_i) = \bigcup (AB_i)\).
      2. 互斥 : 因 \(B_i\) 互斥，故 \(AB_i\) 也互斥.
      3. 加法 : \(P(A) = P(\bigcup AB_i) = \sum P(AB_i)\).
      4. 乘法 : \(P(AB_i) = P(B_i)P(A|B_i)\).
    • 结论 : \[P(A) = \sum_{i=1}^n P(B_i)P(A|B_i)\]
  3. 贝叶斯公式 ( “执果索因” ) :
    • 推导 : 由条件概率定义与乘法公式 \[P(B_i|A) = \dfrac{P(AB_i)}{P(A)} = \dfrac{P(B_i)P(A|B_i)}{P(A)}\] 再将分母 \(P(A)\) 用全概率公式展开.
    • 结论 : \[P(B_i|A) = \dfrac{P(B_i)P(A|B_i)}{\sum_{j=1}^n P(B_j)P(A|B_j)}\]
    • 应用场景 : 已知结果 \(A\) 发生了，求导致它发生的原因是 \(B_i\) 的概率.

3.3 事件的独立性

  1. 定义 : 若 \(P(AB) = P(A)P(B)\)，称 A, B 相互独立.
  2. 性质 :
    • A, B 独立 \(\iff\) A, \(\overline{B}\) 独立 \(\iff\) \(\overline{A}\), B 独立 \(\iff\) \(\overline{A}\), \(\overline{B}\) 独立.
    • 若 \(P(A)=0\) 或 \(P(A)=1\)，则 A 与任意事件独立.

注意 : 独立 vs 互斥 若 \(P(A)>0, P(B)>0\)

  • 互斥 ( Mutually Exclusive ) : \(AB = \varnothing \implies P(AB) = 0 \neq P(A)P(B)\). ( 肯定不独立 )
  • 独立 ( Independent ) : \(P(AB) = P(A)P(B) > 0 \implies AB \neq \varnothing\). ( 肯定不互斥 )

结论 : 概率非零的两个事件，不可能既互斥又独立.

注意 : 两两独立 vs 相互独立

  • 两两独立 : \(P(AB)=P(A)P(B)\), \(P(AC)=P(A)P(C)\), \(P(BC)=P(B)P(C)\).
  • 相互独立 : 除满足两两独立外，还需满足 \(P(ABC)=P(A)P(B)P(C)\).
  • 相互独立 \(\implies\) 两两独立，反之不成立.

四、经典概型

4.1 古典概型 ( Classical Probability )

  • 特征 : 有限个样本点、等可能性.
  • 公式 : \(P(A) = \dfrac{A \text{包含的样本点数}}{\Omega \text{总样本点数}}\).
  • 常见模型 :
    • 有放回抽样 : \(N^n\).
    • 无放回抽样 : \(P_N^n\) 或 \(C_N^n\).

经典模型 : 鞋子配对问题 ( Shoes Pairing )

有 \(N\) 双不同的鞋子 ( 共 \(2N\) 只 )，从中随机取出 \(2n\) 只 (\(2n \le N\))，求其中恰好有 \(k\) 双配对的概率.

解

  1. 样本空间 : 从 \(2N\) 只鞋中任取 \(2n\) 只，总数 \(C_{2N}^{2n}\).
  2. 有利场合
    • 先选配对的 \(k\) 双 : 从 \(N\) 双鞋中选出 \(k\) 双，方法数 \(C_N^k\). ( 此时已取 \(2k\) 只 )
    • 再选不配对的鞋 : 还需要取 \(2n - 2k\) 只鞋，且这 \(2n - 2k\) 只鞋必须来自不同的鞋双 ( 否则会构成新的一双，与 “恰好 \(k\) 双” 矛盾 ).
      • 从剩下的 \(N-k\) 双鞋中选出 \(2n-2k\) 双，方法数 \(C_{N-k}^{2n-2k}\).
      • 从这 \(2n-2k\) 双中，每双各取 1 只 ( 左或右 )，方法数 \(2^{2n-2k}\).
  3. 计算概率 \[P(A) = \dfrac{C_N^k \cdot C_{N-k}^{2n-2k} \cdot 2^{2n-2k}}{C_{2N}^{2n}}.\]

4.2 几何概型 ( Geometric Probability )

  • 特征 : 样本空间为几何区域 ( 线段、平面、立体 )、等可能性 ( 概率仅与度量成正比 ).
  • 公式 : \(P(A) = \dfrac{S_A}{S_\Omega}\).

经典模型 : 会面问题 ( The Meeting Problem ) 甲乙两人约定 12:00-13:00 在某地会面. 先到者等 20 分钟后离开. 求两人能会面的概率.

解 设甲、乙到达时刻分别为 \(x, y\) ( 分钟 )，则 \(\Omega = \{(x,y) \mid 0 \le x \le 60, 0 \le y \le 60\}\).

会面条件 : \(|x - y| \le 20\).

作图可知，可行域为正方形除去两个边长为 40 的小三角形. \(S_A = 60^2 - 40^2 = 2000\). \(P(A) = \dfrac{2000}{3600} = \dfrac{5}{9}\).

经典模型 : 棍子折断问题 ( Stick Breaking )

一根长为 \(L\) 的棍子随机折成三段，求这三段能构成三角形的概率.

解

设折断点为 \(x, y\) ( 不妨设 \(0 < x < y < L\) )，则三段长度为 \(x, y-x, L-y\). 样本空间 \(\Omega = \{(x,y) \mid 0 < x < y < L\}\)，面积 \(S_\Omega = \dfrac{1}{2}L^2\).

构成三角形条件 ( 任意两边之和大于第三边 )

  1. \(x + (y-x) > L-y \implies y > L/2\)
  2. \(x + (L-y) > y-x \implies y - x < L/2\)
  3. \((y-x) + (L-y) > x \implies x < L/2\)

作图分析 : 可行域是 \(\Omega\) 中由 \(x < L/2, y > L/2, y < x + L/2\) 围成的小三角形. 面积 \(S_A = \dfrac{1}{2} (L/2)^2 = \dfrac{1}{8}L^2\).

概率 : \(P(A) = \dfrac{S_A}{S_\Omega} = \dfrac{1/8}{1/2} = \dfrac{1}{4}\).

4.3 伯努利概型 ( Bernoulli Trials )

  • 独立重复试验 : \(n\) 次试验相互独立，每次只有 “成功/失败” 两种结果，且成功概率 \(p\) 不变.
  • 二项概率公式 : \[P_n(k) = C_n^k p^k (1-p)^{n-k}, \quad k=0, 1, \dots, n\]

经典模型 : 巴拿赫火柴盒问题 ( Banach’s Matchbox Problem )

某数学家有两盒火柴，每盒 \(N\) 根. 每次用火柴时，随机任选一盒取出 1 根 ( 选左、右盒概率均为 \(1/2\) ). 求 : 当他发现某一盒空了时，另一盒恰好还剩 \(r\) (\(0 \le r \le N\)) 根的概率.

解

设 “发现空了” 的时刻，实际上是他 第 \(N+1\) 次 伸手去拿该盒 ( 此时盒中已无，才发现空 ). 设左盒取空，右盒剩 \(r\) 根. 这意味着

  1. 他总共取了 \((N+1) + (N-r) = 2N - r + 1\) 次.
  2. 最后一次 ( 第 \(2N-r+1\) 次 ) 必定是取左盒 ( 从而发现空 ).
  3. 在前 \(2N-r\) 次中，恰好有 \(N\) 次取左盒， \(N-r\) 次取右盒.

这是一次 伯努利试验 ( 每次选盒独立，概率 \(1/2\) ). \(P(\text{左空, 右剩} r) = C_{2N-r}^N (\frac{1}{2})^N (\frac{1}{2})^{N-r} \times \frac{1}{2} = C_{2N-r}^N (\frac{1}{2})^{2N-r+1}\).

同理，右盒空、左盒剩 \(r\) 的概率也是如此. 故总概率为 \[P = 2 \times C_{2N-r}^N \left(\frac{1}{2}\right)^{2N-r+1} = C_{2N-r}^N \left(\frac{1}{2}\right)^{2N-r}\]

五、综合真题解析 ( Selected Problems )

真题 1 ( 概率的不等式与范围 ) 已知 \(P(A)=P(B)=P(C)=\frac{1}{4}\)，且 \(P(AB)=0\)，\(P(AC)=\frac{1}{8}\). 求 \(A, B, C\) 至少有一个发生的概率 \(P(A \cup B \cup C)\) 的最大值与最小值.

解

由容斥原理 \(P(A \cup B \cup C) = P(A)+P(B)+P(C) - P(AB) - P(AC) - P(BC) + P(ABC)\).

  1. 分析隐含条件
    • \(P(AB)=0 \implies A, B\) 互斥 ( \(AB = \varnothing\) ).
    • \(A, B\) 互斥 \(\implies ABC = \varnothing \implies P(ABC)=0\).
    • 代入已知数据 : \(P(A \cup B \cup C) = \frac{1}{4} + \frac{1}{4} + \frac{1}{4} - 0 - \frac{1}{8} - P(BC) + 0 = \frac{5}{8} - P(BC)\).
  2. 确定 \(P(BC)\) 的范围

    • 最大值

      \(B, C\) 的重叠受限于 \(C\) 在 \(\overline{A}\) 中的剩余概率.

      \(P(C \cap \overline{A}) = P(C) - P(AC) = \frac{1}{4} - \frac{1}{8} = \frac{1}{8}\).

      \(B\) 完全在 \(\overline{A}\) 中 ( 因 \(P(AB)=0\) ).

      故 \(P(BC) = P(B \cap C \cap \overline{A}) \le \min(P(B), P(C \cap \overline{A})) = \min(\frac{1}{4}, \frac{1}{8}) = \frac{1}{8}\).

    • 最小值

      考察 \(\overline{A}\) 空间，概率为 \(1 - P(A) = \frac{3}{4}\).

      其中 \(B\) 占 \(\frac{1}{4}\)，\(C\) 的剩余部分占 \(\frac{1}{8}\). \(\frac{1}{4} + \frac{1}{8} = \frac{3}{8} < \frac{3}{4}\)，说明 \(B\) 和 \(C\) 可以在 \(\overline{A}\) 中互不相容.

      故 \(P(BC)\) 最小值为 \(0\).

  3. 结论
    • 最大值 : \(\frac{5}{8} - 0 = \frac{5}{8}\).
    • 最小值 : \(\frac{5}{8} - \frac{1}{8} = \frac{1}{2}\).

真题 2 ( 贝叶斯公式应用 ) 某工厂有甲、乙、丙三个车间生产同一种零件，产量分别占 25%, 35%, 40%. 次品率分别为 5%, 4%, 2%.

  1. 从仓库任取一件产品，求它是次品的概率.
  2. 已知取出的是次品，求它来自甲车间的概率.

解 设 \(A_1, A_2, A_3\) 分别表示 “来自甲、乙、丙车间”，\(B\) 表示 “是次品”. 已知 \(P(A_1)=0.25, P(A_2)=0.35, P(A_3)=0.40\) \(P(B|A_1)=0.05, P(B|A_2)=0.04, P(B|A_3)=0.02\)

  1. 由 全概率公式 \[P(B) = \sum P(A_i)P(B|A_i) = 0.25\times 0.05 + 0.35\times 0.04 + 0.40\times 0.02\] \[= 0.0125 + 0.014 + 0.008 = 0.0345\]

  2. 由 贝叶斯公式 \[P(A_1|B) = \dfrac{P(A_1)P(B|A_1)}{P(B)} = \dfrac{0.0125}{0.0345} = \dfrac{125}{345} = \dfrac{25}{69} \approx 0.362\]

分析 虽然甲车间产量不是最高，但由于次品率最高，所以发现次品时，来自甲的可能性 ( 36.2% ) 显著高于其产量占比 ( 25% ).

真题 3 ( 独立性判定 ) 甲、乙、丙三人同时对飞机进行射击，命中率分别为 \(0.4, 0.5, 0.7\). 飞机被击中一人而击落的概率为 0.2，被击中两人而击落的概率为 0.6，被击中三人肯定击落. 求飞机被击落的概率.

解 设 \(H_i\) 表示 “第 \(i\) 人击中” (\(i=1,2,3\))，\(A\) 表示 “飞机被击落”. \(H_1, H_2, H_3\) 相互独立.

击中 0 人 : \(B_0 = \overline{H_1}\overline{H_2}\overline{H_3}\), \(P(B_0) = 0.6 \times 0.5 \times 0.3 = 0.09\). \(P(A|B_0) = 0\).

击中 1 人 : \(B_1\)，概率 \(P(B_1) = 0.4\times 0.5\times 0.3 + 0.6\times 0.5\times 0.3 + 0.6\times 0.5\times 0.7 = 0.06 + 0.09 + 0.21 = 0.36\).

击中 2 人 : \(B_2\)，概率 \(P(B_2) = 0.4\times 0.5\times 0.3 + 0.4\times 0.5\times 0.7 + 0.6\times 0.5\times 0.7 = 0.06 + 0.14 + 0.21 = 0.41\).

击中 3 人 : \(B_3 = H_1 H_2 H_3\), \(P(B_3) = 0.4 \times 0.5 \times 0.7 = 0.14\).

全概率公式 \(P(A) = P(B_1)\times 0.2 + P(B_2)\times 0.6 + P(B_3)\times 1\) \(= 0.36 \times 0.2 + 0.41 \times 0.6 + 0.14 \times 1\) \(= 0.072 + 0.246 + 0.14 = 0.458\).

六、坑点与注意事项 ( Common Pitfalls )

坑点 1 : 零概率事件与不可能事件

  • \(P(A) = 0 \nRightarrow A = \varnothing\). ( 例如几何概型中，取到某一点的概率为 0，但该点存在 )
  • \(P(A) = 1 \nRightarrow A = \Omega\).
  • 逆命题成立 : \(A = \varnothing \implies P(A) = 0\).

坑点 2 : 条件概率的样本空间

  • 计算 \(P(A|B)\) 时，样本空间缩减为 \(B\).
  • 所有的概率公式 ( 加法、减法、乘法、全概率等 ) 在条件概率下依然成立. 例如 : \(P(A_1 \cup A_2 | B) = P(A_1|B) + P(A_2|B) - P(A_1 A_2 | B)\).

坑点 3 : 独立性的传递性误区

  • \(A, B\) 独立， \(B, C\) 独立 \(\nRightarrow A, C\) 独立. ( 独立性没有传递性 )
  • \(A, B\) 互斥 \(\nRightarrow A, B\) 独立. ( 恰恰相反，若 \(P(A), P(B) > 0\)，互斥一定不独立 )

坑点 4 : “至多/至少” 问题

  • 遇到 “至少有一个” ( \(A \cup B \cup \dots\) ) 或 “至多有一个” 时，优先考虑 对立事件，往往能简化运算.
  • “至多有一个发生” 的对立面是 “至少有两个发生”，而不是 “都发生”.
