一、二次型的核心概念与标准形

1.1 二次型的矩阵表示

含有 \(n\) 个变量 \(x_1, x_2, \cdots, x_n\) 的二次齐次函数 \[ f(x_1, x_2, \cdots, x_n) = \sum_{i = 1}^{n}\sum_{j = 1}^{n}a_{ij}x_ix_j \quad (a_{ij} = a_{ji}) \] 称为 \(n\) 元二次型. 记 \(\boldsymbol{x} = (x_1, \cdots, x_n)^{\text{T}}, \boldsymbol{A} = (a_{ij})_{n \times n}\)，则 \[ f(\boldsymbol{x}) = \boldsymbol{x}^{\text{T}}\boldsymbol{A}\boldsymbol{x} \] 其中 \(\boldsymbol{A}\) 为 实对称矩阵 ( 考研核心隐含条件 )，且二次型的秩 \(r(f) = r(\boldsymbol{A})\).

1.2 标准形与规范形

  • 标准形 : 只含平方项 \(k_1 y_1^2 + \cdots + k_n y_n^2\). ( 系数 \(k_i\) 不唯一 )
  • 规范形 : 系数只能是 \(1, -1, 0\). ( 唯一，由惯性指数决定 )

惯性定理 ( 考研重点 ) 任意二次型都可经可逆线性变换化为标准形，虽然标准形不唯一，但 正惯性指数 \(p\) ( 正系数个数 ) 和 负惯性指数 \(q\) ( 负系数个数 ) 是 唯一确定的. 且 \(p+q = r(\boldsymbol{A})\).

1.3 化标准形的两种方法

方法 变换类型 几何意义 目标矩阵 备注
配方法 可逆变换 \(\boldsymbol{x}=\boldsymbol{C}\boldsymbol{y}\) 坐标轴伸缩/倾斜 合同对角阵 计算简单，系数不一定是特征值
正交变换法 正交变换 \(\boldsymbol{x}=\boldsymbol{Q}\boldsymbol{y}\) 坐标轴旋转/反射 ( 形状不变 ) 相似对角阵 系数 必须 是特征值

易错点辨析 题目若要求 “求一个 正交变换 将二次型化为标准形”，必须求特征值和特征向量； 若仅要求 “化为标准形” 或 “求规范形”，优先使用 配方法 ( 更快 ).

二、矩阵的三大关系 ( 合同、相似、等价 )

这是考研数学一线性代数最容易混淆的难点，需从本质上区分.

关系 定义 充要条件 ( 实对称矩阵 \(\boldsymbol{A},\boldsymbol{B}\) ) 核心不变量
合同 \(\boldsymbol{C}^{\text{T}}\boldsymbol{A}\boldsymbol{C} = \boldsymbol{B}\) 正、负惯性指数相同 \(p, q, r\)
相似 \(\boldsymbol{P}^{-1}\boldsymbol{A}\boldsymbol{P} = \boldsymbol{B}\) 特征值相同 \(\lambda_i, |\boldsymbol{A}|, \text{tr}(\boldsymbol{A}), r\)
等价 \(\boldsymbol{P}\boldsymbol{A}\boldsymbol{Q} = \boldsymbol{B}\) 秩相同 \(r\)

深度结论 ( 数一 必背 ) 对于实对称矩阵 \(\boldsymbol{A}, \boldsymbol{B}\) :

  1. 相似 \(\implies\) 合同. ( 特征值相同 \(\implies\) 正负个数相同 )
  2. 合同 \(\nRightarrow\) 相似. ( 例 : \(\boldsymbol{A}=\text{diag}(2,1), \boldsymbol{B}=\text{diag}(1,1)\). \(p,q\) 相同但 \(\lambda\) 不同 )
  3. 合同 \(\implies\) 等价. ( 秩相同 )

三、正定二次型与正定矩阵

3.1 判别法全集

设 \(\boldsymbol{A}\) 为 \(n\) 阶实对称矩阵，以下命题等价 :

  1. 定义法 : \(\forall \boldsymbol{x} \neq \boldsymbol{0}, \boldsymbol{x}^{\text{T}}\boldsymbol{A}\boldsymbol{x} > 0\).
  2. 特征值法 : 所有特征值 \(\lambda_i > 0\).
  3. 主子式法 : 所有 顺序主子式 全大于 \(0\).
  4. 合同法 : \(\boldsymbol{A}\) 合同于单位阵 \(\boldsymbol{E}\) ( 或规范形为 \(y_1^2+\cdots+y_n^2\) ).
  5. 分解法 : 存在可逆阵 \(\boldsymbol{P}\) 使得 \(\boldsymbol{A} = \boldsymbol{P}^{\text{T}}\boldsymbol{P}\).

3.2 必要条件 ( 快速排除法 )

若 \(\boldsymbol{A}\) 正定，则 :

  • 主对角线元素 \(a_{ii} > 0\).
  • 行列式 \(|\boldsymbol{A}| > 0\).

四、考研数学一 高难例题精讲

题型一 : 参数讨论与正定性

例题 1 ( 经典考题变式 ) 设二次型 \(f(x_1, x_2, x_3) = x_1^2 + ax_2^2 + (a+2)x_3^2 + 2x_1x_2 - 2x_1x_3 + 2x_2x_3\).

  1. 求二次型的矩阵 \(\boldsymbol{A}\);
  2. 问 \(a\) 为何值时，二次型 \(f\) 正定?
  3. 当 \(f\) 正定时，求正交变换 \(\boldsymbol{x}=\boldsymbol{Q}\boldsymbol{y}\) 将 \(f\) 化为标准形.

解 : (1) 对照系数写出矩阵 ( 注意交叉项系数除以 2 ) : \[ \boldsymbol{A} = \begin{pmatrix} 1 & 1 & -1 \\ 1 & a & 1 \\ -1 & 1 & a+2 \end{pmatrix} \]

(2) 利用顺序主子式判别正定性 :

  • 一阶 : \(D_1 = 1 > 0\).
  • 二阶 : \(D_2 = \begin{vmatrix} 1 & 1 \\ 1 & a \end{vmatrix} = a - 1 > 0 \implies a > 1\).
  • 三阶 : \[ D_3 = |\boldsymbol{A}| = \begin{vmatrix} 1 & 1 & -1 \\ 1 & a & 1 \\ -1 & 1 & a+2 \end{vmatrix} \xlongequal{r_2-r_1, r_3+r_1} \begin{vmatrix} 1 & 1 & -1 \\ 0 & a-1 & 2 \\ 0 & 2 & a+1 \end{vmatrix} \] \[ = (a-1)(a+1) - 4 = a^2 - 5 > 0 \implies a > \sqrt{5} \text{ 或 } a < -\sqrt{5}. \] 综上，同时满足 \(a > 1\) 和 \(a^2 > 5\)，故 \(a > \sqrt{5}\) 时 \(f\) 正定.

(3) 正交变换法 ( 需具体 \(a\) 值，题目若未给定通常设特值或讨论，此处设 \(a=3\) 演示 ) : 设 \(a=3\) ( 满足 \(a>\sqrt{5}\) ). \[ \boldsymbol{A} = \begin{pmatrix} 1 & 1 & -1 \\ 1 & 3 & 1 \\ -1 & 1 & 5 \end{pmatrix} \] 求特征值 \(|\lambda \boldsymbol{E} - \boldsymbol{A}| = 0 \implies (\lambda-2)(\lambda-3)(\lambda-4) = 0\). ( 此处为假设数值，实际计算需展开 ). 解得 \(\lambda_1, \lambda_2, \lambda_3\)，求出对应单位特征向量 \(\boldsymbol{\eta}_1, \boldsymbol{\eta}_2, \boldsymbol{\eta}_3\). 构造 \(\boldsymbol{Q} = (\boldsymbol{\eta}_1, \boldsymbol{\eta}_2, \boldsymbol{\eta}_3)\)，标准形为 \(\lambda_1 y_1^2 + \lambda_2 y_2^2 + \lambda_3 y_3^2\).

题型二 : 矩阵合同与相似的综合判定

例题 2 ( 概念辨析深坑 ) 设 \(\boldsymbol{A}, \boldsymbol{B}\) 均为 \(3\) 阶实对称矩阵，且 \(|\boldsymbol{A}| = |\boldsymbol{B}| < 0\). 则 ( )

A. \(\boldsymbol{A}\) 与 \(\boldsymbol{B}\) 一定相似

B. \(\boldsymbol{A}\) 与 \(\boldsymbol{B}\) 一定合同

C. \(\boldsymbol{A}\) 与 \(\boldsymbol{B}\) 既相似又合同

D. \(\boldsymbol{A}\) 与 \(\boldsymbol{B}\) 既不相似也不合同

解析 :

  1. 看相似 : 相似要求特征值完全相同. 仅行列式 ( 特征值之积 ) 相同推不出特征值相同. 排除 A, C.

  2. 看合同 : 合同要求正负惯性指数相同.

    • \(|\boldsymbol{A}| = \lambda_1 \lambda_2 \lambda_3 < 0\).
    • 对于 3 阶矩阵，乘积为负有两种情况 :
      • 情况 1 : 3 个负 ( \(p=0, q=3\) ).
      • 情况 2 : 1 个负，2 个正 ( \(p=2, q=1\) ).
    • 若 \(\boldsymbol{A}\) 属于情况 1，\(\boldsymbol{B}\) 属于情况 2，则 \(p, q\) 不同，不合同.
    • 但是! 题目真的能取到情况 1 吗 ? 让我们反思一下.
    • 若题目给定的是 \(p, q\) 确定的条件，则合同. 这里无法确定，故选 D.

    反例 : \(\boldsymbol{A} = \text{diag}(-1, -1, -1), |\boldsymbol{A}|=-1\). \(\boldsymbol{B} = \text{diag}(1, 1, -1), |\boldsymbol{B}|=-1\). \(\boldsymbol{A}\) 的惯性指数 \((0, 3)\), \(\boldsymbol{B}\) 的惯性指数 \((2, 1)\). 显然不合同.

    变式 : 若 \(|\boldsymbol{A}| = |\boldsymbol{B}| > 0\) 且 \(\boldsymbol{A}, \boldsymbol{B}\) 均有负特征值?

    • 积为正，3 个特征值. 可能 : (3 正) 或 (1 正 2 负).
    • “均有负特征值” 排除了 (3 正).
    • 故 \(\boldsymbol{A}, \boldsymbol{B}\) 均为 (1 正 2 负)，\(p=1, q=2\).
    • 此时 \(\boldsymbol{A}, \boldsymbol{B}\) 一定合同.

题型三 : 二次型最值问题 ( Rayleigh 商 )

例题 3 ( 2023 数一考点趋势 ) 设 \(\boldsymbol{A}\) 是实对称矩阵，其特征值为 \(\lambda_1 \ge \lambda_2 \ge \cdots \ge \lambda_n\). 求 \(f(\boldsymbol{x}) = \boldsymbol{x}^{\text{T}}\boldsymbol{A}\boldsymbol{x}\) 在约束条件 \(\boldsymbol{x}^{\text{T}}\boldsymbol{x} = 1\) 下的最大值和最小值.

结论 : \[ \lambda_{min} \le \frac{\boldsymbol{x}^{\text{T}}\boldsymbol{A}\boldsymbol{x}}{\boldsymbol{x}^{\text{T}}\boldsymbol{x}} \le \lambda_{max} \] 最大值为 \(\lambda_1\) ( 最大特征值 )，最小值为 \(\lambda_n\) ( 最小特征值 ).

应用演示 : 已知 \(\boldsymbol{A} = \begin{pmatrix} 2 & -1 \\ -1 & 2 \end{pmatrix}\)，求 \(\boldsymbol{x}^{\text{T}}\boldsymbol{A}\boldsymbol{x}\) 在单位圆 \(x_1^2+x_2^2=1\) 上的最值. 1. 求特征值 : \(|\lambda \boldsymbol{E} - \boldsymbol{A}| = (\lambda-2)^2 - 1 = 0 \implies \lambda = 1, 3\). 2. 最大值为 3，最小值为 1.

五、总结与备考建议

  1. 计算基本功 : 特征值、特征向量、施密特正交化必须练得滚瓜烂熟，这是拿分基础.
  2. 概念辨析 : 遇到 “合同” 想 “惯性指数”，遇到 “相似” 想 “特征值”.
  3. 正定性 : 大题中常考参数范围讨论，优先用 顺序主子式 ( 计算量小 ) 或 特征值定义 ( 理论推导 ).
