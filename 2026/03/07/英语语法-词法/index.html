数词

数词分为 基数词 和 序数词.

基数词

数字表达规律

  1. 三位分节法 : 从右往左，每 三 位加一个逗号.
    • 倒数第 1 个逗号 : thousand ( 千 )
    • 倒数第 2 个逗号 : million ( 百万 )
    • 倒数第 3 个逗号 : billion ( 十亿 )
    • 倒数第 4 个逗号 : trillion ( 万亿 )
  2. 连接词 : 百位和十位之间、数字隔零时加 and.
数字 英文表达
101 one hundred and one
996 nine hundred and ninety-six
60,263,150 sixty million two hundred and sixty-three thousand one hundred and fifty

确切与不确切表达

类型 规则 例句
确切数字 基数词 + 单位 ( 单数 ) 🫧 three hundred people ( 💫 三百人 )
不确切数字 单位 ( 复数 ) + of 🫧 hundreds of people ( 💫 数以百计的人 )

💡 注意 : 单位词 ( hundred, thousand, million… ) 前有具体数字时不加 s，无具体数字时加 s 并带 of.

序数词

  • the + 序数词 : 表示顺序 ( 第几 ) .
  • a / an + 序数词 : 表示 “又一、再一”.

场景 规则 / 示例 英文

2008 -> two thousand and eight
日期 月 + 日 ( 序数词 ) 9月1日 -> September ( the ) first

倍数表达法

句型结构 例句

A + be + 倍数 + adj_comp + than + B 🫧 This room is three times bigger than that one.
A + be + 倍数 + the size of + B 🫧 The earth is 49 times the size of the moon.

冠词

冠词是置于名词前，用来 限定 名词意义的虚词.

分类 形式 主要功能
定冠词 the 特指，确指，双方都知道
不定冠词 a / an 泛指，不确定，某一个
零冠词 / 泛指，抽象，独一无二的头衔等

定冠词 ( The )

核心含义 : “那一个”，“也就是那…” ( 特指 )

1. 特指与独一无二

规则 说明 例句
特指 上文提到过，或双方都知道的人/物 🫧 The blue shirt is mine. ( 💫 那件 蓝衬衫是我的. )
独一无二 世界上唯一的自然天体等 🫧 the sun / moon / earth / sky / world
方位/顺序 序数词、方位词、最高级前 🫧 on the left / the first man / the best coffee
乐器 演奏乐器时加 the 🫧 play the piano / violin / guitar

2. 专有名词与群体

规则 说明 例句
江河湖海 河流、海洋、群岛、山脉 🫧 the Yellow River / the Alps / the Pacific Ocean
国家/组织 由普通名词构成的专有名词 🫧 the United States / the U.N. / the Great Wall
姓氏复数 表示 “一家人” 🫧 The Smiths ( 💫 史密斯一家 )
类指 the + adj. 表示一类人 🫧 the rich ( 富人 ) / the poor ( 穷人 ) / the old

3. 特殊句型与习语

  • “越…越…” The + 比较级…, the + 比较级…
    • 🫧 The more, the better. ( 💫 越多越好. )
  • “打某人部位” hit sb. in the face ( 💫 打某人的脸 )
    • 💡 注意 : 不用 hit sb’s face.
  • 常见固定搭配
    • in the morning / afternoon ( 在上午/下午 )
    • at the same time ( 同时 )
    • by the way ( 顺便说 )
    • on the whole ( 总的来说 )

不定冠词 ( A / An )

核心含义 : “一个”，“某一个” ( 泛指 )

  • a : 用于 辅音音素 开头的单词前 ( a book, a university ) .
  • an : 用于 元音音素 开头的单词前 ( an apple, an hour ) .

常见用法

规则 说明 例句
数量 “一” 强调数量，但不如 one 强烈 🫧 Rome was not built in a day.
类指 泛指某一类人或事物 🫧 A dog is a faithful animal.
“每一” 相当于 per / every 🫧 three times a day ( 💫 一天三次 )
“又一” 序数词前表示 “再一，又一” 🫧 try a second time ( 💫 再试一次 )
抽象具体化 抽象名词表示具体的人/事 🫧 a success ( 一件成功的事 ) / a pity ( 一件遗憾的事 )

常见固定搭配

短语 含义 短语 含义
a lot of 许多 have a rest 休息
in a hurry 匆忙 make a living 谋生
as a result 结果 have a good time 玩得开心

零冠词 ( / )

核心含义 : 不使用冠词的情况.

常见场景速查表

场景 规则 例句
泛指复数 复数名词表泛指 🫧 Horses are useful animals.
专有名词 人名、地名、国名 ( 大多数 ) 🫧 China / Beijing / Jack
头衔/职位 独一无二的职位作表语/补语 🫧 He was elected president.
学科/语言 学科、语言、球类、棋牌 🫧 study English / play football / play chess
季节/节假日 季节、月份、星期、节假日 🫧 in spring / on Monday / at Christmas
三餐 breakfast / lunch / dinner 🫧 have breakfast
交通方式 by + 交通工具 ( 单数 ) 🫧 by bus / by car / on foot

有无冠词 含义辨析

有冠词 ( 具体/特指 ) 无冠词 ( 抽象/功能 )

in the school ( 在学校里，比如办事 ) in school ( 上学，读书 )
go to the bed ( 走向那张床 ) go to bed ( 睡觉 )

名词

名词分类

分类 定义 示例
专有名词 人名、地名、机构名等 ( 首字母大写 ) Jack, China, IBM, Sunday
普通名词 一类人或东西或抽象概念 book, dog, water, love

注意

专有名词复数使用，表示 “某某一家” 或 “叫某某的人们”.

  • 🫧 There are two Jacks in our class. ( 💫 我们班有两个杰克. )

名词的性 ( Gender )

类别 说明 示例
阳性 指男性 man, father, actor, hero, bull
阴性 指女性 woman, mother, actress, heroine, cow
通性 指男女均可 teacher, student, friend, doctor
中性 指无生命事物 desk, pen, water, love

可数与不可数名词

类型 特点 示例
可数名词 能计数，有单复数，可用 a/an/one apple, car, idea, family
不可数名词 不能计数，只有单数，不可用 a/an water, bread, advice, information

不可数名词的量词搭配

不可数名词需借助 量词 ( Partitives ) 来表示数量.

量词 搭配示例 中文
piece a piece of news / paper / advice 一条新闻/一张纸/一条建议
cup / glass a cup of tea / a glass of water 一杯茶/一杯水
bar a bar of chocolate / soap 一块巧克力/肥皂
loaf / slice a loaf of bread / a slice of bread 一条面包/一片面包
pair a pair of shoes / glasses 一双鞋/一副眼镜
set a set of equipment 一套设备

名词单复数变化

1. 规则变化

规则 变化方式 例词
一般情况 加 -s map -> maps, girl -> girls
s, x, ch, sh 结尾 加 -es box -> boxes, bus -> buses
辅音 + y 结尾 变 y 为 i 加 -es baby -> babies, city -> cities
元音 + y 结尾 直接加 -s boy -> boys, day -> days
f / fe 结尾 变 f / fe 为 v 加 -es knife -> knives, leaf -> leaves
o 结尾 ( 有生命 ) 加 -es ( “英雄爱吃土豆西红柿” ) hero -> heroes, potato -> potatoes
o 结尾 ( 无生命 ) 加 -s photo -> photos, radio -> radios

2. 不规则变化

类型 变化规律 例词
元音变化 oo -> ee, a -> e foot -> feet, man -> men, mouse -> mice
词尾变化 加 -ren child -> children, ox -> oxen
单复同形 不变 sheep, deer, fish, Chinese, means

3. 外来语复数 ( 希腊/拉丁语 )

结尾变化 例词 ( 单数 -> 复数 )
-um -> -a datum -> data ( 数据 ) , medium -> media ( 媒体 )
-is -> -es basis -> bases ( 基础 ) , crisis -> crises ( 危机 )
-on -> -a criterion -> criteria ( 标准 ) , phenomenon -> phenomena ( 现象 )
-us -> -i radius -> radii ( 半径 ) , stimulus -> stimuli ( 刺激 )

4. 复合名词复数

规则 示例
主体名词变复数 son-in-law -> sons-in-law, passer-by -> passers-by
无主体名词，词尾变 grown-up -> grown-ups
man/woman 前缀，两头变 man doctor -> men doctors, woman teacher -> women teachers

名词所有格

1. ’s 所有格 ( 有生命 )

规则 示例 说明
单数名词 Tom’s book 汤姆的书
s 结尾复数 the students’ room 学生们的房间
不以 s 结尾复数 Children’s Day 儿童节
并列 ( 共同所有 ) Tom and Jack’s room 两人共用一间房 ( 最后加’s )
并列 ( 各自所有 ) Tom’s and Jack’s rooms 两人各有一间房 ( 分别加’s )

2. of 所有格 ( 无生命 )

  • 结构 : 名词 + of + 名词
    • 🫧 the leg of the table ( 桌腿 )
    • 🫧 the cover of the book ( 书皮 )

3. 双重所有格

  • 结构 : of + 名词’s / 名词性物主代词
    • 🫧 a friend of my father’s ( 我父亲的一位朋友 )
    • 🫧 a photo of mine ( 我的一张照片 - 照片属于我 )
    • 💡 对比 : a photo of me ( 我的一张照片 - 照片里是我 )

名词主谓一致速查表

规则类型 谓语单/复数 示例
基本原则 单数主语 -> 单数
复数主语 -> 复数		 He is… / They are…
整体概念 国家、机构、书名、时间、距离、金钱 -> 单数 Three miles is not far.
“Friends” is funny.
And 连接 指同一人/事 -> 单数
指不同人/事 -> 复数		 The singer and actor is coming.
The singer and the actor are coming.
集合名词 侧重整体 -> 单数
侧重个体 -> 复数		 The family is big.
The family are watching TV.
不定代词 each, every, no one, either -> 单数 Each boy has a pen.
Many a Many a + 单数名词 -> 单数 Many a student makes mistakes.
The number The number of ( 数量 ) -> 单数
A number of ( 许多 ) -> 复数		 The number of students is 50.
A number of students are playing.
就近原则 There be, Either…or, Neither…nor, Not only…but also There is a pen and two books.
Neither he nor I am right.
分数/百分数 取决于 of 后面的名词 2/3 of the land is…
2/3 of the people are…

1. 就近原则示例 ( Proximity Rule )

结构 示例 分析
or His friends or he was dead.
He or his friends were dead.		 谓语由靠近的 he / friends 决定
Either…or Either my brothers or mother is going.
Either mother or my brothers are going.		 谓语由靠近的 mother / brothers 决定
Neither…nor Neither two apples nor one banana is vegetable.
Neither one apple nor two bananas are vegetables.		 谓语由靠近的 banana / bananas 决定
Not only…but also Not only my friends but also my girlfriend was wrong.
Not only my girlfriend but also my friends were wrong.		 谓语由靠近的 girlfriend / friends 决定

2. 就远原则 ( Remote Rule )

规则 : 谓语动词的单复数由 前面的主语 ( 距离谓语 远 的名词 ) 决定，不受修饰语影响.

常见修饰语

  • but, except, besides, in addition to ( 除了/包括 )
  • with, along with, together with, combined with, as well as ( 和…一起 )
  • including ( 包括 ) , rather than, instead of ( 而不是 )
关键词 示例 分析
but / except Peter, but his roommates, is friendly.
Peter, except his roommates, is friendly.		 主语是 Peter ( 单数 )
besides / in addition to Peter, besides his roommates, is friendly. 主语是 Peter ( 单数 )
with / along with Peter, with his roommates, is friendly.
Peter, along with his roommates, is friendly.		 主语是 Peter ( 单数 )
together with Peter, together with his roommates, is friendly. 主语是 Peter ( 单数 )
as well as Peter, as well as his roommates, is friendly. 主语是 Peter ( 单数 )
including Peter, including his roommates, is friendly. 主语是 Peter ( 单数 )
rather than Peter, rather than his roommates, is friendly. 主语是 Peter ( 单数 )

名词的句法功能 ( Syntactic Functions )

名词在句中主要作 主语、宾语、表语、同位语，有时也可作 定语 或 状语.

成分 示例 说明
主语 Time flies.
宾语 I love music.
表语 He is a teacher.
同位语 Mr. Smith, our teacher, is here. 解释说明前面的名词
定语 apple tree / woman doctor 名词修饰名词
状语 See you next Monday. 表示时间/地点/方式等

代词 ( Pronouns )

代词是代替名词或一句话的一种词类. 大多数代词具有名词和形容词的功能.

1. 人称代词 ( Personal Pronouns )

人称代词用于指代人或事物，分为 主格 ( 作主语 ) 和 宾格 ( 作宾语 ).

人称 单/复数 主格 ( Subjective ) 宾格 ( Objective ) 示例
第一人称 单数 I me I love her. / She loves me.
复数 we us We help them. / They help us.
第二人称 单/复数 you you You look great. / I call you.
第三人称 单数 he / she / it him / her / it He is tall. / Give it to him.
复数 they them They are here. / Watch them.

💡 排列顺序 : 单数 ( 二 -> 三 -> 一 ) You, he and I; 复数 ( 一 -> 二 -> 三 ) We, you and they.

It 的特殊用法

用法 说明 示例
指时间/天气/距离 作非人称主语 It is raining. / It is 5 miles away.
指代上文事物 避免重复 I have a cat. It is cute.
指代身份不明的人 敲门、电话等 Who is it? / It’s me.
形式主语 真正主语后置 ( to do / that ) It is important to learn English.
形式宾语 真正宾语后置 ( to do / that ) I find it hard to finish the work.
强调句型 It is/was + 被强调部分 + that/who It was Jack who broke the window.

2. 物主代词 ( Possessive Pronouns )

表示所有关系的代词，分为 形容词性 ( 作定语 ) 和 名词性 ( 作主/宾/表语 ).

人称 形容词性物主代词 ( Adjective ) 名词性物主代词 ( Nominal ) 区别与用法
第一人称 my / our mine / ours 形容词性 + 名词 ( My book )
第二人称 your yours 名词性 = 形容词性 + 名词
第三人称 his / her / its / their his / hers / its / theirs This is my pen. = This pen is mine.

双重所有格 of + 名词性物主代词 - a friend of mine ( 我的一个朋友 ) - a book of yours ( 你的一本书 )

3. 反身代词 ( Reflexive Pronouns )

表示 “某人自己”，作宾语或同位语.

人称 单数 复数 常用短语
第一人称 myself ourselves by oneself ( 独自地 )
第二人称 yourself yourselves enjoy oneself ( 过得愉快 )
第三人称 himself / herself / itself themselves help oneself to ( 随便吃/用 )

4. 指示代词 ( Demonstrative Pronouns )

表示 “这个”、“那个”、“这些”、“那些”.

单数 复数 用法
近指 this ( 这个 ) these ( 这些 ) 指时间/空间上较近的人或物
远指 that ( 那个 ) those ( 那些 ) 指时间/空间上较远的人或物

用法辨析

  • 打电话时 : 我是…用 this ( This is Mary speaking ) , 对方是…用 that ( Is that Jack? ) .
  • 比较级避免重复 : The weather here is better than that in London. ( 指代 weather )

5. 不定代词 ( Indefinite Pronouns )

不指明代替任何特定名词或形容词的代词.

常用不定代词辨析

组别 词汇 区别 示例
Some / Any some 用于肯定句; 疑问句表请求/建议 Would you like some tea?
any 用于否定句、疑问句、条件句 I don’t have any money.
Many / Much many 修饰可数名词复数 Many students
much 修饰不可数名词 Much water
Few / Little ( a ) few 修饰可数; a few ( 有一些, 肯定 ); few ( 几乎没有, 否定 ) I have a few friends.
( a ) little 修饰不可数; a little ( 有一点, 肯定 ); little ( 几乎没有, 否定 ) There is little time left.
Each / Every each 强调个体 ( 2个或以上 ); 可作主/宾/定/同位语 Each has a book.
every 强调整体 ( 3个或以上 ); 只作定语 Every student is here.
Other / Another other 泛指 “其他的” ( 后接复数名词 ) other people
the other 特指 “两者中的另一个” ( one…, the other… ) One is red, the other is blue.
another 泛指 “三者或以上中的另一个” Have another cup of tea.
Both / All both 两者都 ( Keep both eyes open ) Both are good.
all 三者或以上都 ( We all agree ) All is well.
Either / Neither either 两者中任意一个 Either is okay.
neither 两者都不 ( 全部否定 ) Neither is correct.
None / No one no one 只能指人, 不接 of No one knows.
none 指人或物, 可接 of, 强调数量 ( None of… ) None of us went.

5.2 复合不定代词 ( Compound Indefinite Pronouns )

由 some-, any-, no-, every- 与 -one, -body, -thing 组合而成.

组合 -one / -body ( 指人 ) -thing ( 指物 ) 谓语动词
some- someone / somebody ( 某人 ) something ( 某事/物 ) 单数
any- anyone / anybody ( 任何人 ) anything ( 任何事/物 ) 单数
no- no one / nobody ( 没人 ) nothing ( 什么也没有 ) 单数
every- everyone / everybody ( 人人 ) everything ( 一切 ) 单数

注意

  • 不能用进行时的动词 : 表示情感 ( love, hate ) , 归属 ( have, own ) , 感官 ( see, hear ) , 思想 ( know, believe ) 等静态动词.

3.2 过去时态 ( Past Tenses )

时态 构成 用法 示例
一般过去时
( Simple Past )		 did
否定 : did not do		 过去某个时间发生的动作或状态 - I learned English.
- The UFO was flying over my head.
过去进行时
( Past Continuous )		 was/were + doing 过去某时刻正在进行的动作 - The UFO was flying over my head last midnight.
过去完成时
( Past Perfect )		 had + done 过去的过去 ( Before past ) - I had lived in Nanjing before I moved to Kunming.
过去完成进行时
( Past Perfect Continuous )		 had + been + doing 过去某时刻之前一直进行的动作 - She had been carrying out her plot.

3.3 将来时态 ( Future Tenses )

时态 构成 用法 示例
一般将来时
( Simple Future )		 will + do 将要发生的动作 - I will buy a new car.
将来进行时
( Future Continuous )		 will + be + doing 将来某时刻正在进行的动作 - Alibaba users will be growing.
将来完成时
( Future Perfect )		 will + have + done 将来某时刻之前完成的动作 - I will have reached Beijing.
将来完成进行时
( Future Perfect Continuous )		 will + have + been + doing 将来某时刻之前一直进行的动作 - By then, the match will have been running for three months.

3.4 过去将来时态 ( Past Future Tenses )

多用于间接引语或虚拟语气中，立足于过去看将来.

时态 构成 示例
一般过去将来时 would + do I said I would win the match.
过去将来进行时 would + be + doing Dad said I would be fighting for my future.
过去将来完成时 would + have + done I guessed you would have told her.
过去将来完成进行时 would + have + been + doing He would have been translating those books.

4. 动词语态 ( Verb Voices )

英语有两种语态：主动语态 ( Active Voice ) 和 被动语态 ( Passive Voice ).

语态 定义 示例
主动语态 主语是动作的执行者 I give you a car.
被动语态 主语是动作的承受者 You are given a car ( by me ).

注意

只有 及物动词 ( vt. ) 才有被动语态.

4.1 被动语态构成 ( Formation )

基本结构 : be + 过去分词 ( done )

时态 被动语态构成 示例 ( I kill you -> You … by me )
一般现在时 am/is/are + done You are killed by me.
一般过去时 was/were + done You were killed by me.
一般将来时 will be + done You will be killed by me.
现在进行时 am/is/are + being + done You are being killed by me.
过去进行时 was/were + being + done You were being killed by me.
现在完成时 have/has + been + done You have been killed by me.
过去完成时 had + been + done You had been killed by me.
将来完成时 will have + been + done You will have been killed by me.
情态动词 modal + be + done You can be killed by me.

4.2 主动改被动步骤 ( Conversion Steps )

  1. 宾变主 : 把主动句的 宾语 变为被动句的 主语.
  2. 动变被 : 把谓语动词变为 be + done，注意 时态不变，人称/数 随新主语变化.
  3. 主变宾 : 把主动句的 主语 变为 by + 宾格，放在谓语后 ( 若执行者不重要可省略 ).
  4. 其他不变 : 其他成分 ( 状语等 ) 保持不变.

示例

  • Active : My mother sweeps the floor every morning.
  • Passive : The floor is swept by my mother every morning.

4.3 特殊被动结构 ( Special Forms )

类型 规则 示例
双宾语
( give sb sth )		 1. 间接宾语( 人 ) 作主语 ( 常用 )
2. 直接宾语( 物 ) 作主语 ( +to/for )		 - She was given some money.
- Some money was given to her.
宾语补足语
( make sb do )		 宾语作主语，宾补保留
( 感官/使役动词省去的 to 需还原 )		 - We made him manager -> He was made manager.
- I saw him enter -> He was seen to enter.
动词短语
( take care of )		 把短语看作整体，介词不可省 - The trees must be taken good care of.

4.4 何时使用被动语态 ( Usage )

  1. 不知道 动作执行者 : The diamond was stolen.
  2. 强调 动作承受者 : Peter was grown in this country.
  3. 客观说明 事实 : The iPhone was created by Steve Jobs.

5. 非谓语动词 ( Non-finite Verbs )

5.1 分类 ( Classification )

类型 形式 含义 示例
不定式 ( Infinitive ) to do 将来 / 目的 The teacher told us to do the homework.
动名词 ( Gerund ) doing 名词性质 / 抽象动作 I like swimming.
分词 ( Participle ) doing ( Present )
done ( Past )		 主动 / 进行
被动 / 完成		 He sat there reading a newspaper.
We heard about polluted air.

5.2 不定式 ( Infinitive )

基本形式 : to + do ( 有时省略 to )

5.2.1 省略 to 的情况 ( Omission of “to” )
情况 关键词 示例
助动词后 can, must, may, will, shall I can do it.
感官动词后 see, hear, watch, feel, notice I saw him enter the house.
使役动词后 let, make, have Let me try it.
固定搭配 Why not... Why not go out for a walk?
Know 句型 know + obj + ( to ) do ( 过去/完成时 ) I have never known her tell lies.
介词后 except, but ( 前有 do ) She tries everything except cook.
主语带 do All... do is... All you do now is complete the form.
并列结构 and, or He told me to stay and wait for him.

注意 感官动词和使役动词在 被动语态 中需 还原 to.

  • Active : She made him feel happy.
  • Passive : He was made to feel happy.
5.2.2 时态与语态 ( Tense & Voice )
时态 \ 语态 主动 ( Active ) 被动 ( Passive ) 含义
一般式 ( Simple ) to do to be done 动作发生在谓语 之后 ( 将来/目的 )
完成式 ( Perfect ) to have done to have been done 动作发生在谓语 之前 ( 已完成 )
进行式 ( Continuous ) to be doing / 动作 正在进行
完成进行式 to have been doing / 动作发生在谓语之前且 持续
5.2.3 句子成分 ( Usage )
成分 示例 备注
主语 To learn a foreign language is difficult.
It is difficult to learn a foreign language.		 常用 it 作形式主语
宾语 I want to go home.
I find it difficult to learn English.		 常用 it 作形式宾语
表语 His wish is to be a scientist. 说明主语的内容或目标
定语 I have nothing to say. 放在名词后，表将来或目的
状语 They went to see their aunt. 表目的、结果或原因
宾补 The teacher told us to do homework. 补充说明宾语的动作

5.3 动名词 ( Gerund )

基本形式 : verb + ing ( 具有名词性质 )

5.3.1 时态与语态 ( Tense & Voice )
时态 \ 语态 主动 ( Active ) 被动 ( Passive ) 含义
一般式 ( Simple ) doing being done 泛指或与谓语同时发生
完成式 ( Perfect ) having done having been done 动作发生在谓语 之前
5.3.2 句子成分 ( Usage )
成分 示例 备注
主语 Learning English is important. 表示抽象动作或习惯
宾语 I enjoy dancing. 部分动词后只能接 doing
表语 His job is driving a bus. 说明主语的内容
定语 This is a swimming pool. 说明用途 ( a pool for swimming )
宾补 I call this robbing Peter to pay Paul.

5.4 分词 ( Participle )

包括 现在分词 ( Present Participle ) 和 过去分词 ( Past Participle ).

5.4.1 现在分词 ( Present Participle )

含义 : 主动、进行

时态 \ 语态 主动 ( Active ) 被动 ( Passive ) 含义
一般式 ( Simple ) doing being done 与谓语同时进行
完成式 ( Perfect ) having done having been done 发生在谓语 之前

用法示例

  • 表语 : The story is interesting. ( 令人有趣的 )
  • 定语 : The area being studied is large. ( 正在被研究的 )
  • 状语 : When leaving the airport, she waved to us. ( 时间/原因/伴随 )
  • 宾补 : I saw him stealing money. ( 正在做 )
5.4.2 过去分词 ( Past Participle )

含义 : 被动、完成

成分 示例 含义
表语 He is interested in the news. 感到感兴趣的 ( 人作主语 )
定语 The polluted river. 被污染的 ( 被动/完成 )
状语 Given more time, I can do better. 条件/原因/时间 ( 被给时间 )
宾补 I found my car stolen. 发现车被偷了

5.5 易混淆用法 ( Common Confusions )

动词 接 to do ( 含义 ) 接 doing ( 含义 )
forget / remember 还没做 / 将要做 ( Remind )
Don’t forget to lock the door.		 已经做了 ( Recall )
I remember locking the door.
regret 遗憾将要做 ( Inform )
I regret to tell you…		 后悔做了 ( Repent )
I regret telling him.
mean 打算 ( Intend )
I mean to go.		 意味着 ( Involve )
Success means working hard.
try 尽力 ( Effort )
Try to learn.		 尝试 ( Experiment )
Try knocking.
stop 停下来去做另一件事 ( Purpose )
He stopped to smoke.		 停止正在做的事 ( Cease )
He stopped smoking.
go on 继续做另一件事 ( Switch )
He went on to talk history.		 继续做同一件事 ( Continue )
He went on talking history.

6. 独立主格结构 ( Independent Nominative )

定义 : 由名词/代词 ( 逻辑主语 ) + 非谓语/adj/adv/prep 构成，作状语.

特点 : 逻辑主语与主句主语 不一致；用逗号隔开；无连词.

结构类型 示例 分析
N. + doing Time permitting, we will go out. 时间允许 ( 主动/将来 )
N. + done All things considered, it’s a good plan. 事情被考虑 ( 被动/完成 )
N. + to do So many things to do, I can’t leave. 有事要做 ( 将来 )
N. + adj. She sat there, her face pale. 脸苍白 ( 状态 )
N. + adv. He sat there, head down. 头低着
N. + prep. The teacher entered, book in hand. 书在手里
with 复合结构 He stood there with his hand raised. 手举着 ( with + 宾语 + 宾补 )

注意

  1. There being + 名词 : There being no bus, we walked.
  2. It being + 名词/形容词 : It being Sunday, the shop is closed.

连词 ( Conjunctions )

连词是用来连接词与词、短语与短语、句子与句子的虚词.

1. 并列连词 ( Coordinating Conjunctions )

连接两个语法地位平等的成分 ( 词、短语、分句 ).

连词 含义 逻辑关系 示例
and 和，并且 并列 / 顺承 You and I.
Stand up and answer.
but 但是 转折 Short but strong.
or 或者，否则 选择 / 否定的并列 Tea or coffee?
Hurry up, or you’ll be late.
so 所以 因果 ( 结果 ) It was cold, so I wore a coat.
for 因为 因果 ( 推测/补充 ) It must be morning, for the birds are singing.
yet 然而 转折 ( 虽…但… ) Tired yet happy.
nor 也不 否定并列 ( 倒装 ) He doesn’t know, nor do I.

2. 从属连词 ( Subordinating Conjunctions )

用于引导从句 ( 状语从句、名词性从句 ).

分类 连词 示例
时间 when, while, after, before, since, until Wait until I come back.
原因 because, since, as I did it because I like it.
条件 if, unless ( 除非 ) , as long as If it rains, I won’t go.
让步 although, though, even if Although he is old, he is strong.
目的 so that, in order that Speak loud so that we can hear.
结果 so...that, such...that He is so kind that we all like him.
比较 than, as...as He works harder than I.

3. 关联连词 ( Correlative Conjunctions )

成对出现的连词词组.

连词组 含义 谓语一致性 ( 作主语时 ) 示例
both…and… 两者都… 复数 Both he and I are right.
either…or… 或者…或者… 就近原则 Either you or he is wrong.
neither…nor… 既不…也不… 就近原则 Neither he nor I am going.
not only…but also… 不但…而且… 就近原则 Not only he but also I am interested.

感叹词 ( Interjections )

表示喜怒哀乐等强烈感情的词.

词汇 情感 示例
Oh 惊讶、恐惧、痛苦 Oh! What a beautiful flower!
Ah 惊奇、高兴、遗憾 Ah, I see.
Wow 极大的惊奇、钦佩 Wow! You did a great job!
Alas 悲伤、遗憾 ( 唉 ) Alas! He is dead.
Ouch 突然的疼痛 ( 哎呦 ) Ouch! You stepped on my foot.
Well 惊讶、无奈、让步 Well, let’s go.
