线性代数 第五章 特征值与特征向量

本章是线性代数承上启下的核心章节，也是考研数学一的命题热点. 不仅考察特征值与特征向量的计算，更深入考察 矩阵相似对角化 的判定条件以及 实对称矩阵 的特殊性质.

一、核心概念与性质

1.1 定义与几何直观

设 \(A\) 是 \(n\) 阶矩阵，若存在数 \(\lambda\) 和 \(n\) 维 非零 列向量 \(\alpha\)，使得 \[ A\alpha = \lambda\alpha \quad (\alpha \neq 0) \] 则称 \(\lambda\) 是 \(A\) 的 特征值，\(\alpha\) 是对应于 \(\lambda\) 的 特征向量.

  • 几何意义 : 矩阵 \(A\) 代表一种线性变换. 特征向量 \(\alpha\) 是在该变换下 方向保持不变 ( 或反向 ) 的向量，而特征值 \(\lambda\) 则是该方向上的 伸缩比例.
  • 计算核心 : 解特征方程 \(|\lambda E - A| = 0\).

1.2 核心性质全解

设 \(\lambda_1, \cdots, \lambda_n\) 是 \(A\) 的 \(n\) 个特征值，\(\alpha\) 是对应的特征向量.

1. 数值关系 ( 韦达定理推广 )

  • 和 : \(\displaystyle\sum_{i=1}^n \lambda_i = \displaystyle\sum_{i=1}^n a_{ii} = \text{tr}(A)\) ( 迹 ).
  • 积 : \(\displaystyle\prod_{i=1}^n \lambda_i = |A|\) ( 行列式 ).
  • 推论 : \(A\) 可逆 \(\iff\) 所有 \(\lambda_i \neq 0\).

2. 运算传导 ( 谱映射定理 )

若 \(A\alpha = \lambda\alpha\)，则对于多项式 \(f(x)\)，有 \(f(A)\alpha = f(\lambda)\alpha\).

  • 幂 : \(A^k \alpha = \lambda^k \alpha\).
  • 逆 : \(A^{-1} \alpha = \lambda^{-1} \alpha\) ( 若 \(A\) 可逆 ).
  • 伴随 : \(A^* \alpha = \frac{|A|}{\lambda} \alpha\) ( 若 \(\lambda \neq 0\) ).
  • 转置 : \(A^T\) 与 \(A\) 具有 相同 的特征值 ( 但特征向量通常不同 ).

3. 结构特征

  • 三角阵 : 上 ( 下 ) 三角矩阵或对角矩阵的特征值即为 主对角线元素.
  • 秩 1 矩阵 : 若 \(r(A)=1\)，则特征值为 \(\text{tr}(A), 0, \cdots, 0\).

1.3 代数重数与几何重数 ( 难点辨析 )

对于 \(A\) 的 \(k\) 重特征值 \(\lambda_0\) ( 代数重数 \(= k\) ) :

  • 几何重数 : 属于 \(\lambda_0\) 的线性无关特征向量的个数 \(m = n - r(\lambda_0 E - A)\).
  • 核心不等式 : \(1 \le m \le k\).
    • 当 \(m=k\) 时，该特征值对应的特征子空间“够维数”，是矩阵可对角化的关键条件.
    • 不同 特征值对应的特征向量 线性无关.

易错点 特征向量 \(\alpha\) 必须是非零向量，但特征值 \(\lambda\) 可以为 0. 若 \(\lambda=0\) 是 \(A\) 的特征值 \(\iff |A|=0 \iff A\) 不可逆.

二、相似矩阵与对角化

2.1 相似矩阵全解析

若存在 可逆 矩阵 \(P\)，使得 \(P^{-1}AP = B\)，则称 \(A\) 与 \(B\) 相似，记为 \(A \sim B\).

1. 相似的不变量 ( 必要条件 )

若 \(A \sim B\)，则它们共享以下核心属性 ( 简称“四同” ) :

属性 符号表示 备注
特征多项式 \(|\lambda E - A| = |\lambda E - B|\) 最强必要条件，推导出 \(\lambda\) 相同
行列式 \(|A| = |B|\) \(\prod \lambda_i\)
迹 \(\text{tr}(A) = \text{tr}(B)\) \(\sum \lambda_i\)
秩 \(r(A) = r(B)\) 相似变换不改变秩

警惕误区 上述条件均为 必要 条件. 即便“四同”也不一定相似 ( 例如单位阵 \(E\) 与二阶若尔当块 \(J_2\) 不相似，虽特征值相同 ).

2.2 矩阵可对角化的判定

矩阵 \(A\) 能对角化 ( 即 \(A \sim \Lambda\) ) \(\iff\) \(A\) 有 \(n\) 个线性无关的特征向量.

判定逻辑流程图

  1. 第一步 : 看特征值
    • 若 \(A\) 有 \(n\) 个 互不相等 的特征值 \(\implies\) 必可对角化.
    • 若有重根 \(\implies\) 进入第二步.
  2. 第二步 : 算几何重数 ( 核心判据 )
    • 对于每个 \(k_i\) 重特征值 \(\lambda_i\)，计算 \(n - r(\lambda_i E - A)\).
    • 若 \(n - r(\lambda_i E - A) = k_i\) 对所有重根成立 \(\implies\) 可对角化.
    • 若任一重根不满足 \(\implies\) 不可对角化.

2.3 实对称矩阵 ( 完美的矩阵 )

实对称矩阵 \(A\) ( 即 \(A^T=A\) ) 是考研线性代数中的“天选之子”，拥有其他矩阵无法比拟的完美性质.

1. 四大核心性质 ( 必背 )

性质 描述 考研考法
实根性 特征值 \(\lambda\) 全为 实数 排除复数根
必可对角化 一定 相似于对角阵 无需验证几何重数
正交性 不同 特征值对应的特征向量 天然正交 利用正交性求参数
正交对角化 存在 正交矩阵 \(Q\) 使得 \(Q^{-1}AQ = \Lambda\) 二次型标准化核心

2. 谱分解定理 ( 进阶视角 )

若 \(A\) 是实对称矩阵，且 \(A = Q \Lambda Q^T\)，其中 \(Q=(\xi_1, \cdots, \xi_n)\) 为标准正交特征向量组，则 \(A\) 可分解为 : \[ A = \lambda_1 \xi_1 \xi_1^T + \lambda_2 \xi_2 \xi_2^T + \cdots + \lambda_n \xi_n \xi_n^T \]

  • 物理意义 : 任意实对称矩阵都可以分解为 \(n\) 个秩 1 矩阵 ( 投影算子 ) 的线性组合.
  • 应用 : 既然 \(A\) 可分解，那么 \(A^k = \sum \lambda_i^k \xi_i \xi_i^T\).

3. 正交对角化解题 S.O.P ( 标准作业程序 )

要求正交矩阵 \(Q\) 使得 \(Q^TAQ = \Lambda\) :

  1. 求特征值 : 解 \(|\lambda E - A|=0\).
  2. 求特征向量 : 解 \((\lambda E - A)x=0\).
  3. 正交化 ( 关键 ) :
    • 对于 单根 : 特征向量只需 单位化.
    • 对于 \(k\) 重根 : 得到的 \(k\) 个线性无关特征向量必须先进行 施密特正交化，再 单位化.
  4. 构造 \(Q\) : 将处理后的标准正交向量拼成 \(Q = (\eta_1, \eta_2, \cdots, \eta_n)\).

瑞利商 ( Rayleigh Quotient ) 对于实对称矩阵 \(A\)，其二次型 \(f(x) = x^TAx\) 的取值范围受特征值控制 : \[ \lambda_{\min} x^Tx \le x^TAx \le \lambda_{\max} x^Tx \] 即 : \(\lambda_{\min} \le \frac{x^TAx}{x^Tx} \le \lambda_{\max}\). 这是求解二次型最值问题的理论基础.

三、考研数学一 高难例题精讲

题型一 : 含参矩阵的相似对角化 ( 重难点 )

例题 1 ( 2024 重点预测 ) 设矩阵 \(A = \begin{pmatrix} 1 & 0 & 1 \\ 0 & 2 & 0 \\ 1 & 0 & a \end{pmatrix}\). (1) 求 \(A\) 的特征值; (2) 问 \(a\) 为何值时，矩阵 \(A\) 可相似对角化?

解 : (1) 计算特征多项式 : \[ |\lambda E - A| = \begin{vmatrix} \lambda-1 & 0 & -1 \\ 0 & \lambda-2 & 0 \\ -1 & 0 & \lambda-a \end{vmatrix} \] 按第二行展开 : \[ = (\lambda-2) [(\lambda-1)(\lambda-a) - 1] = (\lambda-2) [\lambda^2 - (a+1)\lambda + a - 1] \] 令 \(|\lambda E - A| = 0\)，得 \(\lambda_1 = 2\). 解方程 \(\lambda^2 - (a+1)\lambda + a - 1 = 0\)，判别式 \(\Delta = (a+1)^2 - 4(a-1) = a^2 - 2a + 5 = (a-1)^2 + 4 > 0\). 故方程必有两个不等的实根 \(\lambda_2, \lambda_3\). 特征值为 \(\lambda_1=2\)，\(\lambda_{2,3} = \frac{a+1 \pm \sqrt{a^2-2a+5}}{2}\).

(2) 分类讨论 : 要使 \(A\) 可对角化，需讨论是否有重根.

  • 情形 1 : \(\lambda_2 \neq \lambda_3\). 由 \(\Delta > 0\) 恒成立，故 \(\lambda_2, \lambda_3\) 永不相等.

  • 情形 2 : \(\lambda_1\) 与 \(\lambda_2\) 或 \(\lambda_3\) 重合. 令 \(\lambda = 2\) 代入方程 \(\lambda^2 - (a+1)\lambda + a - 1 = 0\) : \(4 - 2(a+1) + a - 1 = 0 \implies 4 - 2a - 2 + a - 1 = 0 \implies 1 - a = 0 \implies a = 1\).

    检验 \(a=1\) 时的情况 : 当 \(a=1\) 时，\(\lambda_1=2\)，方程变为 \(\lambda^2 - 2\lambda = 0 \implies \lambda(\lambda-2)=0\). 特征值为 \(2, 2, 0\). ( 2 为二重根 ). 此时需验证 \(\lambda=2\) 的几何重数是否等于 2. \[ 2E - A = \begin{pmatrix} 1 & 0 & -1 \\ 0 & 0 & 0 \\ -1 & 0 & 1 \end{pmatrix} \sim \begin{pmatrix} 1 & 0 & -1 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix} \] 秩 \(r(2E-A) = 1\). 几何重数 \(m = 3 - 1 = 2\). 因为 \(m = k = 2\)，所以 \(a=1\) 时 \(A\) 可对角化.

结论 : 对任意实数 \(a\)，矩阵 \(A\) 均可对角化. ( 注意 : 此题 \(A\) 是实对称矩阵，其实可以直接由实对称矩阵性质秒杀，但如果是普通矩阵则必须按上述步骤讨论 ).

题型二 : 实对称矩阵的反求与正交对角化

例题 2 ( 综合逆向题 ) 设 3 阶实对称矩阵 \(A\) 的特征值 \(\lambda_1 = 1, \lambda_2 = 2, \lambda_3 = -2\). 对应于 \(\lambda_1\) 的特征向量 \(\alpha_1 = (1, 1, 1)^T\). 对应于 \(\lambda_2\) 的特征向量 \(\alpha_2 = (-2, 1, 1)^T\). (1) 求 \(\lambda_3\) 对应的特征向量 \(\alpha_3\); (2) 求矩阵 \(A\).

解 : (1) 利用 实对称矩阵不同特征值对应的特征向量正交. 设 \(\alpha_3 = (x_1, x_2, x_3)^T\). \[ \begin{cases} \alpha_3 \cdot \alpha_1 = x_1 + x_2 + x_3 = 0 \\[.5em] \alpha_3 \cdot \alpha_2 = -2x_1 + x_2 + x_3 = 0 \end{cases} \] 解得基础解系 \(\alpha_3 = (0, -1, 1)^T\) ( 或其倍数 ).

(2) 利用谱分解公式 \(A = P \Lambda P^{-1}\). 由于 \(A\) 是实对称矩阵，可构造正交矩阵 \(Q\) 使 \(A = Q \Lambda Q^T\). 将 \(\alpha_1, \alpha_2, \alpha_3\) 单位化 : \(\xi_1 = \frac{1}{\sqrt{3}}(1,1,1)^T, \xi_2 = \frac{1}{\sqrt{6}}(-2,1,1)^T, \xi_3 = \frac{1}{\sqrt{2}}(0,-1,1)^T\). \(Q = (\xi_1, \xi_2, \xi_3)\). \[ A = Q \begin{pmatrix} 1 & 0 & 0 \\ 0 & 2 & 0 \\ 0 & 0 & -2 \end{pmatrix} Q^T \] ( 具体矩阵乘法计算略，考研中通常只需列出式子或计算简单情形 ). 另解 ( 待定系数法 ) : 设 \(A(x_1,x_2,x_3)^T = \lambda(x_1,x_2,x_3)^T\)，联立方程组求解 \(a_{ij}\).

题型三 : 秩 1 矩阵的特征值 ( 秒杀技巧 )

结论 若 \(A = \alpha \beta^T\) ( 列向量 \(\times\) 行向量 )，则 \(A\) 是秩为 1 的矩阵. 其特征值为 : \(\lambda_1 = \text{tr}(A) = \beta^T \alpha\) ( 内积 )，\(\lambda_2 = \cdots = \lambda_n = 0\).

实战演练 : 设 \(A = \begin{pmatrix} 2 & 2 & 2 \\ 2 & 2 & 2 \\ 2 & 2 & 2 \end{pmatrix}\). 求 \(A^{100}\).

  1. 观察 : \(A = \begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix} \begin{pmatrix} 2 & 2 & 2 \end{pmatrix} = \alpha \beta^T\).
  2. 特征值 : \(\lambda_1 = \text{tr}(A) = 6\)，\(\lambda_2 = \lambda_3 = 0\).
  3. 特征向量 :
    • \(\lambda_1=6\) : \(\alpha_1 = (1, 1, 1)^T\). ( \(A\alpha = \alpha\beta^T\alpha = \alpha(\beta^T\alpha) = 6\alpha\) ).
    • \(\lambda=0\) : 解 \(x_1+x_2+x_3=0\).
  4. 计算 \(A^{100}\) : 由于 \(A\) 可对角化 ( 实对称 )，\(P^{-1}AP = \text{diag}(6, 0, 0)\). \(A^{100} = P \text{diag}(6^{100}, 0, 0) P^{-1}\). 或者直接利用秩 1 性质 : \(A^2 = (\alpha \beta^T)(\alpha \beta^T) = \alpha (\beta^T \alpha) \beta^T = 6 A\). 递推得 \(A^{100} = 6^{99} A\).

四、总结与备考建议

  1. 计算准确性 : 特征值计算往往涉及三次方程求解，务必熟练掌握“试根法” ( 试 \(\pm 1, \pm 2\) ).
  2. 概念清晰 : 深刻理解“相似”与“合同”的区别 ( 相似看特征值，合同看正负惯性指数 ).
  3. 几何直观 : 实对称矩阵对应着空间中的伸缩变换，且主轴方向互相垂直，这是理解二次型标准化的关键.
