一、多元函数的极限与连续

1.1 二元函数的极限

定义 ( 二重极限 )

设 \(f(x,y)\) 在点 \(P_0(x_0,y_0)\) 的某去心邻域内有定义. 若对 \(\forall \varepsilon > 0\)，\(\exists \delta > 0\)，使得当 \(0 < \sqrt{(x-x_0)^2+(y-y_0)^2} < \delta\) 时，均有 \[|f(x,y)-A|<\varepsilon\] 则称 \(A\) 为 \(f(x,y)\) 当 \((x,y)\to(x_0,y_0)\) 时的极限，记为 \[\lim_{(x,y)\to(x_0,y_0)} f(x,y) = A\]

重要警示 二重极限与累次极限的区别

  • 二重极限 \((x,y)\) 以 任意方式 趋向 \((x_0,y_0)\) 时，\(f(x,y)\) 趋向同一值.
  • 累次极限 先对 \(x\) 取极限，再对 \(y\) 取极限 ( 或反之 ).

三者关系 \[\text{二重极限存在} \;\not\Rightarrow\; \text{两个累次极限都存在}\] \[\text{两个累次极限都存在且相等} \;\not\Rightarrow\; \text{二重极限存在}\] \[\text{二重极限存在，且某一累次极限存在} \;\Rightarrow\; \text{它们相等}\]

证明二重极限不存在的方法 见下方“三大杀手锏”.

判定二重极限不存在的“三大杀手锏”

只要能找到一种方式让极限值不唯一，即可宣判极限不存在 ( 一票否决权 ).

  1. 路径判别法 ( 最常用 )
    • 选取不同的趋近路径 ( 如 \(y=kx, y=x^2, y=x^3\) 等 ).
    • 若沿两条不同路径趋近时极限值不相等，则二重极限不存在.
    • 技巧 观察分子分母次数. 若分子分母齐次 ( 如 \(\frac{xy}{x^2+y^2}\) ) ，试 \(y=kx\); 若次数不齐 ( 如 \(\frac{xy}{x^2+y^4}\) ) ，试 \(x=ky^2\).
  2. 累次极限不相等
    • 若 \(\displaystyle \lim_{x\to x_0}\left(\lim_{y\to y_0}f(x,y)\right) \neq \lim_{y\to y_0}\left(\lim_{x\to x_0}f(x,y)\right)\)，则二重极限不存在.
    • 注 即使累次极限相等，二重极限也未必存在.
  3. 极坐标变换法
    • 令 \(x=x_0+r\cos\theta, \; y=y_0+r\sin\theta\).
    • 当 \(r\to 0\) 时，若极限表达式结果 依赖于角度 \(\theta\) ( 如结果为 \(\cos^2\theta\) ) ，则极限不存在.

示例 1 ( 路径法证极限不存在 )

证明 \(\displaystyle\lim_{(x,y)\to(0,0)}\frac{xy}{x^2+y^2}\) 不存在.

解

沿路径 \(y=kx\) ( \(k\) 为常数，\(x\to 0\) )
\[\lim_{x\to 0}\frac{x\cdot kx}{x^2+k^2x^2} = \frac{k}{1+k^2}\]

不同的 \(k\) 给出不同的值 ( 如 \(k=0\) 得 \(0\)，\(k=1\) 得 \(\frac{1}{2}\) ) ，故极限不存在.

示例 2 ( 夹逼定理 )

求 \(\displaystyle\lim_{(x,y)\to(0,0)}(x^2+y^2)\sin\frac{1}{xy}\).

解

由于 \(\left|\sin\frac{1}{xy}\right|\le 1\)，故 \[0 \le \left|(x^2+y^2)\sin\frac{1}{xy}\right| \le x^2+y^2\]

而 \(\displaystyle\lim_{(x,y)\to(0,0)}(x^2+y^2)=0\)，由夹逼定理知原式 \(=0\).

1.2 连续性

定义

若 \(\displaystyle\lim_{(x,y)\to(x_0,y_0)}f(x,y)=f(x_0,y_0)\)，则称 \(f\) 在点 \((x_0,y_0)\) 连续.

有界闭区域上连续函数的性质 ( 与一元函数类比 )

  1. 最值定理 有界闭区域 \(D\) 上的连续函数必有最大值和最小值.
  2. 介值定理 \(f\) 在 \(D\) 上可取到最大值与最小值之间的任意值.

偏导存在、连续、可微的关系辨析

  1. 强弱链条
    • \(\text{偏导连续} \Rightarrow \text{可微} \Rightarrow \text{连续} \Rightarrow \text{极限存在}\)
    • \(\text{可微} \Rightarrow \text{偏导存在}\)
  2. 常见反例 ( 陷阱 )
    • 偏导存在 \(\nRightarrow\) 连续 如 \(f(x,y)=\frac{xy}{x^2+y^2}\) ( 补充定义0 ).
    • 连续 \(\nRightarrow\) 偏导存在 如 \(z=\sqrt{x^2+y^2}\) ( 原点锥顶 ).

记忆口诀

  • 可微 是 “高富帅” ( 必连续、必有偏导 ).
  • 偏导连续 是 “王炸” ( 直接推出可微 ).
  • 偏导存在 是 “路人甲” ( 甚至推不出连续 ).

二、偏导数

2.1 偏导数的定义与计算

定义

\[f_x(x_0,y_0) = \lim_{\Delta x\to 0}\frac{f(x_0+\Delta x,y_0)-f(x_0,y_0)}{\Delta x}\]

\[f_y(x_0,y_0) = \lim_{\Delta y\to 0}\frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}\]

计算方法
* 对 \(x\) 求偏导时，将 \(y\) 看作常数，对 \(x\) 用一元函数求导法则; 对 \(y\) 亦然.

考研数学一 高难例题 ( 偏导数定义的灵活运用 )

例题 1 分段函数偏导数与可微性判定 设 \(f(x,y) = \begin{cases} (x^2+y^2)\sin\frac{1}{x^2+y^2}, & x^2+y^2 \ne 0 \\[.5em] 0, & x^2+y^2 = 0 \end{cases}\).

  1. 求 \(f_x(0,0)\) 与 \(f_y(0,0)\).
  2. 讨论 \(f_x(x,y)\) 在 \((0,0)\) 处的连续性.
  3. 讨论 \(f(x,y)\) 在 \((0,0)\) 处的可微性.

解

(1) 用定义求 \(f_x(0,0)\) \[ f_x(0,0) = \lim_{x \to 0} \frac{f(x,0) - f(0,0)}{x} = \lim_{x \to 0} \frac{x^2\sin\frac{1}{x^2} - 0}{x} = \lim_{x \to 0} x\sin\frac{1}{x^2} = 0. \] 同理，由对称性知 \(f_y(0,0) = 0\).

(2) 讨论 \(f_x\) 的连续性 当 \((x,y) \ne (0,0)\) 时，直接求导 \[ f_x(x,y) = 2x\sin\frac{1}{x^2+y^2} + (x^2+y^2)\cos\frac{1}{x^2+y^2} \cdot (-\frac{2x}{(x^2+y^2)^2}) \] \[ = 2x\sin\frac{1}{x^2+y^2} - \frac{2x}{x^2+y^2}\cos\frac{1}{x^2+y^2}. \] 考察 \((x,y) \to (0,0)\) 时 \(f_x(x,y)\) 的极限

令 \(y=0, x \to 0\)，则 \(f_x(x,0) = 2x\sin\frac{1}{x^2} - \frac{2}{x}\cos\frac{1}{x^2}\).

第一项趋于 0，但第二项 \(\frac{2}{x}\cos\frac{1}{x^2}\) 振荡发散，极限不存在.

因此 \(\lim_{(x,y)\to(0,0)} f_x(x,y)\) 不存在，故 \(f_x\) 在 \((0,0)\) 处不连续.

(3) 讨论可微性 ( 重要考点 ) 虽然偏导数不连续，但 可微性定义 是 \[ \lim_{\rho \to 0} \frac{\Delta z - [A\Delta x + B\Delta y]}{\rho} = 0 \] 其中 \(A=f_x(0,0)=0, B=f_y(0,0)=0\). \[ \lim_{\rho \to 0} \frac{f(x,y) - f(0,0) - 0}{\sqrt{x^2+y^2}} = \lim_{\rho \to 0} \frac{(x^2+y^2)\sin\frac{1}{x^2+y^2}}{\sqrt{x^2+y^2}} \] 令 \(\rho = \sqrt{x^2+y^2}\)，则极限为 \(\lim_{\rho \to 0} \rho \sin\frac{1}{\rho^2} = 0\). 结论 \(f(x,y)\) 在 \((0,0)\) 处 可微.

点拨 此题完美展示了 偏导数存在、偏导数连续 与 可微 之间的微妙关系.

  • 偏导数连续 \(\implies\) 可微 ( 充分条件 ).
  • 偏导数存在 \(\nRightarrow\) 可微.
  • 可微 \(\nRightarrow\) 偏导数连续 ( 本题即反例 ).

考研数学一 高难例题 ( 抽象函数偏导数 )

例题 2 链式法则与二阶偏导 设 \(z = f(xy, \frac{x}{y}) + g(\frac{y}{x})\)，其中 \(f, g\) 具有二阶连续偏导数，求 \(\frac{\partial^2 z}{\partial x \partial y}\).

解 令 \(u = xy, v = \frac{x}{y}, w = \frac{y}{x}\). 则 \(z = f(u,v) + g(w)\).

Step 1 求一阶偏导 \(z_x\) \[ \frac{\partial z}{\partial x} = f_1' \cdot (u)_x' + f_2' \cdot (v)_x' + g' \cdot (w)_x' \] \[ = y f_1' + \frac{1}{y} f_2' - \frac{y}{x^2} g' \] ( 注 \(f_1'\) 代表 \(f\) 对第一个中间变量求偏导，即 \(f_u\) )

Step 2 求混合偏导 \(z_{xy}\) 对 \(z_x\) 关于 \(y\) 求导 \[ \frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial y} (y f_1') + \frac{\partial}{\partial y} (\frac{1}{y} f_2') - \frac{\partial}{\partial y} (\frac{y}{x^2} g') \] 逐项计算

  1. \(\frac{\partial}{\partial y} (y f_1') = 1 \cdot f_1' + y [f_{11}'' \cdot x + f_{12}'' \cdot (-\frac{x}{y^2})]\) \(= f_1' + xy f_{11}'' - \frac{x}{y} f_{12}''\).
  2. \(\frac{\partial}{\partial y} (\frac{1}{y} f_2') = -\frac{1}{y^2} f_2' + \frac{1}{y} [f_{21}'' \cdot x + f_{22}'' \cdot (-\frac{x}{y^2})]\) \(= -\frac{1}{y^2} f_2' + \frac{x}{y} f_{21}'' - \frac{x}{y^3} f_{22}''\).
  3. \(-\frac{\partial}{\partial y} (\frac{y}{x^2} g') = - [ \frac{1}{x^2} g' + \frac{y}{x^2} g'' \cdot \frac{1}{x} ]\) \(= -\frac{1}{x^2} g' - \frac{y}{x^3} g''\).

整理结果 ( 注意 \(f_{12}'' = f_{21}''\) ) \[ \frac{\partial^2 z}{\partial x \partial y} = f_1' - \frac{1}{y^2} f_2' - \frac{1}{x^2} g' + xy f_{11}'' - \frac{x}{y^3} f_{22}'' - \frac{y}{x^3} g''. \]

分段函数在分界点处的偏导数

对于分界点处，必须用定义计算偏导数，不能直接对表达式求导!

示例 设 \(f(x,y)=\begin{cases}\dfrac{xy}{\sqrt{x^2+y^2}}, & (x,y)\neq(0,0)\\0, & (x,y)=(0,0)\end{cases}\)

在原点处
\[f_x(0,0)=\lim_{\Delta x\to 0}\frac{f(\Delta x,0)-f(0,0)}{\Delta x}=\lim_{\Delta x\to 0}\frac{0}{\Delta x}=0\]

同理 \(f_y(0,0)=0\). 但 \(f\) 在 \((0,0)\) 处不连续 ( 不可微 )!

2.2 高阶偏导数

混合偏导数相等定理

若 \(f_{xy}\) 与 \(f_{yx}\) 在点 \((x_0,y_0)\) 处均 连续，则 \[f_{xy}(x_0,y_0) = f_{yx}(x_0,y_0)\]

注意 不连续时混合偏导数可能不相等!

示例 3 ( 高阶偏导计算 )

设 \(z=x^y\) ( \(x>0\) ) ，求 \(z_{xy}\) 与 \(z_{yx}\).

解

\[z_x = yx^{y-1}, \quad z_{xy} = \frac{\partial}{\partial y}(yx^{y-1}) = x^{y-1}+y\cdot x^{y-1}\ln x = x^{y-1}(1+y\ln x)\]

\[z_y = x^y\ln x, \quad z_{yx} = \frac{\partial}{\partial x}(x^y\ln x) = yx^{y-1}\ln x + x^y\cdot\frac{1}{x} = x^{y-1}(y\ln x+1)\]

两者相等，验证了混合偏导数在连续条件下的对称性.

三、全微分

3.1 全微分的定义与存在条件

定义

设 \(z=f(x,y)\)，若全增量 \[\Delta z = f(x+\Delta x, y+\Delta y)-f(x,y)\] 可以表示为 \[\Delta z = A\Delta x + B\Delta y + o(\rho), \quad \rho=\sqrt{(\Delta x)^2+(\Delta y)^2}\] 则称 \(f\) 在该点 可微，\(dz = A\,dx + B\,dy\) 称为 全微分.

若 \(f\) 可微，则 \(A=f_x\)，\(B=f_y\)，即 \[dz = f_x\,dx + f_y\,dy\]

可微的充分条件与必要条件

  • 必要条件 \(f\) 在点 \((x_0,y_0)\) 可微 \(\Rightarrow\) \(f_x,\,f_y\) 在该点存在，且 \(dz=f_x\,dx+f_y\,dy\).
  • 充分条件 \(f_x,\,f_y\) 在点 \((x_0,y_0)\) 处 连续 \(\Rightarrow\) \(f\) 在该点可微.

可微的判断步骤 ( 分段函数 )

  1. 写出全增量 \(\Delta z = f(x_0+\Delta x,y_0+\Delta y)-f(x_0,y_0)\);
  2. 计算 \(f_x(x_0,y_0)\) 和 \(f_y(x_0,y_0)\) ( 用定义 );
  3. 验证 \(\lim_{\rho\to 0}\dfrac{\Delta z - f_x\Delta x - f_y\Delta y}{\rho}\) 是否为 \(0\)，若为 \(0\) 则可微.

示例 4 ( 验证可微性 )

验证 \(f(x,y)=\sqrt{|xy|}\) 在 \((0,0)\) 处的可微性.

解

\[f_x(0,0)=\lim_{\Delta x\to 0}\frac{\sqrt{|\Delta x\cdot 0|}-0}{\Delta x}=0, \quad f_y(0,0)=0\]

验证
\[\frac{\Delta z - 0\cdot\Delta x - 0\cdot\Delta y}{\rho} = \frac{\sqrt{|\Delta x\cdot\Delta y|}}{\sqrt{(\Delta x)^2+(\Delta y)^2}}\]

令 \(\Delta x=\Delta y=t\to 0\)
\[\frac{\sqrt{t^2}}{\sqrt{2t^2}}=\frac{|t|}{|t|\sqrt{2}}=\frac{1}{\sqrt{2}}\neq 0\]

故 \(f\) 在 \((0,0)\) 处 不可微. ( 但偏导数存在! )

3.2 全微分的应用 近似计算

近似公式

\[f(x_0+\Delta x,\,y_0+\Delta y)\approx f(x_0,y_0)+f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y\]

应用场景 计算 \(f(1.04,\,1.98)\) 等近似值时，取 \((x_0,y_0)\) 为附近容易计算的整点.

四、多元复合函数的求导法则

4.1 链式法则

链式法则 ( 全导数 )

若 \(z=f(u,v)\)，\(u=\varphi(t)\)，\(v=\psi(t)\)，则 \[\frac{dz}{dt}=\frac{\partial z}{\partial u}\frac{du}{dt}+\frac{\partial z}{\partial v}\frac{dv}{dt}\]

偏导数形式 若 \(z=f(u,v)\)，\(u=\varphi(x,y)\)，\(v=\psi(x,y)\)，则 \[\frac{\partial z}{\partial x}=\frac{\partial z}{\partial u}\frac{\partial u}{\partial x}+\frac{\partial z}{\partial v}\frac{\partial v}{\partial x}\] \[\frac{\partial z}{\partial y}=\frac{\partial z}{\partial u}\frac{\partial u}{\partial y}+\frac{\partial z}{\partial v}\frac{\partial v}{\partial y}\]

符号陷阱 \(\partial z/\partial x\) 与 \(f_1'\) 的区别

设 \(z = f(x, x+y)\)，令 \(u=x,\,v=x+y\)，则

\[\frac{\partial z}{\partial x} = f_1'\cdot 1 + f_2'\cdot 1 = f_1'+ f_2'\]

而 \(f_x\) 若直接对 \(f(x,x+y)\) 中第一个 \(x\) 求导则仅为 \(f_1'\)，含义不同.

规范做法 引入中间变量，用 \(f_1',f_2'\) 表示对第一、第二个中间变量的偏导.

示例 5 ( 链式法则与高阶偏导 )

设 \(z = f(2x-y, x^2+y^2)\)，\(f\) 有二阶连续偏导，求 \(\dfrac{\partial^2 z}{\partial x\partial y}\).

解

令 \(u=2x-y,\;v=x^2+y^2\)，则
\[\frac{\partial z}{\partial x}=2f_1'+2xf_2'\]

再对 \(y\) 求偏导 ( 注意 \(u,v\) 均含 \(y\) )
\[\frac{\partial^2 z}{\partial y\partial x}=2\left(f_{11}''\cdot(-1)+f_{12}''\cdot 2y\right)+2x\left(f_{21}''\cdot(-1)+f_{22}''\cdot 2y\right)\] \[=-2f_{11}''+4yf_{12}''-2xf_{21}''+4xyf_{22}''\]

由 \(f_{12}''=f_{21}''\) ( 连续条件 )
\[\frac{\partial^2 z}{\partial x\partial y}=-2f_{11}''+(4y-2x)f_{12}''+4xyf_{22}''\]

4.2 全微分形式不变性

全微分形式不变性

无论 \(u, v\) 是自变量还是中间变量，全微分形式保持不变
\[dz = \frac{\partial z}{\partial u}du + \frac{\partial z}{\partial v}dv\]

应用 利用全微分形式不变性，可将链式法则的计算简化为对微分的直接操作.

五、隐函数求导

5.1 一个方程确定的隐函数

隐函数存在定理 ( 一元 )

设 \(F(x,y)\) 在点 \((x_0,y_0)\) 的邻域内有连续偏导，\(F(x_0,y_0)=0\)，且 \(F_y(x_0,y_0)\neq 0\)，则方程 \(F(x,y)=0\) 在 \((x_0,y_0)\) 附近唯一确定一个连续可微函数 \(y=f(x)\)，且

\[\frac{dy}{dx}=-\frac{F_x}{F_y}\]

隐函数存在定理 ( 二元 )

设 \(F(x,y,z)\) 满足类似条件，\(F_z(x_0,y_0,z_0)\neq 0\)，则 \(F(x,y,z)=0\) 确定 \(z=z(x,y)\)，且

\[\frac{\partial z}{\partial x}=-\frac{F_x}{F_z}, \qquad \frac{\partial z}{\partial y}=-\frac{F_y}{F_z}\]

示例 6 ( 隐函数高阶导数 )

设 \(y=y(x)\) 由 \(e^y + xy - e = 0\) 确定，求 \(y''(0)\).

解

设 \(F=e^y+xy-e\)，\(F_x=y\)，\(F_y=e^y+x\).

\[y'=-\frac{F_x}{F_y}=-\frac{y}{e^y+x}\]

在 \(x=0\) 代入方程 \(e^y-e=0\Rightarrow y(0)=1\)，故 \(y'(0)=-\dfrac{1}{e}\).

对 \(y'=-\dfrac{y}{e^y+x}\) 两边对 \(x\) 再求导

\[y''=-\frac{y'(e^y+x)-y(y'e^y+1)}{(e^y+x)^2}\]

代入 \(x=0,\;y=1,\;y'=-\frac{1}{e}\)

\[y''(0)=-\frac{-\frac{1}{e}\cdot e - 1\cdot(-\frac{1}{e}\cdot e+1)}{e^2}=-\frac{-1-(-1+1)}{e^2}=\frac{1}{e^2}\]

5.2 方程组确定的隐函数

方程组情形

设方程组 \(\begin{cases}F(x,y,u,v)=0\\[.5em]G(x,y,u,v)=0\end{cases}\) 在满足一定条件时，确定 \(u=u(x,y),\;v=v(x,y)\).

雅可比行列式 \(J = \dfrac{\partial(F,G)}{\partial(u,v)}=\begin{vmatrix}F_u & F_v\\G_u & G_v\end{vmatrix}\neq 0\) 是存在条件.

求导时，方程组两边对 \(x\) ( 或 \(y\) ) 求偏导，得关于 \(u_x,v_x\) 的线性方程组，用克莱姆法则求解.

六、多元函数的几何应用

6.1 空间曲线的切线与法平面

参数方程形式

曲线 \(\Gamma:\begin{cases}x=x(t)\\[.5em]y=y(t)\\[.5em]z=z(t)\end{cases}\) 在 \(t=t_0\) 对应点 \(M_0\) 处

  • 切向量 \(\boldsymbol{\tau}=(x'(t_0),\;y'(t_0),\;z'(t_0))\)
  • 切线方程 \(\dfrac{x-x_0}{x'(t_0)}=\dfrac{y-y_0}{y'(t_0)}=\dfrac{z-z_0}{z'(t_0)}\)
  • 法平面方程 \(x'(t_0)(x-x_0)+y'(t_0)(y-y_0)+z'(t_0)(z-z_0)=0\)

两曲面交线形式

曲线 \(\Gamma:\begin{cases}F(x,y,z)=0\\G(x,y,z)=0\end{cases}\) 在点 \(M_0(x_0,y_0,z_0)\) 处的切向量

\[\boldsymbol{\tau}=\nabla F\times\nabla G=\begin{vmatrix}\boldsymbol{i}&\boldsymbol{j}&\boldsymbol{k}\\F_x&F_y&F_z\\G_x&G_y&G_z\end{vmatrix}\Bigg|_{M_0}\]

6.2 曲面的切平面与法线

隐式方程 \(F(x,y,z)=0\)

在点 \(M_0(x_0,y_0,z_0)\) 处

  • 法向量 \(\boldsymbol{n}=(F_x,F_y,F_z)\big|_{M_0}\)
  • 切平面方程 \(F_x(x-x_0)+F_y(y-y_0)+F_z(z-z_0)=0\)
  • 法线方程 \(\dfrac{x-x_0}{F_x}=\dfrac{y-y_0}{F_y}=\dfrac{z-z_0}{F_z}\)

显式方程 \(z=f(x,y)\) 令 \(F=f(x,y)-z\)，则 \(F_x=f_x,\;F_y=f_y,\;F_z=-1\).

切平面 \(z-z_0=f_x(x-x_0)+f_y(y-y_0)\)

示例 7 ( 切平面方程 )

求曲面 \(z=x^2+y^2\) 在点 \((1,1,2)\) 处的切平面和法线方程.

解

令 \(F=x^2+y^2-z\)，则 \(F_x=2x,\;F_y=2y,\;F_z=-1\).

在 \((1,1,2)\) 处 \(F_x=2,\;F_y=2,\;F_z=-1\).

切平面 \(2(x-1)+2(y-1)-(z-2)=0 \Rightarrow 2x+2y-z-2=0\).

法线 \(\dfrac{x-1}{2}=\dfrac{y-1}{2}=\dfrac{z-2}{-1}\).

七、方向导数与梯度

7.1 方向导数

定义

函数 \(f(x,y)\) 在点 \(P_0\) 处沿方向 \(\boldsymbol{l}\) ( 单位向量 \(\boldsymbol{e}_l=(\cos\alpha,\cos\beta)\) ) 的方向导数

\[\frac{\partial f}{\partial \boldsymbol{l}}\bigg|_{P_0}=\lim_{t\to 0^+}\frac{f(x_0+t\cos\alpha,\,y_0+t\cos\beta)-f(x_0,y_0)}{t}\]

计算公式 ( \(f\) 可微时 )
\[\frac{\partial f}{\partial \boldsymbol{l}}=f_x\cos\alpha+f_y\cos\beta\]

三元推广 \(\dfrac{\partial f}{\partial \boldsymbol{l}}=f_x\cos\alpha+f_y\cos\beta+f_z\cos\gamma\)

方向导数与偏导数的关系

  • \(f_x\) 是沿 \(x\) 轴正方向的方向导数;
  • \(-f_x\) 是沿 \(x\) 轴 负 方向的方向导数;
  • 方向导数存在 不等价于 偏导数存在;
  • 偏导数存在且可微，方向导数在所有方向均存在.

7.2 梯度

梯度的定义与性质

\[\mathbf{grad}\,f = \nabla f = \left(f_x,\;f_y\right) \quad \text{ ( 三元为 } (f_x,f_y,f_z)\text{ ) }\]

方向导数与梯度的关系
\[\frac{\partial f}{\partial \boldsymbol{l}} = \nabla f \cdot \boldsymbol{e}_l = |\nabla f|\cos\theta\]

其中 \(\theta\) 为梯度方向与 \(\boldsymbol{l}\) 的夹角.

结论

  • 当 \(\boldsymbol{l}\) 与梯度方向相同时，方向导数取 最大值 \(|\nabla f|\);
  • 当 \(\boldsymbol{l}\) 与梯度方向相反时，方向导数取 最小值 \(-|\nabla f|\);
  • 沿等值线 ( 面 ) 方向，方向导数为 \(0\).

示例 8 ( 方向导数 )

求 \(f(x,y)=x^2+xy\) 在点 \((1,2)\) 处沿从 \((1,2)\) 到 \((3,3)\) 方向的方向导数.

解

方向向量 \(\overrightarrow{AB}=(2,1)\)，单位向量 \(\boldsymbol{e}_l=\left(\dfrac{2}{\sqrt{5}},\dfrac{1}{\sqrt{5}}\right)\).

\[f_x=2x+y, \quad f_y=x\]

在 \((1,2)\) \(f_x=4,\;f_y=1\).

\[\frac{\partial f}{\partial \boldsymbol{l}}=4\cdot\frac{2}{\sqrt{5}}+1\cdot\frac{1}{\sqrt{5}}=\frac{9}{\sqrt{5}}=\frac{9\sqrt{5}}{5}\]

八、多元函数的极值与最值

8.1 无条件极值

极值的必要条件

若 \(f\) 在点 \((x_0,y_0)\) 可偏导且取极值，则 \[f_x(x_0,y_0)=0, \quad f_y(x_0,y_0)=0\]

使偏导数为零的点称为 驻点 ( 稳定点 ).

注意 驻点不一定是极值点; 极值点 ( 若可偏导 ) 一定是驻点.

极值的充分条件 ( 二阶判别法 )

设 \((x_0,y_0)\) 为驻点，令 \[A=f_{xx}(x_0,y_0),\quad B=f_{xy}(x_0,y_0),\quad C=f_{yy}(x_0,y_0)\]

\(\Delta=AC-B^2\) \(A\) 结论
\(\Delta>0\) \(A<0\) 极大值
\(\Delta>0\) \(A>0\) 极小值
\(\Delta<0\) — 不是极值点 ( 鞍点 )
\(\Delta=0\) — 无法判断 ( 需用定义 )

考研数学一 高难概念题 ( 极值判定的陷阱与反例 )

例题 3 极值的必要与充分条件辨析 判断下列命题的真假，并说明理由或举反例.

  1. 若 \(f(x,y)\) 在 \((x_0,y_0)\) 处取极值，则 \((x_0,y_0)\) 必为驻点 ( 即偏导数为 0 ).
  2. 若 \(f(x,y)\) 在 \((x_0,y_0)\) 处取得极值，且偏导数存在，则 \(df|_{(x_0,y_0)} = 0\).
  3. 若驻点 \((x_0,y_0)\) 处的 Hesse 矩阵行列式 \(AC-B^2 = 0\)，则 \(f(x,y)\) 在该点必不取极值.

解

(1) 假 ( False ) 理由 极值点可能是 偏导数不存在的点 ( 尖点/脊线 ). 反例 \(f(x,y) = -\sqrt{x^2+y^2}\) ( 圆锥面 ). 在 \((0,0)\) 处取得极大值 0，但 \(f_x(0,0), f_y(0,0)\) 均不存在 ( 类似 \(y=|x|\) ).

(2) 真 ( True ) 理由 此为 费马引理 的推广. 若极值点处偏导数存在，则必为 0，故 \(df = f_x dx + f_y dy = 0 dx + 0 dy = 0\).

(3) 假 ( False ) 理由 当判别式 \(\Delta = 0\) 时，可能取极值，也可能不取极值，判定法失效，需回归定义. 反例 A ( 取极值 ) \(f(x,y) = x^4 + y^4\). 驻点 \((0,0)\)，二阶偏导 \(A=B=C=0 \implies \Delta=0\). 但显然 \((0,0)\) 是极小值点 ( \(f \ge 0\) ). 反例 B ( 不取极值 ) \(f(x,y) = x^3\). 驻点 \((0,0)\)，\(\Delta=0\). 但在 \(x\) 轴上 \(x^3\) 变号，故无极值.

深度点拨 考研常考“极值点 \(\nLeftrightarrow\) 驻点”，需时刻警惕 不可导点 和 边界点 ( 虽然无条件极值通常指内点 ).

示例 9 ( 求极值 )

求 \(f(x,y)=x^3+y^3-3xy\) 的极值.

解

令 \(\begin{cases}f_x=3x^2-3y=0\\[.5em]f_y=3y^2-3x=0\end{cases}\)，解得 \((0,0)\) 与 \((1,1)\).

\(f_{xx}=6x,\;f_{xy}=-3,\;f_{yy}=6y\).

在 \((0,0)\) \(A=0,B=-3,C=0\)，\(\Delta=AC-B^2=-9<0\)，\((0,0)\) 是鞍点.

在 \((1,1)\) \(A=6,B=-3,C=6\)，\(\Delta=36-9=27>0\)，\(A=6>0\)，故 \((1,1)\) 为极小值点，极小值 \(f(1,1)=-1\).

8.2 条件极值与拉格朗日乘数法

拉格朗日乘数法

求目标函数 \(f(x,y,z)\) 在约束 \(g(x,y,z)=0\) 下的极值

构造拉格朗日函数 \(L=f(x,y,z)+\lambda g(x,y,z)\)，解方程组
\[\begin{cases}L_x=f_x+\lambda g_x=0\\[.5em]L_y=f_y+\lambda g_y=0\\[.5em]L_z=f_z+\lambda g_z=0\\[.5em]g(x,y,z)=0\end{cases}\]

求出所有可能的极值点候选 ( 驻点 ) ，再比较函数值或结合实际背景确定最值.

多约束情形 约束 \(g_1=0,\;g_2=0\) 时，\(L=f+\lambda_1 g_1+\lambda_2 g_2\).

示例 10 ( 拉格朗日乘数法 )

求表面积为 \(a^2\) 的长方体的最大体积.

解

设长、宽、高分别为 \(x,y,z\) ( 均大于零 ) ，目标 \(V=xyz\)，约束 \(2(xy+yz+xz)=a^2\).

构造 \(L=xyz+\lambda(2xy+2yz+2xz-a^2)\).

\[\begin{cases}yz+2\lambda(y+z)=0\\[.5em]xz+2\lambda(x+z)=0\\[.5em]xy+2\lambda(x+y)=0\end{cases}\]

由对称性猜测 \(x=y=z\)，代入约束 \(6x^2=a^2\Rightarrow x=\dfrac{a}{\sqrt{6}}\).

最大体积 \(V=x^3=\dfrac{a^3}{6\sqrt{6}}\).

( 由实际意义知最大值存在，故此为最大值. )

示例 11 ( 有界闭区域上的最值 )

求 \(f(x,y)=x^2+2y^2-x\) 在圆盘 \(D=\{x^2+y^2\leq 1\}\) 上的最大值与最小值.

解

内部驻点 \(f_x=2x-1=0,\;f_y=4y=0 \Rightarrow \left(\frac{1}{2},0\right)\).

\(f\!\left(\frac{1}{2},0\right)=-\frac{1}{4}\).

边界上 ( 令 \(x=\cos\theta,\;y=\sin\theta\) )

\[h(\theta)=\cos^2\theta+2\sin^2\theta-\cos\theta=1+\sin^2\theta-\cos\theta\]

\[h'(\theta)=2\sin\theta\cos\theta+\sin\theta=\sin\theta(2\cos\theta+1)=0\]

\(\theta=0,\pi,\;\theta=\dfrac{2\pi}{3},\dfrac{4\pi}{3}\).

对应点 \((1,0),(-1,0),\left(-\frac{1}{2},\frac{\sqrt{3}}{2}\right),\left(-\frac{1}{2},-\frac{\sqrt{3}}{2}\right)\).

函数值 \(f(1,0)=0,\;f(-1,0)=2,\;f\!\left(-\frac{1}{2},\pm\frac{\sqrt{3}}{2}\right)=\frac{11}{4}\).

结论 最大值 \(\dfrac{11}{4}\)，最小值 \(-\dfrac{1}{4}\).

求多元函数最值 技巧与注意事项 ( 常考 )

目标 在指定区域 \(D\) 上找 \(f\) 的最大值/最小值

  1. 先判定 “最值是否必存在”
    • \(D\) 为 有界闭集 且 \(f\) 连续 \(\Rightarrow\) 最大最小值必存在 ( 最值定理 )
    • 若 \(D\) 非闭或无界，最值可能不存在，要考虑上确界/下确界或极限
  2. 列出 “候选点集合”
    • 内部点 \(\nabla f=0\) 的驻点 + 不可导点
    • 边界 把边界当作低维问题处理 ( 参数化 / 消元 / 拉格朗日 )
  3. 对候选点 “比较函数值” 得出最值

无条件极值 ( 内点 )

  • 一阶 求驻点 \(\nabla f=0\) ( 同时关注不可导点 )
  • 二阶 用 Hessian 判别
    • 二元 \(\Delta=f_{xx}f_{yy}-f_{xy}^2\)
    • \(\Delta>0\) 且 \(f_{xx}>0\) 极小; \(\Delta>0\) 且 \(f_{xx}<0\) 极大; \(\Delta<0\) 鞍点; \(\Delta=0\) 失效
  • \(\Delta=0\) 时的常用补救
    • 直接用定义 找两条路径使函数增减相反
    • 或做高阶展开/配方观察符号

有界闭区域 \(D\) 上的最值

  • 必做两步 内部驻点 + 边界
  • 边界处理常见三种
    • 参数化 如圆 \(x=\cos\theta,y=\sin\theta\)
    • 消元降维 如边界为 \(y=\phi(x)\)，代入变一元
    • 拉格朗日 边界是等式约束 \(g(x,y)=0\) 时，统一求边界驻点
  • 分段边界容易漏点
    • 多边形、分段曲线 每一段边界都要单独做，还要检查各段端点/交点

拉格朗日乘数法 什么时候用、怎么不漏解

  • 适用 等式约束 \(g(x,y,\dots)=0\)，且目标/约束可微
  • 解方程组时两类常见情况要分支
    • 某些偏导项为 0 导致的分支 ( 类似示例中的 \(z=0\) 或 \(\lambda=-1\) )
    • 对称性猜测要回代验证，避免把不满足约束的点当答案
  • 最后一步 比较候选点的 \(f\) 值 ( 只给驻点不算最值 )
  • 只求驻点就下结论，忘了边界
  • 只用二阶判别表，忽略 “\(\Delta=0\) 失效” 的情形
  • 做边界参数化时，忘记参数范围 ( 如 \(\theta\in[0,2\pi]\) )，导致漏点
  • 区域不是光滑边界 角点/交点往往给出最值
  • “最大值” 与 “极大值” 混淆 最值是全局比较，极值是局部性质

九、综合难题选讲 ( Selected Hard Problems )

难题 1 ( 抽象函数的偏导与全微分 )

设 \(f(x,y)\) 可微，\(f(0,0)=0\)，\(f'_x(0,0)=a\)，\(f'_y(0,0)=b\)，且满足 \[f(x+\Delta x, y+\Delta y)-f(x,y)=\Delta x\cdot f'_x(\xi,\eta)+\Delta y\cdot f'_y(\xi,\eta)\] 其中 \((\xi,\eta)\) 在 \((x,y)\) 与 \((x+\Delta x,y+\Delta y)\) 之间. 证明 \(df\big|_{(0,0)}=a\,dx+b\,dy\).

解

由可微性定义，\(f(x,y)\) 在 \((0,0)\) 处的全微分由偏导数决定

\[df\big|_{(0,0)} = f'_x(0,0)\,dx + f'_y(0,0)\,dy = a\,dx + b\,dy \quad \]

难题 2 ( 隐函数与偏导的综合 )

设 \(u=f(x,y,z)\) 由方程 \(F(x-az,\;y-bz)=0\) 隐式确定，其中 \(f,F\) 均有连续偏导，证明 \[a\frac{\partial u}{\partial x}+b\frac{\partial u}{\partial y}+\frac{\partial u}{\partial z}=0.\]

( 此题中 \(u=z\)，由隐函数定理 )

解

令 \(p=x-az,\;q=y-bz\)，则 \(F(p,q)=0\) 确定 \(z=z(x,y)\).

\[\frac{\partial z}{\partial x}=-\frac{F_p\cdot 1}{F_p\cdot(-a)+F_q\cdot(-b)}=\frac{F_p}{aF_p+bF_q}\]

\[\frac{\partial z}{\partial y}=\frac{F_q}{aF_p+bF_q}\]

于是
\[a\frac{\partial z}{\partial x}+b\frac{\partial z}{\partial y}=\frac{aF_p+bF_q}{aF_p+bF_q}=1\]

即 \(a\dfrac{\partial u}{\partial x}+b\dfrac{\partial u}{\partial y}-1=0\)，即所证等式中将 \(\dfrac{\partial u}{\partial z}=1\) ( \(u=z\) ) ，故等式成立.

难题 3 ( 拉格朗日乘数法 椭球上的距离极值 )

求椭球面 \(\dfrac{x^2}{4}+\dfrac{y^2}{9}+z^2=1\) 上到点 \((1,2,0)\) 距离最近的点.

解

目标函数 ( 距离平方 ) \(f=(x-1)^2+(y-2)^2+z^2\)，约束 \(g=\dfrac{x^2}{4}+\dfrac{y^2}{9}+z^2-1=0\).

构造 \(L=f+\lambda g\)，方程组

\[\begin{cases}2(x-1)+\lambda\cdot\dfrac{x}{2}=0\\[0.6em]2(y-2)+\lambda\cdot\dfrac{2y}{9}=0\\[0.6em]2z+2\lambda z=0\\[0.6em]\dfrac{x^2}{4}+\dfrac{y^2}{9}+z^2=1\end{cases}\]

由第三式 \(z=0\) 或 \(\lambda=-1\).

当 \(\lambda=-1\) 第一式 \(2(x-1)-\dfrac{x}{2}=0\Rightarrow x=\dfrac{4}{3}\); 第二式 \(2(y-2)-\dfrac{2y}{9}=0\Rightarrow y=\dfrac{18}{8}=\dfrac{9}{4}\).

代入约束 \(\dfrac{(4/3)^2}{4}+\dfrac{(9/4)^2}{9}=\dfrac{4}{9}+\dfrac{9}{16}=\dfrac{64+81}{144}=\dfrac{145}{144}>1\)，矛盾.

当 \(z=0\) 解方程 \(x=\dfrac{4}{4+\lambda},\;y=\dfrac{18}{9+2\lambda}\)，代入约束迭代数值求解.

( 实际考研中此类题目多要求给出方程组，结合极值存在性用连续函数有界闭集上必取最值论证即可. )

最近点在 \(z=0\) 截面的椭圆上，数值约为 \(\left(\dfrac{4}{3},\dfrac{9}{4},0\right)\) 附近.

难题 4 ( 二阶偏导数与极值判别的边界情形 )

设 \(f(x,y)=\left(y-x^2\right)\left(y-2x^2\right)\)，讨论 \((0,0)\) 是否为极值点.

解

\(f_x(0,0)=0,\;f_y(0,0)=0\)，\((0,0)\) 是驻点.

\(f_{xx}=2\cdot(-1)\cdot(y-2x^2)+2\cdot(-2)(y-x^2)+\ldots\)，在 \((0,0)\) 处

\(A=f_{xx}(0,0)=0,\;B=f_{xy}(0,0)=0,\;C=f_{yy}(0,0)=2\).

\(\Delta=AC-B^2=0\)，判别法失效，需用定义.

取不同路径

  • 沿 \(y=\frac{3}{2}x^2\) ( 两抛物线中间 ) \(f=\left(\frac{1}{2}x^2\right)\left(-\frac{1}{2}x^2\right)=-\frac{1}{4}x^4<0\).
  • 沿 \(y=0\) \(f=(-x^2)(-2x^2)=2x^4>0\).

\((0,0)\) 的任意邻域内 \(f\) 既取正值又取负值，故 \((0,0)\) 不是极值点.
