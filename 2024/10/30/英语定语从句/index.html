英语定语从句从入门到高级 ( Attributive Clause )

一、限制性定语从句 ( Defining Relative Clause )

在复合句中，修饰名词或代词的从句叫做 定语从句.

  • 被定语从句所修饰的词叫 先行词 ( Antecedent ).
  • 引导定语从句的词叫 关系词 ( Relative Words ).

基本结构示例

I fell in love with someone who is considerate.

  • 先行词 someone
  • 关系词 who ( 在从句中作主语，替代 someone )
  • 从句 who is considerate ( 修饰 someone )

1. 关系词的选择步骤

  1. 划出先行词.
  2. 将先行词带入从句，看作何成分.

关系词功能速查

从句成分 词性 候选词 备注
主 / 宾 / 表 关系代词 that ( 人/物 ), which ( 物 ), who ( 人 ), whom ( 人宾 ), as 无 what
定语 关系形容词 whose ( 某人的 ), of which ( 某物的 ) = of whom / of which
状语 关系副词 when ( 时间 ), where ( 地点 ), why ( 原因 ) = 介词 + which

2. 基础实战演练

请尝试填入正确的关系词 ( 点击查看答案 )

  1. I have an apple _____ is green. > that / which
  2. I know a man _____ is a doctor. > that / who
  3. This is the place _____ we visited last time. > that / which
  4. I know the man _____ you just talked with. > that / who / whom
  5. He is still the boy _____ he was ten years ago. > that (人作表语一般不用 who)
  6. Chengdu is no longer the city _____ it used to be. > that / which

二、关系词三大解题技巧

1. 缺宾语 ( 选关系代词 )

如果从句中缺少宾语 ( 动词或介词后 )，应选择关系代词 ( that, which, who, whom ).

  • This is the reason that / which she told me. ( told 缺宾语 )

2. 被动即完成 ( 选关系副词 )

如果从句是被动语态 ( be done )，通常主谓宾齐全 ( 间接宾语除外 )，此时从句不缺主宾表，应选择 关系副词 或 介词+which.

  • This is the place where the party was held. ( was held 被动，不缺宾语 )
  • This is the reason that / which I was told. ( was told 此时保留了直接宾语 reason，故仍选代词 )

3. 关系代词的省略

可省略的情况

在限制性定语从句中，作宾语 的关系代词通常可以省略.

  • This is the place ( that / which ) we visited last time.

不可省略的情况

  1. 介词 + whom / which 介词提前时，关系代词不能省.
    • Do you know any shops nearby in which I can buy flowers?
  2. 非限制性定语从句 逗号后的关系代词不能省.
    • Mr. Green, whom you know, is the tallest.
  3. 特殊句式 the same ... as, such ... as, as ... as.
    • I have bought the same bicycle as you have.
  4. 并列定语从句 第二个及之后的关系代词不能省.
    • This is the book (which) I read yesterday and which I find very interesting.

三、关系副词 vs 介词+Which

关系副词 ( when, where, why ) 在从句中作状语，相当于 “介词 + which”.

对应关系

  • when = at / in / on / during … + which ( 时间 )
  • where = at / in / on / to … + which ( 地点 )
  • why = for + which ( 原因 )

进阶实战演练

请尝试填入正确的关系词

  1. This is the place _____ we visited last time. > that / which (visit 是及物动词，缺宾语)
  2. This is the place _____ we stayed last time. > where (stay 是不及物动词，不缺宾语)
  3. I still remember the time _____ we first met. > when
  4. I still remember the time _____ we spent together. > that / which (spend 缺宾语)
  5. This is the reason _____ she cried. > why
  6. This is the reason _____ she gave me. > that / which (give 缺宾语)

介词选择技巧

介词的选择通常取决于

  1. 先行词的搭配 ( at the airport, in the house )
  2. 从句动词的搭配 ( listen to, depend on )
  3. 句意逻辑
  • Is this the house in which Shakespeare was born? ( born in the house )
  • Villagers here depend on the fishing industry, without which there won’t be much work. ( depend on, live without )

四、关系形容词 Whose

Whose 表所属关系 ( “谁的” )，既可指人也可指物.

  • 指人 I like girls whose hair is black.
  • 指物 I like books whose covers are shiny.

Whose vs Of which 当先行词指 物 时，whose + n. 可以转换为 the + n. + of which 或 of which + the + n.

  • the house whose roof is red
  • = the house, the roof of which is red
  • = the house, of which the roof is red

五、非限制性定语从句 ( Non-defining )

非限制性定语从句是对先行词的 补充说明，去掉后主句意思依然完整. 形式上常有 逗号 隔开.

核心特征

  1. 有逗号 隔开.
  2. 不能用 that 引导，只能用 which ( 指物 ) 或 who / whom ( 指人 ).
  3. 不能省略 关系词.
  4. 可修饰 整个主句 ( which / as ).

As vs Which ( 引导非限制性定语从句 )

当从句修饰 整个主句 时，常用 As 或 Which.

特性 As ( 正如 ) Which ( 这件事 )
位置 可置于 句首、句中或句末 只能置于 句末 ( 逗号后 )
含义 “正如”，“正像” ( 像…一样 ) “这件事”，“这一点” ( 纯粹的事实 )
搭配 as is known to all, as we expect -
否定 不能用于否定意义 可用于否定意义

Examples

  • As is known to all, the earth is round. ( 正如众所周知… )
  • He failed the exam, which is a pity. ( 他考试挂了，这很遗憾. -> which 指代 He failed the exam )
  • The elephant is like a spear, as anybody can see. ( 大象像长矛，正如任何人都能看见的那样. )

六、定语从句中的 “主谓一致”

从句谓语动词的单复数，取决于 先行词.

  • I, who am a student, should study hard. ( 先行词是 I )
  • He is one of the students who are late. ( 先行词是 students )
  • He is the only one of the students who is late. ( 先行词是 the only one )
