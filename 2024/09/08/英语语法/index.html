英语语法基本框架

  • 10 种词性
  • 9 种成分
  • 8 种句型

10 种词性

  1. noun ( 名词 n. )
  2. pronoun ( 代词 pron. )
  3. numeral ( 数词 num. )
  4. article ( 冠词 art. )
  5. verb ( 动词 v. )
  6. adverb ( 副词 adv. )
  7. adjective ( 形容词 adj. )
  8. preposition ( 介词 prep. )
  9. conjunction ( 连词 conj. )
  10. interjection ( 感叹词 int. )

9 种成分

  1. subject ( 主 )
  2. predicate ( 谓 )
  3. object ( 宾 )
  4. attribute ( 定语 )
  5. adverbial ( 状语 )
  6. object complement ( 宾补 )
  7. subject complement ( 表 )
  8. double object ( 双宾 )
  9. apposition ( 同位 )

8 种句型

简单句

  • 主 + 谓
  • 主 + 谓 + 宾
  • 主 + 谓 + 表
  • 主 + 谓 + 双宾
  • 主 + 谓 + 宾 + 宾补

复合句

  • 名词性从句
  • 定语从句
  • 状语从句

4 种简单句

陈述句

陈述句的形式

陈述句分为两种形式，肯定式和否定式.

肯定式

🫧 I have an apple. ( 💫 我有一个苹果. )

否定式

肯定变否定，助动词 ( do ) 后面加否定词 ( not ).

🫧 I don't ( do not ) have an apple. ( 💫 我没有苹果. )

陈述句的句型

陈述句分为五种句型

主 + 谓 ( 不及物动词 )

🫧 I run. ( 💫 我跑了. )

主 + 谓 + 宾

🫧 I love you. ( 💫 我爱你. )

主 + 系 + 表

🫧 I am Spider-Man. ( 💫 我是蜘蛛侠. )

系动词 = be 动词 ( is，am，are... ) / 感官动词 ( look，sound，smell，taste，feel... )

主 + 谓 + 双宾 ( 直接宾语 + 间接宾语 )

🫧 I give you money. ( 💫 我给你钱. )

判定技巧 当我们在宾语后面加上 be 动词，如果能构成一个 语意通顺 的句子，那么这个成分就很可能是 宾语补足语，而不是双宾语中的直接宾语.

这种方法基于的原理是，宾语补足语是用来补充说明宾语的状态、身份或特征的，因此它与宾语之间的关系非常紧密，可以形成一个完整的句子. 而双宾语中的直接宾语和间接宾语之间并没有这种紧密的关系.

对当前例子中，you money 尝试加入 be 动词，变成 you are money 显然语意不通，故为双宾结构.

主 + 谓 + 宾 + 宾补

🫧 I should make the job done. ( 💫 我应该完成这项工作。. )

借用上个例子的判定技巧，可以轻易得出其为宾补.

疑问句

一般疑问句

陈述句变一般疑问句有三种方式 : 提前 系动词 ( be 动词  感官动词 ) 、提前 助动词 ( do，have... ) 、提前 情态动词 ( can，may，will... )

提前 系动词

主 + 系 + 表 系 + 主 + 表

陈述句 You are an idiot. ( 💫 你是笨蛋. )

一般疑问句 Are you an idiot? ( 💫 你是不是笨蛋? )

回答 Yes, I am. / No, I'm not.

提前 助动词

主 + 谓 + ( 宾 ) 助动词 + 主 + 动词原形 + 其他

陈述句 You have got a problem. You have a problem. ( 💫 你有一个问题. )

一般疑问句 Have you got a problem? Do you have a problem? ( 💫 你有什么问题吗? )

回答 Yes, I have. / No, I haven't.

  • 当 have 表示 动作，如表示 吃 ( =eat )，喝 ( =drink )，拿 ( =take ) 收到 ( =receive ) 度过 ( =spend ) 等时只能用 have，后面不能加 got，也不可以缩写，可用于进行时态，但一般不用于被动语态.
  • 表达过去式时不用 have got.

提前 情态动词

主 + 情态动词 + 动词原形 + 其他 情态动词 + 主 + 动词原形 + 其他

陈述句 You can play the piano. ( 💫 你会弹钢琴. )

一般疑问句 Can you play the piano? ( 💫 你会弹钢琴吗? )

回答 Yes, I can. / No, I can't.

特殊疑问句

对句子中某个成分提问，用 特殊疑问词 开头且不可用 yes，no 回答.

特殊提问词 ( who，whose，whom，what，when，where，why，how，how munch / many，which... )

🫧 Who is she? ( 💫 她是谁? )

选择疑问句

提问两个及以上，选择疑问用 or 连接且不可用 yes，no 回答.

🫧 Do you like to play the violin or the piano? ( 💫 你喜欢拉小提琴还是弹钢琴? )

回答 I like to play the violin. / I like to play the piano.

反义疑问句

前肯后否，前否后肯 且可用 yes，no 回答.

陈述句部分是肯定，疑问句部分用否定句

🫧 She is your sister, isn't she? ( 💫 她是你的姐姐，不是吗? )

回答 Yes, she is. / No, she isn't.

陈述句部分是否定，疑问句部分用肯定句

🫧 She isn't your sister, is she?🫧 她不是你的姐姐，是吗? )

回答 Yes, she is. / No, she isn't. ( 根据 事实 回答 )

祈使句 的反义疑问句

用 will you 和 won't you 等.

🫧 Open the door, won't you? ( 💫 打开门，好吗? )

🫧 Don't open the door, will you? ( 💫 不打开门，好吗? )

  • 当祈使句为 Let's... 时，反义疑问句总是用 shall we.

🫧 Let’s phone her now, shall we? ( 💫 我们现在就给她打电话，好吗? )

  • 当祈使句为 Let us... 时，若表示 请求，反意疑问句用 will you，若表示 建议，反意疑问句用 shall we.

🫧 Let us know your address, will you? ( 💫 请把你的地址告诉我们，好吗? )

🫧 Let us go swimming together, shall we? ( 💫 我们一起去游泳，好吗? )

含有 must 的反义疑问句

  • 表 "必须"、"有必要" 时，疑问部分多用 mustn't.

🫧 I must take the medicine, mustn't I? ( 💫 我必须吃了这药，不是吗? )

  • 表 "禁止，不可以" 时，疑问部分用 must.

🫧 I mustn't go out, must I? ( 💫 我不能出门，对吗? )

  • 表 肯定推测 时，分 2 种情况

case-1 must + 动词原形 : 疑问部分跟这个 动词 保持 一致，比如 be 动词 — am / is / are ( 人称 / 肯否看情况 )

🫧 The Gods must be crazy, aren't they? ( 💫 上帝肯定是疯了，不是吗? )

case-2 "过去肯定做过某事" : must have done...，haven't / didn't + 主语?

🫧 She must have heard of the bad news, hasn't she? ( 💫 她肯定听说了这个坏消息，不是吗? )

如果有明确的表示过去的时间状语，疑问部分可以用 did.

感叹句 的反义疑问句

用感叹句陈述，用 否定 提问.

🫧 What a lovely girl, isn't she? ( 💫 多么可爱的女孩，不是吗? )

含有 否定词 的反义疑问句

常见的否定词 never，seldom，hardly，few，little，barely，scarcely，nothing，none，rarely one，nobody，neither...

当陈述句还有 否定词 是，用 肯定 提问.

🫧 He hardly goes to school, does he? ( 💫 他几乎不去学校，是吗? )

复合句 的反义疑问句

复合句反义疑问，都是对 主句 提问与主句相关. ( 除了宾语从句 )

🫧 She never told you that she had a baby, did she? ( 💫 她从来没有告诉过你她有一个孩子，是吗? )

含有 宾语从句 的反义疑问句

全部复合句里，只有宾语从句的反义疑问句，是和从句相关的.

don't think，believe，imagine，suppose 引导的宾语从句，反义疑问句与 从句主语，主谓一致，用 肯定 提问.

🫧 I don't imagine he has got it, has he? ( 💫 我没有想过他有它，他有吗? )

感叹句

What 感叹词

  • what + a / an + 形容词 + 可数名词单数 + 主语 + 谓语

🫧 What a beautiful photo it is ! ( 💫 这照片它多美啊 ! )

  • what + 形容词 + 不可数名词 / 名词复数 + 主语 + 谓语

🫧 What cleanly water it is ! ( 💫 多干净的水啊 ! )

How 感叹词

  • how + 形容词 + a / an + 可数名词单数 + 主语 + 谓语

🫧 How cute a dog it is ! ( 💫 它是多么可爱的一只狗啊 ! )

  • how + 形容词 / 副词 + 主语 + 谓语

🫧 How fast he runs ! ( 💫 他跑得多快啊 ! )

语气词 + 感叹号 的感叹句

🫧 Wow ! That's totally awesome ! ( 💫 哇 !真是厉害 ! )

祈使句

命令、警告 祈使句

🫧 Get out ! ( 💫 滚 ! )

🫧 Be quiet, please. ( 💫 请安静. )

🫧 Be careful ! ( 💫 小心 ! )

请求、祝愿 祈使句

🫧 Have a good day ! ( 💫 祝你有美好的一天 ! )

🫧 Let me help you. ( 💫 让我帮你. )

10 种词性

数词

数词分为 基数词 和 序数词.

基数词

数字表达规律

  • 从右往左，每 三 位加一个逗号，倒数第 一 个逗号之前是 thousand，倒数第 二 个逗号之前是 million，倒数第 三 个逗号之前是 billion，倒数第 四个 逗号之前是 trilion.
  • 逗号之间的百位和十位要加 and，数字隔零要加 and.

🫧 one hundred ( 💫 100 )

🫧 one hundred and one ( 💫 101 )

🫧 nine hundred and ninety-six ( 💫 996 )

🫧 sixty million two hundred and sixty-three thousand one hundred and fifty ( 💫 60，263，150 )

  • 基数词的单位 hundred，thousand，million 和 billion 等用来表示具体数目，前面有具体的基数词时，这些词的后面不加 -s.

🫧 There are nine hundred teachers in our school. ( 💫 我们学校有9000名老师. )

  • 当 hundred，thousand，million 和 billion 等用来表示不具体的数目时用 hundreds of，thousands of，millions of，billions of ＋ 名词 来表示，这些词的前面也不用具体的基数词.

🫧 We have already planted thousands of trees on the farm. ( 💫 我们在农场里已经种植了成千上万棵树. )

序数词

  • 序数词 在句子中主要作 定语，有时也可以用作 主语、宾语 和 表语.

🫧 I am the third in this exam. ( 💫 我这次考试是第一名. )

定冠词 the 和 序数词 连用表示顺序. 不定冠词 a / an 和 序数词 连用表示 "又一、再一" 的意思.

🫧 You are the first and I am the second. ( 💫 你第一，我第二. )

🫧 Would you repeat it a third time? ( 💫 你能再重复一次吗? )

-- 分数 --

分数是以 基数词 代表 分子，序数词 代表 分母，除了分子是 "1" 的情况外，分母 的序数词都要用 复数.

🫧 a half ( 💫 1 / 2 )

🫧 one third ( 💫 1 / 3 )

🫧 two thirds ( 💫 2 / 3 )

🫧 a (one) quarter ( 💫 1 / 4 )

分数在句子中作主语的时候，谓语动词的单、复数形式由 of 后面的名词来决定.

  • 如果 of 后面的名词是可数名词的复数形式，谓语动词用 复数形式.

🫧 Three fifths of the boys in this room are 17 years old. ( 💫 这个房间五分之三的男孩子 17 岁. )

  • 如果 of 后面的名词是 不可数名词，谓语动词用 第三人称单数形式.

🫧 It is said that two thirds of information on the Internet is not the fact. ( 💫 据说互联网上三分之二的信息不是事实. ( Information 是不可数名词 ) )

-- 年 --

🫧 nineteen ninety-four ( 💫 1994 )

🫧 two thousand ( 💫 2000 )

🫧 two thousand and eight ( 💫 2008 )

-- 日期 --

🫧 September the first, two thousand and nine ( 💫 September 1, 2009 )

-- 时刻 --

🫧 nine o’clock ( 💫 9 : 00 )

🫧 ten past nine / nine ten ( 💫 9 : 10 )

🫧 ( a ) quarter past nine / nine fifteen ( 💫 9 : 15 )

🫧 half past nine / nine thirty ( 💫 9 : 30 )

🫧 twenty to ten / nine forty ( 💫 9 : 40 )

🫧 ( a ) quarter to ten / nine forty-five ( 💫 9 : 45 )

-- 电话号码 --

🫧 six four seven - three double o - two five one six ( 💫 647 - 300 - 2516 )

-- 价格 --

通常省去 dollars 和 cents.

🫧 ninety-nine ninety-eight / ninety-nine dollars and ninety-eight cents ( 💫 99.98$ )

-- 小数 --

通常省去 zero 或 读成 o.

🫧 point five / zero point five ( 💫 零点五 )

🫧 point o five (💫 零点零五 )

-- 两个一样的数字，三个一样的数字 --

两个一样用 double，三个一样用 triple.

🫧 double oseven ( 💫 零零七 )

🫧 boeing triple seven ( 💫 波音777 )

冠词

冠词是置于名词前并限定名词的意义的虚词.

定冠词 只有 the 这一种形式.

不定冠词 只有 a 和 an 这两种形式.

名词前若 不用冠词 时，又称作 零冠词.

定冠词 主要特点是 "确指" 或 "特指".

不定冠词 主要特点就是 "不确定" 或 "泛指".

定冠词

  1. 特指 某 ( 些 ) 人 或 某 ( 些 ) 物，上文提到 的，双方都知道的.

🫧 The blue shirt is mine. ( 💫 蓝衬衫是我的. )

  1. 指世界上 独一无二 的事物.

🫧 the sun ( 💫 太阳 )

  1. 用在形容词或副词的 最高级 前.

🫧 This is the best cup of coffee. ( 💫 这个最好的一杯咖啡. )

  1. 用在 姓氏复数 形式前，表示 "全家人".

🫧 The Smiths live in the apartment above ours. ( 💫 史密斯一家住在我们楼上的一套住房里. )

  1. 用在 序数词，方位词 前.

🫧 on the left ( 💫 在左边 )

🫧 He was the first man to think of it. ( 💫 他是第一个想到这一点的人. )

  1. 用在 乐器、棋类 前.

🫧 play the piano ( 💫 弹钢琴 )

  1. 用在表示 海洋，河流，山脉，群岛 及 国家 和 党派 等名词前.

🫧 the Black Sea ( 💫 黑海 )

🫧 the Yangzi River ( 💫 长江 )

  1. 用在由普通名词构成的 专有名词 前.

🫧 the United States ( 💫 美国 )

  1. 用于某些 固定词组.

🫧 all the same ( 💫 仍然 )

🫧 all the time ( 💫 一直 )

🫧 at the moment ( 💫 此刻 )

🫧 at the same time ( 💫 同时 )

🫧 by the way ( 💫 顺便说 )

🫧 do the shopping / washing ( 💫 买东西 / 洗衣服 )

🫧 in the morning / afternoon / evening ( 💫 在上午 / 下午 / 晚上 )

🫧 in the open air ( 💫 在户外，在野外 )

不定冠词

辅音 发音前面加 a，元音 发音前面加 an.

  1. 用来表示 "一" 这个数量.

🫧 Rome was not built in a day. ( 💫 罗马不是一天建成的. )

🫧 He will be back in a day or two. ( 💫 他将在一至两天内返回. )

  1. 泛指 某 一类人、事 或 物.

🫧 A grammar book is necessary to a language learmer. ( 💫 对语言学习者来说，语法书是很必要的. )

  1. 表示 "每一"，相当于 every.

🫧 I go to school five days a week. ( 💫 我一周上五天课. )

  1. 用在序数词前，表示 "又一","再一".

🫧 I have three books, I want to buy a fourth one. ( 💫 我已经有三本书，我想买第四本. )

  1. 用在某些固定词组中.

🫧 a lot ( of ) ( 💫 许多，大量 )

🫧 after a while ( 💫 过一会儿 )

🫧 have a rest ( 💫 休息 )

🫧 have a look ( 💫 瞧瞧 )

🫧 have a good time ( 💫 过得开心 )

🫧 take a shower ( 💫 淋浴 )

🫧 take a break ( 💫 休息 )

🫧 take a bath ( 💫 洗澡 )

🫧 get a cold ( 💫 患感冒 )

🫧 get a fever ( 💫 发烧 )

🫧 make a living ( 💫 谋生 )

🫧 in a moment ( 💫 过一会儿 )

零冠词

  1. 很 广泛 的 概念.

🫧 Sichuan dishes are spicy. ( 💫 四川菜很辣. )

  1. 书名、人名.

🫧 I am reading "Journey to the West". ( 💫 我正在读西游记. )

  1. 某些城市、国家.

🫧 China is a big country. ( 💫 中国是一个大国. )

  1. 某些山河湖海洲.

🫧 Asia ( 💫 亚洲 )

  1. 星球.

🫧 Mars ( 💫 火星 )

  1. 语言、球类、学科、棋牌.

🫧 I have studied English for ten years. ( 💫 我学了十年英语. )

  1. 三餐、日期、季节 ( 一般不加冠词，如果要特指可以加 the )

🫧 It's September 1th ( 💫 这是九月一日. )

名词

专有名词 和 普通名词

专有名词

专有名词大多数是不可数名词，在某些情况下可以复数使用，或者转换为普通名词，单数 / 复数使用，例如 : 人名、地名、国家、组织、单位、大型活动、著名事件 等 ( 首字母大写、如果缩写全部字母大写 )

人名 : Jack ( 💫 杰克 )

单位 : Department of Education ( 💫 教育部 )

地名 : Beiing ( 💫 北京 )

大型活动 : Midi Festival ( 💫 迷笛音乐节 )

国家 : China ( 💫 中国 )

著名事件 : Columbus's discovery of America ( 💫 哥伦布发现新大陆 )

组织 : FBI ( 💫 美国联邦调查局 )

普通名词

专有名词以外的名词.

专有名词复数使用，专业名词转换普通名词

动词原形

动词原始的样子，很多时候需要用动词原形.

  1. 与 助动词 或 情态动词 一起构成 谓语，要用 原形.

🫧 Do you like English? ( 💫 你喜欢英语吗? )

  1. 祈使句 的谓语动词，要用 原形.

🫧 Don't move. ( 💫 别动. )

  1. 感官动词 的 宾语 与 宾补 构成 逻辑上的主谓关系，表示 动作 的 最后结果 或 全过程 时，作 宾补 的动词要 用原形.

🫧 I heard a coin drop. ( 💫 我听到一个硬币掉了. )

  1. 使役动词 后的 宾语 与作 宾补 的动词构成 逻辑上的主谓关系 时，动词要 用原形.

🫧 He let Jack lead the way. ( 💫 他让杰克在前带路. )

  1. 两个以上 的 并列不定式 用 and 或 or 连接起来，动词要用 原形.

🫧 Let's go and play football. ( 💫 我们去踢球. )

  1. 在 why not...和 why... 句型中，not 和 why 后直接跟 动词原形，构成 反问，表示"建议、劝告、责备 或 提醒"...

🫧 Why not use my bicycle? ( 💫 为什么不用我的自行车呢? )

  1. 动词 help 后是 宾 + 宾补，其 宾补 由 动词充当 时，动词用 原形.

🫧 I helped her clean the room. ( 💫 我帮助她打扫了房间. )

当 不定式 用于 使役动词 后作 宾语补足语，不定式 必须 省略 to. (to clean -> clean )

  1. do、does、did 与动词原形连用.

🫧 Do be carefu! ( 💫 一定要当心! )

  1. 不定式 作 介词 but 和 except 的 宾语，当其前有 实意动词 do 时，后面的动词 不定式 要 省略 to.

🫧 Jack had nothing to do but / except watch TV. ( 💫 杰克除了看电视外无事可做. )

  1. 在一些 固定的句型 中，如 had better ( 最好 )，would rather ( 宁愿 )，would rather...that...( 宁愿...而不 )....后面的动词要用 原形.

🫧 You had better have a good rest. ( 💫 你最好好好休息一下. )

第三人称单数

一般现在时，主语 为 第三人称单数 ( 人称代词 he / she / it、单个人名、地名、单数可数名词、不可数名词)，谓语 要用 第三人称单数.

🫧 It looks like a cat. ( 💫 它看起来像只猫. )

🫧 Beijing is in China. ( 💫 北京在中国. )

🫧 The cat is lucy's. ( 💫 这猫是路西的. )

🫧 The water is very cold. ( 💫 这水很冷. )

-- 动词变第三人称单数的规律 --

直接动词结尾加 -s : ask - asks

以 s、x、ch、sh、o 结尾加 -es : watch - watches

辅音字母 加 y 结尾，y 变成 i 加 -es : try - tries

不规则 变化 : have - has、be - is

动名词和现在分词

动名词起着 名词 的功能，常在句子中充当 主语、定语、表语...成分.

现在分词 来构成 进行时态，与助动词 be 搭配构成句子的 谓语，现在分词在句中还可以充当 定语、表语 和 状语.

词尾加 -ing : play - playing

词尾是 e 去掉 e 加 -ing : ride - riding

词尾是 ie 把 ie 变成 y 加 ing : tie - tying

辅音 加 元音加辅音 双写辅音 加 ing : beg - begging

过去式和过去分词

过去式单独 做 谓语，表示 过去某个时间发生动作.

过去分词 是动词的 非谓语形式，表示 被动 或 完成.

加 -ed : play - played

结尾加 -d : love - loved

特殊 : 略

动词语态

主动语态

主语 是 动作执行者，某人某物做什么.

🫧 I give you a car. ( 💫 我给你一辆车. )

被动语态

主语 是 动作承受者，某人某物被做了什么.

只有 及物动词 才 有被动语态，同样地，对于 不定式 来说，也只有 及物动词的不定式 才 有被动式.

如果是 不及物动词的不定式 要用于 被动式，要注意在其后加上 合适的介词. 但并不绝对，凡是 不定式作定语 与 被修饰名词 为 动状关系 ( 被修饰名词 在 逻辑 上是不定式动作的 时间、地点、方式 或 原因状语 ) 时，该 名词 必须是可以被 关系副词 that 引导的 定语从句 修饰的名词. 这类名词后的 不定式作定语 才允许动状关系而 不需要 加 介词.

可被关系副词引导的定语从句修饰的名词例如 : 地点名词 : 只有 place 一个词. / day，month，year，season，time... / money / 货币名称 / reason / way....

🫧 We have to find a place to stay. ( 💫 我们得找个地方住. )

🫧 You are given a car by me. ( 💫 你被我给了一辆车. )

主动句改被动句

主动改被动一共四步

  1. 把主动句的 宾语改 成被动句的 主语.

  2. 把主动句的 谓语动词改 成被动句谓语形式 be + 过去分词，并且 人称 和 单复数 要 随着新主语变化，时态保持不变.

  3. 把主动句的 主语前面加 by，放到 谓语之后，变化 人称宾格，如果看前面就知道是谁被谁做了什么，或 上下文有交代 是谁，可以省略 by 加主语.

  4. 其他 句子成分 保持不变.

🫧 My mother sweeps the floor every day morning. ( 💫 我妈每天早上扫这地. )

🫧 The floor is swept by my mother everyday morning. ( 💫 这地每天早上被我妈扫. )

🫧 I bought some books. ( 💫 我买了一些书. )

🫧 Some books were bought ( by me ). ( 💫 一些书被我买了. )

双宾的被动语态形式

双宾的被动语态可以改两句.

把 间接宾语改 成被动语态的 主语，直接宾语 留着 原位.

把 直接宾语改 成被动语态的 主语，间接宾语 前面加 to 或 for.

🫧 He gave her some money. ( 💫 他给了她一些钱. )

🫧 She was given some money ( by him ). ( 💫 她被 ( 他 ) 给了一些钱. )

🫧 Some money was given to her ( by him ). ( 💫 一些钱被 ( 他 ) 给了她. )

宾补的被动语态形式

宾补的被动语态只能改一句.

主动语态的 宾语改 成 主语，宾补 在被动语态中做 主语的宾补

🫧 We made him manger. ( 💫 我们选他做经理. )

🫧 He was made manger ( by us). ( 💫 他被 ( 我们 ) 选做经理. )

动词短语的被动语态形式

将其看为一个整体.

🫧 We must take good care of the trees. ( 💫 我们必须好好照顾这些树. )

🫧 The trees must be taken good care of (by us). ( 💫 这些树必须被 ( 我们 ) 好好照顾. )

can 和 must 作为情态动词，它本身不能用于被动语态，但它们后面所接动词可以用被动式.

什么时候用被动语态形式

被动语态使用场景.

  1. 不知道动作执行者是谁 的情况用被动语态.

🫧 This diamond was stolen yesterday. ( 💫 这钻石昨天被偷了. )

  1. 强调动作承受者的情况 用被动语态.

🫧 Peter was also grown in this country. ( 💫 皮特也是在这国家长大的. )

  1. 说客观事实 可以用被动语态.

🫧 The IPhone was created by Steven Jobs.Iphone. ( 💫 是由史蒂芬，乔布斯创造的. )

非谓语动词

非谓语动词分类

不做谓语 的 动词，不定式，动名词，( 现在过去 ) 分词.

🫧 The teacher told us to do the homework. ( 💫 老师叫我们做家庭作业. )

🫧 I like swimming. ( 💫 我喜欢游泳. )

🫧 He sat there and reading a newspaper. ( 💫 他坐在那看着报纸. )

🫧 We heard about polluted air information. ( 💫 我们听到了空气被污染的消息. )

不定式

大多数都是 to + 动词原形， 有些情况 to 可省略.

🫧 I like to clean. ( 💫 我喜欢打扫卫生. )

🫧 I can do it. ( 💫 我能做到. )

-- 什么时候 不加 to --

  1. 跟 助动词 ( can，could，might，should，will，may，must，ought to，shall ) 一起 不加 to.

🫧 I can do it. ( 💫 我能做到. )

  1. 跟 感官动词 ( see，watch，observe，hear，listen to，feel... ) 一起 不加 to.

🫧 I saw him enter the house. ( 💫 我看到他走进了房子. )

  1. 跟 使役动词 ( let，make，have，keep，help... ) 一起 不加 to.

🫧 Let me try it. ( 💫 让我试试看. )

  1. 跟 why ( not ) 一起 不加 to.

🫧 Why not go out for a walk on a sunny day? ( 💫 在阳光明媚的日子里出去散步不是很好吗？)

  1. 跟 know 一起有时候不加 to.

-- know + 宾语 + to be ( to be 通常 不省略 )，这类句型可转换为 that 引导的 宾语从句 --

I know him to be a fool. = I know ( that ) he is a fool. ( 💫 我知道他是个傻瓜. )

-- know + 宾语 + to do ( 有时 可省略 to )，在这类句型中 know 一般只限于用 过去时 或 完成时，通常译 "听说过" "见过" --

🫧 I have never known her ( to ) tell lies. ( 💫 我从未听说她撒过谎. )

🫧 I've never known her ( to ) be late before. ( 💫 我以前从不知道她迟到过. )

be known as 意为 "作为...出名".

be known for 意为 "因为...出名".

be known to 意为 "为...所知".

  1. 介词 except、but 后 不加 to.

🫧 She tries everything except cook. ( 💫 除了做饭，她什么都试. )

  1. 主语 部分带 do 不定式做表语 省略 to.

🫧 All you do now is complete the form. ( 💫 你现在所做的就是填写表格. )

  1. 当 两个 或 多个 作用相同 的 不定式并列 时，第一个 不定式前用 to，其余 不定式前的 to 可以 省略.

🫧 He told me to stay there and wait for him. ( 💫 他叫我呆在那里等他. )

-- 当 使役动词、感官动词 用于 被动语态 时，要 补 上在主动语态中 省略 的 to --

🫧 She was made to feel happy by the great news. ( 💫 这个好消息使她感到高兴. )

🫧 The woman was seen to enter a bank. ( 💫 有人看见这个女人进了一家银行. )

完成进行 to have been doing 动作发生在谓语的 动作之前

独立主格结构

独立主格 是一个 名词 或 代词 ( 作为 逻辑主语 ) 加 形容词 / 副词 / 非谓语 / 介词短语 / 名词 在句中作 状语.

逻辑主语 与 主句的主语 不一致.

名词 / 代词 与 后面的部分 构成逻辑上的 主谓关系，用来修饰谓语动词或整个句子.

独立主格结构 一般用 逗号 与主句 分开，且与主句 不能使用 任何 连接词.

独立主格结构 用于补充主句的 时间、原因、条件、方式、伴随状语.

大体分为两类

名词 / 代词 + 非谓语

名词 / 代词 + ( being + ) 形容词 / 副词 / 介词短语 / 名词

  1. 名词 / 代词 + to do 将做 / to be done 将被做...

  2. 名词 / 代词 + doing 主动做、正在做 / being done 正在被做 / having done 已经做了 / having been done 已经被做了

  3. 名词 / 代词 + done 被做

  4. 名词 + 形容词

🫧 She sat down on the ground, her face pale with great pain. ( 💫 她痛苦地跌坐在地上，脸色苍白. )

her face ( n. ) pale ( adj. )

  1. 名词 + 副词

🫧 She sat at the table, collar off, head down, and pen in position, ready to begin the long letter. ( 💫 她坐在桌旁，衣领敞开，低着头，笔已经准备好，准备开始写一封长信. )

collar ( n. ) off ( adv. )

head ( n. ) down ( adv. )

pen ( n. ) in position ( 介词短语 )

ready to begin the long letter ( 伴随状语 )

  1. 名词 + 介词短语

🫧 The teacher came in, book in hand. ( 💫 老师进来了，手里拿着一本书. )

这种结构通常 省略冠词.

  1. 名词 + 名词

🫧 The children in the mountainous school received many gifts, most of them books. ( 💫 山区学校的孩子们收到了许多礼物，其中大多数是书. )

-- 不能省略 being 的情况 --

  • There being + 名词 的结构

🫧 There being no further business, I declare the meeting closed. ( 💫 由于没有其他事务，我宣布会议结束. )

  • 逻辑主语 为 人称代词 不能省略

🫧 It being Christmas, the government offices were closed. ( 💫 因为是圣诞节，政府办公室都关门了. )

名词性从句

用一个句子代替句子里一个名词.

分为 主语从句、宾语从句、表语从句、同位语从句.

主语从句

主语从句 引导词

连词 : that，whether

代词 : who，whoever，whom，whose，what，whatever，which，whichever

副词 : when，where，how，why，whenever，wherever，however

主语从句经常，用 形式主语 it 放主语位置，真的主语后置.

🫧 That he will win the match is certain. ( 💫 他将赢得这场比赛是肯定的. )

🫧 It is certain that he will win the match. ( 💫 他肯定会赢得这场比赛. )

主语从句一定 不能放句首 的情况

  1. If / whether 引导的主语从句

🫧 It is uncertain whether / if he will come this evening. ( 💫 不确定他今晚是否会来. )

  1. It is said 开头的结构

🫧 It is said that he never laughed. ( 💫 据说他从来不笑. )

  1. It happens、It occurs 结构

🫧 It happens to me that I meet my professor. ( 💫 我碰巧遇到我的教授. )

  1. It doesn't matter how / whether 结构

🫧 It doesn't matter how it happened. ( 💫 它发生没有什么关系. )

  1. 疑问句 的时候

🫧 Is it sure that you will come back here? ( 💫 确定你要回到这里吗? )

如何 区别 主语从句 和 强调句

判断方法 : 去掉 it、be、that 之后看 句子是否完整.

🫧 It is certain that he will come here. ( 💫 肯定他会来这里. ) ( 主语从句 )

🫧 It was in the hospital that I saw the Peter. ( 💫 我在医院看见的是皮特. ) ( 强调句 )

主语从句 多数情况 视为 第三人称单数，但是连接代词 what 引导 的主语从句，后面 谓语动词单复数，要 根据句子意思决定.

🫧 What we need is time. ( 💫 我们需要的是时间. )

🫧 What we need are good doctors. ( 💫 我们需要的是好医生. )

主语从句的 时态，不受主句时态的影响

What he did yesterday will never be known by his mother. ( 💫 他昨天做的事将不会被他妈妈知道. )

主句时态 will never be known 是 将来时.

主语从句 what he did yesterday 其时态是过去时.

宾语从句

用一个句子代替句子里一个名词，做句子宾语.

宾语从句 引导词

连词 : that ( that 常可省略 )，whether，if

代词 : who，whose，what，which

副词 : when，where，how，why...

当 主句 是 一般现在时 或 一般将来时 的时候，从句可以 任何时态.

当 主句 是 一般过去时 的时候，从句时态必须是 过去时范围内的时态 ( 主过从过 )

宾语从句种类

  1. 动词 的 宾语从句

🫧 Tell your son that playing game too much is bad for his eyes. ( 💫 告诉你儿子玩太多游戏对他眼睛不好. )

  1. 介词 的 宾语从句

🫧 It all depends on whether it will be fine tomorrow. ( 💫 所有的一切都取决于明天的天气是否晴朗. )

  1. 形容词 的 宾语从句

🫧 I am sure that he wil come soon. ( 💫 我确信他很快就会来了. )

  1. 宾语从句 的 it 做 形式宾语

动词 find、feel、consider、make、believe、think...后面有 宾补 的时候宾语，要用 it 做形式宾语，将 that 宾语从句后置.

🫧 I find it interesting that talking to you. ( 💫 我觉得同你谈话很有意思. )

🫧 I think it best that you should stay here. ( 💫 我认为你最好住这. )

  1. 宾语从句 的 否定转移

主语 为 第一人称 ( I，We ) 时，且 主句的动词 是表示 看法、感觉 的词 think、believe、imagine、suppose、guess...，则属于 that 从句 的 否定 转移 到 主句.

🫧 I don't believe that the car is good. ( 💫 我相信这车不好. )

🫧 I don't think that he will come back now. ( 💫 我认为他现在不会回来. )

that 不能省略 的情形

  • 当 主句 谓语动词带 两个 或 以上的宾语从句 时候，可以 省略 第一个 that，其他 的从句 that 不能省.

🫧 I realize you are a hero and that everybody accepts your leadership. ( 💫 我意识到你是个英雄，每个人都接受你的领导. )

  • 宾语从句 的 主语 是 非谓语动词 或 主语从句，that 不能省.

🫧 He said that studying English is very hard. ( 💫 他说学英语非常难. )

  • 主语 中的 谓语动词 是 固定搭配 时候，that 不能省.

🫧 Mary has made up her mind that she will study hard for the scholarship. ( 💫 玛丽已经打定主意，她将努力学习拿奖学金. )

  • 宾语从句 的 主语 是 this / that 时候，或 this / that 做 主语 的 定语，that 不能省.

🫧 He said that this would lead their ancestors to earth. ( 💫 他说将把他们的祖先带到地球. )

  • 宾语从句放在 in、except、but...介词后面，that 不能省.

🫧 He is a good student except that he is a little careless. ( 💫 他是个好学生，除了有的粗心以外. )

  • it 做 形式宾语 时候，that 不能省.

🫧 We thought it strange that you did not come yesterday. ( 💫 我们认为很奇怪，你昨天没来. )

  • that 宾语从句 放 句首 表示 强调，that 不能省.

🫧 That he ever said such a thing, I can not believe. ( 💫 我不能相信，他曾经说过这样的话. )

  • 主句谓语动词和宾语从句自己有 插入语 时候，that 不能省.

🫧 Everyone could see, I believe, that Peter was really afraid. ( 💫 每个人都可以看到，我相信，彼得很害怕. )

  • 宾语从句 的 之前 或 之后 紧跟一个 状语 的时候，that 不能省.

🫧 He suddenly caught sight of the boy and realized at once that he had seen him before. ( 💫 他突然发现了这个男孩，立刻意识到自己曾经见过他. )

表语从句

用一个句子代替句子里一个名词，做句子表语.

表语从句 引导词

连词 : that，whether，as if，as though

代词 : who，whose，whom，what，which

副词 : when，where，how，why，because

表语从句种类

  1. that 引导

🫧 The reason was that he was late for school. ( 💫 原因是他上学迟到了. )

  1. whether、as、as if 引导

🫧 The question is whether they will be able to help us. ( 💫 问题是他们是否能够为我们提供帮助. )

  1. 连接代词 who、whom、whose、what、which...在表语从句做 主语 和 宾语

🫧 The problem is who we can get to replace her. ( 💫 问题是我们可以代替谁. )

  1. 连接副词 where、when、how、why

🫧 This is where they once lived. ( 💫 这是他们曾经住过的地方. )

  1. 连词 because

🫧 I think it is because you are doing too much. ( 💫 我认为这是因为你做太多了. )

  1. 一些表示 建议劝说命令 的名词后面

🫧 My suggestion is that we should start early tomorrow. ( 💫 我的建议是我们明天早点开始. )

if 和 whether

if 不能引导表语从句，要用 whether 引导.

whether 与 or not 连成 一个 词组 时，whether 不可换用 if.

动词不定式 之前，只能用 whether，不能用 if.

引导宾语从句 时 if 和 whether 可以互换，但是 介词后面 的宾语从句 只能 用 whether.

位于 句首 的 主语从句 只用 whether 来 引导，同位语从句 也只能用 whether.

🫧 The question is whether they will be able to help us. ( 💫 问题是他们是否能够为我们提供帮助. )

同位语从句

用一个句子代替句子里一个名词，做句子同位语，放在名词后面解释名词.

同位语从句 引导词

连词 : that ( 不能省 )，whether

代词 : what，who，whom，whose，which...

副词 : when，where，how，why...

同位语从句结构

  • 主语 + 同位语从句 + 其他

🫧 The news that he will come here is real. ( 💫 他要来这里的消息是真的. )

  • 主语 + 同位语从句

🫧 The news that he will come here. ( 💫 他要来这里的消息. )

同位语从句 和 定语从句 的 区别

-- 先行词 的 区别 --

同位语从句 前面只能是 idea，fact，news，suggestion...有 内在含义的名词.

定语从句 前面可以是 名词、代词、部分句子，整个句子.

🫧 The news that he will come here. ( 💫 他要来这里的消息. ) ( 同位语从句 )

🫧 He will come here, which made us very excited. ( 💫 他会来这里，这让我们非常兴奋. ) ( 定语从句 )

-- 性质 的 区别 --

同位语从句 是对前面的名词做 解释，可以翻译成什么什么是什么.

🫧 The news that our tearn has won the game ( 💫 比赛消息是我们队赢了那场. )

定语从句 是对前面的东西做 修饰，可以翻译成什么样的什么.

🫧 The news that he told me yesterday ( 💫 昨天他告诉我的消息. )

-- 引导词 的 区别 --

how，whether，what 可以 引导同位语从句，不能引导定语从句.

that 引导同位语从句 时候 不能省略，引导定语从句 经常省略.
