4 种简单句

陈述句

陈述句的形式

陈述句分为两种形式，肯定式和否定式.

肯定式

( 💫 I have an apple. ( 💫 我有一个苹果. )

否定式

肯定变否定，助动词 ( do ) 后面加否定词 ( not ).

( 💫 I don't ( do not ) have an apple. ( 💫 我没有苹果. )

陈述句的句型

陈述句分为五种句型

主 + 谓 ( 不及物动词 )

( 💫 I run. ( 💫 我跑了. )

主 + 谓 + 宾

( 💫 I love you. ( 💫 我爱你. )

主 + 系 + 表

( 💫 I am Spider-Man. ( 💫 我是蜘蛛侠. )

系动词 = be 动词 ( is，am，are... ) / 感官动词 ( look，sound，smell，taste，feel... )

主 + 谓 + 双宾 ( 直接宾语 + 间接宾语 )

( 💫 I give you money. ( 💫 我给你钱. )

判定技巧 当我们在宾语后面加上 be 动词，如果能构成一个 语意通顺 的句子，那么这个成分就很可能是 宾语补足语，而不是双宾语中的直接宾语.

这种方法基于的原理是，宾语补足语是用来补充说明宾语的状态、身份或特征的，因此它与宾语之间的关系非常紧密，可以形成一个完整的句子. 而双宾语中的直接宾语和间接宾语之间并没有这种紧密的关系.

对当前例子中，you money 尝试加入 be 动词，变成 you are money 显然语意不通，故为双宾结构.

主 + 谓 + 宾 + 宾补

( 💫 They ordered hamburger their lunch. ( 💫 他们的午餐他们点了汉堡. )

借用上个例子的判定技巧，可以轻易得出其为宾补.

疑问句

一般疑问句

陈述句变一般疑问句有三种方式 : 提前 系动词 ( be 动词  感官动词 ) 、提前 助动词 ( do，have... ) 、提前 情态动词 ( can，may，will... )

提前 系动词

主 + 系 + 表 系 + 主 + 表

陈述句 You are an idiot. ( 💫 你是笨蛋. )

一般疑问句 Are you an idiot? ( 💫 你是不是笨蛋? )

回答 Yes, I am. / No, I'm not.

提前 助动词

主 + 谓 + ( 宾 ) 助动词 + 主 + 动词原形 + 其他

陈述句 You have got a problem. You have a problem. ( 💫 你有一个问题. )

一般疑问句 Have you got a problem? Do you have a problem? ( 💫 你有什么问题吗? )

回答 Yes, I have. / No, I haven't.

  • 当 have 表示 动作，如表示 吃 ( =eat )，喝 ( =drink )，拿 ( =take ) 收到 ( =receive ) 度过 ( =spend ) 等时只能用 have，后面不能加 got，也不可以缩写，可用于进行时态，但一般不用于被动语态.
  • 表达过去式时不用 have got.

提前 情态动词

主 + 情态动词 + 动词原形 + 其他 情态动词 + 主 + 动词原形 + 其他

陈述句 You can play the piano. ( 💫 你会弹钢琴. )

一般疑问句 Can you play the piano? ( 💫 你会弹钢琴吗? )

回答 Yes, I can. / No, I can't.

特殊疑问句

对句子中某个成分提问，用 特殊疑问词 开头且不可用 yes，no 回答.

特殊提问词 ( who，whose，whom，what，when，where，why，how，how munch / many，which... )

( 💫 Who is she? ( 💫 她是谁? )

选择疑问句

提问两个及以上，选择疑问用 or 连接且不可用 yes，no 回答.

( 💫 Do you like to play the violin or the piano? ( 💫 你喜欢拉小提琴还是弹钢琴? )

回答 I like to play the violin. / I like to play the piano.

反义疑问句

前肯后否，前否后肯 且可用 yes，no 回答.

陈述句部分是肯定，疑问句部分用否定句

( 💫 She is your sister, isn't she? ( 💫 她是你的姐姐，不是吗? )

回答 Yes, she is. / No, she isn't.

陈述句部分是否定，疑问句部分用肯定句 ( 💫 她不是你的姐姐，是吗? )

( 💫 She isn't your sister, is she?

回答 Yes, she is. / No, she isn't. ( 根据 事实 回答 )

祈使句 的反义疑问句

用 will you 和 won't you 等.

( 💫 Open the door, won't you? ( 💫 打开门，好吗? )

( 💫 Don't open the door, will you? ( 💫 不打开门，好吗? )

  • 当祈使句为 Let's... 时，反义疑问句总是用 shall we.

( 💫 Let’s phone her now, shall we? ( 💫 我们现在就给她打电话，好吗? )

  • 当祈使句为 Let us... 时，若表示 请求，反意疑问句用 will you，若表示 建议，反意疑问句用 shall we.

( 💫 Let us know your address, will you? ( 💫 请把你的地址告诉我们，好吗? )

( 💫 Let us go swimming together, shall we? ( 💫 我们一起去游泳，好吗? )

含有 must 的反义疑问句

  • 表 "必须"、"有必要" 时，疑问部分多用 mustn't.

( 💫 I must take the medicine, mustn't I? ( 💫 我必须吃了这药，不是吗? )

  • 表 "禁止，不可以" 时，疑问部分用 must.

( 💫 I mustn't go out, must I? ( 💫 我不能出门，对吗? )

  • 表 肯定推测 时，分 2 种情况

case-1 must + 动词原形 : 疑问部分跟这个 动词 保持 一致，比如 be 动词 — am / is / are ( 人称 / 肯否看情况 )

( 💫 The Gods must be crazy, aren't they? ( 💫 上帝肯定是疯了，不是吗? )

case-2 "过去肯定做过某事" : must have done...，haven't / didn't + 主语?

( 💫 She must have heard of the bad news, hasn't she? ( 💫 她肯定听说了这个坏消息，不是吗? )

如果有明确的表示过去的时间状语，疑问部分可以用 did.

感叹句 的反义疑问句

用感叹句陈述，用 否定 提问.

( 💫 What a lovely girl, isn't she? ( 💫 多么可爱的女孩，不是吗? )

含有 否定词 的反义疑问句

常见的否定词 never，seldom，hardly，few，little，barely，scarcely，nothing，none，rarely one，nobody，neither...

当陈述句还有 否定词 是，用 肯定 提问.

( 💫 He hardly goes to school, does he? ( 💫 他几乎不去学校，是吗? )

复合句 的反义疑问句

复合句反义疑问，都是对 主句 提问与主句相关. ( 除了宾语从句 )

( 💫 She never told you that she had a baby, did she? ( 💫 她从来没有告诉过你她有一个孩子，是吗? )

含有 宾语从句 的反义疑问句

全部复合句里，只有宾语从句的反义疑问句，是和从句相关的.

don't think，believe，imagine，suppose 引导的宾语从句，反义疑问句与 从句主语，主谓一致，用 肯定 提问.

( 💫 I don't imagine he has got it, has he? ( 💫 我没有想过他有它，他有吗? )

感叹句

What 感叹词

  • what + a / an + 形容词 + 可数名词单数 + 主语 + 谓语

( 💫 What a beautiful photo it is ! ( 💫 这照片它多美啊 ! )

  • what + 形容词 + 不可数名词 / 名词复数 + 主语 + 谓语

( 💫 What cleanly water it is ! ( 💫 多干净的水啊 ! )

How 感叹词

  • how + 形容词 + a / an + 可数名词单数 + 主语 + 谓语

( 💫 How cute a dog it is ! ( 💫 它是多么可爱的一只狗啊 ! )

  • how + 形容词 / 副词 + 主语 + 谓语

( 💫 How fast he runs ! ( 💫 他跑得多快啊 ! )

语气词 + 感叹号 的感叹句

( 💫 Wow ! That's totally awesome ! ( 💫 哇 !真是厉害 ! )

祈使句

命令、警告 祈使句

( 💫 Get out ! ( 💫 滚 ! )

( 💫 Be quiet, please. ( 💫 请安静. )

( 💫 Be careful ! ( 💫 小心 ! )

请求、祝愿 祈使句

( 💫 Have a good day ! ( 💫 祝你有美好的一天 ! )

( 💫 Let me help you. ( 💫 让我帮你. )

10 种词性

数词

数词分为 基数词 和 序数词.

基数词

数字表达规律

  • 从右往左，每 三 位加一个逗号，倒数第 一 个逗号之前是 thousand，倒数第 二 个逗号之前是 million，倒数第 三 个逗号之前是 billion，倒数第 四个 逗号之前是 trilion.
  • 逗号之间的百位和十位要加 and，数字隔零要加 and.

( 💫 one hundred ( 💫 100 )

( 💫 one hundred and one ( 💫 101 )

( 💫 nine hundred and ninety-six ( 💫 996 )

( 💫 sixty million two hundred and sixty-three thousand one hundred and fifty ( 💫 60，263，150 )

  • 基数词的单位 hundred，thousand，million 和 billion 等用来表示具体数目，前面有具体的基数词时，这些词的后面不加 -s.

( 💫 There are nine hundred teachers in our school. ( 💫 我们学校有9000名老师. )

  • 当 hundred，thousand，million 和 billion 等用来表示不具体的数目时用 hundreds of，thousands of，millions of，billions of ＋ 名词 来表示，这些词的前面也不用具体的基数词.

( 💫 We have already planted thousands of trees on the farm. ( 💫 我们在农场里已经种植了成千上万棵树. )

序数词

  • 序数词 在句子中主要作 定语，有时也可以用作 主语、宾语 和 表语.

( 💫 I am the third in this exam. ( 💫 我这次考试是第一名. )

定冠词 the 和 序数词 连用表示顺序. 不定冠词 a / an 和 序数词 连用表示 "又一、再一" 的意思.

( 💫 You are the first and I am the second. ( 💫 你第一，我第二. )

( 💫 Would you repeat it a third time? ( 💫 你能再重复一次吗? )

-- 分数 --

分数是以 基数词 代表 分子，序数词 代表 分母，除了分子是 "1" 的情况外，分母 的序数词都要用 复数.

( 💫 a half ( 💫 1 / 2 )

( 💫 one third ( 💫 1 / 3 )

( 💫 two thirds ( 💫 2 / 3 )

( 💫 a (one) quarter ( 💫 1 / 4 )

分数在句子中作主语的时候，谓语动词的单、复数形式由 of 后面的名词来决定.

  • 如果 of 后面的名词是可数名词的复数形式，谓语动词用 复数形式.

( 💫 Three fifths of the boys in this room are 17 years old. ( 💫 这个房间五分之三的男孩子 17 岁. )

  • 如果 of 后面的名词是 不可数名词，谓语动词用 第三人称单数形式.

( 💫 It is said that two thirds of information on the Internet is not the fact. ( 💫 据说互联网上三分之二的信息不是事实. ( Information 是不可数名词 ) )

-- 年 --

( 💫 nineteen ninety-four ( 💫 1994 )

( 💫 two thousand ( 💫 2000 )

( 💫 two thousand and eight ( 💫 2008 )

-- 日期 --

( 💫 September the first, two thousand and nine ( 💫 September 1, 2009 )

-- 时刻 --

( 💫 nine o’clock ( 💫 9 : 00 )

( 💫 ten past nine / nine ten ( 💫 9 : 10 )

( 💫 ( a ) quarter past nine / nine fifteen ( 💫 9 : 15 )

( 💫 half past nine / nine thirty ( 💫 9 : 30 )

( 💫 twenty to ten / nine forty ( 💫 9 : 40 )

( 💫 ( a ) quarter to ten / nine forty-five ( 💫 9 : 45 )

-- 电话号码 --

( 💫 six four seven - three double o - two five one six ( 💫 647 - 300 - 2516 )

-- 价格 --

通常省去 dollars 和 cents.

( 💫 ninety-nine ninety-eight / ninety-nine dollars and ninety-eight cents ( 💫 99.98$ )

-- 小数 --

通常省去 zero 或 读成 o.

( 💫 point five / zero point five ( 💫 零点五 )

( 💫 point o five (💫 零点零五 )

-- 两个一样的数字，三个一样的数字 --

两个一样用 double，三个一样用 triple.

( 💫 double oseven ( 💫 零零七 ) ( 💫 boeing triple seven ( 💫 波音777 )

冠词

冠词是置于名词前并限定名词的意义的虚词.

定冠词 只有 the 这一种形式.

不定冠词 只有 a 和 an 这两种形式.

名词前若 不用冠词 时，又称作 零冠词.

定冠词 主要特点是 "确指" 或 "特指".

不定冠词 主要特点就是 "不确定" 或 "泛指".

定冠词

  1. 特指 某 ( 些 ) 人 或 某 ( 些 ) 物，上文提到 的，双方都知道的.

( 💫 The blue shirt is mine. ( 💫 蓝衬衫是我的. )

  1. 指世界上 独一无二 的事物.

( 💫 the sun ( 💫 太阳 )

  1. 用在形容词或副词的 最高级 前.

( 💫 This is the best cup of coffee. ( 💫 这个最好的一杯咖啡. )

  1. 用在 姓氏复数 形式前，表示 "全家人".

( 💫 The Smiths live in the apartment above ours. ( 💫 史密斯一家住在我们楼上的一套住房里. )

  1. 用在 序数词，方位词 前.

( 💫 on the left ( 💫 在左边 )

( 💫 He was the first man to think of it. ( 💫 他是第一个想到这一点的人. )

  1. 用在 乐器、棋类 前.

( 💫 play the piano ( 💫 弹钢琴 )

  1. 用在表示 海洋，河流，山脉，群岛 及 国家 和 党派 等名词前.

( 💫 the Black Sea ( 💫 黑海 )

( 💫 the Yangzi River ( 💫 长江 )

  1. 用在由普通名词构成的 专有名词 前.

( 💫 the United States ( 💫 美国 )

  1. 用于某些 固定词组.

( 💫 all the same ( 💫 仍然 )

( 💫 all the time ( 💫 一直 )

( 💫 at the moment ( 💫 此刻 )

( 💫 at the same time ( 💫 同时 )

( 💫 by the way ( 💫 顺便说 )

( 💫 do the shopping / washing ( 💫 买东西 / 洗衣服 )

( 💫 in the morning / afternoon / evening ( 💫 在上午 / 下午 / 晚上 )

( 💫 in the open air ( 💫 在户外，在野外 )

不定冠词

辅音 发音前面加 a，元音 发音前面加 an.

  1. 用来表示 "一" 这个数量.

( 💫 Rome was not built in a day. ( 💫 罗马不是一天建成的. )

( 💫 He will be back in a day or two. ( 💫 他将在一至两天内返回. )

  1. 泛指 某 一类人、事 或 物.

( 💫 A grammar book is necessary to a language learmer. ( 💫 对语言学习者来说，语法书是很必要的. )

  1. 表示 "每一"，相当于 every.

( 💫 I go to school five days a week. ( 💫 我一周上五天课. )

  1. 用在序数词前，表示 "又一","再一".

( 💫 I have three books, I want to buy a fourth one. ( 💫 我已经有三本书，我想买第四本. )

  1. 用在某些固定词组中.

( 💫 a lot ( of ) ( 💫 许多，大量 )

( 💫 after a while ( 💫 过一会儿 )

( 💫 have a rest ( 💫 休息 )

( 💫 have a look ( 💫 瞧瞧 )

( 💫 have a good time ( 💫 过得开心 )

( 💫 take a shower ( 💫 淋浴 )

( 💫 take a break ( 💫 休息 )

( 💫 take a bath ( 💫 洗澡 )

( 💫 get a cold ( 💫 患感冒 )

( 💫 get a fever ( 💫 发烧 )

( 💫 make a living ( 💫 谋生 )

( 💫 in a moment ( 💫 过一会儿 )

零冠词

  1. 很 广泛 的 概念.

( 💫 Sichuan dishes are spicy. ( 💫 四川菜很辣. )

  1. 书名、人名.

( 💫 I am reading "Journey to the West". ( 💫 我正在读西游记. )

  1. 某些城市、国家.

( 💫 China is a big country. ( 💫 中国是一个大国. )

  1. 某些山河湖海洲.

( 💫 Asia ( 💫 亚洲 )

  1. 星球.

( 💫 Mars ( 💫 火星 )

  1. 语言、球类、学科、棋牌.

( 💫 I have studied English for ten years. ( 💫 我学了十年英语. )

  1. 三餐、日期、季节 ( 一般不加冠词，如果要特指可以加 the )

( 💫 It's September 1th ( 💫 这是九月一日. )

名词

专有名词 和 普通名词

专有名词

专有名词大多数是不可数名词，在某些情况下可以复数使用，或者转换为普通名词，单数 / 复数使用，例如 : 人名、地名、国家、组织、单位、大型活动、著名事件 等 ( 首字母大写、如果缩写全部字母大写 )

人名 : Jack ( 💫 杰克 )

单位 : Department of Education ( 💫 教育部 )

地名 : Beiing ( 💫 北京 )

大型活动 : Midi Festival ( 💫 迷笛音乐节 )

国家 : China ( 💫 中国 )

著名事件 : Columbus's discovery of America ( 💫 哥伦布发现新大陆 )

组织 : FBI ( 💫 美国联邦调查局 )

普通名词

专有名词以外的名词.

专有名词复数使用，专业名词转换普通名词

( 💫 There are two Jacks in our class. ( 💫 我们班有两个杰克. )

( 💫 Millions of Lei Fengs have emerged in China. ( 💫 中国出现了数百万的活雷锋. )

( 💫 Hey, Tom, have a Mrs. Carl is waiting for you. ( 💫 嘿，汤姆，有一位卡尔夫人正在等你. )

可数名词 和 不可数名词

可数名词

单数复数都可以，可以加数词计算 例如 : apple，egg，boy，tree，family，class，police，team...

不可数名词

只能单数，不可以加数词计算 例如 : water，fire，time，hope，knowledge，intelligence...

具体名词 和 抽象名词

具体名词

看得见摸得着，可数不可数都有 例如 : man，Beijing，book，car，water，fire...

抽象名词

either : 否定句句尾，用 逗号隔开.

nor : 放在 句首.

🫧 Are you Chinese, too? ( 💫 你也是中国人? )

🫧 You are also Chinese. ( 💫 你也是中国人. )

🫧 I am not lucky, either. ( 💫 我也不幸运. )

🫧 Nor is Peter anywhere a good man. ( 💫 皮特也不是个好人. )

  1. sometimes、sometime、some times、some time

sometimes : 有时候.

sometime : 未来的某个时候.

some times : 很多次. ( 一次 once、两次 twice )

some time : 一些时间.

🫧 Sometimes I go shopping in the mall. ( 💫 有时候去我大商场购物. )

🫧 Peter,I will meet your mother sometime. ( 💫 皮特,我哪天去见下你的妈妈. )

🫧 You have treated me quite some times recently. Now it's my turn. ( 💫 你已经请我好几次了，这次轮到我请你了. )

🫧 I should stay home some time. ( 💫 我应该待在家一段时间. )

  1. farther、further

表示 地点距离远 时候，两个词 都能用，表示 更远.

如果要表示 更多、额外，要用 further.

具体的远 用 farther 或 further，抽象的远 用 further.

🫧 Can we walk farther / further tomorrow? ( 💫 我们明天能走远点吗? )

🫧 See under for further information. ( 💫 更多内容详见下文. )

  1. most、mostly

most ( 名词、形容词、副词 )表示 最、很、十分.

mostly ( 副词 ) 表示 主要地、大部分地.

🫧 The most outlandish ideas. ( 💫 最奇特的想法. )

🫧 The atoms themselves are mostly empty space. ( 💫 原子本身大部分的空间是空隙. )
