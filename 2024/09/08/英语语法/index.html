英语语法

英语语法

sparkle5202024-09-082024-10-02

🫧 指代 例子. 💫 指代 翻译.

英语语法基本框架

  • 10 种词性
  • 9 种成分
  • 8 种句型

10 种词性

  1. noun ( 名词 n. )
  2. pronoun ( 代词 pron. )
  3. numeral ( 数词 num. )
  4. article ( 冠词 art. )
  5. verb ( 动词 v. )
  6. adverb ( 副词 adv. )
  7. adjective ( 形容词 adj. )
  8. preposition ( 介词 prep. )
  9. conjunction ( 连词 conj. )
  10. interjection ( 感叹词 int. )

9 种成分

  1. subject ( 主 )
  2. predicate ( 谓 )
  3. object ( 宾 )
  4. attribute ( 定语 )
  5. adverbial ( 状语 )
  6. object complement ( 宾补 )
  7. subject complement ( 表 )
  8. double object ( 双宾 )
  9. apposition ( 同位 )

8 种句型

简单句

  • 主 + 谓
  • 主 + 谓 + 宾
  • 主 + 谓 + 表
  • 主 + 谓 + 双宾
  • 主 + 谓 + 宾 + 宾补

复合句

  • 名词性从句
  • 定语从句
  • 状语从句

4 种简单句

陈述句

陈述句的形式

陈述句分为两种形式，肯定式和否定式.

肯定式

🫧 I have an apple. ( 💫 我有一个苹果. )

否定式

肯定变否定，助动词 ( do ) 后面加否定词 ( not ).

🫧 I dont't ( do not ) have an apple. ( 💫 我没有苹果. )

陈述句的句型

陈述句分为五种句型

主 + 谓 ( 不及物动词 )

🫧 I run. ( 💫 我跑了. )

主 + 谓 + 宾

🫧 I love you. ( 💫 我爱你. )

主 + 系 + 表

🫧 I am Spider-Man. ( 💫 我是蜘蛛侠. )

系动词 = be 动词 ( is，am，are... ) / 感官动词 ( look，sound，smell，taste，feel... )

主 + 谓 + 双宾 ( 直接宾语 + 间接宾语 )

🫧 I give you money. ( 💫 我给你钱. )

判定技巧 当我们在宾语后面加上 be 动词，如果能构成一个语意通顺的句子，那么这个成分就很可能是宾语补足语，而不是双宾语中的直接宾语.

这种方法基于的原理是，宾语补足语是用来补充说明宾语的状态、身份或特征的，因此它与宾语之间的关系非常紧密，可以形成一个完整的句子. 而双宾语中的直接宾语和间接宾语之间并没有这种紧密的关系.

对当前例子中，you money 尝试加入 be 动词，变成 you are money 显然语意不通，故为双宾结构.

主 + 谓 + 宾 + 宾补

🫧 They ordered hamburger their lunch. ( 💫 他们的午餐他们点了汉堡. )

借用上个例子的判定技巧，可以轻易得出其为宾补.

疑问句

一般疑问句

陈述句变一般疑问句有三种方式 : 提前 系动词 ( be 动词  感官动词 ) 、提前 助动词 ( do，have... ) 、提前 情态动词 ( can，may，will... )

提前 系动词

主 + 系 + 表 系 + 主 + 表

陈述句 You are an idiot. ( 💫 你是笨蛋. )

一般疑问句 Are you an idiot? ( 💫 你是不是笨蛋? )

回答 Yes, I am. / No, I'm not.

提前 助动词

主 + 谓 + ( 宾 ) 助动词 + 主 + 动词原形 + 其他

陈述句 You have got a problem. You have a problem. ( 💫 你有一个问题. )

一般疑问句 Have you got a problem? Do you have a problem? ( 💫 你有什么问题吗? )

回答 Yes, I have. / No, I haven't.

  • 当 have 表示 动作，如表示 吃 ( =eat )，喝 ( =drink )，拿 ( =take ) 收到 ( =receive ) 度过 ( =spend ) 等时只能用 have，后面不能加 got，也不可以缩写，可用于进行时态，但一般不用于被动语态.
  • 表达过去式时不用 have got.

提前 情态动词

主 + 情态动词 + 动词原形 + 其他 情态动词 + 主 + 动词原形 + 其他

陈述句 You can play the piano. ( 💫 你会弹钢琴. )

一般疑问句 Can you play the piano? ( 💫 你会弹钢琴吗? )

回答 Yes, I can. / No, I can't.

特殊疑问句

对句子中某个成分提问，用 特殊疑问词 开头且不可用 yes，no 回答.

特殊提问词 ( who，whose，whom，what，when，where，why，how，how munch / many，which... )

🫧 Who is she? ( 💫 她是谁? )

选择疑问句

提问两个及以上，选择疑问用 or 连接且不可用 yes，no 回答.

🫧 Do you like to play the violin or the piano? ( 💫 你喜欢拉小提琴还是弹钢琴? )

回答 I like to play the violin. / I like to play the piano.

反义疑问句

前肯后否，前否后肯 且可用 yes，no 回答.

陈述句部分是肯定，疑问句部分用否定句

🫧 She is your sister, isn't she? ( 💫 她是你的姐姐，不是吗? )

回答 Yes, she is. / No, she isn't.

陈述句部分是否定，疑问句部分用肯定句 ( 💫 她不是你的姐姐，是吗? )

🫧 She isn't your sister, is she?

回答 Yes, she is. / No, she isn't. ( 根据 事实 回答 )

祈使句 的反义疑问句

用 will you 和 won't you 等.

🫧 Open the door, won't you? ( 💫 打开门，好吗? )

🫧 Don't open the door, will you? ( 💫 不打开门，好吗? )

  • 当祈使句为 Let's... 时，反义疑问句总是用 shall we.

🫧 Let’s phone her now, shall we? ( 💫 我们现在就给她打电话，好吗? )

  • 当祈使句为 Let us... 时，若表示 请求，反意疑问句用 will you，若表示 建议，反意疑问句用 shall we.

🫧 Let us know your address, will you? ( 💫 请把你的地址告诉我们，好吗? )

🫧 Let us go swimming together, shall we? ( 💫 我们一起去游泳，好吗? )

含有 must 的反义疑问句

  • 表 "必须"、"有必要" 时，疑问部分多用 mustn't.

🫧 I must take the medicine, mustn't I? ( 💫 我必须吃了这药，不是吗? )

  • 表 "禁止，不可以" 时，疑问部分用 must.

🫧 I mustn't go out, must I? ( 💫 我不能出门，对吗? )

  • 表 肯定推测 时，分 2 种情况

case-1 must + 动词原形 : 疑问部分跟这个 动词 保持 一致，比如 be 动词 — am / is / are ( 人称 / 肯否看情况 )

🫧 The Gods must be crazy, aren't they? ( 💫 上帝肯定是疯了，不是吗? )

case-2 "过去肯定做过某事" : must have done...，haven't / didn't + 主语?

🫧 She must have heard of the bad news, hasn't she? ( 💫 她肯定听说了这个坏消息，不是吗? )

如果有明确的表示过去的时间状语，疑问部分可以用 did.

感叹句 的反义疑问句

用感叹句陈述，用 否定 提问.

🫧 What a lovely girl, isn't she? ( 💫 多么可爱的女孩，不是吗? )

含有 否定词 的反义疑问句

常见的否定词 never，seldom，hardly，few，little，barely，scarcely，nothing，none，rarely one，nobody，neither...

当陈述句还有 否定词 是，用 肯定 提问.

🫧 He hardly goes to school, does he? ( 💫 他几乎不去学校，是吗? )

复合句 的反义疑问句

复合句反义疑问，都是对 主句 提问与主句相关. ( 除了宾语从句 )

🫧 She never told you that she had a baby, did she? ( 💫 她从来没有告诉过你她有一个孩子，是吗? )

含有 宾语从句 的反义疑问句

全部复合句里，只有宾语从句的反义疑问句，是和从句相关的.

don't think，believe，imagine，suppose 引导的宾语从句，反义疑问句与 从句主语，主谓一致，用 肯定 提问.

🫧 I don't imagine he has got it, has he? ( 💫 我没有想过他有它，他有吗? )

感叹句

What 感叹词

  • what + a / an + 形容词 + 可数名词单数 + 主语 + 谓语

🫧 What a beautiful photo it is ! ( 💫 这照片它多美啊 ! )

  • what + 形容词 + 不可数名词 / 名词复数 + 主语 + 谓语

🫧 What cleanly water it is ! ( 💫 多干净的水啊 ! )

How 感叹词

  • how + 形容词 + a / an + 可数名词单数 + 主语 + 谓语

🫧 How cute a dog it is ! ( 💫 它是多么可爱的一只狗啊 ! )

  • how + 形容词 / 副词 + 主语 + 谓语

🫧 How fast he runs ! ( 💫 他跑得多快啊 ! )

语气词 + 感叹号 的感叹句

🫧 Wow ! That's totally awesome ! ( 💫 哇 !真是厉害 ! )

祈使句

命令、警告 祈使句

🫧 Get out ! ( 💫 滚 ! )

🫧 Be quiet, please. ( 💫 请安静. )

🫧 Be careful ! ( 💫 小心 ! )

请求、祝愿 祈使句

🫧 Have a good day ! ( 💫 祝你有美好的一天 ! )

🫧 Let me help you. ( 💫 让我帮你. )

10 种词性

数词

数词分为基数词和序数词.

基数词

数字表达规律

  • 从右往左，每三位加一个逗号，倒数第 一 个逗号之前是 thousand，倒数第 二 个逗号之前是 million，倒数第 三 个逗号之前是 billion，倒数第 四个 逗号之前是 trilion.
  • 逗号之间的百位和十位要加 and，数字隔零要加 and.

🫧 one hundred ( 💫 100 )

🫧 one hundred and one ( 💫 101 )

🫧 nine hundred and ninety-six ( 💫 996 )

🫧 sixty million two hundred and sixty-three thousand one hundred and fifty ( 💫 60，263，150 )

  • 基数词的单位 hundred，thousand，million 和 billion 等用来表示具体数目，前面有具体的基数词时，这些词的后面不加 -s.

🫧 There are nine hundred teachers in our school. ( 💫 我们学校有9000名老师. )

  • 当 hundred，thousand，million 和 billion 等用来表示不具体的数目时用 hundreds of，thousands of，millions of，billions of ＋ 名词 来表示，这些词的前面也不用具体的基数词.

🫧 We have already planted thousands of trees on the farm. ( 💫 我们在农场里已经种植了成千上万棵树. )

序数词

  • 序数词在句子中主要作定语，有时也可以用作主语、宾语和表语.

🫧 I am the third in this exam. ( 💫 我这次考试是第一名. )

定冠词 the 和 序数词 连用表示顺序. 不定冠词 a/an 和 序数词 连用表示 "又一、再一" 的意思.

🫧 You are the first and I am the second. ( 💫 你第一，我第二. )

🫧 Would you repeat it a third time? ( 💫 你能再重复一次吗? )

分数

分数是以 基数词 代表 分子，序数词 代表 分母，除了分子是 "1" 的情况外，分母 的序数词都要用 复数.

🫧 a half ( 💫 1 / 2 )

🫧 one third ( 💫 1 / 3 )

🫧 two thirds ( 💫 2 / 3 )

🫧 a (one) quarter ( 💫 1 / 4 )

分数在句子中作主语的时候，谓语动词的单、复数形式由 of 后面的名词来决定.

  • 如果 of 后面的名词是可数名词的复数形式，谓语动词用 复数形式.

🫧 Three fifths of the boys in this room are 17 years old. ( 💫 这个房间五分之三的男孩子 17 岁. )

  • 如果 of 后面的名词是 不可数名词，谓语动词用 第三人称单数形式.

🫧 It is said that two thirds of information on the Internet is not the fact. ( 💫 据说互联网上三分之二的信息不是事实. ( Information 是不可数名词 ) )

年

🫧 nineteen ninety-four ( 💫 1994 )

🫧 two thousand ( 💫 2000 )

🫧 two thousand and eight ( 💫 2008 )

日期

🫧 September the first, two thousand and nine ( 💫 September 1, 2009 )

时刻

🫧 nine o’clock ( 💫 9 : 00 )

🫧 ten past nine / nine ten ( 💫 9 : 10 )

🫧 ( a ) quarter past nine / nine fifteen ( 💫 9 : 15 )

🫧 half past nine / nine thirty ( 💫 9 : 30 )

🫧 twenty to ten / nine forty ( 💫 9 : 40 )

🫧 ( a ) quarter to ten / nine forty-five ( 💫 9 : 45 )

电话号码

🫧 six four seven - three double o - two five one six ( 💫 647 - 300 - 2516 )

价格

通常省去 dollars 和 cents.

🫧 ninety-nine ninety-eight / ninety-nine dollars and ninety-eight cents ( 💫 99.98$ )

小数

通常省去 zero 或 读成 o.

🫧 point five / zero point five ( 💫 零点五 )

🫧 point o five (💫 零点零五 )

两个一样的数字，三个一样的数字

两个一样用 double，三个一样用 triple.

🫧 double oseven ( 💫 零零七 ) 🫧 boeing triple seven ( 💫 波音777 )

冠词

冠词是置于名词前并限定名词的意义的虚词.

定冠词 只有 the 这一种形式.

不定冠词 只有 a 和 an 这两种形式.

名词前若 不用冠词 时，又称作 零冠词.

定冠词 主要特点是 "确指" 或 "特指".

不定冠词 主要特点就是 "不确定" 或 "泛指".

定冠词

  1. 特指 某 ( 些 ) 人 或 某 ( 些 ) 物，上文提到 的，双方都知道的.

🫧 The blue shirt is mine. ( 💫 蓝衬衫是我的. )

  1. 指世界上 独一无二 的事物.

🫧 the sun ( 💫 太阳 )

  1. 用在形容词或副词的 最高级 前.

🫧 This is the best cup of coffee. ( 💫 这个最好的一杯咖啡. )

  1. 用在 姓氏复数 形式前，表示 "全家人".

🫧 The Smiths live in the apartment above ours. ( 💫 史密斯一家住在我们楼上的一套住房里. )

  1. 用在 序数词，方位词 前.

🫧 on the left ( 💫 在左边 )

🫧 He was the first man to think of it. ( 💫 他是第一个想到这一点的人. )

  1. 用在 乐器、棋类 前.

🫧 play the piano ( 💫 弹钢琴 )

  1. 用在表示 海洋，河流，山脉，群岛 及 国家 和 党派 等名词前.

🫧 the Black Sea ( 💫 黑海 )

🫧 the Yangzi River ( 💫 长江 )

  1. 用在由普通名词构成的 专有名词 前.

🫧 the United States ( 💫 美国 )

  1. 用于某些 固定词组.

🫧 all the same ( 💫 仍然 )

🫧 all the time ( 💫 一直 )

🫧 at the moment ( 💫 此刻 )

🫧 at the same time ( 💫 同时 )

🫧 by the way ( 💫 顺便说 )

🫧 do the shopping / washing ( 💫 买东西 / 洗衣服 )

🫧 in the morning / afternoon / evening ( 💫 在上午 / 下午 / 晚上 )

🫧 in the open air ( 💫 在户外，在野外 )

不定冠词

辅音 发音前面加 a，元音 发音前面加 an.

  1. 用来表示 "一" 这个数量.

🫧 Rome was not built in a day. ( 💫 罗马不是一天建成的. )

🫧 He will be back in a day or two. ( 💫 他将在一至两天内返回. )

  1. 泛指 某 一类人、事 或 物.

🫧 A grammar book is necessary to a language learmer. ( 💫 对语言学习者来说，语法书是很必要的. )

  1. 表示 "每一"，相当于 every.

🫧 I go to school five days a week. ( 💫 我一周上五天课. )

  1. 用在序数词前，表示 "又一","再一".

🫧 I have three books, I want to buy a fourth one. ( 💫 我已经有三本书，我想买第四本. )

  1. 用在某些固定词组中.

🫧 a lot ( of ) ( 💫 许多，大量 )

🫧 after a while ( 💫 过一会儿 )

🫧 have a rest ( 💫 休息 )

🫧 have a look ( 💫 瞧瞧 )

🫧 have a good time ( 💫 过得开心 )

🫧 take a shower ( 💫 淋浴 )

🫧 take a break ( 💫 休息 )

🫧 take a bath ( 💫 洗澡 )

🫧 get a cold ( 💫 患感冒 )

🫧 get a fever ( 💫 发烧 )

🫧 make a living ( 💫 谋生 )

🫧 in a moment ( 💫 过一会儿 )

零冠词

  1. 很 广泛 的 概念.

🫧 Sichuan dishes are spicy. ( 💫 四川菜很辣. )

  1. 书名、人名.

🫧 I am reading "Journey to the West". ( 💫 我正在读西游记. )

  1. 某些城市、国家.

🫧 China is a big country. ( 💫 中国是一个大国. )

  1. 某些山河湖海洲.

🫧 Asia ( 💫 亚洲 )

  1. 星球.

🫧 Mars ( 💫 火星 )

  1. 语言、球类、学科、棋牌.

🫧 I have studied English for ten years. ( 💫 我学了十年英语. )

  1. 三餐、日期、季节 ( 一般不加冠词，如果要特指可以加 the )

🫧 It's September 1th ( 💫 这是九月一日. )

名词

专有名词 和 普通名词

专有名词

专有名词大多数是不可数名词，在某些情况下可以复数使用，或者转换为普通名词，单数 / 复数使用，例如 : 人名、地名、国家、组织、单位、大型活动、著名事件 等 ( 首字母大写、如果缩写全部字母大写 )

人名 : Jack ( 💫 杰克 )

单位 : Department of Education ( 💫 教育部 )

地名 : Beiing ( 💫 北京 )

大型活动 : Midi Festival ( 💫 迷笛音乐节 )

国家 : China ( 💫 中国 )

著名事件 : Columbus's discovery of America ( 💫 哥伦布发现新大陆 )

组织 : FBI ( 💫 美国联邦调查局 )

普通名词

专有名词以外的名词.

专有名词复数使用，专业名词转换普通名词

🫧 There are two Jacks in our class. ( 💫 我们班有两个杰克. )

🫧 Millions of Lei Fengs have emerged in China. ( 💫 中国出现了数百万的活雷锋. )

🫧 Hey, Tom, have a Mrs. Carl is waiting for you. ( 💫 嘿，汤姆，有一位卡尔夫人正在等你. )

可数名词 和 不可数名词

可数名词

单数复数都可以，可以加数词计算 例如 : apple，egg，boy，tree，family，class，police，team...

不可数名词

只能单数，不可以加数词计算 例如 : water，fire，time，hope，knowledge，intelligence...

可数名词 和 不可数名词 用法注意

可数名词 能 计数.

可数名词 能 加不定冠词 ( a / an ) 和数词 ( one，two... ).

可数名词 能 加定冠词 ( the ).

不可数名词 不能 计数.

不可数名词 不能 加不定冠词 ( a / an ) 和数词 ( one，two... ).

不可数名词 能 加定冠词 ( the ).

具体名词 和 抽象名词

具体名词

看得见摸得着，可数不可数都有 例如 : man，Beijing，book，car，water，fire...

抽象名词

无形的各种事物，大多数是不可数 例如 : time，hope，knowledge，love，anger...

名词单复数变化

一般 在名词词尾加 "-s".

🫧 pencils

以 s，x，ch，sh，o 结尾的名词加 "-es”.

🫧 boxes

其中 o 比较特殊，只有单词意思是 动植物 才加 es，如果 不是，就加 s.

🫧 potatoes

🫧 heroes

🫧 pianos

🫧 radios

以 辅音字母 加 y 结尾的名词，变 y 为 i 加 "-es".

🫧 babies

如果以元音字母 加 y 结尾的名词直接加 "-s".

🫧 toys

以 fe 或 f 结尾的名词，把 fe 或 f 变为 v 加 "-es".

🫧 knives

🫧 wives

名词单数变复数的不规则变化

🫧 child—children

🫧 foot—feet

🫧 tooth—teeth

🫧 mouse—mice

🫧 man—men

🫧 woman—women

单复数同形的名词

🫧 deer

🫧 sheep

🫧 fish

🫧 Chinese

🫧 Japanese

🫧 yuan

复合名词的变化

  1. 不可数名词结尾，无复数 形式.

🫧 homework

🫧 newspaper

🫧 greenhouse

  1. man 或 woman 做 前缀，前后 都要 复数.

🫧 actor-men actors

  1. 两个名词，后面 变复数.

🫧 friend-boy friends

  1. 可数名词 + 介词 ( 短语 ) ，前面 名词复数.

🫧 law-sisters in law

  1. 动词 / 过去分词 + 副词，加 "-s".

🫧 by-stand bys

集体名词，以单数形式出现，但实为复数

例如 people police cattle... 本身就是 复数，不能说 a people，a police，a cattle，但可以说 a person，a policeman，a head of cattle.

the English，the British，the French，the Chinese，the Japanese... 表示 国民总称 时，作 复数 用.

🫧 The Chinese are hard-working and brave. ( 💫 中国人民是勤劳勇敢的. )

名词所有格

单数名词 / 复数名词非 s 结尾，后加 's，单数名词 s 结尾，后加 's.

🫧 Jack's ( 💫 杰克的)

🫧 The boss's car ( 💫 老板的车 )

复数名词 s 结尾，后加 '

🫧 Boys' ( 💫 男孩的 )

用 and 连接并列名词时候表示各自的所有关系，分别加 's，共同所有关系，只在最后一个名词加 's.

🫧 Tom's and Jack's rooms ( 🫧 汤姆的房间和杰克的房间 )

🫧 Tom and Jack's room ( 🫧 汤姆和杰克的房间 )

of 所有格

中英语序相反修饰前面的名词，表两个名词之间的所属关系.

🫧 The content of the book = The book's content ( 💫 这本书的内容 )

's + of 双重所有格

of + 名词's 所有格

🫧 This is a photo of my cat's. ( 💫 这是我猫的照片. )

of + 名词性物主代词

🫧 He is a friend of mine. ( 💫 他是我的朋友. )

名词主谓一致

主语的单复数和谓语的单复数要保证一致.

🫧 I ( 单数 ) am ( 单数 ) student. ( 💫 我是学生. )

🫧 They ( 复数 ) are ( 复数 ) students. ( 💫 他们是学生. )

国家、机构、书名 等 专有名词 的复数形式看作 整体，谓语用 单数.

🫧 The United State was founded in 1776. ( 💫 美国建立于 1776 年. )

🫧 "Three Kingdoms" is a good story. ( 💫 三国演义是个好故事. )

many a... 或 more than one + 名词 做主语，谓语用 单数.

🫧 Many a man is smoking. ( 💫 很多男人在抽烟. )

🫧 More than one man is smoking. ( 💫 不止一个男人在抽烟. )

one and a half + 复数名词 做主语，谓语用 单数.

🫧 One and a half watermelons is in the fridge. ( 💫 冰箱里有一个半西瓜. )

时间、钱、距离 等各种 单位 看作 整体，谓语用 单数.

🫧 Two dollars is no big deal. ( 💫 两美元没什么大不了. )

🫧 Four miles is not a long way. ( 💫 四英里的路不远. )

加减乘除 运算中，谓语用 单数.

🫧 Five times five is twenty-five. ( 💫 5 乘 5 是 25. )

🫧 One plus one is two. ( 💫 1 加 1 是 2. )

each，one，no one，some ( any，no，every ) + body ( one，thing ) 作主语或作限定词，谓语用 单数.

🫧 Each student is in the classroom. ( 💫 每 1 个学生都在教室里. )

🫧 Someone student is in the classroom. ( 💫 某个学生在教室里. )

one + of + 复数名词 结构中，谓语用 单数.

🫧 One of the students in our school is from the United States. ( 💫 我们学校有一名来自美国的学生. )

含有集体概念的名词，根据上下文判断，整体概念 谓语用 单数，个体概念 谓语用 复数.

🫧 The family was in a poor way. ( 💫 这个家庭生活贫困. ) ( 整体概念，这个家庭. )

🫧 The family gose to the movies together. ( 💫 这家人一起去看电影. ) ( 个体概念，这个家的人. )

含有单复数形式一样的名词作主语，根据 上下文判断，谓语用单数还是复数.

🫧 A fish was back in the water. ( 💫 一条鱼回到了水中. )

🫧 Some fish are still alive. ( 💫 有些鱼还活着. )

a number of + 可数名词复数 作主语，泛指 许多不一定多少，谓语用 复数.

🫧 A number of scientists are single. ( 💫 许多科学家都是单身. )

the number of + 可数名词复数 作主语，特指 数量是多少，谓语用 单数.

🫧 The number of scientists is in the lab is five. ( 💫 实验室里的科学家人数是五人. )

and 或 both...and...，连接两个单数名词做主语代表复数的概念，大多数情况谓语用 复数.

🫧 He and him families are at the funeral. ( 💫 他和他的家人在参加葬礼. )

🫧 Both China and the United States are big countries. ( 💫 中国和美国都是大国. )

and 连接两个单数词，指 同一人同一事物 或 同一概念，两个名词共用一个冠词，谓语动词用 单数.

🫧 The Youtuber and traveler is he friend. ( 💫 这位旅行家兼 YouTube 视频博主是他的朋友. )

and 连接两个单数词，主语前有 every，each，no，many a... 修饰时，谓语用 单数.

🫧 Each day and night is busy. ( 💫 每天都从早忙到晚. )

a / an + 单数名词 + or two 做主语，谓语用 单数.

🫧 An apple or two is not enough. ( 💫 一两个苹果是不够的. )

one or two + 复数名词 做主语，谓语用 复数.

🫧 One or two apples are not enough. ( 💫 一两个苹果是不够的. )

some of，plenty of，a lot of，lots of，most of，the rest of，all of，half of，part of，分数 / 百分百 + of + 名词短语，作主语，谓语动词与 of 后 的名词，保持单复数.

🫧 Most of the student was gone. ( 💫 大多数学生都走了. )

🫧 Most of the students were gone. ( 💫 大多数学生都走了. )

few，many，several，all，some 后面加名词做主语，谓语用复数.

🫧 The few words are difficult. ( 💫 这几句话好难. )

就近原则 ( 距离谓语 近 的名词，决定谓语单复数. ) There be / Here be.

🫧 There is one computer and two phones. ( 💫 那有一台电脑和两台手机. )

🫧 There are two computers and one phone. ( 💫 那有两台电脑和一台手机. )

🫧 Here is one apple and two bananas. ( 💫 这有一个苹果和两个香蕉. )

🫧 Here are two apples and one banana. ( 💫 这有两个苹果和一个香蕉. )

就近原则 ( 距离谓语 近 的名词，决定谓语单复数.)

or，either...or... ( 二选一 )，neither...nor... ( 两个都不选，选另外一个 )，not only...but... ( also ) ( 不仅而且，两个都 ) 连接并列做主语.

🫧 His friends or he was dead. ( 💫 他的朋友们或者他已经死了. )

🫧 He or his friends were dead. ( 💫 作也或者他的朋友们已经死了. )

🫧 Either my brothers or mother is going to the garden. ( 💫 我弟弟或者妈妈要去花园. )

🫧 Either mother or my brothers are going to the garden. ( 💫 妈妈或者我弟弟要去花园. )

🫧 Neither two apples nor one banana is vegetable .( 💫 两个苹果和一个香蕉都不是蔬菜. )

🫧 Neither one apple nor two bananas are vegetables. ( 💫 一个苹果和两个香蕉都不是蔬菜. )

🫧 Not only my friends but also my girlfriend was wrong. ( 💫 不仅是我朋友我女朋友也错了. )

🫧 Not only my girlfriend but also my friends were wrong. ( 💫 不仅是我女朋友我朋友也错了. )

就远原则 ( 距离谓语 远 的名词，决定谓语单复数. )

  • 四除 ( but，except，besides，in addition to )

  • 五和 ( with，along with，together with，combined with，as well as )

  • 包括 ( including )

  • 而不是 ( rather than，instead of )
