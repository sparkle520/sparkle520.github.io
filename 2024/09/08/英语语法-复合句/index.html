复合句 ( Compound Sentences )

复合句 由一个 主句 ( Main Clause ) 和一个或多个 从句 ( Subordinate Clause ) 构成. 从句在主句中充当某种成分 ( 名词、形容词或副词 )，因此分为三大类 名词性从句、定语从句 和 状语从句.

一、名词性从句 ( Noun Clauses )

名词性从句在句子中起 名词 的作用，充当主语、宾语、表语或同位语. 在考研英语中，识别从句边界、理解 “What” 的指代意义 以及 长难句中的嵌套结构 是考察重点.

1.1 引导词体系构建

三类引导词的功能辨析

类别 词汇 句法功能 语义功能
连词 ( Conjunctions ) that 不充当成分 ( 仅起连接作用 ) 无词义 ( 仅表陈述事实 )
whether / if 不充当成分 表 “是否” ( 不确定性 )
连接代词 ( Pronouns ) what / whatever 主语 / 宾语 / 表语 什么 / 无论什么 ( = the thing that )
who / whoever 主语 / 宾语 / 表语 谁 / 无论谁 ( = the person who )
which / whichever 定语 ( 修饰名词 ) 哪一个
连接副词 ( Adverbs ) when, where, why, how 状语 何时，何地，原因，方式

1.2 主语从句 ( Subject Clauses )

考研侧重 形式主语 it 的识别与 What 从句的翻译.

三种常见形态

  1. It 作形式主语 ( 平衡句式 )
    • 常用于 “It is + adj./noun/done + that…” 结构.
    • 🫧 It is common knowledge that the earth is round. ( 💫 众所周知… )
    • 高频句型 It is reported that… ( 据报道 ), It occurs to me that… ( 我突然想到 ).
  2. That 引导 ( 句首不可省 )
    • 置于句首表达正式、强调的语气.
    • 🫧 That he refused the offer surprised us. ( 💫 他拒绝了邀请，这让我们很惊讶. )
  3. What 引导 ( 代词性 )
    • What = The thing that. 谓语动词单复数视语义而定.
    • 🫧 What we need is time. ( 💫 我们需要的是时间. )
    • 🫧 What matter are not words but actions. ( 💫 重要的不是言语而是行动. )

1.3 宾语从句 ( Object Clauses )

考研侧重 否定转移、介词宾语从句及形式宾语.

核心考点

  1. 否定转移 ( Negative Transfer )
    • 主句主语是第一人称，谓语是 think, believe, suppose, expect 等时，否定词需前移.
    • 🫧 I don’t think you are right. ( 💫 我认为你是不对的. )
  2. 介词后的宾语从句
    • 介词后通常接 wh- 类引导词，慎用 that ( 除 except that, in that, but that 等少数固定搭配外 ).
    • 🫧 He has no idea of what he should do. ( 💫 他不知道该做什么. )
  3. 形式宾语 it
    • 结构 Subject + Verb + it ( 形式宾语 ) + Adj./Noun + that/to do ( 真正宾语 ).
    • 🫧 I make it clear that I object to the plan. ( 💫 我明确表示反对这个计划. )

1.4 表语从句 ( Predicative Clauses )

位于系动词 ( be, remain, look, seem ) 之后，解释主语的内容.

  • The reason is that… ( 考研写作推荐，避免用 because ).
    • 🫧 The reason for his failure is that he was lazy.
  • It looks/seems as if/as though… ( 看起来好像… ).
    • 若描述非真实情况，需用 虚拟语气.

1.5 同位语从句 ( Appositive Clauses )

考研侧重 抽象名词后的解释说明，常被误判为定语从句.

同位语从句 vs 定语从句 ( 必考辨析 )

维度 同位语从句 定语从句
先行词 抽象名词 ( fact, evidence, belief, doubt, possibility ) 任何名词
That 的作用 连词 ( 只连接，不充当成分，不可省 ) 关系代词 ( 充当主/宾语，作宾语可省 )
语义关系 解释 名词的具体内容 ( Equal ) 修饰/限定 名词的属性 ( Attribute )
判定口诀 后面句子 完整 \(\rightarrow\) 同位语 后面句子 缺成分 \(\rightarrow\) 定语
示例 The fact that he is dead suprises me.
( 他死了这个事实 \(\rightarrow\) 句意完整 )		 The fact (that) he told me is true.
( 他告诉我的事实 \(\rightarrow\) told 缺宾语 )

考研难点攻关

1. What vs That 的终极判决

  • What 既是引导词，又是从句的一个 成分 ( 主/宾/表 )，有实际含义.
  • That 只是引导词，不充当成分，无实际含义.
  • 🫧 I don’t know what he said. ( said 缺宾语 \(\rightarrow\) 用 what )
  • 🫧 I don’t know that he had left. ( had left 完整 \(\rightarrow\) 用 that )

2. Whether vs If 的使用禁区

虽然都表 “是否”，但以下情况 只能用 Whether

  • 位于句首作主语时 ( Whether it rains matters little. )
  • 作介词宾语时 ( It depends on whether… )
  • 与 or not 紧密连用时 ( whether or not… )
  • 引导表语从句或同位语从句时

3. 名词性从句中的虚拟语气

在表示 “建议/命令/要求” (suggest, order, demand, propose) 的动词后的宾语从句，或其名词形式 (suggestion, order) 后的表语/同位语从句中，谓语用 “( should ) + 动词原形”.

  • 🫧 He suggested that we (should) go at once.

二、定语从句 ( Attributive Clauses )

定语从句在句中作 定语，修饰名词或代词 ( 称为 先行词 ). 考研英语中，定语从句常用于从句套从句的长难句中，需重点掌握 关系词选择、介词+关系代词 及 特殊句型.

2.1 关系词体系构建

关系代词 & 关系副词

词汇 先行词类型 在从句中的成分 备注
who 人 主语 / 宾语 主格
whom 人 宾语 宾格 ( 口语常用 who 代替 )
whose 人 / 物 定语 表 “……的”
which 物 主语 / 宾语
that 人 / 物 主语 / 宾语 万能词 ( 有限制条件 )
as 句子 / 物 主语 / 宾语 表 “正如”，位置灵活
when 时间 状语 = at/in/on + which
where 地点 状语 = at/in/on + which
why 原因 状语 = for + which

2.2 考研必考 介词 + 关系代词

在正式文体或考研长难句中，常出现 “介词 + 关系代词” 结构.

核心规则

  1. 介词前置 时，关系代词只能用 which ( 指物 ) 或 whom ( 指人 )，不可用 that.
  2. 介词的选择依据
    • 与从句动词/形容词搭配 The book on which I spent 5 dollars. ( spend…on )
    • 与先行词搭配 The reason for which he was late. ( reason for )
    • 根据句意逻辑 The telescope through which we observe stars. ( 通过望远镜 )
  3. 复杂结构 “名词/代词/数词 + of + which/whom”
    • I have many friends, most of whom are students. ( = most of the friends )

2.3 限制性 vs 非限制性

两种定语从句的对比

比较项目 限制性定语从句 ( Restrictive ) 非限制性定语从句 ( Non-restrictive )
形式 无逗号，紧跟先行词 有逗号，与主句隔开
功能 修饰/限定 先行词，是句子必要成分 补充说明 先行词，去掉不影响主句完整性
关系词 可用 that 不可用 that，常用 which, as, who
翻译 译为 “…的…” 译为主句的并列句 ( “，他/它…” )
示例 🫧 He has a son who is a doctor.
( 💫 他有一个当医生的儿子. )
> 暗示可能有多个儿子，其中一个是医生.		 🫧 He has a son, who is a doctor.
( 💫 他有一个儿子，他是医生. )
> 暗示只有一个儿子.

2.4 考研难点 As 引导的定语从句

as 引导非限制性定语从句时，指代 整个主句 的内容，意为 “正如，正像”.

As vs Which ( 非限制性定语从句 )

特性 As Which
位置 灵活 ( 句首、句中、句末 ) 通常在 句末 ( 紧跟主句 )
含义 “正如，正像” ( 表示主句内容符合预期/规律 ) “这” ( 单纯指代主句这件事 )
搭配 as is known to all ( 众所周知 )
as we expected ( 正如我们需要 )		 which makes me angry ( 这让我很生气 )
which is a fact ( 这是一个事实 )
示例 🫧 As is known to all, the earth is round.
( 💫 众所周知，地球是圆的. )		 🫧 He was late, which made the boss angry.
( 💫 他迟到了，这让老板很生气. )

固定搭配 such … as, the same … as ( 像 … 一样的 )

  • 🫧 I like the same book as you do. ( 💫 我喜欢和你一样的书. )

2.5 考研长难句 分隔定语从句

先行词与定语从句之间被其他成分 ( 如状语、同位语 ) 隔开，造成阅读障碍.

  • 🫧 The days are gone forever when we could do whatever we wanted.
    • 解析 when … 引导的定语从句修饰 The days，被 are gone forever 隔开.
    • 翻译 我们随心所欲的日子一去不复返了.

三、状语从句 ( Adverbial Clauses )

状语从句在复合句中修饰主句中的动词、形容词或副词，表示时间、地点、原因、条件、让步、方式、比较、目的或结果. 考研英语中，逻辑关系词的识别、特殊句式 ( 倒装/省略 ) 及 长难句的切分 是核心考点.

3.1 九大状语从句逻辑图谱

核心引导词与逻辑功能

类型 逻辑关键词 ( 考研高频 ) 考点备注
时间 when, while, as ( 当 … 时 )
before, after ( 在 … 前后 )
until, not … until ( 直到 … 才 )
as soon as, the moment ( 一 … 就 )		 while 强调同时进行
until 延续性动词限制
the moment 名词作连词
地点 where ( 在 … 地方 )
wherever ( 无论哪里 )		 Where there is a will, there is a way.
原因 because ( 语气最强 )
since, as ( 既然/由于 )
in that, now that ( 既然/因为 )		 in that 书面语常用
since 侧重已知事实
条件 if ( 如果 )
unless ( 除非 = if not )
as long as, provided that ( 只要 )		 unless 引导否定条件
provided that 正式文体
让步 although, though, even if/though ( 虽然/即使 )
while ( 尽管/虽然 )
no matter wh- / however ( 无论 )		 while 放句首表让步
as 引导让步需倒装
方式 as ( 按照 )
as if / as though ( 好像 )		 as if 常接虚拟语气
比较 as … as ( 和 … 一样 )
than ( 比 )
the more … the more ( 越 … 越 … )		 涉及省略和倒装结构
目的 so that, in order that ( 为了 )
lest, for fear that ( 以免 )		 lest 后接 ( should ) do
结果 so … that ( 如此 … 以至于 )
such … that ( 如此 … 以至于 )		 区分 so ( 副词 ) 与 such ( 形容词 )

3.2 考研必考 易混逻辑词辨析

易混逻辑词辨析

1. While 的 “三副面孔”

  • 时间 当 … 时候 ( 强调动作延续 ).
    • Strike while the iron is hot. ( 趁热打铁. )
  • 让步 虽然/尽管 ( 通常置于 句首 ).
    • While I understand your point, I disagree. ( 虽然我懂你的意思，但我不同意. )
  • 对比 而/然而 ( 连接两个对立事物 ).
    • He likes sports, while I prefer reading. ( 他喜欢运动，而我喜欢阅读. )

2. As 的 “万能用法”

  • 时间 随着/当 … 时 ( As time goes by… )
  • 原因 由于 ( As it is raining… )
  • 方式 按照 ( Do as I say. )
  • 让步 虽然 ( 需倒装 Child as he is, he knows a lot. )

3. Since 的 “双重身份”

  • 时间 自从 ( 主句常用完成时 ).
    • We have been friends since we met.
  • 原因 既然/由于 ( 侧重已知原因 ).
    • Since everyone is here, let’s start.

3.3 考研难点 特殊句式

特殊句式规则

1. “主将从现” 原则 ( 时态呼应 )

在 时间 和 条件 状语从句中，若主句是将来时，从句必须用 一般现在时 表将来.

  • 🫧 I will tell him as soon as he arrives. ( \(\checkmark\) )

2. As 引导让步状语从句的倒装

结构 Adj./Adv./Noun/Verb + as + 主语 + 谓语

  • 🫧 Young as he is, he is experienced. ( 虽然他年轻 … )
  • 🫧 Try as he might, he couldn’t open the door. ( 尽管他试过 … )

3. So … that vs Such … that

  • so + adj./adv. + that
    • so beautiful that…
  • such + (a/an) + adj. + noun + that
    • such a beautiful flower that…
  • 特例 当名词前有 many, much, few, little 修饰时，用 so
    • so many people that…

3.4 考研长难句实战 状语从句的省略

当状语从句的主语与主句 主语一致，且从句谓语含 be 动词 时，可省略 “主语 + be”.

  • 省略前 When he was asked about the accident, he kept silent.
  • 省略后 When asked about the accident, he kept silent.
    • ( 当被问及事故时，他保持沉默. )
  • 常见结构 if possible ( 如果可能 ), when necessary ( 必要时 ), while reading ( 阅读时 ).

全文总结 ( Summary )

  • 名词性从句 : 抓 “成分” ( 主/宾/表/同 )，辨 “What” ( 成分+含义 )，防 “陷阱” ( 否定转移/虚拟 ).
  • 定语从句 : 定 “先行词”，选 “关系词” ( 介词前用 which )，分 “限/非限” ( 逗号/As ).
  • 状语从句 : 识 “逻辑” ( 9大类 )，通 “特殊” ( 倒装/省略/While多义 )，守 “时态” ( 主将从现 ).
