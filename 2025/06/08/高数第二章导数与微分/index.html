第二章 导数与微分

本章主要介绍导数与微分的概念、计算法则以及导数的应用 ( 极值、凹凸性、曲率等 )，是考研数学的重点章节，也是后续积分学的基础.

一、导数与微分的概念 ( Concept )

1.1 微分的定义

设 \(y=f(x)\) 在 \(x=x_0\) 的某邻域 \(U(x_0)\) 内有定义，并设 \(x_{0}+\Delta x\in U(x_{0}).\) 如果

\[\Delta y=f(x_{0}+\Delta x)-f(x_{0})\xrightarrow{可写成}A\Delta x+o(\Delta x)\:,\]

其中 \(A\) 与 \(\Delta x\) 无关，\(\displaystyle\lim_{\Delta x\to_0}\displaystyle\frac{o(\Delta x)}{\Delta x}=0\)，则称 \(f(x)\) 在点 \(x=x_0\) 处可微，并称 \(A\Delta x\) 为 \(f(x)\) 在\(x=x_0\) 处的微分，记为 \(dy= A\Delta x.\) 又因自变量的增量 \(\Delta x\) 等于自变量的微分 \(dx\)，于是 \(dy\) 又可写成 \(dy=Adx.\)

注 以 直 代 曲，这便是微分的含义.

1.2 微分与增量的几何关系

首先先要明确的是，\(d\) 代表的是微观的意义 ( 微 )，而 \(\Delta\) 代表的是宏观的概念 ( 宏 ).

由下图，易知

  1. \(dx = \Delta x.\)
  2. \(dy \neq \Delta y.\)
  3. \(ds \neq \Delta s\) ( \(ds\) 为直线 \(T\) ，\(\Delta s\) 为曲线 \(f(x)\) ).

额外的，还有 \((\Delta s)^2 \approx (\Delta x)^2 + (\Delta y)^2\quad (ds)^2 = (dx)^2 + (dy)^2\) ( 此处 “\(=\)” 代表化曲为直 ).

1.3 可导、可微与连续的关系

在一元函数微分学中，可导 \(=\) 可微 ，可导 \(\Rightarrow\) 连续 ，连续 \(\nRightarrow\) 可导.

且对任意的 \(x\) 有，

\[dy = f'(x)dx.\]

1.4 微分与增量的近似关系

设 \(y=f(x)\) 在 \(x_0\) 处可导 ( 可微 ) ，则有

\[\Delta y=dy+ o( \Delta x)\quad\text{或者}\quad \Delta y=f'(x_0)\Delta x+o(\Delta x).\]

若又设在含有 \(x_0\) 的某区间内存在二阶导数，则由拉格朗日余项泰勒公式有

\[\Delta y-dy=\displaystyle\frac{1}{2}f^{\prime\prime}\left(\xi\right)\left(\Delta x\right)^{2},\]

其中 \(\xi\) 介于 \(x_0\) 与 \(x_0 + \Delta x\) 之间.

1.5 微分形式不变性

对于 \(y = f(u)\) 不管 \(u\) 是自变量还是中间变量，函数的微分的表达式都有形式不变性，即 \(dy = f'(u)du.\)

二、导数的计算 ( Calculation )

2.1 变限积分求导

设 \(f(t)\) 为连续函数，\(\varphi_1(x)\) 与 \(\varphi_2(x)\) 均可导，则有

\[\left[\int_{\varphi_1(x)}^{\varphi_2(x)}f\left(t\right)\mathrm{d}t\right]^{\prime}=f\left(\varphi_2(x)\right)\varphi_2^{\prime}(x)-f\left(\varphi_1(x)\right)\varphi_1^{\prime}(x).\]

2.2 高阶导数运算法则

以下均设 \(u,v\) 为 \(n\) 阶可导，则有

  1. \((u\pm v)^{(n)}=u^{(n)}\pm v^{(n)}.\)
  2. \((cu)^{(n)}=cu^{(n)}.\)
  3. \((uv)^{(n)}=u^{(n)}v+\mathrm{C}_{n}^{1}u^{(n-1)}v^{\prime}+\cdots+\mathrm{C}_{n}^{k}u^{(n-k)}v^{(k)}+\cdots+uv^{(n)}.\)

最后一个公式称为乘积的高阶导数的 莱布尼茨公式.

2.3 常见函数的高阶导数

  1. \(\left(e^{ax}\right)^{\left(n\right)}=a^{n}e^{ax}.\)
  2. \((\sin ax)^{(n)}=a^{n}\sin\left(\displaystyle\frac{n\pi}{2}+ax\right).\)
  3. \((\cos ax)^{(n)}=a^{n}\cos\left(\displaystyle\frac{n\pi}{2}+ax\right).\)
  4. \(\left(\ln\left(1+x\right)\right)^{\left(n\right)}=\displaystyle\frac{\left(-1\right)^{n-1}\left(n-1\right)!}{\left(1+x\right)^{n}}.\)
  5. \(\left(\left(1+x\right)^{a}\right)^{\left(n\right)}=\alpha\left(\alpha-1\right)\cdots\left(\alpha-n+1\right)\left(1+x\right)^{a-n}.\)

其中第 \(5\) 条中，若 \(\alpha\) 为某一正整数 \(n\)，则 \(((1+x)^n)^{(n)}=n!\)，\(((1+x)^n)^{(n+j)}=0,j=1,2,\cdots.\)

2.4 反函数求导法则

设 \(y=f\left(x\right)\) 可导且 \(f^{\prime}\left(x\right)\neq0\)，则存在反函数 \(x=\varphi\left(y\right)\)，且

\[\displaystyle\frac{\mathrm{d}x}{\mathrm{d}y}=\displaystyle\frac{1}{\displaystyle\frac{\mathrm{d}y}{dx}},\quad \text{即} \quad \varphi^{\prime}\left(y\right)=\displaystyle\frac{1}{f^{\prime}\left(x\right)}.\]

若又设 \(y=f\left(x\right)\) 存在二阶导数，则 \(\varphi^{\prime\prime}\left(y\right)=-\displaystyle\frac{f^{\prime\prime}\left(x\right)}{\left(f^{\prime}\left(x\right)\right)^{3}}.\)

2.5 弧微分公式

  • 直角坐标系形式 \[ds=\sqrt{1+(\displaystyle\frac{dy}{dx})^2}dx=\sqrt{1+f^{'}(x)^2}dx.\]

  • 参数方程形式 \[ds =\sqrt{(dx)^2+(dy)^2}=\sqrt{(\displaystyle\frac{dx}{dt})^2+(\displaystyle\frac{dy}{dt})^2}dt=\sqrt{x^{^{\prime}}(t)^2+y^{^{\prime}}(t)^2}dt.\]

  • 极坐标形式 \[ds=\sqrt{\rho^2+\rho^{^{\prime}2}}d\theta.\]

三、微分中值定理 ( Mean Value Theorems )

微分中值定理是导数应用的桥梁，也是解决证明题的关键工具.

3.1 罗尔 ( Rolle ) 定理

如果 \(f(x)\) 满足 :

  1. 在 \([a, b]\) 上连续;
  2. 在 \((a, b)\) 内可导;
  3. \(f(a) = f(b)\);

则至少存在一点 \(\xi \in (a, b)\)，使得 \(f'(\xi) = 0\).

几何意义 两端点高度相等，中间至少有一点切线水平.

3.2 拉格朗日 ( Lagrange ) 中值定理

如果 \(f(x)\) 满足 :

  1. 在 \([a, b]\) 上连续;
  2. 在 \((a, b)\) 内可导;

则至少存在一点 \(\xi \in (a, b)\)，使得 \(f(b) - f(a) = f'(\xi)(b - a)\).

几何意义 连接两端点的割线斜率等于中间某点的切线斜率.

推论 若在区间 \(I\) 上 \(f'(x) \equiv 0\)，则 \(f(x) \equiv C\) ( 常数 ).

3.3 柯西 ( Cauchy ) 中值定理

如果 \(f(x), g(x)\) 满足 :

  1. 在 \([a, b]\) 上连续;
  2. 在 \((a, b)\) 内可导;
  3. 对任意 \(x \in (a, b)\)，\(g'(x) \neq 0\);

则至少存在一点 \(\xi \in (a, b)\)，使得 \(\displaystyle\frac{f(b) - f(a)}{g(b) - g(a)} = \displaystyle\frac{f'(\xi)}{g'(\xi)}\).

注 柯西中值定理是拉格朗日中值定理的推广 ( 取 \(g(x) = x\) ).

3.4 泰勒 ( Taylor ) 公式

设 \(f(x)\) 在 \(x_0\) 处具有 \(n\) 阶导数，则有

\[f(x) = f(x_0) + f'(x_0)(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)\]

其中 \(R_n(x)\) 为余项 :

  • 佩亚诺 ( Peano ) 余项 : \(R_n(x) = o((x-x_0)^n)\) ( 适用于极限计算 )
  • 拉格朗日 ( Lagrange ) 余项 : \(R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}\)，其中 \(\xi\) 介于 \(x\) 与 \(x_0\) 之间 ( 适用于不等式证明、估计误差 ).

四、导数的应用 ( Applications )

4.1 极值 ( Extrema )

1. 极值的定义

设 \(f(x)\) 在 \(x=x_0\) 的某邻域有定义，如果存在一个邻域 \(U(x_0)\)，当 \(x\in U(x_0)\) 时有 \(f(x) \ge (\le) f(x_0)\)，称 \(f(x_0)\) 为 \(f(x)\) 的一个 极小 ( 极大 ) 值，点 \(x=x_0\) 称为 \(f(x)\) 的一个 极小 ( 极大 ) 值点.

2. 极值的第一充分条件

设 \(f(x)\) 在 \(x=x_0\) 处连续，在 \(x=x_0\) 的去心邻域内可导.

  1. 若在 \(x=x_0\) 的左侧邻域内 \(f^{\prime}(x)>0\)，右侧邻域内 \(f^{\prime}(x)<0\)，则 \(f(x_0)\) 为 极大值;
  2. 若在 \(x=x_0\) 的左侧邻域内 \(f^{\prime}(x)<0\)，右侧邻域内 \(f^{\prime}(x)>0\)，则 \(f(x_0)\) 为 极小值.

3. 极值的第二充分条件

设 \(f(x)\) 在 \(x=x_0\) 处存在二阶导数，且 \(f^{\prime}\left(x_{0}\right)=0,f^{\prime\prime}\left(x_{0}\right)\neq0,\)

  1. 若 \(f^{\prime\prime}(x_0)<0\)，则 \(f(x_0)\) 为 极大值.
  2. 若 \(f^{\prime\prime}(x_0)>0\)，则 \(f(x_0)\) 为 极小值.

4. 极值的第三充分条件

设 \(f(x)\) 在 \(x=x_0\) 处 \(n\) 阶可导，且 \(f^{(m)}(x_0) = 0\) (\(m=1, 2, \dots, n-1\))，\(f^{(n)}(x_0) \neq 0\) (\(n \ge 2\))，则

  1. 当 \(n\) 为偶数且 \(f^{(n)}(x_0) < 0\) 时，\(f(x)\) 在 \(x_0\) 处取得 极大值.
  2. 当 \(n\) 为偶数且 \(f^{(n)}(x_0) > 0\) 时，\(f(x)\) 在 \(x_0\) 处取得 极小值.

注 当极值点为 \(x_0\) 时，只有 \(f^{\prime}(x_0)=0\) 和 \(f^{\prime}(x_0)\) 不存在 两种情况.

例题 1 设 \(f(x)\) 对一切 \(x\in(-\infty,+\infty)\) 满足方程 \((x-1)f^{\prime\prime}(x)+2(x-1)[f^{\prime}(x)]^3=1-\mathrm{e}^{1-x}\)，且 \(f(x)\) 在 \(x=a(a\neq1)\) 处 \(f^{\prime}(a)=0\)，则 \(x=a\) 是否为极值点？

解 将 \(f'(a)=0\) 带入原方程有， \[f^{\prime\prime}(a)=\displaystyle\frac{e^{1-a}-1}{1-a}\] 且注意到，无论 \(a>1\) 或 \(a<1\) 均有 \(\displaystyle\frac{e^{1-a}-1}{1-a}>0\)，此时 \(f''(a) >0\)，则 \(f(a)\) 为极小值.

4.2 凹凸性与拐点 ( Concavity & Inflection )

1. 凹凸性定义

设函数 \(f(x)\) 在区间 \(I\) 上连续. 如果对 \(I\) 上任意不同两点 \(x_1, x_2\)，恒有

\[f\left(\frac{x_1+x_2}{2}\right) < \frac{f(x_1)+f(x_2)}{2},\]

则称 \(y=f(x)\) 在 \(I\) 上的图形是 凹的 ( 或 凹弧 )，如左图所示; 如果恒有

\[f\left(\frac{x_1+x_2}{2}\right) > \frac{f(x_1)+f(x_2)}{2},\]

则称 \(y=f(x)\) 在 \(I\) 上的图形是 凸的 ( 或凸弧 ) 如右图所示.

注 事实上，当图形为凹 ( 凸 ) 时，可以将 \(f\left(\frac{1}{2}x_1 + \frac{1}{2}x_2\right) \underset{(\ge)}{<} \frac{1}{2}f(x_1) + \frac{1}{2}f(x_2)\) 更一般地写为 \(f(\lambda_1 x_1 + \lambda_2 x_2) \underset{(\ge)}{<} \lambda_1 f(x_1) + \lambda_2 f(x_2)\)，其中 \(0 < \lambda_1 < 1\), \(0 < \lambda_2 < 1\), \(\lambda_1 + \lambda_2 = 1\).

2. 拐点定义

设函数 \(f(x)\) 在点 \(x_0\) 连续，若存在 \(\delta > 0\) 使得 \(f(x)\) 在区间 \((x_0 - \delta, x_0)\) 与 \((x_0, x_0 + \delta)\) 上的 凹凸性相反，则称点 \((x_0, f(x_0))\) 为函数曲线的 拐点.

3. 凹凸性与拐点的判别

  • 判别凹凸性 设函数 \(f(x)\) 在 \(I\) 上 二阶可导.
    1. 若在 \(I\) 上 \(f''(x) > 0\)，则 \(f(x)\) 在 \(I\) 上的图形是 凹 的.
    2. 若在 \(I\) 上 \(f''(x) < 0\)，则 \(f(x)\) 在 \(I\) 上的图形是 凸 的.
  • 判别拐点
    • 第一充分条件 : 设 \(f(x)\) 在点 \(x = x_0\) 处 连续，在点 \(x = x_0\) 的某去心邻域 \(\mathring{U}(x_0, \delta)\) 内 二阶导数存在，且在该点的左、右邻域内 \(f''(x)\) 变号 ( 无论是由正变负，还是由负变正 )，则点 \((x_0, f(x_0))\) 为曲线的拐点.
    • 第二充分条件 : 设 \(f(x)\) 在 \(x=x_0\) 的某邻域内三阶可导，且 \(f''(x_0) = 0\), \(f'''(x_0) \neq 0\)，则点 \((x_0, f(x_0))\) 为曲线的拐点.
    • 第三充分条件 : 设 \(f(x)\) 在 \(x_0\) 处 \(n\) 阶可导，且 \(f^{(m)}(x_0) = 0 (m=2, \cdots, n-1)\), \(f^{(n)}(x_0) \neq 0 (n \ge 3)\)，则当 \(n\) 为 奇数 时，点 \((x_0, f(x_0))\) 为曲线的拐点.

注 \((x_0, f(x_0))\) 为曲线 \(y=f(x)\) 的拐点，并不要求 \(f(x)\) 在点 \(x_0\) 的导数存在，如 \(y=\sqrt[3]{x}\) 在 \(x=0\) 的情形，其中 \(y'=\frac{1}{3}x^{-\frac{2}{3}}\)，\(y''=-\frac{2}{9}x^{-\frac{5}{3}}\)，当 \(x>0\) 时，\(y''<0\); 当 \(x<0\) 时，\(y''>0\)，故点 \((0,0)\) 为曲线 \(y=\sqrt[3]{x}\) 的拐点，但在该点的导数不存在.

4. 极值点与拐点推论

  1. 曲线的可导点不可同时为极值点和拐点; 曲线的不可导点可同时为极值点和拐点.
  2. 设多项式函数 \(f(x)=(x-a)^n g(x)(n>1)\)，且 \(g(a) \neq 0\)，则当 \(n\) 为 偶数 时，\(x=a\) 是 \(f(x)\) 的 极值点; 当 \(n\) 为 奇数 时，点 \((a, 0)\) 是曲线 \(f(x)\) 的 拐点.
  3. 设多项式函数 \(f(x)=(x-a_1)^{n_1}(x-a_2)^{n_2}\cdots(x-a_k)^{n_k}\)，其中 \(n_i\) 是正整数，\(a_i\) 是实数且 \(a_i\) 两两不等，\(i=1, 2, \cdots, k\). 记 \(k_1\) 为 \(n_i=1\) 的个数，\(k_2\) 为 \(n_i>1\) 且 \(n_i\) 为偶数的个数，\(k_3\) 为 \(n_i>1\) 且 \(n_i\) 为奇数的个数，则 \(f(x)\) 的极值点个数为 \(k_1+2k_2+k_3-1\)，拐点个数为 \(k_1+2k_2+3k_3-2\).

4.3 渐近线 ( Asymptotes )

1. 水平渐近线

若 \(\displaystyle\lim_{x\to+\infty}f\left(x\right)=b_{1}\)，则 \(y=b_{1}\) 是一条 水平渐近线; 若又有 \(\displaystyle\lim_{x\to-\infty}f\left(x\right)=b_{2}\)，则 \(y=b_{2}\) 也是一条水平渐近线 ( 若 \(b_1 = b_2\)，则当然只能当作一条 ).

2. 铅直渐近线

若存在 \(x_0\)，使 \(\displaystyle\lim_{x\to x_{0}^{-}} f(x)=\infty\) ( 或 \(\displaystyle\lim_{x\to x_{0}^{+}} f(x)=\infty\) )，则 \(x=x_0\) 是一条 铅直渐近线. 这里的 \(x_0\) 先由观察法观得，一般考虑函数无定义点，端点，分段函数分段点.

3. 斜渐近线

\(y=ax+b\) 是曲线 \(y=f\left(x\right)\) 的一条斜渐近线的充要条件是 \(\displaystyle\lim_{x\to+\infty}\displaystyle\frac{f\left(x\right)}x=a,\lim_{x\to+\infty}(f(x)-ax)=b\). 这里 \(x\to+\infty\) 也可以改成 \(x\to-\infty\). 若 $a=0 $上式成立，即为 水平渐近线.

  • 泰勒公式求斜渐近线 若当 \(x\to +\infty\)时 ( \(-\infty\) 同理 )，\(f(x)\) 可展开成 \(f(x)=ax+b+o(1)\) 或 \(f(x)=x(a+\displaystyle\frac{b}{x}+o(\displaystyle\frac{1}{x}))\) ( 其中 \(o(1)\)、\(o(\displaystyle\frac{1}{x})\) 为 无穷小 )，此时 \(ax+b\) 为该曲线 \(f(x)\) 在 \(x\to +\infty\)时 ( 或 \(-\infty\) ) 的斜渐近线.

例题 2 求 \(y=\displaystyle\displaystyle\frac{x^{1+x}}{(1+x)^{x}}(x>0)\) 的斜渐近线.

解 考虑提出来一个 \(x\)，则有 \(x[\displaystyle\frac{x^{x}}{(1+x)^{x}}]\)，对上下同除 \(x^x\) 则有 \(x[\displaystyle\frac{1}{(1+\displaystyle\frac{1}{x})^{x}}]\) 考虑对 \((1+\displaystyle\frac{1}{x})^{x}\) 泰勒展开，有 \((1+\displaystyle\frac{1}{x})^{x}=e(1-\displaystyle\frac{1}{2x}+o(\displaystyle\frac{1}{x}))\) 带入原式有，\(\displaystyle\frac{x}{e}[\displaystyle\frac{1}{1-\displaystyle\frac{1}{2x}}]\) 再对 \(\displaystyle\frac{1}{1- 2x}\) 泰勒展开，则有 \(\displaystyle\frac{1}{1- 2x}=1+\displaystyle\frac{1}{2x}+o(\displaystyle\frac{1}{x})\)，代入最终解得斜渐近线为 \(y=\displaystyle\frac{x}{e}+\displaystyle\frac{1}{2e}\).

例题 3 ( 隐函数斜渐近线 ) 求解笛卡尔叶形线 \(y^{3} + x^{3} - 3 axy=0\) 所确定的斜渐近线.

解 法一 : 将隐函数方程转化为参数方程 令 \(y=tx\)，则 \(t^{3} x^{3}+ x^{3} - 3 ax\cdot tx=0\)，则有 \[\begin{cases} x=\displaystyle\frac{3 at}{1+t^3},\\[.5em] y=\displaystyle\frac{3 at^2}{1+t^3}.\end{cases}\] 当 \(t\to 1^{-}\) 时，有 \(x\to \infty\) 由斜渐近线公式得， \[k=\lim_{x\to \infty}\displaystyle\frac{y}{x}=\lim_{t\to 1^{-}}\displaystyle\frac{\displaystyle\frac{3 at^2}{1+t^3}}{\displaystyle\frac{3 at}{1+t^3}}=-1\] \[b=\lim_{x\to \infty}(y-kx)=\lim_{t\to 1^{-}}(\displaystyle\frac{3 at^2}{1+t^3}+\displaystyle\frac{3 at}{1+t^3})=-a\] 由此可得 斜渐近线为 \(y=-x-a.\)

法二 : 直接代入斜渐近线公式计算 不妨令 \(\displaystyle\lim_{x\to \infty}\displaystyle\frac{y}{x}=k\)， 对原式方程两边同时除以 \(x^3\) 得 \((\displaystyle\frac{y}{x})^3+1-3a\displaystyle\frac{y}{x^2}\) 取极限得 \(k=-1\)， 由于 \(\displaystyle\lim_{x\to \infty}(y-kx)=\displaystyle\lim_{x\to \infty}(y+x)\)，由原方程可得 \(x+y=\displaystyle\frac{3axy}{x^2+y^2-xy}\)， 对右边上下同时除以 \(x^2\) 得 \(x+y=\displaystyle\frac{3a\cdot \displaystyle\frac{y}{x}}{1+(\displaystyle\frac{y}{x})^2-\displaystyle\frac{y}{x}}\) 此时，取极限有 \[\lim_{x\to \infty}(x+y)=\lim_{x\to \infty}\displaystyle\frac{3a\cdot \displaystyle\frac{y}{x}}{1+(\displaystyle\frac{y}{x})^2-\displaystyle\frac{y}{x}}=\lim_{x\to \infty}\displaystyle\frac{-3a}{1+1-(-1)}=-a.\] 由此可得 斜渐近线为 \(y=-x-a.\)

法三 : 直接代入法 将 \(y=kx+b\) 直接带入原方程有 \[(kx+b)^3+x^3-3ax(kx+b)=0\] 整理得 \[(1+k^3)x^3+(3k^2b-3ak)x^2+(3kb^2-3ab)x+b^3=0\] 令 \(x\) 的最高次的系数和次高次系数为 \(0\)，有
\[\begin{cases} 1+k^3=0,\\[.5em] 3k^2b-3ak=0.\end{cases}\] 解得 \(k=a\)，\(b=-1\). 由此可得 斜渐近线为 \(y=-x-a.\)

结论

  1. 由二元高次方程所确定的隐函数存在斜渐近线的必要条件是二元高次方程中 最高次幂项至少 有 两项. 如例题所示，方程 \(y^3+x^3-3axy=0\) 中，最高次幂为 \(3\) 的项有两项.
  2. 若 \(F(x,kx+b)=0\)，令 \(x\) 的最高次幂与次最高次幂的系数为 \(0\)，得关于 \(k\)与 \(b\) 的联立方程组，解得 \(k\) 与 \(6\)，则直线 \(y=kx+b\) 是曲线的斜渐近线.

4.4 曲率 ( Curvature )

1. 曲率的定义

曲率 \(K\) 是描述曲线弯曲程度的量. 几何上，它是切线倾角 \(\alpha\) 对弧长 \(s\) 的变化率的大小.

\[K = \lim_{\Delta s \to 0} \left| \frac{\Delta \alpha}{\Delta s} \right|\]

直观理解

  • 直线 : 不弯曲，曲率为 \(0\).
  • 圆 : 弯曲程度处处相等，半径越小，弯曲越厉害，曲率越大 (\(K = 1/R\)).

2. 曲率计算公式

  • 直角坐标形式 ( 最常用 ) 设 \(y=f(x)\) 二阶可导，则 \[K = \frac{|y''|}{(1+y'^2)^{\frac{3}{2}}}\]

  • 参数方程形式 设 \(\begin{cases}x=\varphi(t)\\[.5em]y=\psi(t)\end{cases}\) 二阶可导，则 \[K = \frac{|x'y'' - x''y'|}{(x'^2 + y'^2)^{\frac{3}{2}}}\]

    记忆技巧 分母是速度平方的 \(3/2\) 次方，分子是类似于 “行列式” 的交叉相乘形式.

  • 极坐标形式 设 \(r=\rho(\theta)\)，则 \[K = \frac{|\rho^2 + 2\rho'^2 - \rho\rho''|}{(\rho^2 + \rho'^2)^{\frac{3}{2}}}\]

3. 曲率半径与曲率圆

  • 曲率半径 ( Radius of Curvature ) \[R = \frac{1}{K}\]

  • 曲率圆 ( Osculating Circle ) 在曲线 \(y=f(x)\) 上点 \(M(x, y)\) 处，与曲线 相切 且 曲率相同 的圆，称为该点的曲率圆 ( 密切圆 ).

    • 圆心 : 称为曲率中心 \((\alpha, \beta)\).
    • 半径 : 即曲率半径 \(R\).

    曲率中心坐标公式 : \[\begin{cases} \alpha = x - \displaystyle\frac{y'(1+y'^2)}{y''} \\[.8em] \beta = y + \displaystyle\frac{1+y'^2}{y''} \end{cases}\]

    推导思路 ( 避免死记硬背 ) 圆心 \((\alpha, \beta)\) 位于曲线的法线上，且在曲线的凹侧，距离切点距离为 \(R\). 利用向量关系或法线方程求解.

例题 4 ( 摆线曲率圆 )

求摆线 \(\begin{cases} x = t - \sin t \\ y = 1 - \cos t \end{cases}\) 在 \(t = \displaystyle\frac{\pi}{2}\) 处的曲率圆方程.

分析 : 本题涉及参数方程求导、曲率计算以及利用法线向量求解曲率中心，是考察曲率相关知识的综合性题目.

核心思路 : 先求出曲率半径 \(R\)，再求出切点处的法向量 \(\vec{n}\)，利用向量平移确定圆心 : \(\vec{OC} = \vec{OP} + R \cdot \vec{n}_0\).

( 几何含义 : 圆心坐标 = 切点坐标 + 半径 \(\times\) 指向凹侧的单位法向量 )

解

  1. 计算二阶导数

    \(x' = 1 - \cos t, \quad x'' = \sin t\).

    \(y' = \sin t, \quad y'' = \cos t\).

    当 \(t = \displaystyle\frac{\pi}{2}\) 时，\(x_0 = \frac{\pi}{2} - 1, \quad y_0 = 1\). \(x' = 1, \quad x'' = 1\). \(y' = 1, \quad y'' = 0\).

  2. 计算曲率 \(K\) 与半径 \(R\) \[K = \frac{|x'y'' - x''y'|}{(x'^2 + y'^2)^{\frac{3}{2}}} = \frac{|1\cdot 0 - 1\cdot 1|}{(1^2 + 1^2)^{\frac{3}{2}}} = \frac{1}{2\sqrt{2}}\] \[R = \frac{1}{K} = 2\sqrt{2}\]

  3. 确定法向量方向 ( 关键步骤 ) 切向量 \(\vec{\tau} = (x', y') = (1, 1)\). 法向量 \(\vec{n}\) 应与 \(\vec{\tau}\) 垂直. 可取 \((1, -1)\) 或 \((-1, 1)\). 判断凹凸性 : \[\frac{d^2y}{dx^2} = \frac{x'y'' - x''y'}{x'^3} = \frac{-1}{1} = -1 < 0\] 曲线 向上凸 ( 凹侧向下 )，故圆心在切线下方.

    法向量 \(\vec{n}\) 的 \(y\) 分量应为 负.

    故取指向凹侧的法向量 \(\vec{n} = (1, -1)\).

    单位化 : \(\vec{n}_0 = \displaystyle\frac{1}{\sqrt{2}}(1, -1)\).

  4. 利用向量求圆心 \(C(\alpha, \beta)\) \[\vec{OC} = \vec{OP} + R \cdot \vec{n}_0\] \[(\alpha, \beta) = (\frac{\pi}{2} - 1, 1) + 2\sqrt{2} \cdot \frac{1}{\sqrt{2}}(1, -1)\] \[(\alpha, \beta) = (\frac{\pi}{2} - 1, 1) + (2, -2) = (\frac{\pi}{2} + 1, -1)\]

  5. 写出曲率圆方程 \[(x - (\frac{\pi}{2} + 1))^2 + (y + 1)^2 = 8\]

五、易错点剖析 ( Common Pitfalls )

1. 导数定义的“陷阱”

很多同学认为 \(\displaystyle\lim_{\Delta x \to 0} \displaystyle\frac{f(x_0 + \Delta x) - f(x_0 - \Delta x)}{2\Delta x}\) 存在即意味着 \(f'(x_0)\) 存在.

  • 错! 反例 : \(f(x) = |x|\) 在 \(x=0\) 处. 上述极限为 0，但 \(f'(0)\) 不存在.
  • 正解 : 必须严格按照导数定义 \(\displaystyle\lim_{\Delta x \to 0} \displaystyle\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}\) 判断.

2. 极值点 \(\neq\) 驻点

  • 驻点 (\(f'(x)=0\)) 不一定是极值点 ( 如 \(y=x^3\) 在 \(x=0\) ).
  • 极值点 不一定是驻点 ( 如 \(y=|x|\) 在 \(x=0\) ).
  • 结论 : 极值点只可能发生在 驻点 或 不可导点.

3. 拐点 \(\neq f''(x)=0\)

  • \(f''(x_0)=0\) 不一定是拐点 ( 如 \(y=x^4\) 在 \(x=0\) ).
  • 拐点处二阶导数不一定存在 ( 如 \(y=\sqrt[3]{x}\) 在 \(x=0\) ).
  • 判别核心 : 必须检查 \(f''(x)\) 在 \(x_0\) 两侧是否 变号.

4. 洛必达法则的适用条件

\(\displaystyle\lim_{x \to x_0} \displaystyle\frac{f'(x)}{g'(x)}\) 不存在时，不能 说明原极限不存在!

  • 反例 : \(\displaystyle\lim_{x \to \infty} \displaystyle\frac{x + \sin x}{x}\)，洛必达法则失效 ( 导数极限震荡 )，但原极限存在且为 1.

六、例题 ( Examples )

例题 5 ( 泰勒公式与二阶导数估计 )

设 \(f(x)\) 在 \([0, 1]\) 上二阶可导，且 \(f(0) = f(1) = 0\)，\(\displaystyle\min_{0 \le x \le 1} f(x) = -1\). 证明 : \(\exists \xi \in (0, 1)\)，使得 \(f''(\xi) \ge 8\).

分析 : 题目给出了函数在区间的最小值 \(-1\)，这意味着存在一点 \(c \in (0, 1)\) 使得 \(f(c) = -1\) 且 \(f'(c) = 0\) ( 极值点 ). 要建立 \(f(c)\) 与 \(f''(\xi)\) 的联系，最自然的工具是 泰勒公式. 我们需要在 \(c\) 点将函数展开.

证明 : 设 \(f(x)\) 在 \(c \in (0, 1)\) 处取得最小值 \(-1\). 则 \(f(c) = -1, f'(c) = 0\). 分别在 \(x = 0\) 和 \(x = 1\) 处对 \(f(x)\) 应用带有拉格朗日余项的泰勒公式 ( 展开点选在 \(c\) )

  1. \(f(0) = f(c) + f'(c)(0 - c) + \displaystyle\frac{f''(\xi_1)}{2}(0 - c)^2\) 代入已知条件 : \(0 = -1 + 0 + \displaystyle\frac{f''(\xi_1)}{2}c^2 \implies f''(\xi_1) = \displaystyle\frac{2}{c^2}\).
  2. \(f(1) = f(c) + f'(c)(1 - c) + \displaystyle\frac{f''(\xi_2)}{2}(1 - c)^2\) 代入已知条件 : \(0 = -1 + 0 + \displaystyle\frac{f''(\xi_2)}{2}(1 - c)^2 \implies f''(\xi_2) = \displaystyle\frac{2}{(1 - c)^2}\).

其中 \(\xi_1 \in (0, c), \xi_2 \in (c, 1)\). 现在我们要证明 \(\max \{ f''(\xi_1), f''(\xi_2) \} \ge 8\). 只需证明 \(\max \{ \displaystyle\frac{2}{c^2}, \displaystyle\frac{2}{(1-c)^2} \} \ge 8\). 我们通过比较 \(c\) 与 \(\frac{1}{2}\) 的大小进行分类讨论 :

  1. 当 \(c \le \displaystyle\frac{1}{2}\) 时 : \(c^2 \le (\frac{1}{2})^2 = \frac{1}{4} \implies \frac{1}{c^2} \ge 4 \implies f''(\xi_1) = \frac{2}{c^2} \ge 8\).
  2. 当 \(c > \displaystyle\frac{1}{2}\) 时 : \(1 - c < \frac{1}{2} \implies (1 - c)^2 < \frac{1}{4} \implies \frac{1}{(1 - c)^2} > 4 \implies f''(\xi_2) = \frac{2}{(1 - c)^2} > 8\).

综上所述，无论最小值点 \(c\) 在何处，\(\xi_1\) 和 \(\xi_2\) 中至少有一个点的二阶导数 \(\ge 8\). 证毕.

注 : 常数 8 是不可改进的. 例如函数 \(f(x) = 4x^2 - 4x = 4x(x - 1)\)，在 \([0, 1]\) 上满足条件，\(f(1/2) = -1\)，且 \(f''(x) \equiv 8\).

例题 6 ( 朗道-科尔莫戈罗夫不等式 )

设 \(f(x)\) 在 \((-\infty, +\infty)\) 上二阶可导，且 \(|f(x)| \le M_0, |f''(x)| \le M_2\). 证明 : \[|f'(x)| \le \sqrt{2 M_0 M_2}\]

分析 : 这是一个涉及函数及其各阶导数界之间关系的经典不等式. 关键在于利用泰勒公式将 \(f'(x)\) 用 \(f(x)\) 和 \(f''(x)\) 表示出来，并通过引入任意步长 \(h\) 进行优化.

证明 : 对任意 \(x \in \mathbb{R}\) 和任意 \(h > 0\)，由泰勒公式

  1. \(f(x+h) = f(x) + h f'(x) + \displaystyle\frac{h^2}{2} f''(\xi_1)\)
  2. \(f(x-h) = f(x) - h f'(x) + \displaystyle\frac{h^2}{2} f''(\xi_2)\)

(1)-(2) 得 \(f(x+h) - f(x-h) = 2h f'(x) + \displaystyle\frac{h^2}{2} (f''(\xi_1) - f''(\xi_2))\) 整理得到 \(f'(x)\) 的表达式 \(2h f'(x) = f(x+h) - f(x-h) - \displaystyle\frac{h^2}{2} (f''(\xi_1) - f''(\xi_2))\) 两边取绝对值并利用三角不等式 \(2h |f'(x)| \le |f(x+h)| + |f(x-h)| + \displaystyle\frac{h^2}{2} (|f''(\xi_1)| + |f''(\xi_2)|)\) 代入有界条件 \(M_0, M_2\) \(2h |f'(x)| \le 2 M_0 + \displaystyle\frac{h^2}{2} (2 M_2) = 2 M_0 + h^2 M_2\)

除以 \(2h\)

\(|f'(x)| \le \displaystyle\frac{M_0}{h} + \displaystyle\frac{h}{2} M_2\)

由于上式对任意 \(h > 0\) 成立，我们可以选取最优的 \(h\) 使右边最小.

令 \(g(h) = \displaystyle\frac{M_0}{h} + \displaystyle\frac{M_2}{2} h\). 由均值不等式

\(g(h) \ge 2 \sqrt{\displaystyle\frac{M_0}{h} \cdot \displaystyle\frac{M_2}{2} h} = 2 \sqrt{\displaystyle\frac{M_0 M_2}{2}} = \sqrt{2 M_0 M_2}\).

当且仅当 \(\displaystyle\frac{M_0}{h} = \displaystyle\frac{M_2}{2} h\) 即 \(h = \sqrt{\displaystyle\frac{2 M_0}{M_2}}\) 时取等号. 故 \(|f'(x)| \le \sqrt{2 M_0 M_2}\). 证毕.

例题 7 ( 中值定理中间点的极限 )

设 \(f(x)\) 在 \(x=0\) 处二阶可导，且 \(f''(0) \neq 0\). 若 \(f(x) = f(0) + x f'(\theta x)\) (\(0 < \theta < 1\))，求 \(\displaystyle\lim_{x \to 0} \theta\).

分析 :

这是拉格朗日中值定理中 \(\theta\) 的渐近性质. 当 \(x \to 0\) 时，\(\theta x\) 也趋于 \(0\). 我们可以利用泰勒公式将 \(f(x)\) 和 \(f'(\theta x)\) 都在 \(0\) 处展开，通过比较系数求出极限.

解

一方面，将 \(f(x)\) 在 \(x=0\) 处展开到二阶皮亚诺余项 : \(f(x) = f(0) + f'(0)x + \displaystyle\frac{f''(0)}{2}x^2 + o(x^2)\) (1)

另一方面，根据题目条件

\(f(x) = f(0) + x f'(\theta x)\) (2)

将 \(f'(\theta x)\) 在 \(0\) 处展开 ( 注意 \(f'\) 在 \(0\) 处可导，导数为 \(f''(0)\) )

\(f'(u) = f'(0) + f''(0)u + o(u)\)

令 \(u = \theta x\)，则 \(f'(\theta x) = f'(0) + f''(0)\theta x + o(\theta x)\).

代入 (2) 式
\(f(x) = f(0) + x [ f'(0) + f''(0)\theta x + o(\theta x) ]\) \(f(x) = f(0) + f'(0)x + f''(0)\theta x^2 + o(\theta x^2)\) (3)

比较 (1) 和 (3) 的 \(x^2\) 项系数 ( 因为 \(f(x)\) 表达式唯一 )

\(\displaystyle\frac{f''(0)}{2} x^2 \approx f''(0)\theta x^2\)

准确来说，由 (1) 和 (3) 相等得

\(\displaystyle\frac{f''(0)}{2}x^2 + o(x^2) = f''(0)\theta x^2 + o(x^2)\)

两边同时除以 \(x^2 f''(0)\) ( 因 \(f''(0) \neq 0\) ) \(\displaystyle\frac{1}{2} + \displaystyle\frac{o(x^2)}{x^2} = \theta + \displaystyle\frac{o(x^2)}{x^2}\)

令 \(x \to 0\)，则 \(\theta \to \displaystyle\frac{1}{2}\).

故 \(\displaystyle\lim_{x \to 0} \theta = \displaystyle\frac{1}{2}\).

推广 : 若 \(f(x)\) 在 \(0\) 处 \(n+1\) 阶可导且 \(f^{(n+1)}(0) \neq 0\)，则泰勒公式余项中的 \(\theta\) 满足 \(\displaystyle\lim_{x \to 0} \theta = \displaystyle\frac{1}{n+1}\).
