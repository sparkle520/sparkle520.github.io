第二章 导数与微分

本章节不提供详细的知识点.

函数的微分

微分的定义

设 \(y=f(x)\) 在 \(x=x_0\) 的某邻域 \(U(x_0)\) 内有定义，并设 \(x_{0}+\Delta x\in U(x_{0}).\) 如果

\[\Delta y=f(x_{0}+\Delta x)-f(x_{0})\xrightarrow{可写成}A\Delta x+o(\Delta x)\:,\]

其中 \(A\) 与 \(\Delta x\) 无关，\(\displaystyle\lim_{\Delta x\to_0}\displaystyle\frac{o(\Delta x)}{\Delta x}=0\)，则称 \(f(x)\) 在点 \(x=x_0\) 处可微，并称 \(A\Delta x\) 为 \(f(x)\) 在\(x=x_0\) 处的微分，记为 \(dy= A\Delta x.\) 又因自变量的增量 \(\Delta x\) 等于自变量的微分 \(dx\)，于是 \(dy\) 又可写成 \(dy=Adx.\)

注

以 直 代 曲，这便是微分的含义.

dx ∆x，dy ∆x，ds ∆s 关系

首先先要明确的是，\(d\) 代表的是微观的意义 ( 微 )，而 \(\Delta\) 代表的是宏观的概念 ( 宏 ).

由下图，易知

  • \(dx = \Delta x.\)

  • \(dy ≠ \Delta y.\)

  • \(ds ≠ \Delta s\) ( \(ds\) 为直线 \(T\) ，\(\Delta s\) 为曲线 \(f(x)\) ).

额外的，还有 \((\Delta s)^2 ≈ (\Delta x)^2 + (\Delta y)^2\quad (ds)^2 = (dx)^2 + (dy)^2\) ( 此处 “\(=\)” 代表化曲为直 ).

弧微分

直角坐标系形式

\[ds=\sqrt{1+(\displaystyle\frac{dy}{dx})^2}dx=\sqrt{1+f^{'}(x)^2}dx.\]

参数方程形式

\[ds =\sqrt{(dx)^2+(dy)^2}=\sqrt{(\displaystyle\frac{dx}{dt})^2+(\displaystyle\frac{dy}{dt})^2}dt=\sqrt{x^{^{\prime}}(t)^2+y^{^{\prime}}(t)^2}dt.\]

极坐标形式

\[ds=\sqrt{\rho^2+\rho^{^{\prime}2}}d\theta.\]

可导、可微，连续 三者关系

在一元函数微分学中，可导 \(=\) 可微 ，可导 \(\Rightarrow\) 连续 ，连续 \(\nRightarrow\) 可导.

且对任意的 \(x\) 有，

\[dy = f'(x)dx.\]

函数的微分 和 函数的增量 关系

设 \(y=f(x)\) 在 \(x_0\) 处可导 ( 可微 ) ，则有

\[\Delta y=dy+ o( \Delta x)\quad或者\quad \Delta y=f'(x_0)\Delta x+o(\Delta x).\]

若又设在含有 \(x_0\) 的某区间内存在二阶导数，则由拉格朗日余项泰勒公式有

\[\Delta y-dy=\displaystyle\frac{1}{2}f^{\prime\prime}\left(\xi\right)\left(\Delta x\right)^{2},\]

其中 \(\xi\) 介于 \(x_0\) 与 \(x_0 + \Delta x\) 之间.

微分形式不变性

对于 \(y = f(u)\) 不管 \(u\) 是自变量还是中间变量，函数的微分的表达式都有形式不变性，即 \(dy = f'(u)du.\)

变限积分求导公式

设 \(f(t)\) 为连续函数，\(\varphi_1(x)\) 与 \(\varphi_2(x)\) 均可导，则有

\[\left[\int_{\varphi_1(x)}^{\varphi_2(x)}f\left(t\right)\mathrm{d}t\right]^{\prime}=f\left(\varphi_2(x)\right)\varphi_2^{\prime}(x)-f\left(\varphi_1(x)\right)\varphi_1^{\prime}(x).\]

\(n\) 阶导数运算法则

以下均设 \(u,v\) 为 \(n\) 阶可导，则有

  • \((u\pm v)^{(n)}=u^{(n)}\pm v^{(n)}.\)
  • \((cu)^{(n)}=cu^{(n)}.\)
  • \((uv)^{(n)}=u^{(n)}v+\mathrm{C}_{n}^{1}u^{(n-1)}v^{\prime}+\cdots+\mathrm{C}_{n}^{k}u^{(n-k)}v^{(k)}+\cdots+uv^{(n)}.\)

最后一个公式称为乘积的高阶导数的 莱布尼茨公式.

常见的 n 阶导数公式

  1. \(\left(e^{ax}\right)^{\left(n\right)}=a^{n}e^{ax}.\)
  2. \((\sin ax)^{(n)}=a^{n}\sin\left(\displaystyle\frac{n\pi}{2}+ax\right).\)
  3. \((\cos ax)^{(n)}=a^{n}\cos\left(\displaystyle\frac{n\pi}{2}+ax\right).\)
  4. \(\left(\ln\left(1+x\right)\right)^{\left(n\right)}=\displaystyle\frac{\left(-1\right)^{n-1}\left(n-1\right)!}{\left(1+x\right)^{n}}.\)
  5. \(\left(\left(1+x\right)^{a}\right)^{\left(n\right)}=\alpha\left(\alpha-1\right)\cdots\left(\alpha-n+1\right)\left(1+x\right)^{a-n}.\)

其中第 \(5\) 条中，若 \(\alpha\) 为某一正整数 \(n\)，则 \(((1+x)^n)^{(n)}=n!\)，\(((1+x)^n)^{(n+j)}=0,j=1,2,\cdots.\)

反函数导数

设 \(y=f\left(x\right)\) 可导且 \(f^{\prime}\left(x\right)\neq0\)，则存在反函数 \(x=\varphi\left(y\right)\)，且

\[\displaystyle\frac{\mathrm{d}x}{\mathrm{d}y}=\displaystyle\frac{1}{\displaystyle\frac{\mathrm{d}y}{dx}},即 \varphi^{\prime}\left(y\right)=\displaystyle\frac{1}{f^{\prime}\left(x\right)}.\]

若又设 \(y=f\left(x\right)\) 存在二阶导数，则 \(\varphi^{\prime\prime}\left(y\right)=-\displaystyle\frac{f^{\prime\prime}\left(x\right)}{\left(f^{\prime}\left(x\right)\right)^{3}}.\)

极值

极值的定义

设 \(f(x)\) 在 \(x=x_0\) 的某邻域有定义，如果存在一个邻域 \(U(x_0)\)，当 \(x\in U(x_0)\) 时有 \(f(x)≥(≤)f(x_0)\)，称 \(f(x_0)\) 为 \(f(x)\) 的一个 极小 ( 极大 ) 值，点 \(x=x_0\) 称为 \(f(x)\) 的一个 极小 ( 极大 ) 值点.

极值的第一充分条件

设 \(f(x)\) 在 \(x=x_0\) 处连续，在 \(x=x_0\) 的去心邻域内可导.

  1. 若在 \(x=x_0\) 的左侧邻域内 \(f^{\prime}(x)>0\)，右侧邻域内 \(f^{\prime}(x)<0\)，则 \(f(x_0)\) 为 极大值;
  2. 若在 \(x=x_0\) 的左侧邻域内 \(f^{\prime}(x)<0\)，右侧邻域内 \(f^{\prime}(x)>0\)，则 \(f(x_0)\) 为 极小值.

极值的第二充分条件

设 \(f(x)\) 在 \(x=x_0\) 处存在二阶导数，

\[f^{\prime}\left(x_{0}\right)=0,f^{\prime\prime}\left(x_{0}\right)\neq0,\]

  1. 若 \(f^{\prime\prime}(x_0)<0\)，则 \(f(x_0)\) 为 极大值.

  2. 若 \(f^{\prime\prime}(x_0)>0\)，则 \(f(x_0)\) 为 极小值.

极值的第三充分条件

设 \(f(x)\) 在 \(x=x_0\) 处 \(n\) 阶可导，且 \(f^{(m)}(x_0) = 0\) (\(m=1, 2, \dots, n-1\))，\(f^{(n)}(x_0) \neq 0\) (\(n \ge 2\))，则

  1. 当 \(n\) 为偶数且 \(f^{(n)}(x_0) < 0\) 时，\(f(x)\) 在 \(x_0\) 处取得 极大值.

  2. 当 \(n\) 为偶数且 \(f^{(n)}(x_0) > 0\) 时，\(f(x)\) 在 \(x_0\) 处取得 极小值.

注

当极值点为 \(x_0\) 时，只有 \(f^{\prime}(x_0)=0\) 和 \(f^{\prime}(x_0)\) 不存在 两种情况.

-+—-< 例题1 >—-+-

设 \(f(x)\) 对一切 \(x\in(-\infty,+\infty)\) 满足方程 \((x-1)f^{\prime\prime}(x)+2(x-1)[f^{\prime}(x)]^3=1-\mathrm{e}^{1-x}\)，且 \(f(x)\) 在 \(x=a(a\neq1)\) 处 \(f^{\prime}(a)=0\)，则 \(x=a\) 是否为极值点？

解

将 \(f'(a)=0\) 带入原方程有， \[f^{\prime\prime}(a)=\displaystyle\frac{e^{1-a}-1}{1-a}\]

且注意到，无论 \(a>1\) 或 \(a<1\) 均有 \(\displaystyle\frac{e^{1-a}-1}{1-a}>0\)，此时 \(f''(a) >0\)，则 \(f(a)\) 为极小值.

凹凸性与拐点

凹凸性定义

设函数 \(f(x)\) 在区间 \(I\) 上连续. 如果对 \(I\) 上任意不同两点 \(x_1, x_2\)，恒有

\[f\left(\frac{x_1+x_2}{2}\right) < \frac{f(x_1)+f(x_2)}{2},\]

则称 \(y=f(x)\) 在 \(I\) 上的图形是 凹的 ( 或 凹弧 )，如左图所示；如果恒有

\[f\left(\frac{x_1+x_2}{2}\right) > \frac{f(x_1)+f(x_2)}{2},\]

则称 \(y=f(x)\) 在 \(I\) 上的图形是 凸的 ( 或凸弧 ) 如右图所示.

注

事实上，当图形为凹 ( 凸 ) 时，可以将 \(f\left(\frac{1}{2}x_1 + \frac{1}{2}x_2\right) \underset{(\ge)}{<} \frac{1}{2}f(x_1) + \frac{1}{2}f(x_2)\) 更一般地写为 \(f(\lambda_1 x_1 + \lambda_2 x_2) \underset{(\ge)}{<} \lambda_1 f(x_1) + \lambda_2 f(x_2)\)，其中 \(0 < \lambda_1 < 1\), \(0 < \lambda_2 < 1\), \(\lambda_1 + \lambda_2 = 1\).

拐点定义

设函数 \(f(x)\) 在点 \(x_0\) 连续，若存在 \(\delta > 0\) 使得 \(f(x)\) 在区间 \((x_0 - \delta, x_0)\) 与 \((x_0, x_0 + \delta)\) 上的 凹凸性相反，则称点 \((x_0, f(x_0))\) 为函数曲线的 拐点.

凹凸性与拐点的判别

判别凹凸性

设函数 \(f(x)\) 在 \(I\) 上 二阶可导.

  1. 若在 \(I\) 上 \(f''(x) > 0\)，则 \(f(x)\) 在 \(I\) 上的图形是 凹 的.

  2. 若在 \(I\) 上 \(f''(x) < 0\)，则 \(f(x)\) 在 \(I\) 上的图形是 凸 的.

判别拐点

判别拐点的第一充分条件

设 \(f(x)\) 在点 \(x = x_0\) 处 连续，在点 \(x = x_0\) 的某去心邻域 \(\mathring{U}(x_0, \delta)\) 内 二阶导数存在，且在该点的左、右邻域内 \(f''(x)\) 变号 ( 无论是由正变负，还是由负变正 )，则点 \((x_0, f(x_0))\) 为曲线的拐点.

判别拐点的第二充分条件

设 \(f(x)\) 在 \(x=x_0\) 的某邻域内三阶可导，且 \(f''(x_0) = 0\), \(f'''(x_0) \neq 0\)，则点 \((x_0, f(x_0))\) 为曲线的拐点.

判别拐点的第三充分条件

设 \(f(x)\) 在 \(x_0\) 处 \(n\) 阶可导，且 \(f^{(m)}(x_0) = 0 (m=2, \cdots, n-1)\), \(f^{(n)}(x_0) \neq 0 (n \ge 3)\)，则当 \(n\) 为 奇数 时，点 \((x_0, f(x_0))\) 为曲线的拐点.

注

\((x_0, f(x_0))\) 为曲线 \(y=f(x)\) 的拐点，并不要求 \(f(x)\) 在点 \(x_0\) 的导数存在，如 \(y=\sqrt[3]{x}\) 在 \(x=0\) 的情形，其中 \(y'=\frac{1}{3}x^{-\frac{2}{3}}\)，\(y''=-\frac{2}{9}x^{-\frac{5}{3}}\)，当 \(x>0\) 时，\(y''<0\)；当 \(x<0\) 时，\(y''>0\)，故点 \((0,0)\) 为曲线 \(y=\sqrt[3]{x}\) 的拐点，但在该点的导数不存在.

极值点与拐点推论

  1. 曲线的可导点不可同时为极值点和拐点；曲线的不可导点可同时为极值点和拐点.

  2. 设多项式函数 \(f(x)=(x-a)^n g(x)(n>1)\)，且 \(g(a) \neq 0\)，则当 \(n\) 为 偶数 时，\(x=a\) 是 \(f(x)\) 的 极值点； 当 \(n\) 为 奇数 时，点 \((a, 0)\) 是曲线 \(f(x)\) 的 拐点.

  3. 设多项式函数 \(f(x)=(x-a_1)^{n_1}(x-a_2)^{n_2}\cdots(x-a_k)^{n_k}\)，其中 \(n_i\) 是正整数，\(a_i\) 是实数且 \(a_i\) 两两不等，\(i=1, 2, \cdots, k\). 记 \(k_1\) 为 \(n_i=1\) 的个数，\(k_2\) 为 \(n_i>1\) 且 \(n_i\) 为偶数的个数，\(k_3\) 为 \(n_i>1\) 且 \(n_i\) 为奇数的个数，则 \(f(x)\) 的极值点个数为 \(k_1+2k_2+k_3-1\)，拐点个数为 \(k_1+2k_2+3k_3-2\).

渐近线

水平渐近线

若 \(\displaystyle\lim_{x\to+\infty}f\left(x\right)=b_{1}\)，则 \(y=b_{1}\) 是一条 水平渐近线;

若又有 \(\displaystyle\lim_{x\to-\infty}f\left(x\right)=b_{2}\)，则 \(y=b_{2}\) 也是一条水平渐近线 ( 若 \(b_1 = b_2\)，则当然只能当作一条 ).

铅直渐近线

若存在 \(x_0\)，使 \(\displaystyle\lim_{x\to x_{0}^{-}} f(x)=\infty\) ( 或 \(\displaystyle\lim_{x\to x_{0}^{+}} f(x)=\infty\) )，则 \(x=x_0\) 是一条 铅直渐近线. 这里的 \(x_0\) 先由观察法观得，一般考虑函数无定义点，端点，分段函数分段点.

斜渐近线

\(y=ax+b\) 是曲线 \(y=f\left(x\right)\) 的一条斜渐近线的充要条件是 \(\displaystyle\lim_{x\to+\infty}\displaystyle\frac{f\left(x\right)}x=a,\lim_{x\to+\infty}(f(x)-ax)=b\). 这里 \(x\to+\infty\) 也可以改成 \(x\to-\infty\). 若 $a=0 $上式成立，即为 水平渐近线.

泰勒公式 求 斜渐近线

若当 \(x\to +\infty\)时 ( \(-\infty\) 同理 )，\(f(x)\) 可展开成 \(f(x)=ax+b+o(1)\) 或 \(f(x)=x(a+\displaystyle\frac{b}{x}+o(\displaystyle\frac{1}{x}))\) ( 其中 \(o(1)\)、\(o(\displaystyle\frac{1}{x})\) 为 无穷小 )，此时 \(ax+b\) 为该曲线 \(f(x)\) 在 \(x\to +\infty\)时 ( 或 \(-\infty\) ) 的斜渐近线.

-+—-< 例题2 >—-+-

求 \(y=\displaystyle\displaystyle\frac{x^{1+x}}{(1+x)^{x}}(x>0)\) 的斜渐近线.

分析 考虑提出来一个 \(x\)，则有 \(x[\displaystyle\frac{x^{x}}{(1+x)^{x}}]\)，对上下同除 \(x^x\) 则有 \(x[\displaystyle\frac{1}{(1+\displaystyle\frac{1}{x})^{x}}]\) 考虑对 \((1+\displaystyle\frac{1}{x})^{x}\) 泰勒展开，有 \((1+\displaystyle\frac{1}{x})^{x}=e(1-\displaystyle\frac{1}{2x}+o(\displaystyle\frac{1}{x}))\) 带入原式有，\(\displaystyle\frac{x}{e}[\displaystyle\frac{1}{1-\displaystyle\frac{1}{2x}}]\) 再对 \(\displaystyle\frac{1}{1- 2x}\) 泰勒展开，则有 \(\displaystyle\frac{1}{1- 2x}=1+\displaystyle\frac{1}{2x}+o(\displaystyle\frac{1}{x})\)，代入最终解得斜渐近线为 \(y=\displaystyle\frac{x}{e}+\displaystyle\frac{1}{2e}\).

求 隐函数 斜渐近线

-+—-< 例题3 >—-+-

求解笛卡尔叶形线 \(y^{3} + x^{3} - 3 axy=0\) 所确定的斜渐近线.

  1. 法一 将隐函数方程转化为参数方程

令 \(y=tx\)，则 \(t^{3} x^{3}+ x^{3} - 3 ax\cdot tx=0\)，则有

\[\begin{cases} x=\displaystyle\frac{3 at}{1+t^3},\\[.5em] y=\displaystyle\frac{3 at^2}{1+t^3}.\end{cases}\]

当 \(t\to 1^{-}\) 时，有 \(x\to \infty\)

由斜渐近线公式得，

\[k=\lim_{x\to \infty}\displaystyle\frac{y}{x}=\lim_{t\to 1^{-}}\displaystyle\frac{\displaystyle\frac{3 at^2}{1+t^3}}{\displaystyle\frac{3 at}{1+t^3}}=-1\]

\[b=\lim_{x\to \infty}(y-kx)=\lim_{t\to 1^{-}}(\displaystyle\frac{3 at^2}{1+t^3}+\displaystyle\frac{3 at}{1+t^3})=-a\]

由此可得 斜渐近线为 \(y=-x-a.\)

  1. 法二 直接代入斜渐近线公式计算

不妨令 \(\displaystyle\lim_{x\to \infty}\displaystyle\frac{y}{x}=k\)，

对原式方程两边同时除以 \(x^3\) 得 \((\displaystyle\frac{y}{x})^3+1-3a\displaystyle\frac{y}{x^2}\) 取极限得 \(k=-1\)，

由于 \(\displaystyle\lim_{x\to \infty}(y-kx)=\displaystyle\lim_{x\to \infty}(y+x)\)，由原方程可得 \(x+y=\displaystyle\frac{3axy}{x^2+y^2-xy}\)，

对右边上下同时除以 \(x^2\) 得 \(x+y=\displaystyle\frac{3a\cdot \displaystyle\frac{y}{x}}{1+(\displaystyle\frac{y}{x})^2-\displaystyle\frac{y}{x}}\)

此时，取极限有

\[\lim_{x\to \infty}(x+y)=\lim_{x\to \infty}\displaystyle\frac{3a\cdot \displaystyle\frac{y}{x}}{1+(\displaystyle\frac{y}{x})^2-\displaystyle\frac{y}{x}}=\lim_{x\to \infty}\displaystyle\frac{-3a}{1+1-(-1)}=-a.\]

由此可得 斜渐近线为 \(y=-x-a.\)

  1. 直接代入法

将 \(y=kx+b\) 直接带入原方程有

\[(kx+b)^3+x^3-3ax(kx+b)=0\]

整理得

\[(1+k^3)x^3+(3k^2b-3ak)x^2+(3kb^2-3ab)x+b^3=0\]

令 \(x\) 的最高次的系数和次高次系数为 \(0\)，有

\[\begin{cases} 1+k^3=0,\\[.5em] 3k^2b-3ak=0.\end{cases}\]

解得 \(k=a\)，\(b=-1\).

由此可得 斜渐近线为 \(y=-x-a.\)

结论

  1. 由二元高次方程所确定的隐函数存在斜渐近线的必要条件是二元高次方程中 最高次幂项至少 有 两项. 如例题所示，方程 \(y^3+x^3-3axy=0\) 中，最高次幂为 \(3\) 的项有两项.

  2. 若 \(F(x,kx+b)=0\)，令 \(x\) 的最高次幂与次最高次幂的系数为 \(0\)，得关于 \(k\)与 \(b\) 的联立方程组，解得 \(k\) 与 \(6\)，则直线 \(y=kx+b\) 是曲线的斜渐近线.

曲率、曲率圆 与 曲率半径

曲率

一般形式求曲率

设 \(y=f(x)\)，则有

\[K=\frac{|f^{\prime\prime}(x_0)|}{[1+f^{\prime2}(x_0)]^{\displaystyle\frac32}}\]

参数方程求曲率

设参数方程 \(\begin{cases}x=\varphi(t)\\y=\omega(t)\end{cases}\)，则有

\[K=\displaystyle\frac{|\varphi^{\prime}(t)\omega^{\prime\prime}(t)-\omega^{\prime}(t)\varphi^{\prime\prime}(t)|}{[\varphi^{\prime2}(t)+\omega^{\prime2}(t)]^{\displaystyle\frac32}}\]

极坐标求曲率

设极坐标 \(r=\rho(\theta)\)，则有

\[K=\displaystyle\frac{|\rho^2+2\rho^{\prime2}–\rho\rho^{\prime\prime}|}{(\rho^2+\rho^{\prime2})^{\displaystyle\frac32}}\]

曲率圆 与 曲率半径

设 \(f(x)\) 存在二阶导数，曲线 \(y=f(x)\) 在其上点 \((x_0,y_0)\) 处的曲率圆表达式则为

\[(x–\alpha)^2+(x–\beta)^2=R^2\]

其中 \(R\) 为 \(\displaystyle\frac{1}{k}.\)

\((\alpha,\beta)\) 表示该曲率圆的圆心 ( 点 \((x_0,y_0)\) 的曲率中心 )，且：

\[\alpha=x_0-\frac{f^{\prime}\left(x_0\right)\left\{1+\left[f^{\prime}\left(x_0\right)\right]^2\right\}}{f^{\prime\prime}\left(x_0\right)}\quad\beta=y_0+\frac{1+[f^{\prime}\left(x_0\right)]^2}{f^{\prime\prime}\left(x_0\right)}\]

但注意到，曲率圆和曲线相切 ( 圆心到切点为半径 ) 且圆心在曲线的法线上，此时联立方程组会求出两组解，根据二阶导的凹凸性可以判断圆心所在位置.
