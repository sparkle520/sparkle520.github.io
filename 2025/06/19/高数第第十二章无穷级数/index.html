第十二章 无穷级数

本章重点掌握 常数项级数 的敛散性判别 ( 特别是正项级数与交错级数 )，以及 幂级数 的收敛域求法与函数展开. 注意事项 判别法的条件必须严格满足，级数收敛的必要条件 (\(\lim u_n = 0\)) 常用于快速排除发散级数.

总结与速查 ( Cheat Sheet )

判别法 条件 结论 备注
比值判别法 \(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = \rho\) \(\rho<1\) 收敛; \(\rho>1\) 发散 \(\rho=1\) 失效
根值判别法 \(\lim_{n\to\infty} \sqrt[n]{u_n} = \rho\) \(\rho<1\) 收敛; \(\rho>1\) 发散 \(\rho=1\) 失效
莱布尼茨判别法 \(u_n \ge u_{n+1}, \lim u_n=0\) 交错级数收敛 条件充分不必要
\(p\)-级数 \(\sum \frac{1}{n^p}\) \(p>1\) 收敛; \(p\le 1\) 发散 常用比较对象
几何级数 \(\sum aq^n\) \(|q|<1\) 收敛; \(|q|\ge 1\) 发散 常用比较对象

一、常数项级数的概念和性质

1.1 常数项级数

设数列 \(\{ a_n \}\) 为常数列，称 \(\displaystyle\sum_{n=1}^{\infty}a_n\) 为 常数项级数.

1.2 敛散性定义

把 \(S_n=a_1+a_2+\cdots+a_n\) 称为级数 \(\displaystyle\sum_{n=1}^{\infty}a_n\) 的 部分和.

定义 1. 若 \(\displaystyle\lim_{n \to \infty}S_n = S\) 存在，则称级数 收敛，记为 \(\displaystyle\sum_{n=1}^{\infty}a_n=S\). 2. 若 \(\displaystyle\lim_{n \to \infty}S_n\) 不存在 ( 含无穷大 )，则称级数 发散.

1.3 基本性质

  1. 和差法则 \(\displaystyle\sum (u_n \pm v_n) = \displaystyle\sum u_n \pm \displaystyle\sum v_n\). ( 前提 : 两级数均收敛 )

    注意事项

    • 收敛 \(\pm\) 发散 \(\Rightarrow\) 发散.
    • 发散 \(\pm\) 发散 \(\Rightarrow\) 不一定 ( 可能收敛也可能发散 ).

  2. 常数倍法则 \(\displaystyle\sum ku_n = k\displaystyle\sum u_n\). ( \(k \ne 0\) 时敛散性不变 )

  3. 有限项无关性 级数增加、减少、改变有限项不改变级数的敛散性 ( 但可能改变和 \(S\) ).

级数收敛的必要条件 若 \(\displaystyle\sum_{n=1}^{\infty}u_n\) 收敛，则 \(\displaystyle\lim_{n \to \infty}u_n=0\). ( 逆否命题 若 \(\lim u_n \ne 0\)，则级数 必发散. )

陷阱提示 \(\lim u_n = 0\) 推不出 级数收敛.

  • 反例 调和级数 \(\sum_{n=1}^{\infty} \frac{1}{n}\) 发散，尽管 \(\lim \frac{1}{n} = 0\).

1.4 两个重要级数 ( 比较判别的基准 )

两个重要级数

1. \(p\)-级数 \(\displaystyle\sum_{n=1}^{\infty}\frac{1}{n^p}\)

  • \(p > 1\) 收敛 ( 如 \(\sum \frac{1}{n^2}\) )
  • \(p \le 1\) 发散 ( 如 \(\sum \frac{1}{\sqrt{n}}\) )

2. 几何级数 (等比级数) \(\displaystyle\sum_{n=0}^{\infty}aq^n\)

  • \(|q| < 1\) 收敛，和 \(S = \frac{a}{1-q}\)
  • \(|q| \ge 1\) 发散

二、正项级数及其审敛法

定义 若 \(u_n \ge 0\)，则称 \(\sum u_n\) 为正项级数. 充要条件 部分和数列 \(\{S_n\}\) 有界.

2.1 比较审敛法

  1. 基本形式
    • \(u_n \le v_n\) 大收 \(\Rightarrow\) 小收; 小散 \(\Rightarrow\) 大散.
  2. 极限形式 ( 更常用 ) 设 \(\lim_{n\to\infty} \frac{u_n}{v_n} = l\).
    • \(0 < l < +\infty\) 两级数敛散性相同.
    • \(l=0\) : \(v_n\) 收敛 \(\Rightarrow u_n\) 收敛. ( \(u_n\) 是 \(v_n\) 的高阶无穷小 )
    • \(l=+\infty\) : \(v_n\) 发散 \(\Rightarrow u_n\) 发散.

解题技巧 遇到 \(\frac{1}{n^\alpha \pm \dots}\) 形式，通常与 \(p\)-级数 \(\frac{1}{n^\alpha}\) 比较.

利用 等价无穷小替换 可以简化极限计算.

2.2 比值与根值审敛法

比值判别法 (D’Alembert) 设 \(\lim_{n\to\infty} \frac{u_{n+1}}{u_n} = \rho\).

  • \(\rho < 1\) 收敛
  • \(\rho > 1\) 发散
  • \(\rho = 1\) 失效 ( 需改用其他方法，如比较法或定义 )

根值判别法 (Cauchy) 设 \(\lim_{n\to\infty} \sqrt[n]{u_n} = \rho\).

  • \(\rho < 1\) 收敛
  • \(\rho > 1\) 发散
  • \(\rho = 1\) 失效

2.3 积分判别法

设 \(f(x)\) 单调递减非负，且 \(u_n = f(n)\).

则 \(\sum u_n\) 与 \(\int_1^{+\infty} f(x)dx\) 同敛散.

( 常用于 \(u_n = \frac{1}{n \ln n}\) 这类无法用比值/根值法的级数 )

三、交错级数与任意项级数

3.1 交错级数 ( 莱布尼茨判别法 )

形如 \(\sum (-1)^{n-1}u_n\) (\(u_n > 0\)). 若满足

  1. \(u_n \ge u_{n+1}\) ( 单调递减 )
  2. \(\displaystyle\lim_{n\to\infty} u_n = 0\)

则级数 收敛，且和 \(S \le u_1\).

注意事项

莱布尼茨条件是 充分条件. 若不满足单调递减，级数仍可能收敛.

但若不满足 \(\lim u_n = 0\)，则级数必发散.

3.2 绝对收敛与条件收敛

  • 绝对收敛 \(\sum |u_n|\) 收敛. ( 此时 \(\sum u_n\) 必收敛 )
  • 条件收敛 \(\sum u_n\) 收敛，但 \(\sum |u_n|\) 发散.

性质

绝对收敛的级数，交换项的顺序后敛散性和和都不变.

条件收敛的级数，交换项的顺序后和可能改变 ( 黎曼重排定理 ).

四、幂级数

4.1 收敛半径与收敛域

幂级数 \(\sum a_n (x-x_0)^n\) 存在收敛半径 \(R\).

  • \(|x-x_0| < R\) 绝对收敛
  • \(|x-x_0| > R\) 发散
  • \(|x-x_0| = R\) 需单独讨论 ( 代入验证 )

求法 ( 针对 \(x_0=0\) 标准形 \(\sum a_n x^n\) )

\[ R = \lim_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right| \quad \text{或} \quad R = \lim_{n\to\infty} \frac{1}{\sqrt[n]{|a_n|}} \]

缺项级数注意

对于 \(\sum a_n x^{2n}\) 或 \(\sum a_n (x-1)^{3n}\) 这类缺项级数，建议直接使用 比值判别法 求收敛区间，而不要直接套公式算 \(R\).

例如 \(\lim \left| \frac{u_{n+1}}{u_n} \right| < 1 \Rightarrow \lim \left| \frac{a_{n+1} x^{2n+2}}{a_n x^{2n}} \right| < 1 \Rightarrow x^2 < \rho\).

4.2 幂级数的性质

在收敛区间 \((-R, R)\) 内

  1. 连续性 和函数 \(S(x)\) 连续.
  2. 可积性 可逐项积分，半径 \(R\) 不变 ( 端点敛散性可能改善 ).
  3. 可导性 可逐项求导，半径 \(R\) 不变 ( 端点敛散性可能恶化 ).

4.3 函数展开 ( 常用公式 - 需熟记 )

$$ \begin{aligned} e^x &= \sum_{n=0}^{\infty} \frac{x^n}{n!} && (-\infty < x < +\infty) \\ \sin x &= \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!} && (-\infty < x < +\infty) \\ \cos x &= \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!} && (-\infty < x < +\infty) \\ \frac{1}{1-x} &= \sum_{n=0}^{\infty} x^n && (-1 < x < 1) \\ \frac{1}{1+x} &= \sum_{n=0}^{\infty} (-1)^n x^n && (-1 < x < 1) \\ \ln(1+x) &= \sum_{n=0}^{\infty} (-1)^{n-1} \frac{x^n}{n} && (-1 < x \le 1) \end{aligned} $$

五、顶级例题 ( Top-Level Examples )

例题 1 ( 敛散性判定 )

判定级数 \(\displaystyle \sum_{n=1}^{\infty} \frac{n!}{n^n}\) 的敛散性.

解 出现 \(n!\) 和 \(n^n\)，首选 比值判别法. \[ \begin{aligned} \lim_{n\to\infty} \frac{u_{n+1}}{u_n} &= \lim_{n\to\infty} \frac{(n+1)!}{(n+1)^{n+1}} \cdot \frac{n^n}{n!} \\[.5em] &= \lim_{n\to\infty} \frac{(n+1)n^n}{(n+1)^{n+1}} = \lim_{n\to\infty} \frac{n^n}{(n+1)^n} \\[.5em] &= \lim_{n\to\infty} \frac{1}{(1+\frac{1}{n})^n} = \frac{1}{e} \end{aligned} \] 因为 \(\rho = \frac{1}{e} < 1\)，所以该级数 收敛.

例题 2 ( 求收敛域 - 完整步骤 )

求幂级数 \(\displaystyle \sum_{n=1}^{\infty} \frac{(x-1)^n}{n \cdot 2^n}\) 的收敛域.

解 1. 求收敛半径

系数 \(a_n = \frac{1}{n \cdot 2^n}\). \[ R = \lim_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right| = \lim_{n\to\infty} \frac{(n+1)2^{n+1}}{n 2^n} = \lim_{n\to\infty} \frac{n+1}{n} \cdot 2 = 2 \] 收敛中心 \(x_0 = 1\)，收敛区间为 \((1-2, 1+2) = (-1, 3)\).

2. 讨论端点 ( 必须步骤 )

  • 当 \(x=3\) 时，级数为 \(\sum \frac{(3-1)^n}{n 2^n} = \sum \frac{2^n}{n 2^n} = \sum \frac{1}{n}\). 这是调和级数，发散.
  • 当 \(x=-1\) 时，级数为 \(\sum \frac{(-1-1)^n}{n 2^n} = \sum \frac{(-2)^n}{n 2^n} = \sum \frac{(-1)^n}{n}\). 这是交错调和级数，满足莱布尼茨条件 ( 单调递减且趋于0 )，收敛.

结论 收敛域为 \([-1, 3)\).

例题 3 ( 抽象幂级数性质 )

设级数 \(\displaystyle\sum_{n=1}^{\infty}a_n(x-3)^n\) 在 \(x=-6\) 处收敛，在 \(x=12\) 处发散.

试求级数 \(\displaystyle\sum_{n=1}^{\infty}a_nx^{2n-1}\) 的收敛半径.

解 1. 分析原级数 中心 \(x_0 = 3\).

  • 在 \(x=-6\) 处收敛 \(\Rightarrow\) 此点在收敛域内 ( 或端点 ). 距离 \(|-6-3| = 9\). 故 \(R \ge 9\).
  • 在 \(x=12\) 处发散 \(\Rightarrow\) 此点在收敛域外 ( 或端点 ). 距离 \(|12-3| = 9\). 故 \(R \le 9\). 综上，必有收敛半径 \(R=9\). 即 \(\displaystyle\lim_{n\to\infty} |a_n|^{-1/n} = 9\) ( 柯西-阿达马公式推论 ).

2. 分析新级数

新级数 \(\sum a_n x^{2n-1} = \frac{1}{x} \sum a_n (x^2)^n\).

其敛散性主要取决于 \(\sum a_n (x^2)^n\).

令 \(t = x^2\)，则级数变为 \(\sum a_n t^n\).

因为系数 \(a_n\) 没变，所以关于 \(t\) 的收敛半径 \(R_t = 9\).

即当 \(|t| < 9\) 时收敛.

代回 \(x\) \[ |x^2| < 9 \Rightarrow |x| < 3 \]

结论 新级数 \(\displaystyle\sum_{n=1}^{\infty}a_nx^{2n-1}\) 的收敛半径为 3.

例题 4 ( 概念辨析 - 级数性质 )

判断下列命题的真假

  1. 若 \(u_n \ge 0\) 且 \(\displaystyle\sum u_n\) 收敛，则 \(\displaystyle\sum u_n^2\) 必收敛.
  2. 若 \(\displaystyle\sum u_n\) 收敛，则 \(\displaystyle\sum u_n^2\) 必收敛.
  3. 若 \(\displaystyle\sum u_n\) 发散，则 \(\displaystyle\sum u_n^2\) 必发散.

解

1. 命题 (1) 为真.

因为 \(\sum u_n\) 收敛，所以 \(\lim_{n\to\infty} u_n = 0\).

故存在 \(N\)，当 \(n > N\) 时，有 \(0 \le u_n < 1\).

此时 \(0 \le u_n^2 \le u_n\).

根据 比较判别法，大的收敛 \(\Rightarrow\) 小的收敛.

所以 \(\sum u_n^2\) 收敛.

2. 命题 (2) 为假.

反例 交错调和级数 \(\sum u_n = \sum \frac{(-1)^n}{\sqrt{n}}\).

该级数根据莱布尼茨判别法 收敛.

但 \(\sum u_n^2 = \sum \frac{1}{n}\) 为调和级数，发散.

( 关键点：\(u_n\) 有正有负时，绝对收敛性未知 )

3. 命题 (3) 为假.

反例 \(\sum u_n = \sum \frac{1}{n}\) 发散.

但 \(\sum u_n^2 = \sum \frac{1}{n^2}\) 收敛 (\(p=2 > 1\)).

( 关键点：\(u_n \to 0\) 的速度不够快导致发散，平方后可能就够快了 )

例题 5 ( 正项级数审敛法进阶 )

判定级数 \(\displaystyle \sum_{n=1}^{\infty} \ln\left(1 + \frac{1}{n^2}\right)\) 的敛散性.

解

观察通项 \(u_n = \ln(1 + \frac{1}{n^2})\).

当 \(n \to \infty\) 时，\(\frac{1}{n^2} \to 0\).

利用 等价无穷小 \(\ln(1+x) \sim x \ (x \to 0)\).

故 \(u_n \sim \frac{1}{n^2}\).

构造比较对象 \(v_n = \frac{1}{n^2}\). \[ \lim_{n\to\infty} \frac{u_n}{v_n} = \lim_{n\to\infty} \frac{\ln(1 + \frac{1}{n^2})}{\frac{1}{n^2}} = 1 \] 因为极限为常数 \(1\) (\(0 < 1 < +\infty\))，所以 \(\sum u_n\) 与 \(\sum v_n\) 同敛散.

由于 \(\sum \frac{1}{n^2}\) 是 \(p=2\) 的 \(p\)-级数，收敛.

故原级数 收敛.
