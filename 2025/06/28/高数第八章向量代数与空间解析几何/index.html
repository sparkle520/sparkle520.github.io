一、 向量代数

1. 与向量有关的基本概念

  • 向量 既有大小又有方向的量. 模记为 \(|\boldsymbol{a}|\).
  • 坐标表示 \(\boldsymbol{a} = \{a_x, a_y, a_z\} = a_x\boldsymbol{i} + a_y\boldsymbol{j} + a_z\boldsymbol{k}\)，模长 \(|\boldsymbol{a}| = \sqrt{a_x^2 + a_y^2 + a_z^2}\).
  • 方向角与方向余弦 向量与三个坐标轴正向的夹角 \(\alpha, \beta, \gamma\) 称为方向角. 其方向余弦分别为 \(\cos\alpha = \frac{a_x}{|\boldsymbol{a}|}, \cos\beta = \frac{a_y}{|\boldsymbol{a}|}, \cos\gamma = \frac{a_z}{|\boldsymbol{a}|}\)，且满足恒等式 \(\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1\).
  • 单位向量 \(\boldsymbol{a}^{\circ} = \frac{\boldsymbol{a}}{|\boldsymbol{a}|} = \{\cos\alpha, \cos\beta, \cos\gamma\}\).

2. 向量的运算及性质

1. 向量的加减法

  • 几何意义 平行四边形法则 ( 或三角形法则 ).
  • 代数表示 坐标对应相加减. \[\boldsymbol{a} \pm \boldsymbol{b} = \{a_x \pm b_x, a_y \pm b_y, a_z \pm b_z\}\]

2. 数量积 ( 点积、内积 \(\boldsymbol{a} \cdot \boldsymbol{b}\) )

  • 几何意义 投影与做功. \(\boldsymbol{a} \cdot \boldsymbol{b} = |\boldsymbol{a}||\boldsymbol{b}|\cos\theta\).
  • 投影与投影向量
    • 投影 (标量) \(\text{Prj}_{\boldsymbol{b}}\boldsymbol{a} = |\boldsymbol{a}|\cos\theta = \frac{\boldsymbol{a} \cdot \boldsymbol{b}}{|\boldsymbol{b}|}\)，表示 \(\boldsymbol{a}\) 在 \(\boldsymbol{b}\) 上的投影大小及正负.
    • 投影向量 (矢量) 向量 \(\boldsymbol{a}\) 在 \(\boldsymbol{b}\) 方向上的投影向量为 \(\left( \frac{\boldsymbol{a} \cdot \boldsymbol{b}}{|\boldsymbol{b}|} \right) \frac{\boldsymbol{b}}{|\boldsymbol{b}|} = \frac{\boldsymbol{a} \cdot \boldsymbol{b}}{|\boldsymbol{b}|^2} \boldsymbol{b}\).
  • 代数表示 对应坐标乘积之和. \[\boldsymbol{a} \cdot \boldsymbol{b} = a_x b_x + a_y b_y + a_z b_z\]
  • 核心应用
    • 求夹角 \(\cos\theta = \frac{\boldsymbol{a} \cdot \boldsymbol{b}}{|\boldsymbol{a}||\boldsymbol{b}|}\)
    • 判断垂直 \(\boldsymbol{a} \perp \boldsymbol{b} \iff \boldsymbol{a} \cdot \boldsymbol{b} = 0\)

3. 向量积 ( 叉积、外积 \(\boldsymbol{a} \times \boldsymbol{b}\) )

  • 几何意义 结果为一个 向量，方向由右手定则决定 ( 垂直于 \(\boldsymbol{a}\) 和 \(\boldsymbol{b}\) 所在的平面 )，模长为 \(|\boldsymbol{a}||\boldsymbol{b}|\sin\theta\)，即以 \(\boldsymbol{a},\boldsymbol{b}\) 为邻边的平行四边形面积.
  • 代数表示 三阶行列式. \[\boldsymbol{a} \times \boldsymbol{b} = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\[.5em] a_x & a_y & a_z \\[.5em] b_x & b_y & b_z \end{vmatrix}\]
  • 核心应用
    • 求平面的 法向量 ( 同时垂直于平面内两不共线向量 ).
    • 求面积 三角形面积 \(S_{\triangle} = \frac{1}{2} |\boldsymbol{a} \times \boldsymbol{b}|\).
    • 判断平行 \(\boldsymbol{a} \parallel \boldsymbol{b} \iff \boldsymbol{a} \times \boldsymbol{b} = \boldsymbol{0}\)

4. 混合积 \((\boldsymbol{abc})\)

  • 定义 \((\boldsymbol{abc}) = (\boldsymbol{a} \times \boldsymbol{b}) \cdot \boldsymbol{c}\)
  • 几何意义 绝对值表示以 \(\boldsymbol{a},\boldsymbol{b},\boldsymbol{c}\) 为棱的平行六面体体积.
  • 代数表示 坐标构成的三阶行列式. \[(\boldsymbol{abc}) = \begin{vmatrix} a_x & a_y & a_z \\[.5em] b_x & b_y & b_z \\[.5em] c_x & c_y & c_z \end{vmatrix}\]
  • 核心应用
    • 判断共面 \(\boldsymbol{a},\boldsymbol{b},\boldsymbol{c}\) 共面 \(\iff (\boldsymbol{abc}) = 0\)

5. 深入 : 二重向量积 ( 向量三重积 )

  • 定义 连续进行两次叉乘，即 \(\boldsymbol{a} \times (\boldsymbol{b} \times \boldsymbol{c})\). 结果仍为一个向量.
  • 几何意义 向量 \(\boldsymbol{b} \times \boldsymbol{c}\) 垂直于 \(\boldsymbol{b}\) 和 \(\boldsymbol{c}\) 所在的平面，而 \(\boldsymbol{a} \times (\boldsymbol{b} \times \boldsymbol{c})\) 又垂直于 \(\boldsymbol{b} \times \boldsymbol{c}\). 因此，最终结果必在 \(\boldsymbol{b}\) 和 \(\boldsymbol{c}\) 所在的平面内，可以表示为 \(\boldsymbol{b}\) 和 \(\boldsymbol{c}\) 的线性组合.
  • BAC-CAB 展开公式 \[ \boldsymbol{a} \times (\boldsymbol{b} \times \boldsymbol{c}) = \boldsymbol{b}(\boldsymbol{a} \cdot \boldsymbol{c}) - \boldsymbol{c}(\boldsymbol{a} \cdot \boldsymbol{b}) \] ( 记忆口诀 “远近原则” 或 “BAC减CAB”，即中间向量 \(\boldsymbol{b}\) 乘以 \((\boldsymbol{a}\cdot\boldsymbol{c})\)，减去后面向量 \(\boldsymbol{c}\) 乘以 \((\boldsymbol{a}\cdot\boldsymbol{b})\) )
  • 注意 叉乘不满足结合律，即 \(\boldsymbol{a} \times (\boldsymbol{b} \times \boldsymbol{c}) \neq (\boldsymbol{a} \times \boldsymbol{b}) \times \boldsymbol{c}\).

二、 平面与直线

1. 平面方程

核心形式与特征

  • 一般式方程 \(Ax + By + Cz + D = 0\) 平面的法向量为 \(\boldsymbol{n} = \{A, B, C\}\). ( 其中 \(A, B, C\) 不全为零 )
  • 点法式方程 \(A(x - x_0) + B(y - y_0) + C(z - z_0) = 0\) 已知平面过点 \(M_0(x_0, y_0, z_0)\)，且法向量为 \(\boldsymbol{n} = \{A, B, C\}\).
  • 截距式方程 \(\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1\) 平面在 \(x, y, z\) 轴上的截距分别为 \(a, b, c\) ( 均不为零 ).
  • 三点式方程 ( 由不共线的三点 \(P_1, P_2, P_3\) 确定 ) 直接利用混合积为 0 的共面条件：\(|(\overrightarrow{P_1P}\;\overrightarrow{P_1P_2}\;\overrightarrow{P_1P_3})| = 0\)，即构成三阶行列式等于 0.

平面束方程 ( 常用于求过直线的平面 ) 设直线 \(L\) 由平面 \(\Pi_1: A_1x + B_1y + C_1z + D_1 = 0\) 与 \(\Pi_2: A_2x + B_2y + C_2z + D_2 = 0\) 交线确定.

则通过直线 \(L\) 的所有平面 ( 平面束 ) 的方程为 \[ (A_1x + B_1y + C_1z + D_1) + \lambda (A_2x + B_2y + C_2z + D_2) = 0 \quad (\lambda \in \mathbb{R}) \] ( 注意 此方程不包含平面 \(\Pi_2\) 本身 )

2. 直线方程

核心形式与特征

  • 一般式方程 ( 两平面交线 ) \[\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\[.5em] A_2x + B_2y + C_2z + D_2 = 0 \end{cases}\] 直线的 方向向量 为 \(\boldsymbol{s} = \boldsymbol{n_1} \times \boldsymbol{n_2}\) ( 即两个平面法向量的叉乘 ).
  • 对称式方程 ( 点向式 ) \[\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}\] 已知直线过点 \(M_0(x_0, y_0, z_0)\)，且方向向量为 \(\boldsymbol{s} = \{l, m, n\}\). ( 补充说明：若分母中某一项为 0，例如 \(l=0\)，则其数学含义为 \(x - x_0 = 0\) 且 \(\frac{y - y_0}{m} = \frac{z - z_0}{n}\)，即直线位于平面 \(x=x_0\) 内 )
  • 两点式方程 已知直线过两点 \(P_1(x_1, y_1, z_1)\) 与 \(P_2(x_2, y_2, z_2)\)，直接取方向向量 \(\boldsymbol{s} = \overrightarrow{P_1P_2}\)： \[\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}\]
  • 参数式方程 \[\begin{cases} x = x_0 + lt \\[.5em] y = y_0 + mt \\[.5em] z = z_0 + nt \end{cases}\] ( 通常由对称式方程令其等于参数 \(t\) 得到，常用于求交点 )

3. 平面与直线间的位置关系

3.1 平面与平面间的位置关系

设平面 \(\Pi_1: A_1 x + B_1 y + C_1 z + D_1 = 0\) ( 法向量 \(\boldsymbol{n_1}\) )，\(\Pi_2: A_2 x + B_2 y + C_2 z + D_2 = 0\) ( 法向量 \(\boldsymbol{n_2}\) )，则

  1. 平行 \(\Pi_1 \parallel \Pi_2 \iff \boldsymbol{n_1} \parallel \boldsymbol{n_2} \iff \frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}\) ( 若某分母为0，则分子亦为0 )
  2. 垂直 \(\Pi_1 \perp \Pi_2 \iff \boldsymbol{n_1} \perp \boldsymbol{n_2} \iff A_1 A_2 + B_1 B_2 + C_1 C_2 = 0\)
  3. 夹角 \(\theta\) ( 二面角的锐角 ) \[\cos \theta = \frac{|\boldsymbol{n_1} \cdot \boldsymbol{n_2}|}{|\boldsymbol{n_1}||\boldsymbol{n_2}|} = \frac{|A_1 A_2 + B_1 B_2 + C_1 C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2} \sqrt{A_2^2 + B_2^2 + C_2^2}} \quad \left(0 \leqslant \theta \leqslant \frac{\pi}{2}\right)\]

3.2 直线与直线间的位置关系

设直线 \(L_1\) 方向向量为 \(\boldsymbol{s_1} = \{l_1, m_1, n_1\}\)，直线 \(L_2\) 方向向量为 \(\boldsymbol{s_2} = \{l_2, m_2, n_2\}\)，则

  1. 平行 \(L_1 \parallel L_2 \iff \boldsymbol{s_1} \parallel \boldsymbol{s_2} \iff \frac{l_1}{l_2} = \frac{m_1}{m_2} = \frac{n_1}{n_2}\)
  2. 垂直 \(L_1 \perp L_2 \iff \boldsymbol{s_1} \perp \boldsymbol{s_2} \iff l_1 l_2 + m_1 m_2 + n_1 n_2 = 0\)
  3. 夹角 \(\theta\) ( 锐角 ) \[\cos \theta = \frac{|\boldsymbol{s_1} \cdot \boldsymbol{s_2}|}{|\boldsymbol{s_1}||\boldsymbol{s_2}|} = \frac{|l_1 l_2 + m_1 m_2 + n_1 n_2|}{\sqrt{l_1^2 + m_1^2 + n_1^2} \sqrt{l_2^2 + m_2^2 + n_2^2}} \quad \left(0 \leqslant \theta \leqslant \frac{\pi}{2}\right)\]

3.3 平面与直线的位置关系

设平面 \(\Pi\) 法向量为 \(\boldsymbol{n} = \{A, B, C\}\)，直线 \(L\) 方向向量为 \(\boldsymbol{s} = \{l, m, n\}\)，则 ( 注意 线面关系的条件与法向量、方向向量的关系是“反”的 )

  1. 线面平行 \(\Pi \parallel L \iff \boldsymbol{n} \perp \boldsymbol{s} \iff Al + Bm + Cn = 0\)
  2. 线面垂直 \(\Pi \perp L \iff \boldsymbol{n} \parallel \boldsymbol{s} \iff \frac{A}{l} = \frac{B}{m} = \frac{C}{n}\)
  3. 夹角 \(\theta\) ( 直线与它在平面上投影的夹角 ) \[\sin \theta = \frac{|\boldsymbol{n} \cdot \boldsymbol{s}|}{|\boldsymbol{n}||\boldsymbol{s}|} = \frac{|Al + Bm + Cn|}{\sqrt{A^2 + B^2 + C^2} \sqrt{l^2 + m^2 + n^2}} \quad \left(0 \leqslant \theta \leqslant \frac{\pi}{2}\right)\]

3.4 常见的线面组合求法 ( 投影与对称 )

1. 求直线在平面上的投影直线 已知直线 \(L\) ( 方向向量 \(\boldsymbol{s}\) ) 与平面 \(\Pi\) ( 法向量 \(\boldsymbol{n}\) )，求 \(L\) 在 \(\Pi\) 上的投影直线 \(L'\).

解法 ( 投影平面法 ) 投影直线 \(L'\) 是平面 \(\Pi\) 与 过 \(L\) 且垂直于 \(\Pi\) 的投影平面 \(\Pi'\) 的交线.

  • 第一步 找投影平面 \(\Pi'\) 的法向量 \(\boldsymbol{n'} = \boldsymbol{s} \times \boldsymbol{n}\).
  • 第二步 在直线 \(L\) 上任取一点 \(P_0\)，该点必在 \(\Pi'\) 上. 用点法式写出 \(\Pi'\) 的方程.
  • 第三步 联立 \(\Pi\) 与 \(\Pi'\) 的方程，即得到投影直线 \(L'\) 的一般式方程.

2. 求点关于平面的对称点 已知点 \(P\) 及平面 \(\Pi\)，求 \(P\) 关于 \(\Pi\) 的对称点 \(P'\).

解法 ( 法线交点法 )

  • 第一步 写出过点 \(P\) 且垂直于平面 \(\Pi\) 的直线方程 ( 法线 ). 其方向向量即为 \(\Pi\) 的法向量 \(\boldsymbol{n}\).
  • 第二步 将法线方程化为参数式，代入平面 \(\Pi\) 的方程，求出交点 \(M\) ( 即 \(P\) 在平面上的投影点 ).
  • 第三步 利用中点坐标公式：\(M\) 是 \(PP'\) 的中点，即 \(x_M = \frac{x_P + x_{P'}}{2}\)，从而求出对称点 \(P'\) 的坐标.

3. 求直线关于平面的对称直线 已知直线 \(L\) 及平面 \(\Pi\)，求 \(L\) 关于 \(\Pi\) 的对称直线 \(L'\).

解法

  • 情形 A ( \(L\) 与 \(\Pi\) 相交 )
    1. 求出 \(L\) 与 \(\Pi\) 的交点 \(A\) ( 将 \(L\) 的参数式代入 \(\Pi\) ). 交点 \(A\) 必在对称直线 \(L'\) 上.
    2. 在 \(L\) 上另取一点 \(P\)，利用上述“点关于平面的对称点”方法求出 \(P\) 关于 \(\Pi\) 的对称点 \(P'\).
    3. 两点 \(A\) 和 \(P'\) 确定对称直线 \(L'\).
  • 情形 B ( \(L \parallel \Pi\) ) 对称直线 \(L'\) 必平行于 \(L\)，因此方向向量相同. 只需在 \(L\) 上取一点求对称点，再结合方向向量写出方程.

4. 空间距离公式速查

1. 点到平面的距离公式及几何推导 点 \(M_0(x_0, y_0, z_0)\) 到平面 \(\Pi: Ax + By + Cz + D = 0\) 的距离 \[d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}\]

深入解析 ( 投影法 ) 在平面 \(\Pi\) 上任取一点 \(M_1(x_1, y_1, z_1)\)，则平面法向量 \(\boldsymbol{n} = \{A, B, C\}\).

点 \(M_0\) 到平面的距离 \(d\)，本质上就是向量 \(\overrightarrow{M_1 M_0}\) 在法向量 \(\boldsymbol{n}\) 上的投影的绝对值!

\[ d = |\text{prj}_{\boldsymbol{n}}\overrightarrow{M_1 M_0}| = \frac{|\overrightarrow{M_1 M_0} \cdot \boldsymbol{n}|}{|\boldsymbol{n}|} \]

因为 \(M_1\) 在平面上，满足 \(Ax_1 + By_1 + Cz_1 + D = 0 \implies D = -(Ax_1 + By_1 + Cz_1)\). 展开点乘 \[ \begin{aligned} \overrightarrow{M_1 M_0} \cdot \boldsymbol{n} &= A(x_0 - x_1) + B(y_0 - y_1) + C(z_0 - z_1) \\[.5em] &= Ax_0 + By_0 + Cz_0 - (Ax_1 + By_1 + Cz_1) \\[.5em] &= Ax_0 + By_0 + Cz_0 + D \end{aligned} \] 代入投影公式即得上述标准公式，极大地体现了向量投影在距离计算中的降维打击能力.

2. 点到直线的距离公式解析 点 \((x_0, y_0, z_0)\) 到直线 \(\frac{x - x_1}{l} = \frac{y - y_1}{m} = \frac{z - z_1}{n}\) 的距离可以利用 平行四边形面积 求解

设直线上已知点为 \(M_1(x_1, y_1, z_1)\)，直线的方向向量为 \(\boldsymbol{s} = \{l, m, n\}\).

点 \(M_0(x_0, y_0, z_0)\) 到直线的距离 \(d\) 就是以向量 \(\overrightarrow{M_1 M_0}\) 和 \(\boldsymbol{s}\) 为邻边的平行四边形的高.

根据平行四边形面积公式 \(S = |\overrightarrow{M_1 M_0} \times \boldsymbol{s}| = |\boldsymbol{s}| \cdot d\)，可得 \[d = \frac{|\overrightarrow{M_1 M_0} \times \boldsymbol{s}|}{|\boldsymbol{s}|}\]

3. 两异面直线间的距离公式解析

设直线 \(L_1\) 与 \(L_2\) 为异面直线，方向向量分别为 \(\boldsymbol{s_1}\) 与 \(\boldsymbol{s_2}\)，直线上分别有点 \(M_1\) 和 \(M_2\).

这两条异面直线之间的距离 \(d\)，就等于以 \(\boldsymbol{s_1}\)、\(\boldsymbol{s_2}\) 和 \(\overrightarrow{M_1 M_2}\) 为相邻三条棱的平行六面体的高.

平行六面体的体积 \(V = |(\boldsymbol{s_1}\;\boldsymbol{s_2}\;\overrightarrow{M_1 M_2})|\) ( 混合积 ).

底面积 \(S = |\boldsymbol{s_1} \times \boldsymbol{s_2}|\).

根据 \(V = S \cdot d\)，得距离公式 \[d = \frac{|(\boldsymbol{s_1}\;\boldsymbol{s_2}\;\overrightarrow{M_1 M_2})|}{|\boldsymbol{s_1} \times \boldsymbol{s_2}|}\]

4. 平行平面之间的距离公式

设两个平行平面方程分别为 \(\Pi_1: Ax + By + Cz + D_1 = 0\) 与 \(\Pi_2: Ax + By + Cz + D_2 = 0\)。 ( 注意：使用公式前必须将两平面的法向量系数 \(A, B, C\) 化为完全一致! )

两平面之间的距离为 \[d = \frac{|D_1 - D_2|}{\sqrt{A^2 + B^2 + C^2}}\]

推导说明 在 \(\Pi_1\) 上任取一点 \(M_0(x_0, y_0, z_0)\)，则满足 \(Ax_0 + By_0 + Cz_0 = -D_1\)。 该点到 \(\Pi_2\) 的距离即为两平行平面的距离，代入点到平面距离公式： \(d = \frac{|Ax_0 + By_0 + Cz_0 + D_2|}{\sqrt{A^2 + B^2 + C^2}} = \frac{|-D_1 + D_2|}{\sqrt{A^2 + B^2 + C^2}}\)，即得该公式。

5. 平行直线之间的距离公式

设两直线 \(L_1 \parallel L_2\)，方向向量均为 \(\boldsymbol{s}\)。在 \(L_1\) 上取已知点 \(M_1\)，在 \(L_2\) 上取已知点 \(M_2\)。

两平行直线之间的距离，等于以向量 \(\overrightarrow{M_1 M_2}\) 和 \(\boldsymbol{s}\) 为邻边的平行四边形的高： \[d = \frac{|\overrightarrow{M_1 M_2} \times \boldsymbol{s}|}{|\boldsymbol{s}|}\] ( 本质上与点到直线的距离公式完全一致 )

三、 空间曲面与曲线

1. 旋转面与柱面

旋转面方程

由曲线绕一条定直线 ( 旋转轴 ) 旋转一周所生成的曲面.

1. 坐标面内的平面曲线绕坐标轴旋转 设 \(xOy\) 面上的曲线方程为 \(\begin{cases} f(x,y)=0 \\ z=0 \end{cases}\). * 绕 \(x\) 轴旋转 方程为 \(f(x, \pm\sqrt{y^2+z^2}) = 0\). ( 即将 \(y\) 替换为 \(\pm\sqrt{y^2+z^2}\) ) * 绕 \(y\) 轴旋转 方程为 \(f(\pm\sqrt{x^2+z^2}, y) = 0\). ( 即将 \(x\) 替换为 \(\pm\sqrt{x^2+z^2}\) ) ( 口诀：绕谁转谁不变，另一个换成“另两个字母的平方和开根号” )

2. 空间曲线绕坐标轴旋转 ( 一般情况 ) 设空间曲线 \(\Gamma: \begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}\) 绕 \(z\) 轴旋转. * 几何特征 旋转面上任意一点 \((x,y,z)\) 都是曲线上某一点 \((x_0,y_0,z_0)\) 旋转而来的. * 不变量关系 绕 \(z\) 轴旋转时，具有两个不变性：高度不变 (\(z = z_0\))，到 \(z\) 轴的距离不变 (\(x^2 + y^2 = x_0^2 + y_0^2\)). * 求解方法 联立以下四个方程组 \[ \begin{cases} F(x_0, y_0, z_0) = 0 \\[.5em] G(x_0, y_0, z_0) = 0 \\[.5em] z_0 = z \\[.5em] x_0^2 + y_0^2 = x^2 + y^2 \end{cases} \] 从中消去参数 \(x_0, y_0, z_0\)，即可得到仅含 \(x,y,z\) 的旋转面方程 \(\Phi(x,y,z) = 0\).

柱面方程 平行于定直线并沿定曲线移动形成的曲面.

  • 常见特例 准线在 \(xOy\) 面上为 \(f(x,y)=0\)，母线平行于 \(z\) 轴，则柱面方程即为 \(f(x,y)=0\) ( 在空间解析几何中，缺哪个字母，母线就平行于哪个轴 ).
  • 常见柱面
    • 圆柱面 \(x^2 + y^2 = R^2\)
    • 椭圆柱面 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\)
    • 抛物柱面 \(y^2 = 2px\)

2. 常见的二次曲面及图形

二次曲面方程速记

  1. 球面 \((x-x_0)^2 + (y-y_0)^2 + (z-z_0)^2 = R^2\) ( 球心 \((x_0, y_0, z_0)\)，半径 \(R\) )
  2. 椭球面 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\) ( 全是正号 )
  3. 单叶双曲面 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1\) ( 一个负号，图形连通 )
  4. 双叶双曲面 \(-\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1\) ( 两个负号，图形分为两片 )
  5. 椭圆抛物面 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2pz\ (p > 0)\) ( 一次项，且二次项同号 )
  6. 双曲抛物面 ( 马鞍面 ) \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2pz\ (p > 0)\) ( 一次项，且二次项异号 )
  7. 二次锥面 \(\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 0\) ( 常数项为0 )

深入解析 : 截痕法与马鞍面

截痕法 ( 平行截面法 ) 是研究二次曲面形状最有效的手段，即用平行于坐标面的平面 ( 如 \(z=k\) ) 去截曲面，观察截得的曲线形状，然后将这些截痕“堆叠”起来想象整体图形.

以双曲抛物面 ( 马鞍面 ) \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2pz\ (p>0)\) 为例 : 1. 用 \(z = k\) 截 : 截痕为 \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2pk\). 当 \(k>0\) 时，是实轴平行于 \(x\) 轴的双曲线; 当 \(k<0\) 时，是实轴平行于 \(y\) 轴的双曲线; 当 \(k=0\) 时，是两条相交直线 \(\frac{x}{a} = \pm \frac{y}{b}\). 2. 用 \(y = k\) 截 : 截痕为 \(\frac{x^2}{a^2} = 2pz + \frac{k^2}{b^2}\)，即开口向上的抛物线. 3. 用 \(x = k\) 截 : 截痕为 \(-\frac{y^2}{b^2} = 2pz - \frac{k^2}{a^2}\)，即开口向下的抛物线.

直观感受 : 想象你正骑在一匹马上 ( \(z\) 轴正向朝上 ). 马的脊背是沿 \(y\) 轴方向下凹的 ( 开口向上的抛物线 )，而你两侧的腿是沿 \(x\) 轴方向下垂的 ( 开口向下的抛物线 )，这就构成了经典的“马鞍”形状.

3. 空间曲线及其方程

空间曲线方程常见形式

  1. 参数式 \[\begin{cases} x = x(t) \\[.5em] y = y(t) \\[.5em] z = z(t) \end{cases}\]
  2. 一般式 ( 两曲面方程联立 ) \[\begin{cases} F(x,y,z) = 0 \\[.5em] G(x,y,z) = 0 \end{cases}\]

一般式化为参数式的方法 * 若一般式中含有圆/椭圆的形式 (如 \(x^2+y^2=R^2\))，常设 \(x=R\cos t, y=R\sin t\)，再代入另一方程求出 \(z(t)\). * 若方程易于解出某一个变量 (如 \(y=f(x)\))，则可直接设 \(x=t\)，进而得到 \(y=f(t)\) 及 \(z=g(t)\).

4. 空间曲线的投影

求投影柱面与投影曲线的方法 设有空间曲线 \(\Gamma: \begin{cases} F(x,y,z) = 0 \\[.5em] G(x,y,z) = 0 \end{cases}\)

  • 求在 \(xOy\) 面的投影 联立方程组消去 \(z\)，得到关于 \(x,y\) 的方程 \(\varphi(x,y) = 0\). 此方程即为 投影柱面方程.
  • 投影曲线方程 将投影柱面与 \(xOy\) 面 ( \(z=0\) ) 联立，即 \[\begin{cases} \varphi(x,y) = 0 \\[.5em] z = 0 \end{cases}\] ( 求向哪个面投影，就消去没出现的那个字母，并加上该字母等于0的条件 )

个人笔记

空间曲线奔赴平面投影，像一颗心奔赴归处，舍去一个维度的纷繁，在简洁里，藏起曾遍历空间的风尘.

例题

例题 1 过点 \((1,0,1)\) 与 \((0,1,1)\) 且与曲面 \(z = 1 + x^2 + y^2\) 相切的平面为 \(\underline{\quad\quad\quad}\).

分析 设切点，求切面法向量，利用平面过点及向量垂直关系，结合曲面方程求解，得平面方程.

解 设切点为 \((x_0,y_0,z_0)\)，法向量为 \((-2x_0,-2y_0,1)\)，则
\[ \begin{cases} z_0 = 1 + x_0^2 + y_0^2, \\[.5em] -2x_0(x_0 - 0) - 2y_0(y_0 - 1) + (z_0 - 1) = 0, \\[.5em] -2x_0(1 - 0) - 2y_0(0 - 1) + (1 - 1) = 0, \end{cases} \]

解得
\[ \begin{cases} x_0 = 0, \\[.5em] y_0 = 0, \\[.5em] z_0 = 1. \end{cases} \quad\text{或}\quad \begin{cases} x_0 = 1, \\[.5em] y_0 = 1, \\[.5em] z_0 = 3. \end{cases} \]

故法向量为 \((0,0,1)\) 或 \((-2,-2,1)\)，所以平面方程为
\[ z = 1 \ \text{或} \ z - 2x - 2y + 1 = 0. \]

例题 2 设两直线 \(L_1:\begin{cases}x - 3y + z = 0, \\[.5em] 2x - 4y + z + 1 = 0,\end{cases}\) \(L_2:\displaystyle\frac{x}{1} = \frac{y + 1}{3} = \frac{z - 2}{4}\).

1.证明 \(L_1\) 与 \(L_2\) 是异面直线;
2.求 \(L_1\) 与 \(L_2\) 之间的距离;
3.求过 \(L_1\) 且平行于 \(L_2\) 的平面方程.

分析 第一问 异面直线是既不相交，又不平行的直线. 显然，连接两条直线作为一个向量，此时必然和两条直线的方向向量构成一个平行六面体. 由行列式几何意义可知，行列式不为 \(0\). 第二问 直线到直线距离公式. 第三问 由题意显然可知，所求平面方程法向量与两条直线的方向向量都垂直，故法向量取两个直线的方向向量的向量积. 任取 \(L_1\) 一点即可求解.

解

1.证明 \(L_1\) 与 \(L_2\) 是异面直线 取 \(L_1\) 上点 \(P(0,1,3)\) ( 联立 \(L_1\) 方程，令 \(x = 0\) 代入得 \(y=1,z = 3\) )，求 \(L_1\) 方向向量 \(\boldsymbol{s_1}\) 由 \(L_1\) 两平面法向量 \(\boldsymbol{n_1}=(1,-3,1)\)，\(\boldsymbol{n_2}=(2,-4,1)\)，叉乘得 \(\boldsymbol{s_1}=\boldsymbol{n_1} \times \boldsymbol{n_2}=\begin{vmatrix}\boldsymbol{i}&\boldsymbol{j}&\boldsymbol{k}\\[.5em]1&-3&1\\[.5em]2&-4&1\end{vmatrix}=(1,1,2)\).
\(L_2\) 过点 \(Q(0,-1,2)\)，方向向量 \(\boldsymbol{s_2}=(1,3,4)\)，向量 \(\overrightarrow{PQ}=(0,-2,-1)\).
计算混合积 \([\overrightarrow{PQ},\boldsymbol{s_1},\boldsymbol{s_2}]=\begin{vmatrix}0&-2&-1\\[.5em]1&1&2\\[.5em]1&3&4\end{vmatrix}=2\neq0\)，故异面.

2.求 \(L_1\) 与 \(L_2\) 之间的距离 异面直线距离公式 \(d = \displaystyle\frac{|\overrightarrow{PQ} \cdot (\boldsymbol{s_1} \times \boldsymbol{s_2})|}{|\boldsymbol{s_1} \times \boldsymbol{s_2}|}\).
先算 \(\boldsymbol{s_1} \times \boldsymbol{s_2}=\begin{vmatrix}\boldsymbol{i}&\boldsymbol{j}&\boldsymbol{k}\\[.5em]1&1&2\\[.5em]1&3&4\end{vmatrix}=(-2,-2,2)\)，模长 \(|\boldsymbol{s_1} \times \boldsymbol{s_2}|=\sqrt{(-2)^2+(-2)^2+2^2}=2\sqrt{3}\). \(\overrightarrow{PQ} \cdot (\boldsymbol{s_1} \times \boldsymbol{s_2})=0\times(-2)+(-2)\times(-2)+(-1)\times2 = 4 - 2 = 2\)，绝对值为 \(2\).
距离 \(d = \frac{2}{2\sqrt{3}}=\frac{1}{\sqrt{3}}\).

3.求过 \(L_1\) 且平行于 \(L_2\) 的平面方程 平面法向量 \(\boldsymbol{n}=\boldsymbol{s_1} \times \boldsymbol{s_2}=(-2,-2,2)\).
取 \(L_1\) 上点 \(P(0,1,3)\)，由点法式得平面方程 \(1\times(x - 0)+1\times(y - 1)-1\times(z - 3)=0\)，即 \(x + y - z + 2 = 0\).

例题 3 求两条异面直线的公垂线方程 设直线 \(L_1: \frac{x-1}{1} = \frac{y}{-1} = \frac{z-2}{2}\)，直线 \(L_2: \frac{x}{-1} = \frac{y-1}{2} = \frac{z-1}{1}\). 求 \(L_1\) 与 \(L_2\) 的公垂线 \(L\) 的方程.

分析 公垂线 \(L\) 与 \(L_1, L_2\) 都垂直且相交. 其方向向量 \(\boldsymbol{s} = \boldsymbol{s_1} \times \boldsymbol{s_2}\). 可以通过构造两个平面来求，平面 \(\Pi_1\) 包含 \(L_1\) 且平行于 \(\boldsymbol{s}\)，平面 \(\Pi_2\) 包含 \(L_2\) 且平行于 \(\boldsymbol{s}\). 这两个平面的交线即为公垂线.

解 1. 求公垂线方向向量 \(L_1\) 的方向向量 \(\boldsymbol{s_1} = (1, -1, 2)\)，过点 \(P_1(1, 0, 2)\). \(L_2\) 的方向向量 \(\boldsymbol{s_2} = (-1, 2, 1)\)，过点 \(P_2(0, 1, 1)\). 公垂线的方向向量 \(\boldsymbol{s} = \boldsymbol{s_1} \times \boldsymbol{s_2} = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\[.5em] 1 & -1 & 2 \\[.5em] -1 & 2 & 1 \end{vmatrix} = (-5, -3, 1)\).

2. 构造包含 \(L_1\) 与公垂线的平面 \(\Pi_1\) 平面 \(\Pi_1\) 的法向量 \(\boldsymbol{n_1} = \boldsymbol{s_1} \times \boldsymbol{s} = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\[.5em] 1 & -1 & 2 \\[.5em] -5 & -3 & 1 \end{vmatrix} = (5, -11, -8)\). 利用点法式，过点 \(P_1(1, 0, 2)\)，\(\Pi_1\) 方程为 \(5(x-1) - 11(y-0) - 8(z-2) = 0\)，即 \(5x - 11y - 8z + 11 = 0\).

3. 构造包含 \(L_2\) 与公垂线的平面 \(\Pi_2\) 平面 \(\Pi_2\) 的法向量 \(\boldsymbol{n_2} = \boldsymbol{s_2} \times \boldsymbol{s} = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\[.5em] -1 & 2 & 1 \\[.5em] -5 & -3 & 1 \end{vmatrix} = (5, -4, 13)\). 利用点法式，过点 \(P_2(0, 1, 1)\)，\(\Pi_2\) 方程为 \(5(x-0) - 4(y-1) + 13(z-1) = 0\)，即 \(5x - 4y + 13z - 9 = 0\).

4. 得公垂线方程 公垂线 \(L\) 为 \(\Pi_1\) 与 \(\Pi_2\) 的交线，其一般式方程为 \[ \begin{cases} 5x - 11y - 8z + 11 = 0, \\[.5em] 5x - 4y + 13z - 9 = 0. \end{cases} \] ( 核心知识点 叉乘求法向量、平面束与线面结合思想 )

例题 4 求空间曲线的投影曲线 求球面 \(x^2 + y^2 + z^2 = 1\) 与平面 \(x + y + z = 0\) 的交线在 \(xOy\) 平面上的投影曲线方程，并指出该投影曲线的形状.

分析 投影曲线的求法是联立方程消去不需要的变量 ( 向 \(xOy\) 面投影即消去 \(z\) )，得到投影柱面，再结合 \(z=0\) 即可. 投影形状可通过化简投影柱面的方程来判断.

解 1. 求投影柱面 联立方程组 \[ \begin{cases} x^2 + y^2 + z^2 = 1, \\[.5em] x + y + z = 0. \end{cases} \] 由第二个方程得 \(z = -(x + y)\)，代入第一个方程得 \(x^2 + y^2 + (x+y)^2 = 1 \implies 2x^2 + 2xy + 2y^2 = 1\). 此即为投影柱面的方程.

2. 得投影曲线 将投影柱面与 \(xOy\) 面 ( \(z=0\) ) 联立，投影曲线方程为 \[ \begin{cases} 2x^2 + 2xy + 2y^2 = 1, \\[.5em] z = 0. \end{cases} \]

3. 判定形状 对于 \(xOy\) 面内的曲线 \(2x^2 + 2xy + 2y^2 = 1\)，由于判别式 \(B^2 - 4AC = 2^2 - 4 \times 2 \times 2 = -12 < 0\)，且 \(A = 2 > 0\)，因此该投影曲线是一个 椭圆. ( 核心知识点 空间曲线一般式求投影、二次曲线不变量判定 )

例题 5 求直线旋转面方程 求直线 \(L: \begin{cases} x = z \\[.5em] y = 1 \end{cases}\) 绕 \(z\) 轴旋转一周所生成的旋转面方程，并指出曲面名称.

分析 空间曲线或直线绕坐标轴旋转，设直线上任意一点为 \((x_0, y_0, z_0)\)，旋转后到达点 \((x, y, z)\). 绕 \(z\) 轴旋转时，高度 \(z\) 不变，且到 \(z\) 轴的距离平方不变. 利用此不变量关系即可求出旋转面.

解

1. 设点找关系 设 \((x_0, y_0, z_0)\) 为直线 \(L\) 上任意一点，则有 \(x_0 = z_0\) 且 \(y_0 = 1\). 该点绕 \(z\) 轴旋转至点 \((x, y, z)\)，根据旋转的几何性质 高度不变 \(z = z_0\) 到 \(z\) 轴距离不变 \(x^2 + y^2 = x_0^2 + y_0^2\)

2. 代入消参 将 \(x_0 = z_0 = z\) 与 \(y_0 = 1\) 代入距离等式，得 \(x^2 + y^2 = z^2 + 1\) 整理得旋转面方程为 \(\frac{x^2}{1} + \frac{y^2}{1} - \frac{z^2}{1} = 1\)

3. 判定曲面 对照二次曲面标准方程，这是一个 单叶双曲面. ( 核心知识点 参数法求旋转面方程、单叶双曲面的直纹面特性 )
