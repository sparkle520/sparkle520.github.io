第7章 高级汇编语言技术

章节概述

高级汇编技术是汇编器 ( 如MASM ) 提供的 预编译特性，通过宏、重复汇编、条件汇编实现 代码复用、重复代码自动生成、条件化编译，大幅提升汇编程序的开发效率与可维护性. 本章核心是掌握汇编器的"编译期逻辑控制"能力，区别于运行期的子程序/循环逻辑.

7.1 宏汇编

宏是 汇编期展开的代码模板，将重复的短代码封装为"宏指令"，调用时汇编器会将宏体直接替换到调用位置 ( 区别于子程序 : 子程序 是 运行期调用，宏 是 编译期展开 ).

7.1.1 宏定义、宏调用和宏展开

1. 核心概念

  • 宏定义 : 用 ( 宏开始 ) 和 ( 宏结束 ) 封装代码模板；
  • 宏调用 : 直接使用宏名 ( 类似指令 );
  • 宏展开 : 汇编器将宏体代码替换到调用位置，生成最终的机器指令.

4. 宏 vs 子程序 : 关键区别

特性 宏 子程序
执行时机 汇编期展开 ( 代码直接嵌入 ) 运行期调用 ( 栈跳转 )
代码体积 调用次数越多，代码越膨胀 代码仅一份，体积紧凑
运行效率 无调用/返回开销，速度快 有栈操作开销，速度略慢
适用场景 短代码、高频调用的简单逻辑 长代码、多参数的复杂逻辑
