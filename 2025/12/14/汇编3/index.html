第3章 80x86的指令系统和寻址方式笔记

3.1 80x86的寻址方式

寻址方式是指 \(CPU\) 获取指令/操作数的内存地址的方法，分为”与数据有关”和”与转移地址有关”两类.

3.1.1 与数据有关的寻址方式

这类寻址方式用于定位 操作数的内存地址, 是汇编指令的核心基础 ( 以 \(8086\) \(16\) 位指令为例 ) :

寻址方式 有效地址 (EA) 计算规则 默认段 指令示例 & 注释
立即寻址 操作数包含在指令中 ( 立即数 ) - MOV AX, 1234H
; 源操作数是立即数
寄存器寻址 操作数在通用寄存器中 ( AX, BX… ) - MOV AX, BX
; 源操作数在寄存器
直接寻址 EA = 指令给出的 16 位偏移地址 DS MOV AX, [2000H]
; [DS:2000H] -> AX
寄存器间接 EA = BX, SI, DI
EA = BP		 DS
SS		 MOV AX, [BX]
MOV AX, [BP] ( 默认SS )
寄存器相对 EA = (BX, SI, DI) + 位移量

SS		 MOV AX, [SI+10H]
MOV AX, [BP-4]

MOV AX, [BP+DI]
相对基址变址 EA = 基址 + 变址 + 位移量 DS / SS MOV AX, [BX+SI+10H]
; [DS:BX+SI+10H] -> AX

3.1.2 与转移地址有关的寻址方式

这类寻址方式用于定位 程序跳转的目标地址，仅用于控制转移指令. 8086 支持”段内相对”“段内间接”“段间绝对/间接”等形式 :

寻址方式 规则说明 典型编码/范围 适用指令示例
段内短跳转 (短相对) 目标 = \(IP\) + \(disp8\) ( 有符号 \(8\) 位位移 ) 范围 \(-128 \ldots +127\) \(JMP\) \(SHORT\)、\(Jcc\)、\(LOOP\)
段内近跳转 (近相对) 目标 = \(IP\) + \(disp16\) ( 有符号 \(16\) 位位移 ) 范围 \(-32768 \ldots +32767\) \(JMP\) \(NEAR\) \(PTR\)、\(CALL\) \(NEAR\)
段内间接跳转 (近间接) \(IP\) 从寄存器/内存取 \(r/m16\)，段寄存器不变 读取 \(WORD\) \(PTR\) 目标 \(JMP\) \(BX\)、\(JMP\) \(WORD\) \(PTR\) [ \(SI\) ]
段间直接跳转 (远绝对) 立即数给出 \(seg:off\)，同时更新 \(CS:IP\) 编码携带 \(m16:16\) 立即数 \(JMP\) \(FAR\) \(PTR\) \(2000\mathrm{H}:0100\mathrm{H}\)、\(CALL\) \(FAR\) \(1234\mathrm{H}:0000\mathrm{H}\)
段间间接跳转 (远间接) 从内存读取 \(m16:16\) ( 低 2 字节偏移，高 2 字节段 ) 读取 \(DWORD\) \(PTR\) 目标 \(JMP\) \(DWORD\) \(PTR\) [ \(BX\) ]、\(CALL\) \(FAR\) \(DWORD\) \(PTR\) [ \(DI\) ]
  • 选择与范围
    • \(Jcc\) 条件跳转和 \(LOOP\) 仅支持短跳转 ( \(disp8\) )，超出范围需改为逻辑重构或使用“跳转到近跳转的中转”.
    • \(JMP\) 支持短/近/远；汇编器可根据标签距离选择短或近，也可用 \(NEAR\ PTR\)/\(FAR\ PTR\) 强制.
    • 近跳转/近调用使用 \(disp16\)，编码更长但范围更大；短跳更省空间与周期.
  • \(CALL\)/\(RET\) 协议
    • 近调用 : 仅压栈返回偏移 \(IP\)，使用 \(RET\) 返回.
    • 远调用 : 压栈 \(CS:IP\)，使用 \(RETF\) 返回；常用于跨段子程序.
  • 指针大小与语法
    • 段内间接目标用 \(WORD \ PTR\) ( 16 位偏移 )；段间间接目标用 \(DWORD \ PTR\) ( 16 位偏移 + 16 位段 ).
    • \(FAR \ PTR\)/\(NEAR PTR\) 用于在源代码中明确指针种类，避免汇编器歧义.

  • 段内跳转不改变 \(CS\)；段间跳转会改变 \(CS\).
  • 目标距离超范围时，汇编器报错或自动扩展为近跳 ( 取决于汇编器 )；跨段则必须使用远跳或远调用.
  • 构建“跳转表”时，段内用 \(WORD\ PTR\) 表，跨段用 \(DWORD\ PTR\) 表.

3.2 程序占有的空间和执行时间

这部分是程序性能分析的基础指标 :

  • 程序占有空间 : 程序的机器指令在内存中占用的字节数 ( 汇编指令的长度由寻址方式决定, 如立即寻址指令比直接寻址指令短 ).
  • 程序执行时间 : 程序中每条指令的 ” 时钟周期数 ” 之和 ( 不同指令的时钟周期不同, 如 \(MOV\) 占 \(2\) 个时钟周期, \(MUL\) 占 \(15\) 个时钟周期 ).

3.3 80x86的指令系统

80x86指令系统按功能分为6类，覆盖数据处理、流程控制…核心操作 :

3.3.1 数据传送指令

核心功能是在寄存器、内存、\(I/O\) 端口之间传递数据，是最常用的指令类型 :

指令 语法示例 核心功能 标志位影响 备注
\(MOV\) \(MOV\) 目的, 源 传送数据 无影响 不支持内存↔︎内存；段寄存器不可接收立即数
\(XCHG\) \(XCHG\) 操作数 1, 操作数 2 交换两个操作数的值 无影响 支持寄存器↔︎寄存器、寄存器↔︎内存；与 \(AX\) 交换有短编码
\(LEA\) \(LEA\) 寄存器, 内存地址 取地址偏移量到寄存器 无影响 不访问内存，仅计算偏移
\(LDS\) \(LDS\) 寄存器, \(m16:16\) 载入远指针 : 偏移到寄存器，段到 \(DS\) 无影响 从内存连续 4 字节取偏移与段
\(LES\) \(LES\) 寄存器, \(m16:16\) 载入远指针 : 偏移到寄存器，段到 \(ES\) 无影响 从内存连续 4 字节取偏移与段
\(PUSH\) \(PUSH\) 寄存器/内存/段寄存器 压栈 无影响 \(SP = SP - 2\)；8086 不支持立即数压栈
\(POP\) \(POP\) 寄存器/内存/段寄存器 弹栈 无影响 \(SP = SP + 2\)；不可弹到 \(CS\)
\(IN\) \(IN\) \(AL/AX\), 端口号/\(DX\) 从 \(I/O\) 端口读到累加器 无影响 端口号可为 8 位立即数或在 \(DX\) 中
\(OUT\) \(OUT\) 端口号/\(DX\), \(AL/AX\) 将累加器写到 \(I/O\) 端口 无影响 端口号可为 8 位立即数或在 \(DX\) 中
\(XLAT\) \(XLAT\) \(AL \leftarrow [DS:BX + AL]\) 无影响 使用 \(DS:BX\) 作为表基地址进行查表
\(LAHF\) \(LAHF\) 将标志低字节送入 \(AH\) 无影响 读取 \(SF, ZF, AF, PF, CF\) 到 \(AH\)
\(SAHF\) \(SAHF\) 用 \(AH\) 设置标志低字节 影响 \(SF, ZF, AF, PF, CF\) 其他标志位不变

注

\(MOV\) 与 \(LEA\) 的区别与示例

对比项 \(MOV\) \(LEA\)
核心作用 传送数据值 计算有效地址偏移
内存访问 源/目的为内存时会访存 不访问内存，仅计算偏移
目的限制 可为寄存器或内存 必须是通用寄存器
源形式 寄存器/内存/立即数 内存地址表达式
段影响 按默认/覆盖段读写内存 与段无关，仅算偏移
典型用法 取值/存值 构造指针/计算数组元素地址

3.3.2 算术指令

用于执行 算术运算, 会影响 \(FLAGS\) 标志寄存器 ( \(CF\)/\(ZF\)/\(SF\)/\(OF\)… ):

指令 语法示例 核心功能 标志位影响 备注
\(ADD\) \(ADD\) 目的, 源 加法 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) 常用于无进位加法
\(ADC\) \(ADC\) 目的, 源 带进位加法 (含 \(CF\)) 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) 多字节/多字加法
\(SUB\) \(SUB\) 目的, 源 减法 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) \(CF\) 表示借位
\(SBB\) \(SBB\) 目的, 源 带借位减法 (含 \(CF\)) 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) 多字节/多字减法
\(INC\) \(INC\) 操作数 自增 1 影响 \(ZF\)/\(SF\)/\(OF\) 不影响 \(CF\)
\(DEC\) \(DEC\) 操作数 自减 1 影响 \(ZF\)/\(SF\)/\(OF\) 不影响 \(CF\)
\(NEG\) \(NEG\) 操作数 求二进制补码 (取负) 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) 操作数 \(≠0\) 时 \(CF=1\)
\(CMP\) \(CMP\) 目的, 源 比较 (等价 \(SUB\) 不写回) 影响 \(CF\)/\(ZF\)/\(SF\)/\(OF\) 常配合 \(Jcc\)
\(MUL\) \(MUL\) 源 无符号乘法 影响 \(CF\)/\(OF\) 其他标志未定义
\(IMUL\) \(IMUL\) 源 有符号乘法 影响 \(CF\)/\(OF\) 其他标志未定义
\(DIV\) \(DIV\) 源 无符号除法 未定义 商余见说明
\(IDIV\) \(IDIV\) 源 有符号除法 未定义 商余见说明
\(CBW\) \(CBW\) \(AL \to AH\) 符号扩展 无影响 8 位扩 16 位
\(CWD\) \(CWD\) \(AX \to DX\) 符号扩展 无影响 16 位扩 32 位
\(DAA\) \(DAA\) 加法后 \(BCD\) 调整 影响 \(CF\)/\(AF\) 更新 \(ZF\)/\(SF\)/\(PF\)
\(DAS\) \(DAS\) 减法后 \(BCD\) 调整 影响 \(CF\)/\(AF\) 更新 \(ZF\)/\(SF\)/\(PF\)
\(AAA\) \(AAA\) 加法后 \(ASCII-BCD\) 调整 影响 \(CF\)/\(AF\) 调整 \(AL\)/\(AH\)
\(AAS\) \(AAS\) 减法后 \(ASCII-BCD\) 调整 影响 \(CF\)/\(AF\) 调整 \(AL\)/\(AH\)
\(AAD\) \(AAD\) 除法前 \(ASCII-BCD\) 预调 影响 \(ZF\)/\(SF\)/\(PF\) \(AH/AL\) 合并
\(AAM\) \(AAM\) 乘法后 \(ASCII-BCD\) 调整 影响 \(ZF\)/\(SF\)/\(PF\) \(AL\) 分解到 \(AH:AL\)

注

  • \(DAA\) ( 十进制加法调整 )
    • 目的 : 将 \(AL\) 的加法结果调整为压缩 \(BCD\) (packed BCD).
    • 规则 : 若 \((AL \land 0\mathrm{F}\mathrm{H}) > 9\) 或 \(AF=1\)，则 \(AL \leftarrow AL + 06\mathrm{H}\) 并置 \(AF=1\)；若 \(AL > 99\mathrm{H}\) 或 \(CF=1\)，则 \(AL \leftarrow AL + 60\mathrm{H}\) 并置 \(CF=1\).
    • 标志 : 更新 \(ZF\)/\(SF\)/\(PF\)；\(OF\) 未定义.
  • \(DAS\) ( 十进制减法调整 )
    • 目的 : 将 \(AL\) 的减法结果调整为压缩 \(BCD\).
    • 规则 : 若 \((AL \land 0\mathrm{F}\mathrm{H}) > 9\) 或 \(AF=1\)，则 \(AL \leftarrow AL - 06\mathrm{H}\) 并置 \(AF=1\)；若 \(AL > 99\mathrm{H}\) 或 \(CF=1\)，则 \(AL \leftarrow AL - 60\mathrm{H}\) 并置 \(CF=1\).
    • 标志 : 更新 \(ZF\)/\(SF\)/\(PF\)；\(OF\) 未定义.
  • \(AAA\) ( 加法后 \(ASCII-BCD\) 调整，非压缩 \(BCD\) )
    • 目的 : 把 \(AL\) 调整为 \(0–9\) 的数字，进位到 \(AH\)，用于无压缩 \(BCD\).
    • 规则 : 若 \((AL \land 0\mathrm{F}\mathrm{H}) > 9\) 或 \(AF=1\)，则 \(AL \leftarrow AL + 06\mathrm{H}\)，\(AH \leftarrow AH + 1\)，置 \(AF=1, CF=1\)；否则清 \(AF, CF\)；随后 \(AL \leftarrow AL \land 0\mathrm{F}\mathrm{H}\).
    • 标志 : 更新 \(ZF\)/\(SF\)/\(PF\).
  • \(AAS\) ( 减法后 \(ASCII-BCD\) 调整，非压缩 \(BCD\) )
    • 目的 : 把 \(AL\) 调整为 \(0–9\) 的数字，借位从 \(AH\)，用于无压缩 \(BCD\).
    • 规则 : 若 \((AL \land 0\mathrm{F}\mathrm{H}) > 9\) 或 \(AF=1\)，则 \(AL \leftarrow AL - 06\mathrm{H}\)，\(AH \leftarrow AH - 1\)，置 \(AF=1, CF=1\)；否则清 \(AF, CF\)；随后 \(AL \leftarrow AL \land 0\mathrm{F}\mathrm{H}\).
    • 标志 : 更新 \(ZF\)/\(SF\)/\(PF\).
  • \(AAD\) ( 除法前 \(ASCII-BCD\) 预调整 )
    • 目的 : 将无压缩 \(BCD\) 的两位数 \(AH:AL\) 合并为一个二进制数以便后续除法.
    • 规则 : \(AL \leftarrow AH \times 10 + AL\)，\(AH \leftarrow 0\)；更新 \(ZF\)/\(SF\)/\(PF\).
    • 基数 : 8086 默认基数为 \(10\).
  • \(AAM\) ( 乘法后 \(ASCII-BCD\) 调整 )
    • 目的 : 将乘法结果 \(AL\) 拆分为两个无压缩 \(BCD\) 数字到 \(AH:AL\).
    • 规则 : \(AH \leftarrow AL \div 10\)，\(AL \leftarrow AL \bmod 10\)；更新 \(ZF\)/\(SF\)/\(PF\).
    • 基数 : 8086 默认基数为 \(10\).

3.3.3 逻辑指令

用于执行 位运算，常用于数据的位操作、屏蔽、测试 :

指令 语法示例 核心功能 标志位影响 备注
\(AND\) \(AND\) 目的, 源 按位与 \(CF=0\)，更新 \(ZF\)/\(SF\)；\(OF=0\) 屏蔽某些位
\(OR\) \(OR\) 目的, 源 按位或 \(CF=0\)，更新 \(ZF\)/\(SF\)；\(OF=0\) 置 1 某些位
\(XOR\) \(XOR\) 目的, 源 按位异或 \(CF=0\)，更新 \(ZF\)/\(SF\)；\(OF=0\) 翻转某些位
\(NOT\) \(NOT\) 目的 按位取反 无影响 单操作数
\(TEST\) \(TEST\) 操作数 1, 操作数 2 与但不写回 \(CF=0\)，更新 \(ZF\)/\(SF\)；\(OF=0\) 测试某些位是否为 1

ʚ 示例 ɞ : 逻辑指令用例细节

  • 移位/旋转指令一览
指令 语法示例 核心效果 标志位影响 备注
\(SHL/SAL\) \(SHL/SAL\) 目的, 位数 左移，低 位补 \(0\)，溢出位入 \(CF\) 影响 \(CF\)/\(ZF\)/\(SF\)；\(OF\) 对位数 \(=1\) 有定义 左移 \(1\) 位相当于 \(\times 2\)
\(SHR\) \(SHR\) 目的, 位数 逻辑右移，高 位补 \(0\)，溢出位入 \(CF\) 影响 \(CF\)/\(ZF\)/\(SF\)；\(OF\) 对位数 \(=1\) 有定义 右移 \(1\) 位相当于 \(\div 2\)
\(SAR\) \(SAR\) 目的, 位数 算术右移，高位补 符号位，溢出位入 \(CF\) 影响 \(CF\)/\(ZF\)/\(SF\)；\(OF=0\) 适用于有符号数的 \(\div 2\)
\(ROL\) \(ROL\) 目的, 位数 循环左移，最高位 旋入 最低位 影响 \(CF\)；\(OF\) 仅当位数 \(=1\) 有定义 不更新 \(ZF\)/\(SF\)/\(PF\)
\(ROR\) \(ROR\) 目的, 位数 循环右移，最低位 旋入 最高位 影响 \(CF\)；\(OF\) 仅当位数 \(=1\) 有定义 不更新 \(ZF\)/\(SF\)/\(PF\)
\(RCL\) \(RCL\) 目的, 位数 通过 \(CF\) 循环左移 ( 含 进位位 ) 影响 \(CF\)；\(OF\) 仅当位数 \(=1\) 有定义 不更新 \(ZF\)/\(SF\)/\(PF\)
\(RCR\) \(RCR\) 目的, 位数 通过 \(CF\) 循环右移 ( 含 进位位 ) 影响 \(CF\)；\(OF\) 仅当位数 \(=1\) 有定义 不更新 \(ZF\)/\(SF\)/\(PF\)

ʚ 示例 ɞ : 移位/旋转用例细节

3.3.4 串处理指令

用于 连续内存块 ( 串 ) 的批量操作, 默认用 \(SI\) ( 源串, 搭配 \(DS\) ) 和 \(DI\) ( 目的串, 搭配 \(ES\) ) 作为偏移地址 :

指令 语法示例 源/目的地址 核心效果 标志位影响 指针更新
\(MOVSB\) \(MOVSB\) \(DS:SI \to ES:DI\) 传送 1 字节 无影响 \(SI \pm 1\), \(DI \pm 1\)
\(MOVSW\) \(MOVSW\) \(DS:SI \to ES:DI\) 传送 2 字节 无影响 \(SI \pm 2\), \(DI \pm 2\)
\(CMPSB\) \(CMPSB\) \(DS:SI\) vs \(ES:DI\) 比较 1 字节 更新 \(ZF\)/\(SF\)/\(CF\)/\(OF\) \(SI \pm 1\), \(DI \pm 1\)
\(CMPSW\) \(CMPSW\) \(DS:SI\) vs \(ES:DI\) 比较 2 字节 更新 \(ZF\)/\(SF\)/\(CF\)/\(OF\) \(SI \pm 2\), \(DI \pm 2\)
\(SCASB\) \(SCASB\) \(AL\) vs \(ES:DI\) 搜索 1 字节 更新 \(ZF\)/\(SF\)/\(CF\)/\(OF\) \(DI \pm 1\)
\(SCASW\) \(SCASW\) \(AX\) vs \(ES:DI\) 搜索 2 字节 更新 \(ZF\)/\(SF\)/\(CF\)/\(OF\) \(DI \pm 2\)
\(LODSB\) \(LODSB\) \(DS:SI \to AL\) 加载 1 字节到累加器 无影响 \(SI \pm 1\)
\(LODSW\) \(LODSW\) \(DS:SI \to AX\) 加载 2 字节到累加器 无影响 \(SI \pm 2\)
\(STOSB\) \(STOSB\) \(AL \to ES:DI\) 存储 1 字节 无影响 \(DI \pm 1\)
\(STOSW\) \(STOSW\) \(AX \to ES:DI\) 存储 2 字节 无影响 \(DI \pm 2\)
\(REP\) \(REP\) 前缀 + 串指令 依 \(CX\) 次数重复 串指令重复 无影响 每次按指令更新
\(REPE\)/\(REPZ\) \(REPE\) 前缀 + \(CMPS/SCAS\) \(ZF=1\) 且 \(CX\ne0\) 继续 比较/搜索重复 无影响 每次按指令更新
\(REPNE\)/\(REPNZ\) \(REPNE\) 前缀 + \(CMPS/SCAS\) \(ZF=0\) 且 \(CX\ne0\) 继续 比较/搜索重复 无影响 每次按指令更新

ʚ 示例 ɞ : 串处理指令典型用法 ( \(MOVS\)/\(STOS\)/\(LODS\)/\(CMPS\)/\(SCAS\) + \(REP\) 家族 )

注

  • 方向标志 \(DF\) : \(DF=0\) 时 \(SI/DI\) 递增；\(DF=1\) 时 \(SI/DI\) 递减. 可用 \(CLD\) 清零、\(STD\) 置位.
  • 段寄存器 : 源默认 \(DS:SI\)，目的默认 \(ES:DI\). 可通过覆盖前缀改变段.
  • 重复前缀 : \(REP\) 适用于 \(MOVS\)/\(LODS\)/\(STOS\)；\(REPE\)/\(REPNE\) 主要配合 \(CMPS\)/\(SCAS\)，满足条件且 \(CX\ne0\) 才继续.

3.3.5 控制转移指令

用于 改变程序执行流程 ( 跳转, 循环, 调用/返回 ) :

指令 语法示例 类型/范围 标志位条件/影响 备注
\(JMP\) \(SHORT\) \(JMP\) \(SHORT\) \(label\) 段内短相对 ( \(disp8\) ) 无影响 \(CS\) 不变
\(JMP\) \(NEAR\) \(JMP\) \(NEAR\) \(PTR\) \(label\) 段内近相对 ( \(disp16\) ) 无影响 \(CS\) 不变
\(JMP\) \(FAR\) \(JMP\) \(FAR\) \(PTR\) \(2000\mathrm{H}:0100\mathrm{H}\) 段间远绝对 ( \(m16:16\) ) 无影响 更新 \(CS:IP\)
\(JMP\) 间接近 \(JMP\) \(WORD\) \(PTR\) \([ SI ]\) 段内近间接 ( \(r/m16\) ) 无影响 \(CS\) 不变
\(JMP\) 间接远 \(JMP\) \(DWORD\) \(PTR\) \([ BX ]\) 段间远间接 ( \(m16:16\) ) 无影响 更新 \(CS:IP\)
\(Jcc\) \(JZ\)/\(JNZ\)/\(JC\)/\(JNC\) … \(label\) 段内短相对 ( \(disp8\) ) 依条件标志 仅短跳
\(LOOP\) \(LOOP\) \(label\) 段内短相对 ( \(disp8\) ) \(CX-1 \ne 0\) 则跳转 仅短跳
\(LOOPE\)/\(REPZ\) \(LOOPE\) \(label\) 段内短相对 ( \(disp8\) ) \(CX-1 \ne 0\) 且 \(ZF=1\) 仅短跳
\(LOOPNE\)/\(REPNZ\) \(LOOPNE\) \(label\) 段内短相对 ( \(disp8\) ) \(CX-1 \ne 0\) 且 \(ZF=0\) 仅短跳
\(CALL\) 近 \(CALL\) \(NEAR\) \(label\) 段内近调用 无影响 压栈返回偏移 \(IP\)
\(CALL\) 远 \(CALL\) \(FAR\) \(1234\mathrm{H}:0000\mathrm{H}\) 段间远调用 无影响 压栈 \(CS:IP\)
\(RET\) \(RET\) 段内返回 无影响 弹出 \(IP\)；可带立即数
\(RETF\) \(RETF\) 段间返回 无影响 弹出 \(IP\) 和 \(CS\)
\(INT\) \(INT\) \(n\) 软件中断 无影响 压栈 \(FLAGS\)/\(CS\)/\(IP\)；转向向量
\(IRET\) \(IRET\) 中断返回 恢复 \(FLAGS\) 弹出 \(IP\)/\(CS\)/\(FLAGS\)

注

  • 范围 : \(Jcc\) 与 \(LOOP\) 仅支持 \(disp8\)；\(JMP\) 支持短/近/远；跨段需使用远跳或远调用.
  • 近/远 : 近跳/近调不改变 \(CS\)；远跳/远调会更新 \(CS:IP\).
  • 指针大小 : 段内间接用 \(WORD\ PTR\)；段间间接用 \(DWORD\ PTR\)；\(NEAR\ PTR\)/\(FAR\ PTR\) 可用于强制类型.
  • \(m16\) : \(16\) : 表示内存中的远指针 ( 连续 \(4\) 字节 ) ：低 \(2\) 字节为偏移，高 \(2\) 字节为段；8086 为小端存储 ( 低地址存低字节 ). 例如 : \(JMP\) \(DWORD\) \(PTR\) [ \(BX\) ] 从 [ \(BX\) ] 取偏移、从 [ \(BX+2\) ] 取段，更新 \(IP\) 与 \(CS\)；\(JMP\) \(FAR\) \(PTR\) \(2000\mathrm{H}:0100\mathrm{H}\) 为立即给出 段 : 偏移常量.
  • \(r\) / \(m16\) : 表示 寄存器/内存 \(16\) 位操作数 ( \(ModR/M\) ) ：可为寄存器 \(r16\) ( 如 \(AX\)/\(BX\)/\(CX\)/\(DX\)… ) 或内存 \(m16\) ( 如 \(WORD\ PTR\) [ \(SI\) ], \(WORD\ PTR\) [ \(BX+DI+disp\) ] ). 段默认 : \(BX\)/\(SI\)/\(DI\) 用 \(DS\), \(BP\) 用 \(SS\)；可用段覆盖前缀更改.

ʚ 示例 ɞ : 控制转移指令典型用法 ( \(JMP\)/\(Jcc\)/\(LOOP\)/\(CALL\)/\(RET\)/\(INT\) )

3.3.6 处理机控制与杂项操作指令

用于 控制 \(CPU\) 状态与标志位 :

指令 语法示例 核心功能 标志位影响 备注
\(CLC\) \(CLC\) 清进位标志 \(CF \leftarrow 0\) 其他标志不变
\(STC\) \(STC\) 置进位标志 \(CF \leftarrow 1\) 其他标志不变
\(CMC\) \(CMC\) 取反进位标志 \(CF \leftarrow \neg CF\) 其他标志不变
\(CLD\) \(CLD\) 清方向标志 \(DF \leftarrow 0\) 串指令 \(SI/DI\) 自增
\(STD\) \(STD\) 置方向标志 \(DF \leftarrow 1\) 串指令 \(SI/DI\) 自减
\(CLI\) \(CLI\) 清中断标志 \(IF \leftarrow 0\) 禁用可屏蔽中断
\(STI\) \(STI\) 置中断标志 \(IF \leftarrow 1\) 允许可屏蔽中断
\(PUSHF\) \(PUSHF\) 压栈标志寄存器 无直接影响 栈顶入 \(FLAGS\)
\(POPF\) \(POPF\) 弹栈到标志寄存器 恢复各标志 从栈取回 \(FLAGS\)
\(HLT\) \(HLT\) 处理器暂停 无影响 等待中断/复位唤醒
\(NOP\) \(NOP\) 空操作 无影响 对齐/占位，1 字节
\(WAIT\) \(WAIT\) 等待外部测试引脚 无影响 与协处理器/外设同步
\(ESC\) \(ESC\) \(opcode\), \(r/m\) 逃逸到协处理器 无影响 将操作码/数据交给协处理器

注

  • \(DF\) 影响串指令指针增减；可用 \(CLD\)/\(STD\) 显式控制.
  • \(IF\) 控制可屏蔽中断响应；\(CLI\) 禁止、\(STI\) 允许.
  • \(HLT\) 进入低功耗等待；遇中断或复位恢复执行.
  • \(PUSHF\)/\(POPF\) 常与中断管理、特权切换配合使用.
