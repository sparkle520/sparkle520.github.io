计算机基础知识汇编笔记

第1章 基础知识

1.1 进位记数制与不同基数的数之间的转换

1.1.1 二进制数

  • 定义 ：以 \(2\) 为基数的记数制，仅用 \(0\) 和 \(1\) 表示，运算规则为”逢 \(2\) 进 \(1\)“.
  • 位权 ：第 \(n\) 位的位权为 \(2^{n-1}\) ( 从右往左，从 \(0\) 开始计数 ).
  • 示例 ：二进制数 \(1011\) 的位权展开 ： \[ 1\times 2^3 + 0\times 2^2 + 1\times 2^1 + 1\times 2^0 = 11 \text{ ( 十进制 ) } \]

1.1.2 二进制数和十进制数之间的转换

  • 二进制转十进制 ：按位权展开求和 ( 如上示例 ).
  • 十进制转二进制 ：
    1. 整数部分 ：“除 \(2\) 取余，逆序排列”
      • 示例 ：十进制 13 转二进制 ： \(13\div 2=6 \dots 1 \to 6\div 2=3 \dots 0 \to 3\div 2=1 \dots 1 \to 1\div 2=0 \dots 1\) 结果 ：1101
    2. 小数部分 ：“乘 \(2\) 取整，顺序排列”
      • 示例 ：十进制 0.625 转二进制 ： \(0.625\times 2=1.25 \text{ 取1} \to 0.25\times 2=0.5 \text{ 取0} \to 0.5\times 2=1.0 \text{ 取1}\) 结果 ：0.101

1.1.3 十六进制数及其与二进制、十进制数之间的转换

  • 十六进制数 ：以 \(16\) 为基数，用 \(0\)-\(9\)、\(A\)-\(F\) ( \(A\) = \(10\), \(F\) = \(15\) ) 表示，“逢 \(16\) 进 $1”.
  • 十六进制转二进制 ：每 \(1\) 位 十六进制 对应 \(4\) 位 二进制 ( 不足补 \(0\) ).
    • 示例 ：\(3A\) ( 十六进制 ) \(\to\) \(0011\ 1010\) ( 二进制 )
  • 二进制转十六进制 ：从右往左每 \(4\) 位分组，不足补 \(0\) ，每组对应 \(1\) 位十六进制.
    • 示例 ：\(1101011\) ( 二进制 ) \(\to\) 分组 \(0110\ 1011\) \(\to\) \(6B\) ( 十六进制 )
  • 十六进制转十进制 ：按位权展开 ( 位权为 \(16^{n-1}\) ).
    • 示例 ：\(2F\) ( 十六进制 ) \(\to 2\times 16^1 + 15\times 16^0 = 47\) ( 十进制 )

1.2 二进制数和十六进制数运算

1.2.1 二进制数运算

  • 加法 ：\(0+0=0；0+1=1；1+1=10\) ( 进 \(1\) )
    • 示例 ：\(1011 + 0101 = 10000\)
  • 减法 ：\(0-0=0；1-0=1；1-1=0；0-1=1\) ( 借 \(1\) 当 \(2\) )
    • 示例 ：\(1000 - 0101 = 0011\)
  • 乘法 ：类似十进制，“位乘+移位相加”
    • 示例 ：\(101 \times 11 = 101\times 1 + 101\times 10 = 101 + 1010 = 1111\)
  • 除法 ：类似十进制，“试商+减法”
    • 示例 ：\(1111 \div 101 = 11 \text{ ( 商 ) } \dots 0 \text{ ( 余 ) }\)

1.2.2 十六进制数运算

  • 规则 ：“逢16进1，借1当16”，运算时可先转二进制计算，再转回十六进制.
  • 示例 ：\(3A + 15\) ( 十六进制 ) \(\to\) 转二进制 \(00111010 + 00010101 = 01001111\) \(\to\) 转十六进制 \(4F\)

1.3 计算机中数和字符的表示

1.3.1 数的补码表示

  • 背景 ：计算机用补码统一表示正负数，简化减法为加法.
  • 正数的补码 ：与原码 ( 符号位 \(0\) + 数值位 ) 相同.
    • 示例 ：十进制 5 ( 8位 ) \(\to\) 原码 00000101 \(\to\) 补码 00000101
  • 负数的补码 ：原码 ( 符号位 \(1\) + 数值位 ) 取反 ( 符号位不变 ) 后加 \(1\).
    • 示例 ：十进制 -5 ( \(8\) 位 ) \(\to\) 原码 10000101 \(\to\) 反码 11111010 \(\to\) 补码 11111011

1.3.2 补码的加法和减法

  • 加法 ：\([X+Y]_{\text{补}} = [X]_{\text{补}} + [Y]_{\text{补}}\) ( 结果仍为补码 )
  • 减法 ：\([X-Y]_{\text{补}} = [X]_{\text{补}} + [-Y]_{\text{补}}\) ( 将减法转为”加负数的补码” )
  • 示例 ：5-3 \(\to\)等价 5+ ( -3 ) \(\to\) \([5]_{\text{补}}=00000101 + [-3]_{\text{补}}=11111101 = 100000010\) ( 舍去进位 ) \(\to\) 结果 00000010 ( 补码 ) \(\to\) 十进制 2

1.3.3 无符号整数

  • 定义 ：没有符号位，所有位均为数值位，仅表示非负数.
  • 范围 ：\(n\) 位无符号整数的表示范围是 \(0 \sim 2^n - 1\).
    • 示例 ： \(8\) 位无符号整数 \(\to\) \(0 \sim 255\)

1.3.4 字符表示法

  • \(ASCII\) 码 ： \(7\) 位编码，可表示 \(128\) 个字符 ( 数字、字母、符号… ).
    • 示例 ： ‘\(0\)’ \(\to\) \(48\) ( 十进制 )；‘\(A\)’ \(\to\) \(65\)；‘\(a\)’ \(\to\) \(97\)
  • 汉字编码 ：如 \(GB2312\) 、 \(UTF\)-\(8\) ( \(Unicode\) 的一种实现 ) ，用于表示汉字…非 \(ASCII\) 字符.

1.4 几种基本的逻辑运算

1.4.1 “与”运算 ( \(AND\) )

  • 符号 ：& 或 \(\land\)
  • 规则 ：全 \(1\) 则 \(1\)，有 \(0\) 则 \(0\).
  • 示例 ：\(1011 \ \& \ 0101 = 0001\)

1.4.2 “或”运算 ( \(OR\) )

  • 符号 ：| 或 \(\lor\)
  • 规则 ：有 \(1\) 则 \(1\)，全 \(0\) 则 \(0\).
  • 示例 ：\(1011 \ | \ 0101 = 1111\)

1.4.3 “非”运算 ( \(NOT\) )

  • 符号 ：\(\sim\) 或 \(\neg\)
  • 规则 ：取反 (\(1\to 0\)，\(0\to 1\)).
  • 示例 ：\(\sim 1011 \text{ ( 4位 ) } = 0100\)

1.4.4 “异或”运算 ( \(XOR\) )

  • 符号 ：^ 或 \(\oplus\)
  • 规则 ：相同则 \(0\)，不同则 \(1\).
  • 示例 ：\(1011\) ^ \(0101 = 1110\)

1.5 进阶要点与常见坑

  • 进制转换进阶 :
    • 非终止小数 : 十进制 \(0.1\) 在二进制为无限循环, 实际计算需截断或舍入.
    • 舍入策略 : 常见为向下截断、四舍五入或银行家舍入.
    • 验算 : 将结果再转换回原进制检查误差范围.
  • 二进制算术与溢出 :
    • 无符号加法的进位检测 : 最终进位 \(\to\) \(CF\).
    • 有符号加法的溢出检测 : 同号相加且结果符号改变 \(\to\) \(OF\).
    • 示例 : \([0111\ 1111]_2 + [0000\ 0001]_2 = [1000\ 0000]_2\) , \(OF = 1\) , \(CF = 0\).
  • 补码范围与符号扩展 :
    • \(n\) 位补码范围 : \(-2^{n-1} \sim 2^{n-1} - 1\).
    • 符号扩展 : 将最高位复制到新增的高位.
    • 示例 : \(8\) 位 \([1111\ 1010]_2\) 扩展到 \(16\) 位 \(\to\) \([1111\ 1111\ 1111\ 1010]_2\).
  • 按位逻辑性质 :
    • 交换律 : \(A \land B = B \land A\) , \(A \lor B = B \lor A\).
    • 结合律 : ( \(A \land B\) ) \(\land\) \(C\) = \(A\) \(\land\) ( \(B \land C\) ).
    • 分配律 : \(A \land ( B \lor C ) = ( A \land B ) \lor ( A \land C )\).
    • 德摩根律 : \(\neg ( A \land B ) = ( \neg A ) \lor ( \neg B )\) , \(\neg ( A \lor B ) = ( \neg A ) \land ( \neg B )\).
  • 常用位运算技巧 :
    • 清零某一位 : \(X \land \sim ( 1 \ll k )\).
    • 置位某一位 : \(X \lor ( 1 \ll k )\).
    • 切换某一位 : \(X \oplus ( 1 \ll k )\).
    • 判断奇偶 : \(X \land 1\) 为 \(1\) 则奇数 , 为 \(0\) 则偶数.
  • 进位与借位直观理解 :
    • 加法 : 两数在某位产生和为 \(2\) 或 \(3\) 时向高位进 \(1\).
    • 减法 : 若被减位小于减数位则向高位借 \(1\) , 借来后等效增加 \(2\).
