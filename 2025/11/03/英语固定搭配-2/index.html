Part 11

address a meeting

  • 💫 释义 : 在会议上发言
    • 🫧 例句 : The CEO will address the shareholders’ meeting this afternoon.
    • 🌈 同义 : speak at a meeting, give a speech to

  • 💫 释义 : 向观众致辞
    • 🫧 例句 : Before starting his speech, he addressed the audience briefly.
    • 🌈 同义 : speak to the audience, make an address to

  • 💫 释义 : 在信件上写地址
    • 🫧 例句 : Please make sure you address the envelope correctly.
    • 🌈 同义 : write the address on, direct a letter to

  • 💫 释义 : 回应关切
    • 🫧 例句 : The company needs to address customers’ concerns about privacy.
    • 🌈 同义 : respond to concerns, allay worries, answer questions

  • 💫 释义 : 致力于; 着手处理
    • 🫧 例句 : We should address ourselves to the main task at hand.
    • 🌈 同义 : apply oneself to, devote oneself to, attend to

  • 💫 释义 : 解决问题

  • 💫 释义 : 满足需求

  • 💫 释义 : 退出, 退学, 辍学
