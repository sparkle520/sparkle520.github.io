固定搭配-1

sparkle5202025-11-032026-03-07

说明

  • 💫 释义 ( Meaning )
  • 🫧 例句 ( Example )
  • 🌈 同义 ( Synonym )

本文章较长，可通过 目录 或 ctrl + F 搜索.

Part 1

in connection with…

  • 💫 释义 : 与…相关
    • 🫧 例句 : This document is in connection with the pending merger.

bear on… / has a bearing on…

  • 💫 释义 : 与…有关; 对…有影响
    • 🫧 例句 : Your personal health has a direct bearing on your work performance.

on occasion

  • 💫 释义 : 偶尔; 有时
    • 🫧 例句 : He visits the city on occasion.

on multiple occasions

  • 💫 释义 : 多次; 数次
    • 🫧 例句 : I’ve reminded him on multiple occasions.

have the occasion to do sth

  • 💫 释义 : 有机会做某事
    • 🫧 例句 : I hope you have the occasion to visit our new headquarters.

get across

  • 💫 释义 : 使被理解
    • 🫧 例句 : It’s a difficult idea to get across.

impress upon someone

  • 💫 释义 : 使某人铭记
    • 🫧 例句 : I must impress upon you the need for confidentiality.

drive home

  • 💫 释义 : 深刻认识
    • 🫧 例句 : He used a real-life example to drive home his point.

draw…from…

  • 💫 释义 : 吸引; 引出
    • 🫧 例句 : The speaker drew applause from the audience. / The investigation drew criticism from the public.

treat…as…

  • 💫 释义 : 将…视为…
    • 🫧 例句 : The manager always treats her employees as equals, which fosters a positive work environment.
    • 🌈 同义 : regard…as…, see…as…, view…as…, deem…( to be )…, look upon…as…, think of…as…, label…as…, perceive…as…

Part 2

turn out

  • 💫 释义 : 结果是/生产/出席
    • 🫧 例句 : It turned out to be a great success.

turn up

  • 💫 释义 : 调高/出现
    • 🫧 例句 : Can you turn up the volume? / My keys finally turned up.

turn down

  • 💫 释义 : 调低/拒绝
    • 🫧 例句 : Please turn down the music. / She turned down the job offer.

turn in

  • 💫 释义 : 上交/睡觉
    • 🫧 例句 : You must turn in your homework. / I’m going to turn in for the night.

turn to

  • 💫 释义 : 向…求助/翻到
    • 🫧 例句 : He has no one to turn to. / Please turn to page 10.

turn over

  • 💫 释义 : 翻转/移交
    • 🫧 例句 : He turned over the card. / The thief was turned over to the police.

turn around

  • 💫 释义 : 转身/使好转
    • 🫧 例句 : She turned around to see me. / He turned the company around.

set up

  • 💫 释义 : 建立/安排/安装
    • 🫧 例句 : They set up a new company. / I’ll set up a meeting.

set off

  • 💫 释义 : 触发, 启动 ( 警报、装置 )
    • 🫧 例句 : The smoke set off the fire alarm.
    • 🌈 同义 : activate, trigger
  • 💫 释义 : 引发, 导致 ( 一系列事件 )
    • 🫧 例句 : His resignation set off a chain of management changes.
    • 🌈 同义 : spark, cause to begin
  • 💫 释义 : 出发, 启程
    • 🫧 例句 : They set off on their journey at dawn.
    • 🌈 同义 : begin a journey, start out
  • 💫 释义 : 衬托, 使显眼
    • 🫧 例句 : The blue frame sets off the painting perfectly.
    • 🌈 同义 : enhance, make more noticeable

set out

  • 💫 释义 : 出发/陈述/打算
    • 🫧 例句 : They set out on a journey. / He set out his plans clearly.

Part 3

set about

  • 💫 释义 : 开始着手
    • 🫧 例句 : She set about cleaning the room.

set aside

  • 💫 释义 : 留出/搁置/驳回
    • 🫧 例句 : Set aside some money each month. / Let’s set aside our differences.

set back

  • 💫 释义 : 阻碍/耽误
    • 🫧 例句 : The project was set back by bad weather.

set in

  • 💫 释义 : 开始并持续 ( 指不好的事 )
    • 🫧 例句 : Winter has set in.

set apart

  • 💫 释义 : 使与众不同/留出
    • 🫧 例句 : Her talent sets her apart.

set on / upon

  • 💫 释义 : 袭击, 攻击
    • 🫧 例句 : He was set upon by thieves.

set against

  • 💫 释义 : 使反对/以…为背景
    • 🫧 例句 : The scandal set the public against him. / The city is beautiful set against the mountains.

set forth

  • 💫 释义 : 陈述, 阐明
    • 🫧 例句 : The document sets forth the terms of the agreement.

guideline on

  • 💫 释义 : 关于…的指南
    • 🫧 例句 : Please follow the guideline on data security.

guideline for

  • 💫 释义 : 为…制定的指南
    • 🫧 例句 : Here is a guideline for new employees.

Part 4

guideline regarding

  • 💫 释义 : 有关…的指导方针
    • 🫧 例句 : The government issued a guideline regarding tax filing.

up to it

  • 💫 释义 : 有能力、有精力或状态佳以应对某种挑战或完成某项任务
    • 🫧 例句 : I know the project is tough, but I’m sure you’re up to it.
    • 🌈 同义 : capable of ( doing ) something, fit for something, in fit state, Equal to the task / challenge, have what it takes, in the right state of mind ( or body )

engage in ( sth/doing sth )

  • 💫 释义 : 参与, 从事 ( 某项活动或工作 )
    • 🫧 例句 : It’s not a topic I would engage in debate about.
    • 🌈 同义 : participate in, take part in, be involved in

engage with sb/sth

  • 💫 释义 : 与…建立联系; 吸引; 致力于理解或处理
    • 🫧 例句 : The company uses social media to engage with its customers.
    • 🌈 同义 : connect with, interact with, involve oneself with

engage sb in sth

  • 💫 释义 : 使某人参与/卷入某事
    • 🫧 例句 : The teacher tried to engage all the students in the class discussion.
    • 🌈 同义 : involve sb in, draw sb into, include sb in

engage sb’s interest / attention

  • 💫 释义 : 吸引某人的兴趣/注意力
    • 🫧 例句 : The first chapter of the book is designed to engage the reader’s interest immediately.
    • 🌈 同义 : capture/grab one’s interest/attention, hold one’s attention, fascinate

engage sb to do sth

  • 💫 释义 : 雇佣/聘请某人做某事
    • 🫧 例句 : We will engage a professional designer to create the logo.
    • 🌈 同义 : hire/employ sb to do sth, appoint sb to do sth

engage ( as sth )

  • 💫 释义 : ( 以…身份 ) 聘用
    • 🫧 例句 : She was engaged as a consultant on the project.
    • 🌈 同义 : hired as, employed as, appointed as

be engaged to sb

  • 💫 释义 : 与某人订婚
    • 🫧 例句 : I’m happy to announce that my daughter is engaged to a wonderful man.
    • 🌈 同义 : be betrothed to ( 非常正式 ), have pledged to marry

be engaged in sth

  • 💫 释义 : 正忙于, 正从事于 ( 状态 )
    • 🫧 例句 : He was engaged in a deep conversation and didn’t notice me.
    • 🌈 同义 : be busy with, be occupied with, be absorbed in

Part 5

be forced into ( sth/doing sth )

  • 💫 释义 : 被迫 ( 接受某种状态或做某事 )
    • 🫧 例句 : Many small businesses were forced into bankruptcy during the economic crisis.
    • 🌈 同义 : be compelled to do sth, be coerced into, be pressured into, be left with no choice but to do

come across sb/sth

  • 💫 释义 : 偶然遇见, 碰见
    • 🫧 例句 : I came across this old photo while cleaning the attic.
    • 🌈 同义 : run into, stumble upon, chance upon, find by chance

come across ( as sth )

  • 💫 释义 : 给人留下…印象; 显得
    • 🫧 例句 : He comes across as very confident in interviews.
    • 🌈 同义 : seem, appear, strike someone as, give the impression of being

come across ( with sth )

  • 💫 释义 : 提供 ( 所需之物, 如钱、信息 ); 交出
    • 🫧 例句 : He finally came across with the money he owed me.
    • 🌈 同义 : provide, deliver, hand over, come up with

come across ( sth )

  • 💫 释义 : ( 想法、信息等 ) 被理解, 被传达
    • 🫧 例句 : His main point didn’t come across clearly in the speech.
    • 🌈 同义 : be understood, be communicated, get across, be conveyed

above all else

  • 💫 释义 : 尤其是; 最重要的是
    • 🫧 例句 : Above all else, we must ensure the safety of our staff.
    • 🌈 同义 : most importantly, first and foremost, primarily

above all

  • 💫 释义 : 尤其是; 最重要的是
    • 🫧 例句 : He is a great leader, and above all, a good person.
    • 🌈 同义 : most of all, more than anything else, chiefly

first and foremost

  • 💫 释义 : 首先; 首要的是
    • 🫧 例句 : We are, first and foremost, a technology company.
    • 🌈 同义 : primarily, principally, in the first place

most importantly

  • 💫 释义 : 最重要的是
    • 🫧 例句 : We need to finish the project on time and, most importantly, within budget.
    • 🌈 同义 : crucially, critically, above all

favour A over B

  • 💫 释义 : 偏爱A胜过B; 更倾向于选择A而非B
    • 🫧 例句 : Many investors currently favour bonds over stocks for stability.
    • 🌈 同义 : prefer A to B, choose A over B, value A more than B

Part 6

prefer A to B

  • 💫 释义 : 比起B来更喜欢A
    • 🫧 例句 : I prefer tea to coffee in the morning.
    • 🌈 同义 : favour A over B, like A better than B, would rather A than B

would rather A than B

  • 💫 释义 : 宁愿A也不愿B
    • 🫧 例句 : I would rather stay home than go to a crowded party.
    • 🌈 同义 : prefer to A rather than B, would sooner A than B

opt for A instead of B

  • 💫 释义 : 选择A而不是B
    • 🫧 例句 : She opted for the vegetarian menu instead of the fish dish.
    • 🌈 同义 : choose A over B, go for A rather than B, select A in place of B

give precedence to A over B

  • 💫 释义 : 给予A优先权胜过B ( 正式 )
    • 🫧 例句 : The committee gave precedence to environmental concerns over economic gains.
    • 🌈 同义 : prioritize A above B, rank A higher than B

not just… but also…

  • 💫 释义 : 不仅…而且…
    • 🫧 例句 : The new policy is not just inefficient, but also costly.
    • 🌈 同义 : not only… but also…, not merely… but…

not only… but also…

  • 💫 释义 : 不仅…而且… ( 比 not just 稍正式 )
    • 🫧 例句 : She is not only a talented writer but also an accomplished painter.
    • 🌈 同义 : not just… but also…, not merely… but…, as well as

not merely… but…

  • 💫 释义 : 不仅仅…而是… ( 强调后者更重要 )
    • 🫧 例句 : This is not merely a setback, but a critical failure.
    • 🌈 同义 : not just… but…, not only… but…, not simply… but…

…as well as…

  • 💫 释义 : …和…; 既…又… ( 强调前者 )
    • 🫧 例句 : The system is efficient as well as user-friendly.
    • 🌈 同义 : and also, in addition to, along with

both… and…

  • 💫 释义 : 两者都…; 既…又… ( 强调并列 )
    • 🫧 例句 : The project requires both time and money.
    • 🌈 同义 : and, as well as

over and above

  • 💫 释义 : 除…之外 ( 还 ) ( 正式 )
    • 🫧 例句 : He received a bonus over and above his regular salary.
    • 🌈 同义 : in addition to, besides, on top of

Part 7

bring in sth

  • 💫 释义 : 赚取, 产生 ( 收入 )
    • 🫧 例句 : The new product line is expected to bring in significant revenue.
    • 🌈 同义 : earn, generate, yield, produce

bring in sb

  • 💫 释义 : 邀请…参与; 引入 ( 专家、外部力量 )
    • 🫧 例句 : We need to bring in a consultant to solve this technical issue.
    • 🌈 同义 : call in, enlist, recruit, hire

bring in sth

  • 💫 释义 : 推行, 实施 ( 新法律、规则或系统 )
    • 🫧 例句 : The government plans to bring in new regulations next year.
    • 🌈 同义 : introduce, implement, institute, enact

bring in sb

  • 💫 释义 : 逮捕并带至警局
    • 🫧 例句 : The police brought in two suspects for questioning.
    • 🌈 同义 : arrest, take into custody, apprehend

in light of

  • 💫 释义 : 考虑到, 鉴于
    • 🫧 例句 : The meeting was rescheduled in light of the manager’s sudden illness.
    • 🌈 同义 : considering, taking into account, given, in view of

in view of

  • 💫 释义 : 鉴于, 考虑到
    • 🫧 例句 : In view of recent developments, we need to change our strategy.
    • 🌈 同义 : considering, in light of, taking into account, given

taking into account

  • 💫 释义 : 考虑到, 把…纳入考虑
    • 🫧 例句 : Taking into account all the factors, this is our best option.
    • 🌈 同义 : considering, bearing in mind, in view of, allowing for

given that

  • 💫 释义 : 考虑到, 鉴于 ( 后接从句 )
    • 🫧 例句 : Given that the market is volatile, we should be cautious.
    • 🌈 同义 : considering that, in view of the fact that, seeing that

on account of

  • 💫 释义 : 由于, 因为
    • 🫧 例句 : The event was postponed on account of bad weather.
    • 🌈 同义 : because of, due to, owing to, by reason of

owing to

  • 💫 释义 : 由于, 因为
    • 🫧 例句 : Owing to unforeseen circumstances, the project has been delayed.
    • 🌈 同义 : due to, because of, on account of, as a result of

Part 8

in terms of

  • 💫 释义 : 在…方面; 从…角度来说
    • 🫧 例句 : The project is successful in terms of its final outcome.
    • 🌈 同义 : with regard to, regarding, from the perspective of, as regards, in the aspect of

come to terms with

  • 💫 释义 : 接受 ( 困难或不愉快的现实 ); 与…达成妥协
    • 🫧 例句 : It took him years to come to terms with his loss.
    • 🌈 同义 : accept, reconcile oneself to, learn to live with, make peace with

in the long/short term

  • 💫 释义 : 从长/短期来看
    • 🫧 例句 : This decision will benefit the company in the long term.
    • 🌈 同义 : over the long/short haul, from a long/short-term perspective

on good/bad terms ( with )

  • 💫 释义 : ( 与某人 ) 关系好/不好
    • 🫧 例句 : She is on good terms with all her colleagues.
    • 🌈 同义 : have a good/bad relationship with, be friendly with/be at odds with

a term of art

  • 💫 释义 : 专业术语; 特定领域内的专门用语
    • 🫧 例句 : “Liquidated damages” is a term of art in contract law.
    • 🌈 同义 : technical term, specialized terminology, jargon

be on speaking terms

  • 💫 释义 : 关系尚可 ( 至少能交谈 )
    • 🫧 例句 : They are not friends, but they are on speaking terms.
    • 🌈 同义 : be civil to each other, maintain cordial relations

argument for

  • 💫 释义 : 支持…的论点
    • 🫧 例句 : One strong argument for remote work is increased productivity.
    • 🌈 同义 : case for, reason for, justification for

make a case for

  • 💫 释义 : 提出理由支持
    • 🫧 例句 : The report makes a compelling case for increased funding.
    • 🌈 同义 : argue for, advocate for, present arguments for

in favor of

  • 💫 释义 : 支持, 赞成
    • 🫧 例句 : The committee voted in favor of the proposal.
    • 🌈 同义 : supportive of, for, approving of

justification for

  • 💫 释义 : …的理由/依据
    • 🫧 例句 : There’s no justification for such rude behavior.
    • 🌈 同义 : reason for, grounds for, basis for

Part 9

reason for

  • 💫 释义 : …的原因/理由
    • 🫧 例句 : What was the reason for your decision?
    • 🌈 同义 : cause of, basis for, grounds for

defense of

  • 💫 释义 : 为…辩护
    • 🫧 例句 : She spoke in defense of the new policy.
    • 🌈 同义 : justification of, support for, advocacy for

argue sb into doing sth

  • 💫 释义 : 说服某人做某事
    • 🫧 例句 : She tried to argue him into accepting the job offer.
    • 🌈 同义 : persuade sb to do sth, talk sb into doing sth, convince sb to do sth

argue sb out of doing sth

  • 💫 释义 : 说服某人不做某事
    • 🫧 例句 : I managed to argue him out of quitting his job.
    • 🌈 同义 : dissuade sb from doing sth, talk sb out of doing sth, discourage sb from doing sth

persuade sb to do sth

  • 💫 释义 : 说服某人做某事
    • 🫧 例句 : We need to persuade them to invest in our project.
    • 🌈 同义 : convince sb to do sth, talk sb into doing sth, coax sb into doing sth

talk sb into doing sth

  • 💫 释义 : 说服某人做某事
    • 🫧 例句 : He talked me into going to the party with him.
    • 🌈 同义 : persuade sb to do sth, convince sb to do sth, win sb over to doing sth

talk sb out of doing sth

  • 💫 释义 : 说服某人不做某事
    • 🫧 例句 : I tried to talk her out of making such a risky investment.
    • 🌈 同义 : dissuade sb from doing sth, discourage sb from doing sth, advise sb against doing sth

More often than not

  • 💫 释义 : 通常; 多半
    • 🫧 例句 : More often than not, he takes the bus to work.
    • 🌈 同义 : usually, generally, as often as not, more than half the time

push for

  • 💫 释义 : 争取, 敦促
    • 🫧 例句 : The team is pushing for a quick decision on the matter.
    • 🌈 同义 : advocate for, campaign for, press for

push through

  • 💫 释义 : 强行通过, 努力完成
    • 🫧 例句 : The government pushed through the new law despite opposition.
    • 🌈 同义 : force through, ram through, carry out

Part 10

push back ( against )

  • 💫 释义 : 抵制, 反对
    • 🫧 例句 : The public pushed back against the proposed tax increase.
    • 🌈 同义 : resist, oppose, fight back against

push forward

  • 💫 释义 : 推进, 继续前进
    • 🫧 例句 : We must push forward with the plan despite the difficulties.
    • 🌈 同义 : advance, proceed, move ahead

push aside

  • 💫 释义 : 推开; 忽视
    • 🫧 例句 : She pushed aside her doubts and accepted the challenge.
    • 🌈 同义 : disregard, ignore, set aside

push sb to do sth

  • 💫 释义 : 督促/逼迫某人做某事
    • 🫧 例句 : His parents pushed him to apply to top universities.
    • 🌈 同义 : urge sb to do sth, pressure sb to do sth, compel sb to do sth

get ahead of

  • 💫 释义 : 领先于; 超越; 提前处理
    • 🫧 例句 : We need to innovate constantly to get ahead of our competitors.
    • 🌈 同义 : overtake, outpace, surpass, stay ahead of

get ahead of oneself

  • 💫 释义 : 操之过急; 过早行动 ( 常指因兴奋而考虑不周 )
    • 🫧 例句 : Let’s not get ahead of ourselves —we haven’t even won the contract yet.
    • 🌈 同义 : jump the gun, act prematurely, be overhasty

stay ahead of

  • 💫 释义 : 保持领先于
    • 🫧 例句 : It’s hard to stay ahead of the latest technological trends.
    • 🌈 同义 : maintain a lead over, keep an advantage over, outrun

keep ahead of

  • 💫 释义 : 保持领先于; 抢先一步
    • 🫧 例句 : We read industry reports to keep ahead of market changes.
    • 🌈 同义 : stay ahead of, anticipate, preempt

ahead of the curve

  • 💫 释义 : 领先于潮流 ( 或趋势 )
    • 🫧 例句 : Their investment in renewable energy put them ahead of the curve.
    • 🌈 同义 : trendsetting, pioneering, innovative, cutting-edge

address an issue

  • 💫 释义 : 处理问题
    • 🫧 例句 : The management held a meeting to address the safety issues.
    • 🌈 同义 : deal with a problem, handle a matter, tackle an issue
