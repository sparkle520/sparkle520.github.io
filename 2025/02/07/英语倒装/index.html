一、核心概念

倒装 ( Inversion )

英语句子的自然语序是 主语 + 谓语 ( S + V ). 为了 强调、衔接 或 平衡结构，有时会将谓语或助动词移到主语之前，这种结构称为 倒装.

1. 核心规则

谓语动词 完全移到主语之前.

  • 公式 Adv./Prep. + V + S
  • 关键条件 主语必须是 名词 ( Noun ).
  • 禁区 若主语是 代词 ( Pronoun )，绝不倒装.

2. 常见场景

场景 A : 方位副词置于句首

常见词 : here, there, out, in, up, down, away, now, then

类型 例句 说明
√ 倒装 Here comes the bus. 主语 bus 是名词
√ 倒装 Out rushed the children. 主语 children 是名词
✗ 不倒 Here he comes. 主语 he 是代词
✗ 不倒 Away they went. 主语 they 是代词

场景 B : 介词短语/表语置于句首

目的 : 平衡句式，描写环境.

  • On the wall hangs a picture. ( 墙上挂着一幅画 )
  • South of the river lies a small factory. ( 河的南边坐落着一个小工厂 )

注意主谓一致

谓语动词的数由 后面的主语 决定.

  • On the desk are ( not is ) two books.

场景 C : There be 句型

最典型的完全倒装. be 可替换为 exist, lie, stand, come, go 等.

  • There lies a village at the foot of the hill.

1. 核心规则

只将 助动词/情态动词/be动词 移到主语之前.

  • 公式 Trigger + Aux + S + V
  • 助动词 do/does/did, have/has/had, can/could/will/would…

2. 核心场景速查

场景 触发词/结构 例句
否定词 Never, Seldom, Hardly, Little… Never have I seen him.
Only+状语 Only then, Only by…, Only when… Only then did I realize it.
关联词 Not only…but also… Not only is he a teacher…
承接 So / Neither / Nor So do I. / Nor can I.
程度 So + adj / Such + n So cold was it that…
虚拟 省略 if ( Were/Had/Should ) Had I known it…

3. 重点难点解析

① Only + 状语

陷阱

Only 修饰 主语 时不倒装.

  • Only you can solve it. ( 修饰主语 you → 不倒装 )
  • Only in this way can you solve it. ( 修饰状语 → 倒装 )

② Not only… but also…

前倒后不倒 只有 Not only 所在的分句需要倒装.

  • Not only did he learn English, but he also learned French.

③ 虚拟语气省略 If

当条件句含 were, had, should 时，可省 if 并倒装.

  • If I were you → Were I you…
  • If it had rained → Had it rained…
  • If it should rain → Should it rain…

“As” 引导的让步状语从句

表示 “虽然/尽管”，语气比 though 强烈.

结构 Adj/Adv/n/v + as + S + V ( 强调部分前置 )

  1. 形容词前置
    • Young as he is, he is wise. ( Although he is young… )
  2. 副词前置
    • Much as I like it, I won’t buy it.
  3. 名词前置 ( 注意 去掉冠词 ) :
    • Child as he is, he knows a lot. ( Child 前无 a/an )
  4. 动词前置
    • Try as he might, he couldn’t open the door.

综合练习

将下列句子改为倒装句 :

  1. The bus comes here.
  2. I have never heard such a story.
  3. He cares little about money.
  4. ( Though ) he is a hero, he is humble. ( 用 as 改写 )

参考答案 :

  1. Here comes the bus. ( 完全倒装 )
  2. Never have I heard such a story. ( 否定词提前 )
  3. Little does he care about money. ( 否定副词提前，加 does )
  4. Hero as he is，he is humble. ( 让步倒装，hero 前无冠词 )

完成下列句子或改错 :

  1. Only in this way ( we can ) ______ solve the problem.
  2. Not only ( he likes ) ______ reading, but also he likes writing.
  3. ( If I had ) ______ known it, I would have told you. ( 省略 if )
  4. 改错 Out rushed he when the fire broke out.

参考答案 :

  1. can we ( Only + 状语，部分倒装 )
  2. does he like ( Not only 位于句首，部分倒装 )
  3. Had I ( 虚拟语气省略 if，had 提前 )
  4. Out he rushed ( 主语 he 是代词，不可倒装 )
