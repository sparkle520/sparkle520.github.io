集线器 ( Hub ) · 工作原理与功能
OSI 第一层 · 物理层 · Physical Layer · 多端口中继器
功能
Core Function
将来自一个端口的信号 广播 到所有其他端口. 集线器本身不识别 MAC 地址, 对信号进行 再生整形 后向全部端口转发.
广播传输示意 — 端口 A 发送, 所有其他端口均收到
工作特性
广播式传输
任何端口收到信号, 立即向 所有其他端口 转发. 无 MAC 地址识别能力, 不做任何过滤.
共享带宽
所有设备共享 同一冲突域和带宽. 若接入 $N$ 台主机, 每台平均可用带宽仅为 $\dfrac{1}{N}$.
半双工 · CSMA/CD
同一时刻只允许 一台主机 发送数据. 采用 CSMA/CD 协议检测并解决冲突.
物理星型 · 逻辑总线型
外观为 星型拓扑 ( 线缆汇聚到集线器 ), 逻辑上等价于 共享总线, 行为与总线型完全一致.
