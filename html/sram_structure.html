各管功能说明
M5 / M6 — PMOS 上拉管
与 NMOS 组成 CMOS 反相器，接 VDD，为存储节点提供高电平驱动能力.
M1 / M2 — NMOS 驱动管
与 PMOS 组成反相器，接 GND，为存储节点提供低电平驱动. 两对反相器 交叉耦合 形成双稳态锁存器.
M3 / M4 — NMOS 访问管 ( 传输管 )
受 WL ( 字线 ) 控制. WL=1 时导通，连接存储节点与位线 ( BL/BL̄ ) ，允许读写操作.
双稳态存储原理
Q 和 Q̄ 互补. 若 Q=1 则 Q̄=0，反之亦然. 两种状态都能稳定维持，无需刷新，这就是 SRAM 名称中 Static 的含义.

三种工作状态详解
🔒
保持状态 ( Hold )
  • 条件WL = 0，访问管 M3/M4 断开
  • ①两个反相器互锁，维持 Q 和 Q̄ 的互补状态
  • ②只要 VDD 存在，数据永久保持
  • ③无需刷新操作，功耗仅来自漏电流
📖
读操作 ( Read )
  • ①预充电 : BL 和 BL̄ 均充至 VDD/2 或 VDD
  • ②WL 拉高，M3/M4 导通
  • ③存储节点对位线放电，产生差分电压 ΔV
  • ④灵敏放大器 ( SA ) 检测 BL-BL̄ 差值并放大输出
  • ⚠读扰动 : 需保证驱动管强于访问管 ( β比 )
✏️
写操作 ( Write )
  • ①写驱动器强制 BL / BL̄ 为互补电平
  • ②WL 拉高，M3/M4 导通
  • ③写驱动强行翻转存储节点 Q / Q̄
  • ④WL 拉低，M3/M4 断开，新状态被锁存
  • ⚠写裕量 : 需保证访问管比上拉管更强

工作原理
访问某个单元时，行译码器 激活对应 WL ( 整行通电 ) ，列译码器 将该行中目标列的 BL/BL̄ 连接到灵敏放大器 ( SA ) ，SA 将微小差分电压放大为完整逻辑电平输出.
