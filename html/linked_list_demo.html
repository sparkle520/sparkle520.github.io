线性表结构示意
仅结构展示，不含操作演示
顺序表 ( 顺序存储 ) 连续空间，随机访问 
typedef struct {
  ElemType data[MaxSize];
  int length;
} SqList;
单链表 ( 带头结点 ) 结点含 data 与 next 
typedef struct LNode {
  ElemType data;
  struct LNode* next;
} LNode, *LinkList;
双链表 ( 带头结点 ) prior 与 next 
typedef struct DNode {
  ElemType data;
  struct DNode* prior;
  struct DNode* next;
} DNode, *DLinkList;
循环单链表 尾结点 next 指回头结点 
判空: L->next == L
判尾: p->next == L
循环双链表 next 与 prior 都成环 
判空: L->next == L && L->prior == L
判尾: p->next == L
静态链表 ( 数组模拟链表 ) next 存数组下标 
typedef struct {
  ElemType data;
  int next;
} SNode;

SNode S[MaxSize];
